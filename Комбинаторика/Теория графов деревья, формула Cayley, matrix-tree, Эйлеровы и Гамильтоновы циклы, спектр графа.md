# Теория графов: деревья, формула Cayley, matrix-tree, Эйлеровы / Гамильтоновы циклы, спектр графа

## Scope
Перечисление и характеризация остовных деревьев (spanning trees), эйлеровых и гамильтоновых обходов, оценки и тождества через спектр матрицы смежности и Лапласиана. Покрывает задачи на подсчёт деревьев и обходов, sufficient conditions для Hamiltonicity, оценки $\chi, \alpha, \omega$ через собственные значения, биекции «деревья ↔ последовательности» и применения Cauchy–Binet к подсчёту путей.

## Записи - Core

### E1. Tree characterizations (характеризации деревьев)

Формулировка. Для графа $G$ на $n$ вершинах эквивалентны:
1. $G$ — дерево (связный, без циклов);
2. $G$ связен и $|E(G)| = n-1$;
3. $G$ ациклический и $|E(G)| = n-1$;
4. между любыми $u, v$ ровно один путь;
5. $G$ связен, и удаление любого ребра делает его несвязным;
6. $G$ ациклический, и добавление любого нового ребра создаёт ровно один цикл.

Условия применимости. Конечные простые графы. Для бесконечных деревьев теряется эквивалентность с условием на число рёбер.

Идея доказательства. Цепочка импликаций по индукции / контрапозиции: лист всегда существует в конечном связном графе с $n-1$ рёбрами; удаление листа сохраняет свойства.

Варианты и обобщения. Forest на $n$ вершинах с $c$ компонентами имеет ровно $n-c$ рёбер. Cycle space имеет размерность $|E|-n+c$ (циклосматический ранг, cyclomatic number).

Используется в. [IMC-1999-5 (граф $|V|=2n$, $|E|=2n \Rightarrow$ есть цикл), базовый инструмент].

### E2. Cayley's formula (формула Кэли)

Формулировка. Число помеченных деревьев на $n \ge 1$ вершинах равно $n^{n-2}$. Эквивалентно, число остовных деревьев полного графа $\tau(K_n) = n^{n-2}$.

Условия применимости. Помеченные (labeled) деревья, $n \ge 2$. Для unlabeled деревьев формула не работает (последовательность OEIS A000055).

Идея доказательства. Биекция Прюфера (E3); либо Matrix-Tree (E4) применённый к $K_n$ с $L = nI - J$, спектр $\{0, n, n, \dots, n\}$, любой кофактор $= n^{n-2}$; либо двойной счёт пар (forest $F$ с пометкой направленного рёберного списка) Pitman.

Варианты и обобщения.
- $\tau(K_{m,n}) = m^{n-1} n^{m-1}$ (Scoins, см. E15).
- Число лесов на $n$ вершинах с $k$ корнями $= k \cdot n^{n-k-1}$ (Cayley–Pitman).
- Число деревьев со степенью $d_i$ у вершины $i$ равно $\binom{n-2}{d_1-1,\dots,d_n-1}$.

Используется в. [Putnam-2008-B4, базовый «атом» для перечислительных задач].

### E3. Prüfer correspondence (соответствие Прюфера)

Формулировка. Существует биекция между помеченными деревьями на $\{1,\dots,n\}$ и последовательностями длины $n-2$ из $\{1,\dots,n\}$. Алгоритм: на каждом шаге удалить лист с минимальной меткой и записать метку его соседа.

Условия применимости. $n \ge 2$. Биекция конструктивна и обратима.

Идея доказательства. Удаление листа сохраняет дерево; вершина $v$ появляется в коде ровно $\deg_T(v) - 1$ раз; обратное восстановление: на шаге $i$ вершина с минимальной меткой, не появляющаяся в текущем суффиксе и не использованная — лист, соединяемый с первой меткой суффикса.

Варианты и обобщения. Доказывает E2 моментально. Обобщение: коды для лесов с фиксированными корнями. Для $K_{m,n}$ — двусторонний код длины $(n-1) + (m-1)$.

Используется в. [многочисленные задачи на подсчёт деревьев со свойствами по степеням; IMC уровня — построение биекций].

### E4. Matrix-Tree Theorem (теорема Кирхгофа, теорема о матричном дереве)

Формулировка. Пусть $G$ — связный граф с лапласианом $L = D - A$ ($D$ — diagonal степеней, $A$ — adjacency). Тогда число остовных деревьев
$$\tau(G) = \det \widetilde{L}_{ii} = \frac{1}{n}\lambda_2 \lambda_3 \cdots \lambda_n,$$
где $\widetilde{L}_{ii}$ — главный минор $L$ с вычеркнутой $i$-й строкой и столбцом, $0 = \lambda_1 \le \lambda_2 \le \dots \le \lambda_n$ — спектр $L$.

Условия применимости. Конечный неориентированный граф (allowed multi-edges, no loops). Для weighted: $\tau(G) = \sum_T \prod_{e \in T} w(e)$, где сумма по spanning trees.

Идея доказательства. $L = M M^T$, где $M$ — матрица инцидентности с произвольной ориентацией. По Cauchy–Binet
$$\det \widetilde{L}_{ii} = \sum_{S\subset E,\,|S|=n-1} \det(\widetilde{M}_S)^2,$$
а $\det(\widetilde{M}_S) \in \{0, \pm 1\}$ и ненулевой ровно когда $S$ — остовное дерево.

Варианты и обобщения.
- Directed Matrix-Tree (Tutte): для digraph $\sum_T \prod_{e \in T} w(e)$ по arborescences с корнем $r$ равно $\det \widetilde{L}^{\text{out}}_{rr}$.
- All Minors Matrix-Tree (Chaiken): любые миноры $L$ выражаются через rooted spanning forests.

Используется в. [BEST theorem (E14), вычисление $\tau(K_n)$, $\tau(K_{m,n})$, $\tau$ гиперкуба и кругового графа].

### E5. Euler's theorem on Eulerian circuits (теорема Эйлера об эйлеровых обходах)

Формулировка. Связный граф $G$ имеет эйлеров цикл (проходящий каждое ребро ровно один раз) ⟺ все вершины имеют чётную степень. Эйлеров путь существует ⟺ ровно $0$ или $2$ вершины нечётной степени. Для digraph: эйлеров цикл ⟺ $G$ слабо связен и $\deg^{\text{in}}(v) = \deg^{\text{out}}(v)$ для всех $v$.

Условия применимости. Связный граф (или: каждая компонента, имеющая рёбра, связна). Многократные рёбра допускаются.

Идея доказательства. Необходимость: каждый проход через вершину «сжигает» 2 ребра. Достаточность — алгоритм Hierholzer: построить цикл, вернуться в первую вершину с неизрасходованными рёбрами и вшить туда новый цикл.

Варианты и обобщения.
- Минимальное число эйлеровых путей для покрытия графа $= \max(1, k/2)$, где $k$ — число вершин нечётной степени.
- Reconstruction theorem (van Aardenne-Ehrenfest, de Bruijn) — основа BEST theorem (E14).

Используется в. [классические задачи о Кёнигсбергских мостах, IMC задачи на покрытие рёбер циклами; часто как инструмент в digraph задачах].

### E6. Walks via $A^k$ (подсчёт обходов через степени матрицы смежности)

Формулировка. Для adjacency matrix $A$ простого графа $G$:
$$(A^k)_{ij} = \#\{\text{walks длины } k \text{ из } i \text{ в } j\}.$$
Следствия: $\operatorname{tr}(A^2) = 2|E|$; $\operatorname{tr}(A^3) = 6 \cdot \#\{\text{треугольники}\}$; $\sum_{v} \deg(v)^2 = (A \mathbf 1)^T (A\mathbf 1) = \mathbf{1}^T A^2 \mathbf{1}$.

Условия применимости. Любой граф (включая мультиграфы и петли) — $A_{ii} = 2 \cdot \#\text{петель}$ при стандартном соглашении.

Идея доказательства. Индукция: $(A^{k+1})_{ij} = \sum_l (A^k)_{il} A_{lj}$ — суммирование по предпоследней вершине walk.

Варианты и обобщения.
- $\operatorname{tr}(A^k) = \sum_i \lambda_i^k$ — связь со спектром (E7, E18).
- $\sum_{v} \deg(v)^2 \ge \frac{(2|E|)^2}{n}$ (Cauchy–Schwarz); равенство ⟺ регулярный.
- Для walks с весами — заменяем $A$ на weighted adjacency.

Используется в. [IMC-2009-3 (счёт walks длины 2 через $\sum \deg^2$), частая часть в задачах на треугольники / 4-циклы].

### E7. Spectra of standard families (спектры базовых семейств графов)

Формулировка. Spectrum матрицы смежности:
- $K_n$: $\{n-1,\, -1^{(n-1)}\}$.
- $K_{m,n}$ ($m \le n$): $\{\pm\sqrt{mn},\, 0^{(m+n-2)}\}$.
- $C_n$ (cycle): $\{2\cos(2\pi k/n) : k = 0,\dots,n-1\}$.
- $P_n$ (path): $\{2\cos(\pi k/(n+1)) : k = 1,\dots,n\}$.
- Гиперкуб $Q_d$: $\{d - 2k : k = 0,\dots,d\}$ с кратностями $\binom{d}{k}$.
- Petersen: $\{3, 1^{(5)}, -2^{(4)}\}$.

Условия применимости. Все эти графы вершинно-транзитивны или регулярны — лежат в commutative case (циркулянты для $C_n$, тензорные произведения для $Q_d$).

Идея доказательства. Циркулянты — DFT диагонализует. Гиперкуб — тензор из $K_2$. Сильно регулярные графы (например, Petersen) — спектр определяется параметрами $(n, k, \lambda, \mu)$ через квадратное уравнение.

Варианты и обобщения. Cayley-graph $\Gamma(\mathbb{Z}_n^d, S)$ имеет спектр через characters группы. Eigenvalues тензорного произведения = произведения eigenvalues.

Используется в. [Putnam-1980-A4 (спектр $C_n$), вычисление $\tau$ через E4, оценки на $\alpha, \chi$ через E8–E10].

## Записи - Standard

### E8. Cauchy interlacing for graphs (теорема Коши о чередовании, применённая к графам)

Формулировка. Пусть $A$ — симметричная матрица $n\times n$, $B$ — её главный подматрица $m\times m$. Если $\lambda_1 \ge \dots \ge \lambda_n$ — спектр $A$, $\mu_1 \ge \dots \ge \mu_m$ — спектр $B$, то
$$\lambda_i \ge \mu_i \ge \lambda_{n-m+i}, \quad i = 1, \dots, m.$$
Для графов: индуцированный подграф $H \subset G$ имеет $\lambda_i(H)$, чередующиеся со spectrum $G$.

Условия применимости. Симметричные/эрмитовы матрицы. Аналог для quotient matrix (Haemers interlacing) — при равномерном разбиении вершин.

Идея доказательства. Courant–Fischer: $\lambda_k(A) = \max_{\dim V = k} \min_{x \in V} \frac{x^T A x}{x^T x}$. Подпространства $B$ — подмножества подпространств $A$.

Варианты и обобщения.
- Hoffman ratio bound для $\alpha(G)$ (E9) — следствие interlacing.
- Bound на chromatic number $\chi(G) \ge 1 + \lambda_1/(-\lambda_n)$ для нерегулярного случая (Hoffman).
- Bound на bandwidth, diameter через спектр.

Используется в. [Hoffman (E9), доказательства spectral inequalities в задачах IMC уровня].

### E9. Hoffman ratio bound (граница Хоффмана для независимости)

Формулировка. Пусть $G$ — $k$-регулярный граф на $n$ вершинах с минимальным собственным значением $\lambda_n$. Тогда
$$\alpha(G) \le \frac{n \cdot (-\lambda_n)}{k - \lambda_n} = \frac{-n\lambda_n}{k - \lambda_n}.$$
Эквивалентно $\omega(\bar G) \le n(-\lambda_n)/(k-\lambda_n)$.

Условия применимости. Регулярность критична. Для нерегулярного — Lovász theta или модификация. Bound достижим для strongly regular graphs (Petersen: $\alpha = 4$, $n(-\lambda_n)/(k-\lambda_n) = 10\cdot 2/5 = 4$).

Идея доказательства. Положить $M = A - \lambda_n I + \frac{\lambda_n - k}{n} J$; для индикатора $x$ независимого множества $x^T M x \ge 0$ даёт неравенство.

Варианты и обобщения.
- Cvetković inertia bound: $\alpha(G) \le n_0 + \min(n_+, n_-)$, где $n_+, n_-, n_0$ — число положительных, отрицательных, нулевых собственных значений.
- $k$-independence ratio bound (Tait и др.).

Используется в. [упражнения IMC уровня о максимальных независимых множествах в Cayley-графах; косвенно — IMC-2017-4 как альтернатива probabilistic argument].

### E10. Wilf chromatic bound (граница Уилфа для хроматического числа)

Формулировка. Для любого графа $G$
$$\chi(G) \le 1 + \lambda_1(G),$$
где $\lambda_1(G)$ — наибольшее собственное значение adjacency $A$.

Условия применимости. Любой простой граф. Bound достижим для $K_n$ (где $\lambda_1 = n-1$) и нечётных циклов.

Идея доказательства. Greedy colouring в порядке удаления вершины с минимальной степенью; на каждом шаге $\delta(H) \le \lambda_1(H) \le \lambda_1(G)$ (последнее — interlacing, E8).

Варианты и обобщения.
- Hoffman lower bound: $\chi(G) \ge 1 + \lambda_1/(-\lambda_n)$.
- Brooks: $\chi(G) \le \Delta(G)$ кроме $K_n$ и $C_{2k+1}$.

Используется в. [классическое: $\chi(\text{Petersen}) \le 1 + 3 = 4$, и точно $= 3$; задачи на оценку $\chi$ нерегулярных графов через спектр].

### E11. Hall's marriage theorem (теорема Холла о свадьбах)

Формулировка. Двудольный граф $G = (X \cup Y, E)$ имеет matching, насыщающее $X$, ⟺ для любого $S \subseteq X$
$$|N(S)| \ge |S|.$$

Условия применимости. Конечное $X$ (для бесконечного — нужны дополнительные условия; König–Hall). Defect form: max matching $|X| - \max_S (|S| - |N(S)|)$.

Идея доказательства. Augmenting paths: предположить максимальное matching не насыщает $x \in X$; рассмотреть множество достижимых из $x$ alternating путями — нарушение Hall condition.

Варианты и обобщения.
- König (двудольный): $\max\text{matching} = \min\text{vertex cover}$.
- Tutte (общий случай): max matching $= \frac{1}{2}(n - \max_U (\text{odd}(G\setminus U) - |U|))$.
- $k$-factor existence (Tutte–Berge).
- Latin rectangles, system of distinct representatives — applications.

Используется в. [IMC-2011-DAY2-2 (matching между мужчинами/женщинами/extras), Putnam-1965-B5 (Latin square completion)].

### E12. Dirac / Ore / Chvátal–Erdős conditions (условия Дирака / Оре / Хватала–Эрдёша на гамильтоновость)

Формулировка.
- Dirac: $G$ простой, $n \ge 3$, $\delta(G) \ge n/2$ ⟹ $G$ гамильтонов.
- Ore: $\deg(u) + \deg(v) \ge n$ для всех несмежных $u, v$ ⟹ $G$ гамильтонов.
- Chvátal–Erdős: $\kappa(G) \ge \alpha(G)$, $n \ge 3$ ⟹ $G$ гамильтонов.

Условия применимости. $n \ge 3$. Bound в Dirac точен ($K_{n/2, n/2}$ при $\delta = n/2 - 1$ не гамильтонов для нечётных $n$). Chvátal–Erdős — самый сильный из трёх (Dirac ⟹ Ore; Ore ⟹ Chvátal–Erdős).

Идея доказательства. Все три — через rotation-extension (E13) или Bondy–Chvátal closure: добавлять рёбра между $u, v$ при $\deg u + \deg v \ge n$, пока возможно; полученное замыкание $\hat G$ гамильтоново ⟺ $G$ гамильтоново, и при условии Ore $\hat G = K_n$.

Варианты и обобщения.
- Bondy–Chvátal theorem (1976) — обобщает Dirac/Ore/Chvátal–Erdős.
- $n$ можно усилить до $n+1$ для Hamiltonian-connectedness.
- Pósa's degree condition: упорядоченные степени $d_1 \le \dots \le d_n$, $d_k \ge k+1$ для $k < n/2$ ⟹ гамильтонов.

Используется в. [классические Putnam задачи; типичный «закрывающий аргумент» для Hamiltonicity].

### E13. Pósa rotation-extension (метод поворота-расширения Пошы)

Формулировка. Пусть $P = v_0 v_1 \dots v_k$ — самый длинный путь в $G$, и $v_k v_i \in E$ для некоторого $i < k-1$. Тогда $P' = v_0 \dots v_i v_k v_{k-1} \dots v_{i+1}$ — путь той же длины с новым концом $v_{i+1}$. Множество концов $S$, достижимых поворотами из $v_k$, удовлетворяет $|N(S)| < 2|S|$ (нет рёбер из $S$ в $V(P)\setminus(S \cup T)$).

Условия применимости. $G$ конечный, $P$ — longest path. Метод даёт Hamiltonicity в графах с большим $\delta$, и used для Pósa's theorem о random graph $G(n, p)$ при $p = (\log n + \log\log n + \omega(1))/n$.

Идея доказательства. Если $S$ имеет $> n/2$ вершин и $G$ имеет минимальную степень $\ge n/2$, то существует ребро от конца поворота к началу пути или вне $P$ — противоречие максимальности $P$.

Варианты и обобщения.
- Расширение для bipartite / multipartite случая.
- Дает доказательство Dirac's theorem без closure.
- Используется в Bollobás–Frieze для random graphs.

Используется в. [Pósa's theorem; элегантное доказательство Dirac; IMC задачи на поиск длинных путей в плотных графах].

### E14. BEST theorem (теорема BEST: de Bruijn, Aardenne-Ehrenfest, Smith, Tutte)

Формулировка. Для связного эйлерова digraph $G$ (где $\deg^+(v) = \deg^-(v)$ для всех $v$) число эйлеровых обходов равно
$$ec(G) = t_w(G) \cdot \prod_{v \in V} (\deg^+(v) - 1)!,$$
где $t_w(G) = \det \widetilde{L}^{\text{out}}_{ww}$ — число rooted spanning arborescences с корнем в любой вершине $w$ (по directed matrix-tree).

Условия применимости. Directed эйлеров граф; формула не зависит от выбора корня $w$ (для эйлеровых graphs $t_w$ — инвариант). В неориентированном случае подсчёт эйлеровых циклов #P-hard.

Идея доказательства. Биекция: эйлеров обход $\leftrightarrow$ (arborescence $T$ с корнем $w$) $+$ (для каждой вершины $v \neq w$ — порядок исходящих рёбер с фиксированным «деревным»; для $w$ — полный порядок).

Варианты и обобщения. Для регулярного digraph $K^*_n$ (полный с обоими направлениями): $ec = ((n-2)!)^n \cdot n^{n-2}$. Counting de Bruijn sequences — частный случай.

Используется в. [перечисление de Bruijn последовательностей; задачи на подсчёт эйлеровых обходов через определители].

### E15. Scoins formula for $\tau(K_{m,n})$ (формула Скойнса для остовных деревьев двудольного графа)

Формулировка. $\tau(K_{m,n}) = m^{n-1} n^{m-1}$.

Условия применимости. $m, n \ge 1$; полный двудольный граф.

Идея доказательства. Matrix-Tree: лапласиан $L$ блочный
$$L = \begin{pmatrix} nI_m & -J \\ -J^T & mI_n \end{pmatrix},$$
spectrum: $\{0, m^{(n-1)}, n^{(m-1)}, m+n\}$, $\tau = \frac{1}{m+n} m^{n-1} n^{m-1} (m+n) = m^{n-1} n^{m-1}$. Альтернатива: bipartite Prüfer code длины $(m-1) + (n-1)$.

Варианты и обобщения. $\tau(K_n - e) = (n-2) n^{n-3}$. $\tau(Q_d) = \frac{1}{2^d}\prod_{k=1}^{d}(2k)^{\binom{d}{k}}$ (через спектр гиперкуба).

Используется в. [Putnam-1980 задачи о подсчёте конфигураций в bipartite структурах].

### E16. Lindström–Gessel–Viennot lemma (лемма Линдстрёма–Гесселя–Вьенно)

Формулировка. Пусть $D$ — ациклический directed weighted граф, $A = (a_1,\dots,a_n)$, $B = (b_1,\dots,b_n)$ — кортежи вершин, $M_{ij} = \sum_{P : a_i \to b_j} w(P)$. Тогда
$$\det M = \sum_{\sigma \in S_n} \operatorname{sgn}(\sigma) \sum_{(P_1,\dots,P_n) : P_i: a_i \to b_{\sigma(i)},\,\text{non-intersecting}} \prod_i w(P_i).$$

Условия применимости. Acyclic digraph; «non-intersecting» = vertex-disjoint. Если для всех неидентичных $\sigma$ нет невырожденных кортежей путей — $\det M$ равен числу непересекающихся кортежей $(P_i: a_i \to b_i)$ с весом 1.

Идея доказательства. Sign-reversing involution: на пересекающихся кортежах путей делается swap у первого пересечения, что меняет знак $\sigma$ и устанавливает биекцию между не-identity слагаемыми. Остаются только non-intersecting identity-path tuples.

Варианты и обобщения.
- Вычисление чисел SSYT через произведения определителей (Jacobi–Trudi).
- Подсчёт plane partitions, MacMahon box formula.
- Доказательство Cauchy–Binet через lattice paths.

Используется в. [олимпиадные задачи на подсчёт непересекающихся путей в решётках; иногда — Putnam с вычислением определителей через комбинаторику].

### E17. Friendship theorem (теорема о дружбе, Эрдёш–Реньи–Шош)

Формулировка. Если в конечном графе любые две вершины имеют ровно одного общего соседа, то $G = $ «ветряная мельница» $F_k$ — $k$ треугольников с общей вершиной.

Условия применимости. Конечный простой граф. Бесконечный случай — открыт / отдельные результаты.

Идея доказательства. (i) Не все степени равны (иначе $G$ $k$-регулярный, $A^2 = (k-1)I + J$, спектр $A$ имеет иррациональное значение, противоречие Pisot). (ii) Существует вершина степени $n-1$, остальные степени по 2, что даёт $F_k$.

Варианты и обобщения.
- Erdős–Ko–Rado — родственная extremal задача.
- Generalised friendship problem (с $\lambda$ общими соседями) — открытый круг задач.

Используется в. [классический «spectral trick» в комбинаторике; doppelt в Aigner–Ziegler как Proof from THE BOOK; стиль аргумента воспроизводится в IMC-2009-3].

### E18. Bipartite spectrum criterion (спектральный критерий двудольности)

Формулировка. Связный граф $G$ двудольный ⟺ спектр $A(G)$ симметричен относительно нуля, т.е. $\lambda \in \sigma(A) \Rightarrow -\lambda \in \sigma(A)$ с той же кратностью. Эквивалентно: $\lambda_n = -\lambda_1$ (для regular — необходимое и достаточное условие двудольности).

Условия применимости. Обычно формулируется для связного графа. Для несвязного — каждая компонента двудольна ⟺ её spectrum симметричен.

Идея доказательства. Двудольность ⟺ нет нечётных циклов ⟺ $\operatorname{tr}(A^{2k+1}) = 0$ для всех $k$ ⟺ $\sum \lambda_i^{2k+1} = 0$ для всех $k$ ⟺ спектр симметричен. Альтернативно: с разбиением $V = X \sqcup Y$ матрица $A$ записывается блочно $\begin{pmatrix}0 & B\\B^T & 0\end{pmatrix}$, и diag$(I, -I)$ конъюгирует $A$ к $-A$.

Варианты и обобщения.
- Связный граф $r$-регулярен и двудолен ⟺ $-r \in \sigma(A)$.
- Признак Hoffmann: сильно регулярный граф двудольный ⟺ $\mu = 0$.

Используется в. [быстрая проверка двудольности через сумму нечётных степеней trace; IMC задачи о структуре через спектр].

## Self-test

1. Сколько остовных деревьев у графа $K_n$ минус совершенное паросочетание ($n$ чётно, удалено $n/2$ независимых рёбер)?
2. Граф $G$ на $n$ вершинах: каждые две вершины имеют ровно одного общего соседа. Доказать, что в $G$ существует вершина, смежная со всеми остальными.
3. На клетчатой доске $n \times n$ закрашено $2n$ клеток. Доказать, что найдётся последовательность закрашенных клеток $c_1, c_2, \dots, c_{2k}$ ($k \ge 2$), где $c_{2i-1}, c_{2i}$ в одной строке, а $c_{2i}, c_{2i+1}$ в одном столбце (индексы по модулю $2k$).
4. Турнир (полный ориентированный граф) на $n$ вершинах. Доказать, что в нём существует гамильтонов путь.
5. Пусть $G$ — простой граф на $n$ вершинах без треугольников. Доказать, что $\sum_v \deg(v)^2 \le n \cdot |E(G)|$ и вывести из этого Mantel: $|E(G)| \le n^2/4$.
6. На полу выложено $n$ красных и $n$ синих ковриков, причём каждый красный пересекается ровно с тремя синими и наоборот. Доказать, что коврики можно разбить на $n$ пар (красный, синий) пересекающихся ковриков.
7. Спектр $r$-регулярного графа $G$ содержит $-r$. Доказать, что $G$ имеет двудольную компоненту связности.
8. Найти число $k$-кортежей вершинно-непересекающихся монотонных (только $\to$ и $\uparrow$) решёточных путей $P_i: (0, i-1) \to (a, b + i - 1)$, $i = 1, \ldots, k$, в виде определителя биномиальных коэффициентов.
9. Пусть $G$ — $k$-регулярный простой граф на $n$ вершинах без треугольников и без $C_4$. Доказать, что $k \le \sqrt{n-1}$, и привести равенство для $k = 2$ (Petersen для $k=3$, $n=10$).
10. Пусть $K_n^*$ — directed полный граф (каждое неориент. ребро заменено парой противоположно ориентированных). Вычислить число эйлеровых обходов в $K_n^*$ через BEST theorem.

## Решения

1. $A = (J - I) - M$, $D = (n-2)I$, $L = D - A = (n-1)I - J + M$, где $M$ — adjacency паросочетания. Спектр $M$: $+1$ с кратностью $n/2$ (на симметричных парах, включая $\mathbf{1}$) и $-1$ с кратностью $n/2$. На $\mathbf{1}$: $L\mathbf{1} = 0$. На ортогональном дополнении: $(n-1)I + M$ имеет $n$ (кратность $n/2-1$, остаток $+1$-eigenspace) и $n-2$ (кратность $n/2$). По Matrix-Tree $\tau = \frac{1}{n} \cdot n^{n/2-1} (n-2)^{n/2} = n^{n/2 - 2}(n-2)^{n/2}$.

2. Если все степени равны $k$: $A^2 = (k-1)I + J$ (диагональ — степень, off-diag — общий сосед). $A\mathbf 1 = k\mathbf 1 \Rightarrow k^2 = k - 1 + n$, $n = k^2 - k + 1$. На $\mathbf{1}^\perp$: $\lambda^2 = k-1$, $\lambda = \pm\sqrt{k-1}$ с кратностями $a, b$, $a + b = n-1$. $\operatorname{tr}A = 0$: $(a-b)\sqrt{k-1} = -k$. Если $k-1$ не полный квадрат — $a = b$ и $k = 0$, тривиально. Иначе $k - 1 = m^2$, $m \mid k = m^2 + 1$, $m \mid 1$, $m = 1$, $k = 2$, $n = 3$ — треугольник $K_3 = F_1$. Если же степени не все равны, индукционно показывается: вершина $v$ макс. степени имеет $\deg(v) = n-1$ (рассмотрев двух её несоседей $u, w$ — у них общий сосед, и счёт даёт противоречие при $\deg v < n-1$). Получается ветряная мельница $F_k$.

3. Граф $H$: вершины — строки $r_1,\dots,r_n$ и столбцы $c_1,\dots,c_n$, рёбра — закрашенные клетки. $|V| = 2n$, $|E| = 2n \ge |V|$, по E1 (вариант: связный кусок с $\ge $ числа вершин рёбер) есть цикл. Цикл в двудольном графе чётный, что и даёт чередующуюся последовательность.

4. Индукция по $n$. Удалить вершину $v$, найти гамильтонов путь $v_1 \to \dots \to v_{n-1}$ в оставшемся турнире. Если $v \to v_1$ — вставить в начало; если $v_{n-1} \to v$ — в конец; иначе найти первый индекс $i$ с $v_i \to v$: тогда $v_{i-1} \to v$ из условия на ребра, плюс $v \to v_i$ или $v_i \to v$ — позволяет вставить $v$ между $v_{i-1}$ и $v_i$.

5. $\sum \deg(v)^2 = \mathbf{1}^T A^2 \mathbf{1}$ — число walks длины $2$. Для каждой пары $\{u, w\}$ число общих соседей $\le 1$ при отсутствии треугольника $\Leftrightarrow$ $u, w$ не смежные (иначе общий сосед даёт треугольник). Значит walks длины 2 не превышает $n + 2 \cdot |\{\text{несмежные пары с общим соседом}\}| \le n |E|$. С другой стороны, $\sum \deg(v)^2 \ge (2|E|)^2/n$, так что $(2|E|)^2/n \le n|E|$, $|E| \le n^2/4$.

6. Bipartite граф $G = (R, B, E)$, $|R| = |B| = n$, $3$-регулярный (каждая красная — 3 синие). По Hall: $|N(S)| \ge |S|$ для $S \subseteq R$ — иначе $|E(S, N(S))| = 3|S| > 3|N(S)|$ противоречит регулярности справа. Перфектное паросочетание существует.

7. $r$-регулярный, $-r \in \sigma$ ⟹ $\exists x \neq 0$, $Ax = -rx$. По Perron–Frobenius $\mathbf{1}$ — eigenvector для $r$ с положительными компонентами; компонента, на которой $x$ ненулевой, — двудольна (E18, $-r = -\lambda_1$ компоненты).

8. Число монотонных путей $(0, i-1) \to (a, b + j - 1)$ равно $\binom{a + b + j - i}{a}$. По LGV (E16) для возрастающих source/sink — единственная permutation, дающая non-intersecting tuples, identity, поэтому ответ
$$\det\Big(\binom{a + b + j - i}{a}\Big)_{i, j = 1}^{k}.$$
Для частного случая $b = 0$, $a = k$ — получается число SYT прямоугольной формы $k \times k$ (формула Frobenius–Young).

9. Зафиксировать вершину $v$. Шар радиуса $2$ содержит $v$, $k$ соседей, и для каждого соседа $u$ — $k - 1$ его соседей кроме $v$. Triangle-free ⟹ соседи $v$ попарно несмежны, $C_4$-free ⟹ соседи разных $u_1, u_2 \in N(v)$ не имеют общего соседа кроме $v$. Все $1 + k + k(k-1) = k^2 + 1$ вершин различны и $\le n$. Равенство — Moore graphs: $k = 2$, $C_5$ ($n = 5$); $k = 3$, Petersen ($n = 10$); $k = 7$, Hoffman–Singleton ($n = 50$); открытый случай $k = 57$.

10. В $K_n^*$ каждая вершина имеет $\deg^+ = \deg^- = n - 1$, граф эйлеров. Лапласиан out-Лапласиан: $L = (n-1)I - (J - I) = nI - J$, спектр $\{0, n^{(n-1)}\}$. Число arborescences с любым корнем $t_w = \frac{1}{n}\prod \lambda_i = \frac{1}{n} n^{n-1} = n^{n-2}$ (это и есть Cayley-формула, что согласуется с E2). Применяя BEST: $ec(K_n^*) = n^{n-2} \cdot ((n-2)!)^n$. Для $n = 5$: $5^3 \cdot (3!)^5 = 125 \cdot 7776 = 972\,000$.
