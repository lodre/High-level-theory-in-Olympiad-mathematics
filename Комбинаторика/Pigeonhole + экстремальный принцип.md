# Pigeonhole + экстремальный принцип

## Scope

Подтема покрывает аргументы вида «слишком много объектов — слишком мало ящиков» (pigeonhole / box principle, Schubfachprinzip) и «возьмём минимальный/максимальный объект и докажем, что иначе противоречие» (extremal principle, минимальный контрпример). Сюда же — теоремы, чьё доказательство — pigeonhole + extremal в одну-две строки: Erdős–Szekeres, Dilworth/Mirsky, Δ-system, Erdős–Ko–Rado, Bollobás, Schur, Ramsey. Применения — оценки числа конфигураций, существование монотонных подпоследовательностей, intersecting families, аппроксимация иррационалов.

## Записи - Core

### E1. Усиленный Pigeonhole (generalized box principle)

Формулировка. (a) Если $n$ объектов разложены по $k$ ящикам, то некоторый ящик содержит $\geq \lceil n/k \rceil$ объектов; некоторый — $\leq \lfloor n/k \rfloor$. (b) Усреднение: если $\sum a_i = S$ и индексов $n$, то $\max a_i \geq S/n$ и $\min a_i \leq S/n$. (c) Бесконечный pigeonhole: бесконечное множество, разбитое на конечное число классов, содержит бесконечный класс. (d) Непрерывный: при отображении $f\colon X\to Y$ с $\mu(X) > k\cdot\sup_{y}\mu(f^{-1}(y))$ — нет такого $k$ (счётная версия трюка). (e) Вероятностный pigeonhole: если $\mathbb{E}[X]\geq c$, то $\exists\,\omega\colon X(\omega)\geq c$.

Условия применимости. (a)–(b): конечность множеств; для $\lceil\cdot\rceil$ нижней оценки нужна целочисленность. (c): требует AC/ countable choice в общем случае. Для (e) нужна корректно определённая мера/счётная сумма.

Идея доказательства. Сравнение суммы $\sum a_i$ с $n\cdot\lfloor S/n\rfloor$: иначе $\sum < S$.

Варианты и обобщения.
- Fractional pigeonhole: если средняя «плотность» $\geq d$ — есть подмножество $\geq d$ (часто через усреднение по сдвигам).
- Pigeonhole по группе: для любого $g\in G$ из $|G|+1$ элементов в подгрупповом разбиении два лежат в одном классе $\Rightarrow$ их разность в подгруппе.
- «Hilbert-style» бесконечный pigeonhole для последовательностей: применяется в proof-е Bolzano–Weierstrass через двоичное дробление.

Используется в. [IMC-1994-6, IMC-1999-2, IMC-1999-DAY2-6, IMC-2002-DAY2-2, Putnam-многократно].

### E2. Принцип крайнего (extremal principle / minimal counterexample)

Формулировка. На непустом конечном множестве конфигураций, упорядоченном вещественной функцией $\Phi$, существуют конфигурации с $\Phi=\min$ и $\Phi=\max$. Аналог для well-ordered — minimal element argument. Метод применения: предположить «плохую» конфигурацию, выбрать «худшую/лучшую» по $\Phi$, и из её локальной структуры вывести противоречие или доказать желаемое свойство.

Условия применимости. Требуется well-foundedness порядка (конечность, $\mathbb{N}$, well-ordering, либо компактность для непрерывного $\Phi$). Если множество конфигураций не well-founded, метод не работает напрямую — нужна Zornovskaya модификация или infinite descent.

Идея доказательства. Существование экстремума гарантировано порядком; перевод на функцию $\Phi$ переводит «улучшение» в строгое неравенство, что даёт спуск или противоречие («минимальный контрпример был не минимальным»).

Варианты и обобщения.
- Minimal counterexample (бесконечный спуск, Fermat): берётся минимальный по натуральному параметру плохой объект, строится строго меньший — противоречие.
- Maximal chain / maximal antichain: $\max|C|$ существует, и его дополнение анализируется.
- Variational extremum + perturbation: «возьмём вписанный $n$-угольник максимальной площади; вершина, не лежащая на границе, даёт перпендикулярное варьирование».
- Smallest semi-invariant (monovariant): для последовательностей операций берётся такая функция, которая строго убывает $\Rightarrow$ обрыв процесса.

Используется в. [IMC-1996-DAY2-4 (John ellipse), IMC-2004-5, IMC-2008-4, Putnam регулярно].

### E3. Теорема Эрдеша–Секереша (Erdős–Szekeres theorem)

Формулировка. Любая последовательность $(a_1,\ldots,a_N)$ из $N\geq (r-1)(s-1)+1$ попарно различных вещественных чисел содержит возрастающую подпоследовательность длины $r$ или убывающую длины $s$. Граница точна: при $N=(r-1)(s-1)$ контрпример — лексикографическая решётка $(r-1)\times(s-1)$.

Условия применимости. Различность элементов важна для строгой монотонности; для нестрогой версии нужна замена на $\geq r$ либо $\geq s$. Геометрический эквивалент: $(r-1)(s-1)+1$ точек в общем положении дают $r$ точек по возрастающим $x_i$ и $y_i$ или $s$ по возрастающим $x_i$ и убывающим $y_i$.

Идея доказательства. Каждой $a_i$ сопоставляется пара $(\ell_i, d_i)$ — длины максимальных возрастающей и убывающей подпоследовательностей, оканчивающихся в $i$. При $i<j$, если $a_i<a_j$, то $\ell_i<\ell_j$; иначе $d_i<d_j$. Все пары различны, лежат в $\{1,\ldots,r-1\}\times\{1,\ldots,s-1\}$ — pigeonhole. Альтернатива: следствие из Dilworth для poset $(i,j)\colon i<j, a_i<a_j$.

Варианты и обобщения.
- Геометрический Эрдеш–Секереш (happy ending): любые $\binom{2k-4}{k-2}+1$ точек в общем положении содержат выпуклый $k$-угольник.
- Higher-order Erdős–Szekeres (Bárány, Suk): монотонность для $k$-tuples.
- Positive fraction: положительная доля любых $r$-tuples монотонна (Pach, Solymosi).

Используется в. [IMC-2004-5, Putnam-1966-B4 (вариация)].

### E4. Теорема Дилуорта и теорема Мирски (Dilworth, Mirsky)

Формулировка. Пусть $P$ — конечный частичный порядок. Dilworth: минимальное число цепей в разбиении $P$ равно $\alpha(P)$ — максимальному размеру антицепи. Mirsky (двойственная): минимальное число антицепей в разбиении $P$ равно $h(P)$ — длине максимальной цепи.

Условия применимости. Конечность $P$; для бесконечного — версия Dilworth верна для конечной ширины (Perles). Mirsky для бесконечного $P$ конечной высоты остаётся в силе. Часто применяется в виде следствия: если $|P|\geq ab+1$, то есть цепь длины $a+1$ или антицепь $b+1$ (Erdős–Szekeres-стиль).

Идея доказательства. Dilworth — индукция по $|P|$ через max-flow-min-cut в специальном bipartite графе либо через анализ позиции максимальной антицепи $A$: рассматриваются «нижние конусы» $A^-$, «верхние» $A^+$. Mirsky — простая жадная: уровень $L_k$ = элементы с длиной максимальной цепи под ними равной $k-1$; каждое $L_k$ — антицепь, число уровней $= h(P)$.

Варианты и обобщения.
- Mirsky: разложение $P=\sqcup L_k$ по «высоте» — конструктивно.
- Объединение с pigeonhole: $|P|\geq ab+1\Rightarrow$ цепь $a+1$ или антицепь $b+1$ (даёт Erdős–Szekeres при $P=\{1,\ldots,N\}$ с порядком $i\preceq j\iff i\leq j\,\&\,a_i\leq a_j$).
- Greene–Kleitman: обобщение для $k$-цепей и $k$-антицепей.

Используется в. [IMC-2004-5 (через Erdős–Szekeres), стандартный инструмент в Putnam].

## Записи - Standard

### E5. Теорема Дирихле о приближении (Dirichlet's approximation theorem)

Формулировка. (1D) Для любых $\alpha\in\mathbb{R}$, $N\in\mathbb{N}$ существуют $p\in\mathbb{Z}$, $1\leq q\leq N$ с $|q\alpha-p|<1/N$, т.е. $|\alpha-p/q|<1/(qN)\leq 1/q^2$. Если $\alpha\notin\mathbb{Q}$, то таких $p/q$ бесконечно много. (Simultaneous) Для $\alpha_1,\ldots,\alpha_k\in\mathbb{R}$ и $N$ существуют $p_i\in\mathbb{Z}$, $1\leq q\leq N^k$ с $|q\alpha_i-p_i|<1/N$ для всех $i$.

Условия применимости. Любые $\alpha\in\mathbb{R}$. Точность $1/q^2$ оптимальна по порядку (Liouville/Roth — отдельная история).

Идея доказательства. Рассмотреть дробные части $\{j\alpha\}$ для $j=0,1,\ldots,N$ — это $N+1$ точка в $[0,1)$, разбитом на $N$ интервалов длины $1/N$; pigeonhole даёт две точки в одном интервале $\Rightarrow$ их разность $|(j-i)\alpha-(m_j-m_i)|<1/N$. Многомерно: разбить $[0,1)^k$ на $N^k$ кубов и взять $N^k+1$ точек $\{q\boldsymbol\alpha\}$.

Варианты и обобщения.
- Minkowski's first theorem (через выпуклые тела) — даёт ту же оценку и обобщает.
- Three-distance theorem (Steinhaus): расстояния между соседними точками $\{j\alpha\}$ принимают $\leq 3$ значения.
- Применение: $\liminf q\|q\alpha\|<1$ даёт плотность орбит на торе.

Используется в. [Putnam-задачи на иррациональность $\alpha$, IMC-задачи на плотные орбиты].

### E6. Лемма о подсолнухе (Erdős–Rado sunflower / Δ-system lemma)

Формулировка. Подсолнух (sunflower) с $k$ лепестками — семейство множеств $S_1,\ldots,S_k$, в котором попарные пересечения совпадают: $S_i\cap S_j=Y$ (ядро) для всех $i\neq j$. Лемма Erdős–Rado: любое семейство из $> w!\cdot(k-1)^w$ множеств размера $\leq w$ содержит подсолнух с $k$ лепестками.

Условия применимости. Множества равной/ограниченной мощности; $w$ и $k$ фиксированы. Современная граница (Alweiss–Lovett–Wu–Zhang 2019): $(\log w)^{w(1+o(1))}\cdot(k-1)^w$. Гипотеза Эрдеша: $C^w$.

Идея доказательства. Индукция по $w$. Если $w=1$ — очевидно. Для $w>1$: либо есть $k$ попарно непересекающихся (готовый подсолнух с пустым ядром), либо все они пересекают фиксированный набор размера $<k(w-1)$, и pigeonhole даёт элемент $x$ в $> w!(k-1)^w/k(w-1)$ множествах — применяем индукцию к $\{S\setminus\{x\}\}$.

Варианты и обобщения.
- Δ-system free: множеств без подсолнуха $\leq w!(k-1)^w$.
- Применения: нижние оценки для monotone circuit complexity (Razborov), upper bounds для Erdős–Ko–Rado-стиля задач.

Используется в. [IMC-1997-6 (вариация sunflower-аргумента)].

### E7. Теорема Эрдеша–Ко–Радо (Erdős–Ko–Rado)

Формулировка. Пусть $\mathcal{F}\subseteq\binom{[n]}{k}$ — семейство $k$-подмножеств $\{1,\ldots,n\}$, в котором любые два множества пересекаются ($A\cap B\neq\emptyset$). При $n\geq 2k$: $|\mathcal{F}|\leq\binom{n-1}{k-1}$. Равенство достигается на «звезде» — всех $k$-подмножествах, содержащих фиксированный элемент.

Условия применимости. Условие $n\geq 2k$ существенно: при $n<2k$ вообще $\binom{[n]}{k}$ — intersecting family, и оценка $\binom{n-1}{k-1}$ не работает. Для $t$-intersecting (попарное пересечение $\geq t$) — теорема Frankl–Wilson с порогом $n\geq(t+1)(k-t+1)$.

Идея доказательства. Метод сдвигов (shifting / Frankl): заменить $A\in\mathcal{F}$ на $S_{ij}(A)=(A\setminus\{j\})\cup\{i\}$ при $i<j$, если новое множество не уже в $\mathcal{F}$ — сохраняет $|\mathcal{F}|$ и intersecting-свойство. После всех сдвигов $\mathcal{F}$ становится «компрессированным», на нём оценка очевидна. Альтернатива (Katona): циклический permutation argument.

Варианты и обобщения.
- $t$-intersecting: Ahlswede–Khachatrian (полное решение для всех $n$).
- Векторный аналог: для подпространств $\dim k$ в $\mathbb{F}_q^n$.
- Cross-intersecting (две семьи): $|\mathcal{A}|\cdot|\mathcal{B}|\leq\binom{n-1}{k-1}^2$ при условии $A\cap B\neq\emptyset$ для всех $A,B$.

Используется в. [IMC-задачи на intersecting families, регулярно в Schweitzer].

### E8. Неравенство Боллобаша о парах (Bollobás set-pair inequality)

Формулировка. Пусть $(A_i,B_i)_{i=1}^m$ — пары конечных множеств такие, что $A_i\cap B_i=\emptyset$ для всех $i$, и $A_i\cap B_j\neq\emptyset$ при $i\neq j$. Тогда
$$\sum_{i=1}^m\binom{|A_i|+|B_i|}{|A_i|}^{-1}\leq 1.$$
В частности, если $|A_i|=a$, $|B_i|=b$ для всех $i$, то $m\leq\binom{a+b}{a}$.

Условия применимости. Условие skew (антисимметричное по $i,j$): только для $i\neq j$ требуется пересечение, диагональные пары не пересекаются. Если требовать $A_i\cap B_j\neq\emptyset$ для всех пар включая $i=j$ — задача тривиализуется.

Идея доказательства. Рассмотреть случайный линейный порядок $\pi$ на $\bigcup(A_i\cup B_i)$. Событие $E_i=\{$все элементы $A_i$ идут раньше всех $B_i$ в $\pi\}$ имеет вероятность $\binom{|A_i|+|B_i|}{|A_i|}^{-1}$. Из условия $A_i\cap B_j\neq\emptyset$ при $i\neq j$ получаем, что события $E_i$ попарно несовместны $\Rightarrow$ сумма вероятностей $\leq 1$.

Варианты и обобщения.
- Lovász (skew): достаточно $A_i\cap B_j\neq\emptyset$ только при $i<j$.
- Векторное обобщение: $A_i,B_i$ — наборы векторов в $V$ с $\dim\langle A_i\rangle\cap\dim\langle B_i\rangle=0$, $\langle A_i\rangle\cap\langle B_j\rangle\neq 0$ при $i\neq j$ — даёт ту же оценку.
- Применения: covering numbers, transversals в гиперграфах, нижние оценки для $\tau$-критических гиперграфов.

Используется в. [Schweitzer регулярно, IMC-задачи на intersecting/disjoint configurations].

### E9. Теорема Шура (Schur's theorem)

Формулировка. Для любого $r\in\mathbb{N}$ существует $S(r)\in\mathbb{N}$ такое, что при любой раскраске $\{1,2,\ldots,N\}$ в $r$ цветов с $N\geq S(r)$ найдутся одноцветные $x,y,z$ (не обязательно различные) с $x+y=z$. Граница: $S(r)\leq R_r-1$, где $R_r$ — диагональное число Рамсея для $r$ цветов.

Условия применимости. Конечность множества цветов; «monochromatic Schur triple» допускает $x=y$ (т.е. $z=2x$ — тривиальный случай). Чтобы исключить, нужно требовать $x<y$ или $x\neq y$ — тогда оценки те же по порядку.

Идея доказательства. Рассмотреть полный граф $K_N$ с вершинами $\{1,\ldots,N\}$. Раскрасим ребро $\{i,j\}$ цветом числа $|i-j|$. По Ramsey при $N\geq R_r$ есть монохроматический треугольник $i<j<k$ — числа $a=j-i$, $b=k-j$, $c=k-i$ одного цвета и $a+b=c$.

Варианты и обобщения.
- Rado's theorem: характеризация систем линейных уравнений с гарантированно монохроматическим решением (columns condition).
- Schur-like для $x_1+\cdots+x_{k-1}=x_k$: достаточные размеры через Ramsey.
- Density-Schur (Folkman): любое множество положительной верхней плотности содержит Schur triple.

Используется в. [Putnam раскраски, Schweitzer, IMO-задачи на additive Ramsey].

## Записи - Exotic

### E10. Теорема Рамсея с диагональным bound (Ramsey theorem)

Формулировка. Существует $R(s,t)\in\mathbb{N}$ такое, что любая 2-раскраска рёбер $K_n$ при $n\geq R(s,t)$ содержит красную $K_s$ или синюю $K_t$. Верхняя оценка: $R(s,t)\leq\binom{s+t-2}{s-1}$. В частности, $R(s,s)\leq\binom{2s-2}{s-1}\leq 4^{s-1}/\sqrt{s}$ (по Stirling). Нижняя (Erdős, probabilistic): $R(s,s)\geq 2^{s/2}$.

Условия применимости. Конечно много цветов, конечные графы; для бесконечного — Ramsey даёт бесконечный монохроматический подграф (без явных оценок). Hypergraph-Ramsey требует tower-type функций.

Идея доказательства. Индукция: $R(s,t)\leq R(s-1,t)+R(s,t-1)$. Берём вершину $v$ в $K_{R(s-1,t)+R(s,t-1)}$; либо $v$ имеет $\geq R(s-1,t)$ красных соседей (применяем индукцию), либо $\geq R(s,t-1)$ синих. Pascal-рекуррентность даёт биномиальный bound.

Варианты и обобщения.
- Multi-color Ramsey $R(s_1,\ldots,s_r)$: тот же приём индукцией.
- Off-diagonal: $R(3,t)=\Theta(t^2/\log t)$ (Kim, Ajtai–Komlós–Szemerédi).
- Diagonal bound недавно улучшен: $R(s,s)\leq(4-\varepsilon)^s$ (Campos–Griffiths–Morris–Sahasrabudhe 2023), но для олимпиады актуален биномиальный bound.

Используется в. [Putnam-1953-B6, Schweitzer, через Schur — IMO/IMC].

### E11. Pigeonhole на торе / окружности (combinatorial Fourier)

Формулировка. Пусть $\theta_1,\ldots,\theta_k\in\mathbb{R}/\mathbb{Z}$ и $n\in\mathbb{N}$. Разобьём $\mathbb{T}^k=(\mathbb{R}/\mathbb{Z})^k$ на $M^k$ ячеек (например, $M=6$). При $n>M^k$ среди $t=1,\ldots,n$ найдутся $t_1<t_2$ такие, что $\{t_1\boldsymbol\theta\}$ и $\{t_2\boldsymbol\theta\}$ попадают в одну ячейку. Тогда $r=t_2-t_1$ удовлетворяет $\|r\theta_j\|<1/M$ для всех $j$.

Условия применимости. Реальные $\theta_j$ (в иррациональном случае дают плотную орбиту), $n$ должно расти как $M^k$ — экспоненциально в $k$. Для $\mathbb{Z}/n\mathbb{Z}$ — конечная версия.

Идея доказательства. Прямое pigeonhole в $\mathbb{T}^k$. Применение: $|\sum e^{2\pi i a_j r}|$ как сумма $k$ единичных векторов с углами $r\theta_j$ — если все $\|r\theta_j\|$ малы, сумма близка к $k$. Это даёт нижнюю оценку $\sup_r|\sum e^{2\pi i a_j r}|\geq k/2$ при разбиении окружности на 6 дуг (углы в каждой дуге дают $\cos\geq 1/2$).

Варианты и обобщения.
- Vinogradov's lemma на короткие тригонометрические суммы.
- «Quantitative Weyl»: оценки expo-сумм через mod-$N$ pigeonhole.
- Cauchy–Davenport через pigeonhole в $\mathbb{Z}/p\mathbb{Z}$.

Используется в. [IMC-1999-DAY2-6].

### E12. Bound числа целых точек на выпуклой кривой (Andrews / Bombieri–Pila)

Формулировка. Пусть $\Gamma\subset\mathbb{R}^2$ — строго выпуклая кривая длины $L$ (или ограничивающая выпуклое тело площади $V$). Тогда $|\Gamma\cap\mathbb{Z}^2|\leq c\cdot L^{2/3}$ (Jarník 1925), и для гладкой кривой $|\Gamma\cap\mathbb{Z}^2|\leq c_\varepsilon L^{1/2+\varepsilon}$ (Bombieri–Pila 1989). На олимпиаде используется элементарная версия Jarník.

Условия применимости. Строгая выпуклость существенна — иначе на отрезке прямой можно посадить $L+1$ точку. Для невыпуклой части кривой оценка не работает.

Идея доказательства. Разбить $\Gamma$ на дуги, на каждой $\leq 2$ точек $\mathbb{Z}^2$ дают вектор-разность. Среди $k$ выбранных точек $\binom{k}{2}$ попарных разностей $(b_i-b_j, a_i-a_j)$ — это вектора с $|a|\leq 2L$, $|b|\leq 2L$ и параметром $\arg\in[0,2\pi)$. Из строгой выпуклости все векторы попарно различны как направленные, и оценка плотности дроби $b/a\in(-1,1)$ с $|a|\leq A$ — не более $2A^2$ — даёт $\binom{k}{2}\leq c\cdot L^2$, т.е. $k\leq cL^{2/3}$ после оптимизации (классический IMC-1994-6 trick).

Варианты и обобщения.
- Bombieri–Pila determinant method: для $C^\infty$ выпуклой даёт $L^{1/2+\varepsilon}$.
- Schmidt: $L^{2/3}$ для выпуклых кривых — нельзя улучшить (есть примеры $\geq cL^{2/3}$).
- Применение: оценки на $r_2(n)$ — число представлений $n$ как $a^2+b^2$.

Используется в. [IMC-1994-6].

## Self-test

1. В единичном квадрате отмечены $51$ точка. Докажи, что какие-то $3$ из них лежат внутри круга радиуса $1/7$.

2. Из $\{1,2,\ldots,2n\}$ выбрано $n+1$ числа. Докажи, что среди выбранных есть два, одно из которых делит другое.

3. На вечеринке $6$ человек: каждые двое либо знакомы, либо нет. Докажи, что среди них есть либо $3$ попарно знакомых, либо $3$ попарно незнакомых.

4. Последовательность $a_1,\ldots,a_{n^2+1}$ — попарно различные вещественные числа. Докажи, что существует строго монотонная подпоследовательность длины $n+1$.

5. На плоскости отмечены $n$ точек, не все на одной прямой. Докажи, что существует прямая, проходящая ровно через две из этих точек. (Sylvester–Gallai)

6. (Putnam-style) В семействе из $2^{n-1}+1$ подмножеств $\{1,\ldots,n\}$ найдутся два подмножества, одно из которых содержит другое.

7. Пусть $\alpha\in\mathbb{R}\setminus\mathbb{Q}$. Докажи, что для бесконечно многих натуральных $q$ выполнено $\|q\alpha\|<1/q$, где $\|x\|$ — расстояние до ближайшего целого.

## Решения

1. Разделим квадрат на $25$ квадратов $1/5\times1/5$. По pigeonhole в одном из них $\geq\lceil 51/25\rceil=3$ точки; их диаметр $\leq\sqrt{2}/5<1/\sqrt{12}$, описанный круг имеет радиус $\leq\sqrt{2}/(5\sqrt{2})=1/5$. (Аккуратнее: $25$ квадратов $\Rightarrow$ диаметр $\sqrt{2}/5\approx 0{,}283$; описанный круг радиуса $\sqrt{2}/(2\cdot 5)=1/(5\sqrt 2)\approx 0{,}141<1/7$.)

2. Каждое выбранное $a_i$ запишем как $a_i=2^{e_i}m_i$ с нечётным $m_i$. Нечётных $m_i\in\{1,3,\ldots,2n-1\}$ ровно $n$, выбранных чисел $n+1$ — pigeonhole: два числа $a_i$, $a_j$ имеют один $m$, и одно делит другое.

3. Зафиксируем человека $A$. Из $5$ остальных по pigeonhole $\geq 3$ либо знакомы, либо незнакомы с $A$ (WLOG знакомы — это $B,C,D$). Если среди $B,C,D$ есть знакомая пара, она с $A$ образует тройку знакомых; иначе $B,C,D$ — тройка незнакомых.

4. Erdős–Szekeres при $r=s=n+1$: $(n+1-1)^2+1=n^2+1$ — достаточно.

5. (Не pigeonhole, а чистый extremal принцип.) Среди всех пар (точка $P$, прямая $\ell$ через $\geq 2$ другие точки, $P\notin\ell$) выберем пару с минимальным расстоянием $d(P,\ell)>0$ (существует, иначе все точки коллинеарны). Пусть $\ell$ содержит точки $A_1,\ldots,A_k$ и $Q$ — проекция $P$ на $\ell$. Если $k\geq 3$, по pigeonhole две из $A_i$ лежат по одну сторону от $Q$ — пусть $A,B$, причём $A$ ближе к $Q$. Тогда расстояние от $A$ до прямой $PB$ меньше $d(P,\ell)$ — противоречие. Значит $k=2$.

6. По теореме Sperner максимальная антицепь в $(\mathcal{P}([n]),\subseteq)$ имеет размер $\binom{n}{\lfloor n/2\rfloor}$. Простое неравенство $\binom{n}{\lfloor n/2\rfloor}\leq 2^{n-1}$ при $n\geq 2$ (поскольку сумма всех биномиальных коэффициентов $2^n$ распределена хотя бы на две позиции $\lfloor n/2\rfloor$ и $\lfloor n/2\rfloor\pm 1$ симметрично). Значит при $|\mathcal{F}|\geq 2^{n-1}+1>\binom{n}{\lfloor n/2\rfloor}$ семейство $\mathcal{F}$ не может быть антицепью — есть сравнимая пара $A\subsetneq B$. (Sperner — следствие LYM-неравенства $\sum_S 1/\binom{n}{|S|}\leq 1$ для антицепи $\mathcal{F}$.)

7. Для каждого $N$ Дирихле даёт $1\leq q_N\leq N$ и $p_N\in\mathbb{Z}$ с $|q_N\alpha-p_N|<1/N\leq 1/q_N$. Допустим, множество $\{q\colon \|q\alpha\|<1/q\}$ конечно — обозначим его $\{r_1,\ldots,r_m\}$. Поскольку $\alpha\notin\mathbb{Q}$, все $\|r_i\alpha\|>0$; положим $\delta=\min_i\|r_i\alpha\|>0$. Возьмём $N>1/\delta$: соответствующий $q_N$ удовлетворяет $\|q_N\alpha\|<1/N<\delta$, значит $q_N\notin\{r_1,\ldots,r_m\}$. Но $\|q_N\alpha\|<1/q_N$ — противоречие.
