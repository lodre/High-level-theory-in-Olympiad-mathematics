# Биективные аргументы и double counting

## Scope
Подсчёт мощности конечного множества двумя способами и через явное соответствие с другим множеством. Классы задач: тождества с биномиальными коэффициентами, оценки в экстремальной теории множеств, инцидентности в графах и матрицах, доказательства равенств распределений, контрпримеры через инволюции, оценки числа конфигураций. Включает sign-reversing involutions, reflection / cycle arguments, orbit-counting, transversal arguments.

## Записи - Core

### E1. Double counting / incidence matrices (двойной счёт)

Формулировка. Пусть $|S|$ можно посчитать двумя независимыми способами: $|S|=A=B$. Стандартная реализация — bipartite incidence: для $X,Y$ конечных и $R\subseteq X\times Y$
$$\sum_{x\in X}|\{y:(x,y)\in R\}| = |R| = \sum_{y\in Y}|\{x:(x,y)\in R\}|.$$
Частные случаи: handshake lemma ($\sum_v \deg v = 2|E|$); тождества Vandermonde, hockey-stick; усреднение по строкам/столбцам.

Условия применимости. $X,Y$ конечны (или хотя бы $R$ конечно); $R$ задаёт отношение, для которого подсчёт fibres по обеим осям выполним.

Идея доказательства. Подсчёт $|R|$ как суммы $1$-индикаторов и применение Fubini для конечных сумм.

Варианты и обобщения. Взвешенная версия: $\sum_x f(x,\cdot) = \sum_y f(\cdot,y)$ для $f:X\times Y\to\mathbb{R}$. Двойной подсчёт пар: оценки через "удаление одного элемента" — основа extremal proofs (E14, E15).

Используется в. [IMC-2001-1, IMC-2018-5, IMC-2022-5, Putnam-1996-B1, базовый инструмент].

### E2. Bijective principle и инъекция/сюръекция

Формулировка. Если $\phi:A\to B$ биективна, то $|A|=|B|$. Достаточно доказать $|A|\le|B|$ и $|B|\le|A|$ через инъекции (Cantor–Bernstein для конечных множеств — тривиально). Для тождеств: построить $\phi$, привлечь явный обратный $\psi$, проверить $\psi\circ\phi=\text{id}_A$.

Условия применимости. Конечные множества (или счётные с явной нумерацией). Правило хорошего тона — указывать обратное отображение, иначе риск необоснованного "очевидно биекция".

Идея доказательства. Конструктивная: явная формула; или через sign-reversing involution на $A\sqcup B$ с неподвижными точками, образующими "core" равенства.

Варианты и обобщения.
- **Двойственная биекция**: $A\to B\to A$ — иногда легче, чем прямая.
- **Encoding / decoding**: представить элементы как слова, сравнить алфавиты.
- **"Transfer matrix" биекции**: матричное произведение интерпретируется как путь в графе.

Используется в. [IMC-2003-DAY2-5, IMC-2005-2, IMC-2021-2, Cayley E6].

### E3. Sign-reversing involution (инволюционный принцип)

Формулировка. Пусть $X = X^+ \sqcup X^-$ (signed set), и $\iota:X\to X$ — инволюция, меняющая знак на $X\setminus\text{Fix}(\iota)$, тогда
$$|X^+|-|X^-| = |\text{Fix}(\iota)\cap X^+| - |\text{Fix}(\iota)\cap X^-|.$$
Если все неподвижные точки в $X^+$ — получаем $|X^+|-|X^-|=|\text{Fix}(\iota)|$; если их нет — $|X^+|=|X^-|$.

Условия применимости. $\iota^2=\text{id}$, $\text{sign}(\iota(x))=-\text{sign}(x)$ для $x\notin\text{Fix}$. Инволюция должна быть well-defined: например, удалить минимальный элемент с особым свойством.

Идея доказательства. Спаривание $\{x,\iota(x)\}$ для $x\notin\text{Fix}$ даёт сокращение знаков.

Варианты и обобщения.
- **Franklin's involution** для теоремы Эйлера о пятиугольных числах: на разбиениях с различными частями инволюция действует "сдвинуть последний из ступенек на наклонную часть, или наоборот".
- **Чётность**: если $|X|$ нужно показать чётным — найти инволюцию без неподвижных точек.

Используется в. [IMC-2013-DAY2-5, IMC-2024-9, IMC-2012-DAY2-1, Pentagonal Number Theorem, LGV E10].

### E4. Vandermonde / Chu–Vandermonde и биномиальные тождества

Формулировка.
$$\binom{m+n}{r} = \sum_{k=0}^{r}\binom{m}{k}\binom{n}{r-k}\quad (\text{Vandermonde}).$$
Полиномиальное расширение (Chu–Vandermonde) для $x,y\in\mathbb{C}$:
$$\sum_{k=0}^{r}\binom{x}{k}\binom{y}{r-k}=\binom{x+y}{r}.$$
Hockey-stick: $\sum_{k=r}^{n}\binom{k}{r}=\binom{n+1}{r+1}$. Identity Worpitzky: $x^n = \sum_k \langle{n\atop k}\rangle\binom{x+k}{n}$.

Условия применимости. Целые $r,n,m\ge 0$ для базовой; для Chu–Vandermonde — равенство многочленов от $x,y$ степени $r$ обоснованно через бесконечность совпадений на целочисленных точках.

Идея доказательства. Vandermonde: выбрать $r$ из $m+n$ объектов, разделённых на две группы. Hockey-stick: первый "успех" на позиции $k+1$. Worpitzky: разложение в базисе $\binom{x+k}{n}$.

Варианты и обобщения. Chu–Vandermonde-q: $q$-аналог $\sum_k\binom{x}{k}_q\binom{y}{r-k}_q q^{(x-k)(r-k)}=\binom{x+y}{r}_q$. Аналоги для гипергеометрических сумм $_2F_1$ Гаусса.

Используется в. [Putnam-2002-B5, IMC-2010-DAY2-2 (как часть алгебры), классика].

### E5. Cycle lemma (Дворецкий–Моцкин) и reflection principle (Андре)

Формулировка.
- **Cycle lemma**. В круговой последовательности из $a$ символов $+1$ и $b$ символов $-1$ с $a-b=k>0$ ровно $k$ циклических сдвигов имеют все строго положительные префиксные суммы. Следствие: $C_n=\frac{1}{n+1}\binom{2n}{n}$ — путей Дика длины $2n$.
- **Reflection principle**. # лестничных путей из $A=(a_1,a_2)$ в $B=(b_1,b_2)$, касающихся (или пересекающих) прямую $L$, равно # путей из $A'$ в $B$, где $A'$ — отражение $A$ через $L$.

Условия применимости. Cycle lemma: $a>b\ge 0$ (для $k=a-b>0$). Reflection: лестничные шаги $(+1,0),(0,+1)$, прямая $L$ проходит "снаружи" от концов в нужном смысле; $A'$ строится отражением через "первое касание".

Идея доказательства. Cycle: рассмотреть позицию минимума частичной суммы; Reflection: bijection $\pi\mapsto \pi'$ зеркалит часть пути до первого касания $L$.

Варианты и обобщения. Bertrand ballot: $P(\text{строгое лидерство кандидата } A) = (a-b)/(a+b)$. Chung–Feller: путей с $k$ "плохими" шагами поровну при всех $k=0,\dots,n$. Многомерное обобщение через LGV (E10).

Используется в. [Putnam-2008-A4, Putnam-2017-A5, IMC-2018-8 (frog Catalan-like, см. также OGF E13)].

## Записи - Standard

### E6. Prüfer code и формула Кэли $n^{n-2}$

Формулировка. Существует биекция $\Phi$ между labeled trees на $[n]$ ($n\ge 2$) и последовательностями $(a_1,\dots,a_{n-2})\in[n]^{n-2}$. Следствие: число labeled trees равно $n^{n-2}$.

Условия применимости. Деревья на $n\ge 2$ помеченных вершинах. Для rooted trees: $n\cdot n^{n-2}=n^{n-1}$.

Идея доказательства. Кодирование: пока в дереве $>2$ вершин, удалять лист с минимальной меткой и записывать метку его соседа. Декодирование: жадная реконструкция, начиная с "наименьшей метки, не встретившейся в оставшемся хвосте кода".

Варианты и обобщения.
- **Joyal proof**: биекция labeled doubly rooted trees ↔ functions $[n]\to[n]$ через циклическую часть.
- **Hurwitz formula**: forests with $k$ компонент = $k\cdot n^{n-k-1}$ (Cayley generalisation).
- **Spanning trees произвольного $K_n$ с весами**: Matrix-Tree theorem.

Используется в. [Putnam-1996-B1 (counting via Prüfer-like decomposition), стандарт].

### E7. LYM inequality и теорема Шпернера (Sperner)

Формулировка. Пусть $\mathcal{F}\subseteq 2^{[n]}$ — антицепь (никакие $A\subsetneq B$). Тогда
$$\sum_{F\in\mathcal{F}}\frac{1}{\binom{n}{|F|}}\le 1\quad (\text{LYM}),$$
откуда $|\mathcal{F}|\le\binom{n}{\lfloor n/2\rfloor}$ (Sperner). Равенство достигается на семействе всех $\lfloor n/2\rfloor$-подмножеств (или $\lceil n/2\rceil$).

Условия применимости. $\mathcal{F}$ — антицепь по вложению. Версия для multiset / посет — Mirsky / Dilworth.

Идея доказательства. Двойной счёт максимальных цепочек $\emptyset=C_0\subset C_1\subset\dots\subset C_n=[n]$: всего $n!$, через каждый $F$ их $|F|!(n-|F|)!$. Любая цепочка содержит $\le 1$ элемент антицепи.

Варианты и обобщения. **Bollobás set-pair inequality**: для пар $(A_i,B_i)$ с $A_i\cap B_i=\emptyset$, $A_i\cap B_j\ne\emptyset$ при $i\ne j$ имеет место $\sum 1/\binom{|A_i|+|B_i|}{|A_i|}\le 1$. **Kruskal–Katona** — точная оценка теней. Эрдёш–Ко–Радо (см. E14).

Используется в. [Putnam-2005-B6, IMC-2018-5 (extremal antichain ideas), olympiad антицепи].

### E8. Burnside / Cauchy–Frobenius lemma (orbit counting)

Формулировка. Для конечной группы $G$, действующей на конечном множестве $X$:
$$|X/G| \;=\; \frac{1}{|G|}\sum_{g\in G}|X^g|,$$
где $X^g=\{x\in X: gx=x\}$ — фиксаторы.

Условия применимости. $|G|<\infty$, $X$ конечно. Действие может быть произвольным, не обязательно транзитивным.

Идея доказательства. Двойной счёт пар $(g,x)$ с $gx=x$: $\sum_g|X^g|=\sum_x|G_x|$, $|G_x|=|G|/|\text{orb}(x)|$, орбита размера $k$ даёт $k\cdot|G|/k=|G|$ вкладов; делим на $|G|$.

Варианты и обобщения.
- **Pólya enumeration**: с "цветами" — $|X/G|=\frac{1}{|G|}\sum_g w(g)$, $w(g)=$ produktов по циклам.
- **Frobenius variant**: $\sum_g\chi(g)^k=|G|\cdot\#\{H\le G: |H|=|G|/k,\dots\}$ — для подсчёта решений уравнений в группах.

Используется в. [IMC-2023-5 ($S_k$-orbit lower bound), Putnam-1991-B3, классические задачи о бусинах/раскрасках].

### E9. Hall's marriage theorem / SDR

Формулировка. Семейство $A_1,\dots,A_n$ конечных множеств имеет систему различных представителей (SDR) тогда и только тогда, когда для всякого $S\subseteq[n]$
$$\Big|\bigcup_{i\in S}A_i\Big|\ge|S|\quad (\text{Hall condition}).$$

Условия применимости. $n<\infty$ и каждый $A_i$ конечен (для бесконечного — нужны дополнительные условия типа компактности). Для bipartite графа: perfect matching на $X$ существует ⟺ $|N(W)|\ge|W|$ для всех $W\subseteq X$.

Идея доказательства. Индукция по $n$ + альтернирующий путь: либо строгое неравенство во всех собственных $S$ (тривиально берём представителя из $A_n$ и применяем индукцию), либо есть critical $S$ — разбиваем семейство на две части и применяем индукцию к каждой.

Варианты и обобщения.
- **König's theorem** (bipartite): max matching = min vertex cover.
- **Defect Hall**: max matching $=n-\max_S(|S|-|N(S)|)$.
- **Hall на бесконечных семействах** — нужна compactness (P. Hall 1935, M. Hall 1948).

Используется в. [Putnam-2012-B4 (matching argument), стандартный инструмент для существования совершенных конфигураций].

### E10. Lindström–Gessel–Viennot lemma

Формулировка. В DAG $G$ с вершинами-источниками $A_1,\dots,A_n$ и стоками $B_1,\dots,B_n$, мультипликативным весом $w$ на путях:
$$\det\big[\,e(A_i,B_j)\,\big]_{i,j=1}^{n} \;=\; \sum_{(P_1,\dots,P_n)\text{ непересек.}}\text{sgn}(\sigma)\prod_i w(P_i),$$
где $e(A_i,B_j)=\sum_{P:A_i\to B_j}w(P)$ и $\sigma$ — индуцированная перестановка. На "compatible" конфигурациях (только $\sigma=\text{id}$ даёт непересекающиеся системы) определитель = взвешенное число непересекающихся путей.

Условия применимости. DAG; конечность сумм $e(A_i,B_j)$; "компатибельность" — любая пара пересекающихся путей $P_i,P_j$ допускает swap в первой общей вершине.

Идея доказательства. Sign-reversing involution: swap двух путей в первой точке пересечения меняет $\text{sgn}(\sigma)$ и оставляет вес неизменным; неподвижные = непересекающиеся системы.

Варианты и обобщения. **MacMahon box formula** для plane partitions через $3$-мерные пути. **Jacobi–Trudi** identity для Schur functions.

Используется в. [Putnam-2005-B6, IMC-2018-8, доказательство многих $\det$-тождеств комбинаторно].

### E11. Goodman formula (число одноцветных треугольников)

Формулировка. В $2$-рёберной раскраске $K_n$ число одноцветных треугольников
$$M(n) \;=\; \binom{n}{3} - \frac{1}{2}\sum_{v\in V}\deg_R(v)\deg_B(v),$$
где $\deg_R,\deg_B$ — степени в красном/синем подграфах. Минимум:
$$M(n)\ge \begin{cases}\frac{n(n-2)(n-4)}{24},& n\equiv 0\pmod 2\\[2pt]\frac{n(n-1)(n-5)}{24},& n\equiv 1\pmod 4\\[2pt]\frac{(n+1)(n-3)(n-4)}{24},& n\equiv 3\pmod 4\end{cases}$$

Условия применимости. $K_n$, $2$ цвета на рёбрах; обобщается на больше цветов с потерей точности.

Идея доказательства. "Угол" в вершине = упорядоченная пара рёбер из неё. Двухцветный угол — два разных цвета. Каждый разноцветный треугольник содержит ровно $2$ двухцветных угла; одноцветный — $0$. Итого $2(\binom{n}{3}-M)=\sum_v\deg_R\deg_B$. Минимизация $\deg_R\deg_B$ при $\deg_R+\deg_B=n-1$ даёт оценку (равенство на $\deg_R=\lfloor(n-1)/2\rfloor$).

Варианты и обобщения. Ramsey-type bound: $R(3,3)=6$ — тривиальное следствие ($M(6)\ge 2$). Erdős–Lovász: оценки monochrom $K_4$ через flag algebras (далеко за рамки olympiad).

Используется в. [IMC-2022-5 (точная Goodman-оценка для $n=43$)].

### E12. Plünnecke–Ruzsa inequality

Формулировка. Для конечных $A,B\subset G$ (абелева группа) с $|A+B|\le K|A|$ и любых целых $m,n\ge 0$:
$$|nB-mB|\le K^{n+m}|A|.$$
Частный случай ($A=B$, $|A+A|\le K|A|$): $|kA-\ell A|\le K^{k+\ell}|A|$ для всех $k,\ell\ge 0$.

Условия применимости. $G$ абелева (для неабелевой — слабее, через Tao). Предположение doubling $|A+A|\le K|A|$ — строго.

Идея доказательства. Граф $\Gamma$ с двумя долями $A,A+B$, рёбра $a\sim a+b$. Магическая Ruzsa graph inequality по tensor power; средняя степень и оценка через "компрессирующую" прокладку.

Варианты и обобщения. **Ruzsa triangle inequality**: $|A-C|\cdot|B|\le|A-B|\cdot|B-C|$. **Freiman–Ruzsa theorem**: $|A+A|\le K|A|\Rightarrow A\subseteq P$, $P$ — обобщённая арифметическая прогрессия.

Используется в. [IMC-2012-DAY2-5 (прямое применение с $K=c$ и $k=O(\log)$)].

### E13. Bit-pairing / dyadic complementation

Формулировка. Для $k$ с $\lfloor\log_2 k\rfloor=m$ положим $\bar k=2^{m+1}-1-k$. Тогда $k\mapsto\bar k$ — инволюция на $[2^m,2^{m+1}-1]$, $k+\bar k=2^{m+1}-1$. На каждой dyadic полосе $[2^m,2^{m+1})$ длины $2^m$ суммы парных значений симметричных функций часто легко считаются.

Условия применимости. Функция $f$ "симметрична относительно середины полосы" в нужном смысле (например, биекция полосы в себя или в её отражение).

Идея доказательства. Прямое: разбить $[1,n]$ на полосы $[2^m,2^{m+1}-1]$ (последняя — обрезанная), внутри каждой полосы пара $(k,\bar k)$ даёт сумму $2^{m+1}-1$.

Используется в. [IMC-2015-2 ($\sum f(k)\le n^2/4$ через bit-pairing с равенством на $n=2^k-1$)].

## Записи - Exotic

### E14. Erdős–Ko–Rado theorem

Формулировка. Пусть $\mathcal{F}\subseteq\binom{[n]}{k}$ — пересекающееся семейство ($A\cap B\ne\emptyset$ для всех $A,B\in\mathcal{F}$), $n\ge 2k$. Тогда
$$|\mathcal{F}|\le\binom{n-1}{k-1}.$$
Равенство — "звёзды": все $k$-множества, содержащие фиксированную точку.

Условия применимости. $n\ge 2k$ (иначе всё $\binom{[n]}{k}$ пересекающееся). Для $k$-множеств; обобщения на $t$-intersecting (Frankl).

Идея доказательства (Katona). Разместить $[n]$ на цикле; число "арков" (последовательных $k$-множеств) внутри $\mathcal{F}$ — не более $k$ для одной циклической перестановки. Двойной счёт пар (cyclic permutation, $A\in\mathcal{F}$).

Варианты и обобщения. **Hilton–Milner**: вторая по величине пересекающаяся семья. **Frankl–Wilson** (модулярный EKR): мощные оценки через линейную алгебру. **Ahlswede–Khachatryan** complete intersection.

Используется в. [классические olympiad на пересекающиеся семейства; IMC-shortlists].

### E15. Fisher's inequality / oddtown / eventown

Формулировка.
- **Oddtown**. $A_1,\dots,A_m\subseteq[n]$, $|A_i|$ нечётны, $|A_i\cap A_j|$ чётны при $i\ne j$. Тогда $m\le n$.
- **Fisher's inequality**. Если $|A_i\cap A_j|=\lambda$ постоянна для $i\ne j$, то $m\le n$ (без требования равенства размеров).
- **Eventown**. $|A_i|$ чётны, $|A_i\cap A_j|$ чётны. Тогда $m\le 2^{\lfloor n/2\rfloor}$.

Условия применимости. Подмножества $[n]$; характеристики над $\mathbb{F}_2$ (oddtown/eventown) или $\mathbb{R}$ (Fisher).

Идея доказательства. Векторы-индикаторы $v_i\in\mathbb{F}_2^n$ — Gram-матрица $V V^T = I$ (oddtown) или $J+\text{diag}$ (Fisher) — невырождена, ранг $\ge m$, $m\le n$. Eventown: $V V^T=0$, изотропное подпространство $\le n/2$.

Варианты и обобщения. **Modular Fisher** (Frankl–Wilson): ограничения mod $p$. **Ray-Chaudhuri–Wilson**: пересечения с размерами в фиксированном множестве $L$ — $m\le\binom{n}{|L|}$.

Используется в. [IMC-2002-DAY2-2 (Fisher-style двойной счёт пар), Schweitzer задачи, классика linear algebra method].

### E16. De Bruijn–Erdős theorem (точки и линии)

Формулировка. Пусть $n\ge 2$ точек на плоскости (или абстрактных) и линий (= собственных подмножеств точек) таких, что любые две точки лежат ровно на одной линии. Тогда число линий $\ge n$, с равенством только для (a) почти-пучка (одна линия с $n-1$ точкой плюс $n-1$ линий через одну точку), (b) проективной плоскости порядка $q$ (при $n=q^2+q+1$).

Условия применимости. Точечно-линейная инцидентность с уникальностью. Финитарная версия (Bose–Erdős–de Bruijn 1948).

Идея доказательства. Двойной счёт инциденций + Fisher's inequality на character vectors (или прямая комбинаторная индукция в духе Conway).

Варианты и обобщения. **Motzkin–Rabin**: цветной аналог. **Szemerédi–Trotter incidence bound** $O((np)^{2/3}+n+p)$ для $n$ точек и $p$ линий — далеко за пределами olympiad, но идея инцидентного двойного счёта одна.

Используется в. [Putnam-2011-A6 (вариант инцидентного аргумента), olympiad задачи на коллинеарность].

### E17. Garsia–Milne involution principle

Формулировка. Пусть $X=X^+\sqcup X^-$, $\alpha,\beta:X\to X$ — две инволюции, обе sign-reversing (вне своих fixed sets). Тогда композиция $(\beta\alpha)^k$ "стабилизируется" на $\text{Fix}(\alpha)\cap\text{Fix}(\beta)$, и существует биекция
$$\phi:\,\text{Fix}(\alpha)\cap X^+ \;\longleftrightarrow\; \text{Fix}(\beta)\cap X^+,$$
$\phi(x)=(\beta\alpha)^{N(x)}(x)$ для подходящего $N(x)$.

Условия применимости. $\alpha,\beta$ — инволюции, sign-reversing. Композиции $(\beta\alpha)^k(x)$ должны попадать в одну из неподвижных множеств за конечное число шагов (выполнено для конечных $X$).

Идея доказательства. На $X\setminus(\text{Fix}\alpha\cup\text{Fix}\beta)$ орбиты $\langle\alpha,\beta\rangle$ — конечные двухконтурные последовательности; концы (fixed points) одной чётности связаны траекторией $(\beta\alpha)^k$.

Варианты и обобщения. **Garsia–Milne RR-bijection**: первое биективное доказательство Rogers–Ramanujan; даёт "механизированное" построение биекций из двух sign-reversing involutions.

Используется в. [исследовательский олимпиадный уровень — IMC long-list, Schweitzer; основа bijective combinatorics].

## Self-test

1. Сколько всего пар $(A,B)$ подмножеств $[n]$ удовлетворяют $A\subseteq B$? Дать два независимых выражения и доказать их равенство.

2. В $K_{10}$ каждое ребро покрашено в один из двух цветов. Найти точную нижнюю оценку числа одноцветных треугольников.

3. Пусть $n\ge 4$. Доказать: число помеченных деревьев на $[n]$, в которых вершина $1$ — лист, равно $(n-1)^{n-2}$.

4. $\mathcal{F}\subseteq 2^{[n]}$, никакое $A\in\mathcal{F}$ не содержит другого. Доказать $|\mathcal{F}|\le\binom{n}{\lfloor n/2\rfloor}$.

5. Дано $n$ человек и $n$ работ, каждый человек квалифицирован для некоторого набора работ. Доказать: распределение по принципу "1 человек ↔ 1 работа из его списка" возможно тогда и только тогда, когда любое подмножество людей квалифицировано в совокупности на не меньше работ, чем число этих людей.

6. Доказать, что число самосопряжённых разбиений числа $n$ (диаграмма Юнга $\lambda$ совпадает со своей транспонированной $\lambda^T$) равно числу разбиений $n$ на различные нечётные части.

7. Пусть $a_n$ — число последовательностей $(\epsilon_1,\dots,\epsilon_n)\in\{-1,+1\}^n$ с $\sum\epsilon_i\ge 0$ и всеми частичными суммами $\ge 0$. Найти $a_n$.

8. Пусть $n\ge 2$ и $m$ — число способов разбить $K_n$ на гамильтонов цикл и оставшиеся рёбра. Доказать, что $m=n!/(2n)$ (для $n=$нечётного), используя Burnside на действии $D_n$.

9. Биективно доказать тождество $\displaystyle\sum_{k=0}^{n}\binom{n}{k}^2 = \binom{2n}{n}$.

10. Найти количество пар $(A,B)$ непустых $\subseteq[n]$, где $A\cap B=\emptyset$ и $\max A<\min B$. Привести ответ в виде закрытой формулы.

## Решения

1. LHS: $|\{(A,B): A\subseteq B\subseteq[n]\}| = \sum_{B}2^{|B|} = \sum_{k=0}^{n}\binom{n}{k}2^k = 3^n$. RHS (независимый счёт): для каждого $i\in[n]$ — $3$ варианта: $i\notin B$, $i\in B\setminus A$, $i\in A$. Итого $3^n$.

2. По E11: $M(10)=\binom{10}{3}-\frac{1}{2}\sum_v\deg_R(v)\deg_B(v)$, $\deg_R+\deg_B=9$. Максимум произведения $\deg_R\deg_B$ при $\{4,5\}$: $20$. Сумма $\le 10\cdot 20=200$, $M(10)\ge 120-100=20$. Точная Goodman при $n\equiv 2\pmod 4$: $M(10)\ge\frac{10\cdot 8\cdot 6}{24}=20$, достижимо.

3. По E6 (Prüfer): вершина $1$ — лист ⟺ $1$ не появляется в Prüfer-коде. Кодов длины $n-2$ в алфавите $[n]\setminus\{1\}$ — $(n-1)^{n-2}$.

4. По E7 (LYM): $\sum_{F\in\mathcal{F}}1/\binom{n}{|F|}\le 1$; макс. $\binom{n}{|F|}=\binom{n}{\lfloor n/2\rfloor}$, отсюда $|\mathcal{F}|\le\binom{n}{\lfloor n/2\rfloor}$.

5. По E9 (Hall) — прямая формулировка. $(\Rightarrow)$: если SDR $r_i\in A_i$ существует, то $\bigcup_{i\in S}A_i\supseteq\{r_i: i\in S\}$, $|S|$ различных. $(\Leftarrow)$: индукция по $n$ — стандартный аргумент с разбиением на critical / tight $S$.

6. По E2 (явная биекция Сильвестра). Самосопряжённое разбиение разрезается "крюками" вдоль диагонали: $i$-й крюк — крест из строки $i$ и столбца $i$ начиная с диагональной клетки $(i,i)$. Длина $i$-го крюка $= 2(\lambda_i-i)+1$ — нечётная и строго убывает по $i$. Обратно: упорядочив различные нечётные части $2a_1+1>2a_2+1>\dots$, восстанавливаем диаграмму последовательной "сборкой" крюков от внешнего к внутреннему. Биекция явная и обратимая.

7. По E5 (reflection principle). Путь $(\epsilon_1,\dots,\epsilon_n)$ с $S_k=\sum_{i\le k}\epsilon_i\ge 0$ для всех $k$ — "ballot path" с нестрогим лидерством. Bijective reflection: путь, нарушающий $S_k\ge 0$, отражается через $-1$ при первом касании $-1$ — даёт путь, заканчивающийся на $-2-\sum\epsilon_i$. Подсчёт через "плохие" пути с $\sum=2j-n\ge 0$: $a_n=\sum_{k\ge \lceil n/2\rceil}\big(\binom{n}{k}-\binom{n}{k+1}\big)=\binom{n}{\lceil n/2\rceil}$. Проверка: $n=4$: $\binom{4}{2}=6$ ✓.

8. По E8 (Burnside): множество гамильтоновых циклов в $K_n$ как циклических перестановок $[n]$ — действие $D_n$ (повороты + отражения, $|D_n|=2n$). Каждая орбита соответствует геометрически различимому циклу. Всего нумерованных циклов $(n-1)!$ (фиксируем старт), но как последовательность с направлением — $n!$. По Burnside: $|H|=\frac{1}{2n}\sum_g|H^g|$. При $n$ нечётном только $\text{id}$ имеет неподвижные точки (нет повторов в $n$-цикле), $|H|=n!/(2n)$.

9. По E4 / E1. Считаем подмножества $S\subseteq[2n]$ размера $n$ двумя способами. Прямой счёт: $\binom{2n}{n}$. Параметризация: пусть $k=|S\cap[1,n]|$, тогда $|S\cap[n+1,2n]|=n-k$, отсюда $\binom{n}{k}\binom{n}{n-k}=\binom{n}{k}^2$. Суммируем по $k$. Биективно: $S\leftrightarrow(S_1,S_2)$, где $S_1=S\cap[1,n]$ и $S_2=([n+1,2n]\setminus S)-n$ — оба подмножества $[1,n]$ одинакового размера $k$, что даёт тройку $(k,S_1,S_2)$ с $|S_1|=|S_2|=k$.

10. По E1 (двойной счёт через параметризацию). Фиксируем $\max A=k$, $\min B=l$ с $1\le k<l\le n$. Тогда $A\subseteq[1,k]$, $k\in A$: $2^{k-1}$ вариантов; $B\subseteq[l,n]$, $l\in B$: $2^{n-l}$. Итого
$$a_n=\sum_{1\le k<l\le n}2^{k-1}\cdot 2^{n-l}=\sum_{d=1}^{n-1}(n-d)\cdot 2^{n-d-1}=\sum_{m=1}^{n-1}m\cdot 2^{m-1}=(n-2)\cdot 2^{n-1}+1.$$
Проверка: $n=2$: $a_2=1$ ✓; $n=3$: $a_3=5$ ✓ (пары $\{1\}|\{2\}, \{1\}|\{3\}, \{2\}|\{3\}, \{1,2\}|\{3\}, \{1\}|\{2,3\}$).
