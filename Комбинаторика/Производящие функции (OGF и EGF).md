# Производящие функции (OGF и EGF)

## Scope

Формальные степенные ряды как инструмент перечисления. Сюда входят рекуррентные соотношения, разбиения, пути на решётке, перестановки с ограничениями, фильтрация по корням из единицы, операции над помеченными структурами, обращение Лагранжа. Подтема покрывает задачи, где ответ — суммарная статистика по конечному комбинаторному классу, либо рекуррента решается алгебраически.

## Определения

Формальный степенной ряд — выражение $A(x)=\sum_{n\ge 0} a_n x^n$ с коэффициентами $a_n\in\mathbb{C}$. Рассматривается как элемент кольца $\mathbb{C}[[x]]$. Сходимость не требуется: операции определены через формальные правила.

Извлечение коэффициента: $[x^n]A(x):=a_n$.

Обыкновенная производящая функция последовательности $(a_n)$ — ряд $A(x)=\sum a_n x^n$. Далее — ОПФ (англ. OGF).

Экспоненциальная производящая функция последовательности $(a_n)$ — ряд $\hat A(x)=\sum a_n \dfrac{x^n}{n!}$. Далее — ЭПФ (англ. EGF).

Операции в $\mathbb{C}[[x]]$ — сложение, умножение, формальное дифференцирование, формальная подстановка $A(B(x))$ при $B(0)=0$, и обращение $1/A$ при $a_0\ne 0$.

Помеченная (англ. labeled) структура на множестве $[n]=\{1,\dots,n\}$ — комбинаторный объект, в котором элементы базового множества различимы. ЭПФ — естественный инструмент перечисления таких структур.

Непомеченная (англ. unlabeled) структура — объект, в котором различимы только сами элементы как абстрактные точки. ОПФ — естественный инструмент для них.

Свёртка Коши (англ. Cauchy product): $[x^n] A(x)B(x)=\sum_{k=0}^{n}a_k b_{n-k}$.

Биномиальная свёртка: $[x^n/n!]\hat A(x)\hat B(x)=\sum_{k=0}^{n}\binom{n}{k}a_k b_{n-k}$.

Вероятностная производящая функция (англ. probability generating function, PGF): $G_X(z)=\mathbb{E}[z^X]=\sum_k P(X=k)\,z^k$ для целочисленной неотрицательной случайной величины $X$.

## Записи — Основные

### E1. Правила трансфера для ОПФ

Утверждение. Пусть $A(x)=\sum a_n x^n$, $B(x)=\sum b_n x^n$. Тогда:

- $A+B$ соответствует $a_n+b_n$.
- Произведение $AB$ соответствует свёртке $\sum_{k=0}^{n} a_k b_{n-k}$.
- $xA(x)$ соответствует сдвигу $a_{n-1}$ (для $n\ge 1$).
- $A'(x)$ соответствует $(n+1)a_{n+1}$.
- $\dfrac{A(x)}{1-x}$ соответствует частичной сумме $\sum_{k\le n}a_k$.
- Композиция $A(B(x))$ корректна, если $B(0)=0$.

Условия. Композиция требует $B(0)=0$. Пример: при $B(x)=1$ ряд $A(B(x))=A(1)$ — это значение, а не формальный ряд, и для $A(x)=1/(1-x)$ оно не определено.

Идея. Прямой подсчёт коэффициентов.

---

### E2. Правила трансфера для ЭПФ и биномиальная свёртка

Утверждение. Для $\hat A(x)=\sum a_n x^n/n!$, $\hat B(x)=\sum b_n x^n/n!$:

$$\hat A(x)\,\hat B(x)=\sum_{n\ge 0}\Bigl(\sum_{k=0}^{n}\binom{n}{k}a_k b_{n-k}\Bigr)\frac{x^n}{n!}.$$

Производная: $\hat A'(x)$ соответствует $a_{n+1}$. Растяжение: $\hat A(\alpha x)$ соответствует $\alpha^n a_n$.

Условия. Произведение двух ЭПФ перечисляет упорядоченную пару (A-структура на $S$, B-структура на $[n]\setminus S$) с биномиальным коэффициентом — отсюда $\binom{n}{k}$. Условие законности — формальное, никаких ограничений сверх кольца $\mathbb{C}[[x]]$.

Идея. $\dfrac{x^n}{n!}\cdot\dfrac{x^m}{m!}=\binom{n+m}{n}\dfrac{x^{n+m}}{(n+m)!}$.

### E3. Фильтр по корням из единицы (англ. roots of unity filter)

Утверждение. Пусть $\omega=e^{2\pi i/m}$. Тогда

$$\sum_{n\equiv r\pmod m} a_n\,x^n=\frac{1}{m}\sum_{j=0}^{m-1}\omega^{-jr}\,A(\omega^j x).$$

В частности при $x=1$ получаем сумму подпоследовательности

$$\sum_{n\equiv r\pmod m} a_n=\frac{1}{m}\sum_{j=0}^{m-1}\omega^{-jr}A(\omega^j).$$

Условия. Значения $A(\omega^j)$ должны быть определены. Для многочленов и сходящихся рядов — всегда. Для расходящихся формальных рядов работаем коэффициент-за-коэффициентом, без подстановки.

Идея. Свойство дискретного преобразования Фурье на $\mathbb{Z}/m$:

$$\frac{1}{m}\sum_{j=0}^{m-1}\omega^{j(n-r)}=[n\equiv r\pmod m].$$

---

### E4. Линейные рекурренты с постоянными коэффициентами

Утверждение. Если последовательность удовлетворяет рекурренте $\sum_{i=0}^{k} c_i a_{n+i}=0$ с $c_k\ne 0$, то ОПФ имеет вид

$$A(x)=\frac{P(x)}{c_k+c_{k-1}x+\dots+c_0 x^k}=\frac{P(x)}{\prod_j(1-\alpha_j x)^{m_j}},$$

где $\alpha_j$ — корни характеристического многочлена $p(t)=\sum c_i t^{k-i}$ кратностей $m_j$. Разложение на простые дроби даёт

$$a_n=\sum_j Q_j(n)\,\alpha_j^n,\qquad \deg Q_j<m_j.$$

Условия. Рекуррента однородная, $c_k\ne 0$.

Пример нарушения. При $c_k=0$ степень рекурренты падает на $n$-м шаге не у всех индексов, и базис решений не описывается корнями характеристического многочлена.

Идея. Перемножение рекурренты на $x^n$ и суммирование даёт $A(x)\cdot q(x)=P(x)$, где $P$ — многочлен от начальных условий.

---

### E5. Метод змеиного масла (англ. snake oil method, Wilf)

Утверждение. Чтобы вычислить $S(n)=\sum_k f(n,k)$, образуем $F(x)=\sum_n S(n)\,x^n$. Меняем порядок суммирования:

$$F(x)=\sum_k\Bigl(\sum_n f(n,k)\,x^n\Bigr).$$

Внутреннюю сумму выражаем через известную ОПФ. Закрытое выражение для $F(x)$ читается коэффициент-в-коэффициент и даёт $S(n)$.

Условия. Для перестановки сумм в кольце формальных рядов нужно, чтобы для каждого фиксированного $n$ ненулевых членов $f(n,k)$ было конечное число. Иначе требуется аналитическое обоснование (абсолютная сходимость, теорема Фубини).

Пример нарушения. Сумма $\sum_n\sum_k \binom{k}{n}x^n$ с $k$ пробегающим всю $\mathbb{N}$ даёт расходящееся выражение даже формально: коэффициент при $x^n$ — бесконечная сумма.

Идея. Сила метода — не нужно знать ответ заранее, метод его находит.

## Записи — Стандартные

### E6. Уравнение Каталана для производящей функции

Утверждение. ОПФ чисел Каталана $C_n=\dfrac{1}{n+1}\binom{2n}{n}$ удовлетворяет

$$C(x)=1+xC(x)^2,\qquad C(x)=\frac{1-\sqrt{1-4x}}{2x}.$$

Условия. Рекурсивная декомпозиция объекта по схеме «корень + два независимых поддерева такого же типа». К этой схеме сводятся пути Дика (англ. Dyck paths), бинарные деревья, скобочные расстановки, триангуляции выпуклого $(n+2)$-угольника.

Идея. Декомпозиция через первое возвращение пути в ноль даёт квадратное уравнение. Из двух ветвей выбираем ту, в которой $C(0)=1$.

Варианты. $k$-арные деревья дают $T(x)=1+xT(x)^k$, откуда $T_n=\dfrac{1}{(k-1)n+1}\binom{kn}{n}$ — числа Фусса–Каталана.

### E7. Формула обращения Лагранжа (англ. Lagrange inversion formula)

Утверждение. Пусть $\phi\in\mathbb{C}[[t]]$, $\phi(0)\ne 0$, и $w(x)$ — единственное решение уравнения $w=x\phi(w)$ с $w(0)=0$. Для любого формального ряда $H$:

$$[x^n]\,H(w(x))=\frac{1}{n}[t^{n-1}]\bigl(H'(t)\,\phi(t)^n\bigr).$$

В частности при $H(t)=t$: $[x^n]w(x)=\dfrac{1}{n}[t^{n-1}]\phi(t)^n$.

Условия. $\phi(0)\ne 0$.

Пример нарушения. При $\phi(t)=t$ уравнение принимает вид $w=xw$, откуда $w\equiv 0$, и формула вырождается.

Идея. Через формальный вычет: $[x^n]F(w)$ — это вычет дифференциальной формы $F(w)\,dw$ относительно подходящей замены переменной.

---

### E8. Экспоненциальная формула для помеченных структур

Утверждение. Пусть $\mathcal{F}$ — класс помеченных объектов, представимых как неупорядоченное разбиение носителя $[n]$ на блоки с независимой $\mathcal{C}$-структурой на каждом блоке, и $c_0=0$. Тогда

$$\hat F(x)=\exp\bigl(\hat C(x)\bigr).$$

Если фиксировано число компонент $k$:

$$[x^n/n!]\,\frac{\hat C(x)^k}{k!}=\#\{\text{объекты } \mathcal F_n \text{ ровно с } k \text{ компонентами}\}.$$

Условия. Структуры помеченные, компоненты неупорядочены, базовое множество разбивается на блоки, и на каждом блоке $\mathcal C$-структура задаётся независимо. Требование $c_0=0$ нужно, чтобы пустой блок не создавал бесконечного семейства тривиальных компонент.

Пример. Числа Белла $B_n$ — число разбиений $[n]$ на непустые блоки. ЭПФ непустого блока — $\hat C(x)=e^x-1$, откуда $\hat B(x)=\exp(e^x-1)$.

Идея. Произведение ЭПФ — биномиальная свёртка, то есть разбиение носителя на упорядоченную пару частей. Деление на $k!$ симметризует по компонентам.

---

### E9. Символический метод (Flajolet, Sedgewick)

Утверждение. Соответствие комбинаторных конструкций и операций над производящими функциями:

| Конструкция | ОПФ (непомеч.) | ЭПФ (помеч.) |
|---|---|---|
| Последовательность $\mathrm{SEQ}(\mathcal A)$ | $\dfrac{1}{1-A(x)}$ | $\dfrac{1}{1-\hat A(x)}$ |
| Мультимножество $\mathrm{MSET}(\mathcal A)$ | $\exp\sum_{k\ge 1}\dfrac{A(x^k)}{k}$ | $\exp(\hat A(x))$ |
| Множество $\mathrm{PSET}(\mathcal A)$ | $\exp\sum_{k\ge 1}\dfrac{(-1)^{k-1}A(x^k)}{k}$ | $\exp(\hat A(x))$ |
| Цикл $\mathrm{CYC}(\mathcal A)$ | $\sum_{k\ge 1}\dfrac{\varphi(k)}{k}\log\dfrac{1}{1-A(x^k)}$ | $\log\dfrac{1}{1-\hat A(x)}$ |

Условия. $A(0)=0$ — иначе появляются «пустые» атомы и операции теряют комбинаторный смысл.

Идея. Каждая конструкция формализует комбинаторную операцию над классом. Произведение, экспонента и логарифм соответствуют рекуррентным разложениям объектов.

### E10. Дифференциальное уравнение на производящую функцию

Утверждение. Если последовательность $(a_n)$ удовлетворяет рекурренте $\sum_i p_i(n)\,a_{n+i}=0$ с полиномиальными коэффициентами, то её ОПФ удовлетворяет линейному дифференциальному уравнению с полиномиальными коэффициентами. Такие ряды называются $D$-конечными (англ. D-finite).

Условия. Полиномиальные коэффициенты $p_i\in\mathbb{C}[n]$. Свёрточные рекурренты $a_{n+1}=\sum_k a_k\,b_{n-k+c}$ также приводят к ОДУ на ОПФ.

Идея. Соответствия $\sum n a_n x^{n-1}=A'(x)$, $\sum a_{n+1}x^n=(A(x)-a_0)/x$ переводят рекурренту в алгебраическое или дифференциальное соотношение.

### E11. Эйлерово произведение для разбиений

Утверждение. ОПФ числа разбиений $p(n)$:

$$P(q)=\sum_{n\ge 0}p(n)\,q^n=\prod_{k\ge 1}\frac{1}{1-q^k}.$$

С ограничениями:

- части различны: $\prod_{k\ge 1}(1+q^k)$;
- части нечётны: $\prod_{k\ge 0}\dfrac{1}{1-q^{2k+1}}$;
- части не больше $m$: $\prod_{k=1}^{m}\dfrac{1}{1-q^k}$.

Тождество Эйлера: $\prod_{k\ge 1}(1+q^k)=\prod_{k\ge 1}\dfrac{1}{1-q^{2k-1}}$. Эквивалент: число разбиений с различными частями равно числу разбиений с нечётными частями.

Условия. Формальные ряды над $\mathbb{Z}[[q]]$. Бесконечное произведение корректно: на каждый коэффициент при $q^n$ влияет лишь конечное число множителей.

Идея. Множитель $\dfrac{1}{1-q^k}=1+q^k+q^{2k}+\dots$ кодирует выбор кратности части $k$. Эйлерово тождество — телескопирование $1+q^k=\dfrac{1-q^{2k}}{1-q^k}$.

---

### E12. Теорема Эйлера о пятиугольных числах

Утверждение.

$$\prod_{k\ge 1}(1-q^k)=\sum_{m\in\mathbb{Z}}(-1)^m q^{m(3m-1)/2}=1+\sum_{m\ge 1}(-1)^m\bigl(q^{m(3m-1)/2}+q^{m(3m+1)/2}\bigr).$$

Следствие — рекуррента для $p(n)$:

$$p(n)=\sum_{m\ge 1}(-1)^{m-1}\Bigl(p\bigl(n-\tfrac{m(3m-1)}{2}\bigr)+p\bigl(n-\tfrac{m(3m+1)}{2}\bigr)\Bigr).$$

Условия. Формальное тождество в $\mathbb{Z}[[q]]$.

Идея. Биекция Франклина (англ. Franklin involution) на разбиениях с различными частями. Неподвижные точки соответствуют пятиугольным числам.

### E13. Лемма о циклах (Dvoretzky, Motzkin)

Утверждение. В круговой последовательности из $a$ символов $+1$ и $b$ символов $-1$ с $a-b=k>0$ ровно $k$ циклических сдвигов делают все частичные суммы строго положительными.

Условия. $a>b$.

Пример нарушения. При $a=b$ ни один сдвиг не делает все частичные суммы строго положительными — конечная сумма равна нулю.

Идея. В точке глобального минимума частичной суммы единственный сдвиг делает все префиксы положительными. Обобщение на $k>0$ — индукция.

Следствие. Число путей Дика длины $2n$ равно $C_n=\dfrac{1}{n+1}\binom{2n}{n}$ — следует мгновенно из применения леммы к последовательности из $n+1$ единиц и $n$ минусов.

### E14. Числа Стирлинга и Белла через ЭПФ

Утверждение.

- Числа Стирлинга второго рода $S(n,k)$ (число разбиений $[n]$ на $k$ непустых блоков):
$$\sum_{n\ge k}S(n,k)\,\frac{x^n}{n!}=\frac{(e^x-1)^k}{k!}.$$
- Числа Стирлинга первого рода (число перестановок $[n]$ с $k$ циклами, без знака):
$$\sum_{n\ge k}\genfrac[]{0pt}{}{n}{k}\frac{x^n}{n!}=\frac{1}{k!}\Bigl(\log\frac{1}{1-x}\Bigr)^k.$$
- Числа Белла $B_n$: $\hat B(x)=\exp(e^x-1)$. Формула Добиньского (англ. Dobiński):
$$B_n=\frac{1}{e}\sum_{k\ge 0}\frac{k^n}{k!}.$$

Условия. Стандартные тождества над помеченными структурами.

Идея. $(e^x-1)^k/k!$ — ЭПФ упорядоченного разбиения на $k$ непустых блоков, делённая на $k!$ для снятия порядка. Числа Стирлинга первого рода получаются из ЭПФ цикла $\log\tfrac{1}{1-x}$ через символический $\mathrm{MSET}$.

### E15. Геометрическое распределение и гармонические тождества

Утверждение. Пусть $X_i$ — независимые геометрические случайные величины, $P(X=k)=p(1-p)^{k-1}$ для $k\ge 1$. PGF одной величины:

$$G(z)=\frac{pz}{1-(1-p)z}.$$

PGF суммы $n$ независимых — степень $G(z)^n$. Гармоническое тождество для выборки без повторений из бесконечной совокупности:

$$\mathbb{E}[Y_n]=\sum_{i\ge 1}\frac{q^i}{1-q^i},\qquad q=1-p,$$

получается из $\mathbb{E}[Y_n]=\sum_{k\ge 0}P(Y_n\ge k)$ и манипуляций с PGF.

Условия. $0<p\le 1$.

Идея. Сумма $\mathbb{E}[X]=\sum_{k\ge 0}P(X>k)$ для неотрицательных целочисленных величин даёт ramp summation, после чего ОПФ хвостовых вероятностей берётся явно.

## Записи — Редкие

### E16. Тождество Лагранжа на симметрических дробях

Утверждение. Для попарно различных ненулевых $x_1,\dots,x_n$:

$$\sum_{i=1}^{n}\frac{1}{x_i}\prod_{j\ne i}\Bigl(1+\frac{1}{x_j-x_i}\Bigr)+1=\prod_{i=1}^{n}\Bigl(1+\frac{1}{x_i}\Bigr).$$

Условия. $x_i$ различны и не равны нулю.

Идея. Обе части — рациональные функции с теми же полюсами и значениями в бесконечности. Разность равна константе, проверяется в одной точке.

---

### E17. Лемма Линдстрёма–Гесселя–Винно (англ. Lindström–Gessel–Viennot lemma, LGV)

Утверждение. Пусть $G$ — ациклический ориентированный граф, $A_1,\dots,A_n$ и $B_1,\dots,B_n$ — два набора вершин, $w$ — мультипликативный вес. Тогда

$$\det\bigl[e(A_i,B_j)\bigr]=\sum_{(P_1,\dots,P_n)\text{ непересек.}}\operatorname{sgn}(\sigma_P)\,\prod w(P_i),$$

где $e(A_i,B_j)=\sum_{P:A_i\to B_j}w(P)$, а $\sigma_P$ — индуцированная перестановка.

В частности, если для конфигурации $A_i,B_j$ единственная непересекающаяся система путей идёт по тождественной перестановке, то определитель равен числу таких систем.

Условия. Ациклический граф. Пересекающиеся пары путей должны иметь корректную операцию обмена начальных хвостов.

Идея. Инволюция, меняющая знак: на первой точке пересечения двух путей переставляем хвосты.

## Self-test

1. Найти число последовательностей длины $n$ из $\{0,1,\dots,9\}$, сумма которых делится на $5$.

2. Пусть $a_n$ — число способов выложить полосу $1\times n$ плитками $1\times 1$ трёх цветов и плитками $1\times 2$ одного цвета. Найти $a_n$ в замкнутой форме.

3. Доказать тождество $\displaystyle\sum_{k=0}^{n}\binom{2k}{k}\binom{2n-2k}{n-k}=4^n$.

4. Пусть $n$ кубиков подбрасывают независимо. Найти вероятность того, что сумма выпавших очков делится на $5$. Показать, что предел при $n\to\infty$ равен $1/5$, и оценить скорость сходимости.

5. Найти ЭПФ числа способов разбить $[n]$ на непустые упорядоченные блоки и в каждом блоке отметить одну точку.

6. Доказать, что $T_n=\dfrac{n^{n-1}}{n!}$ — коэффициент при $x^n$ в ряде $T(x)$, удовлетворяющем уравнению $T=x\,e^T$ с $T(0)=0$.

7. Вычислить $\displaystyle S_n=\sum_{k\ge 0}\binom{n+k}{2k}\,2^{n-k}$.

8. Используя $\prod\dfrac{1}{1-q^k}$ и тождество Эйлера, доказать равенство числа разбиений $n$ на нечётные части и числа разбиений $n$ на различные части.

---

## Решения

1. ОПФ одной позиции: $g(x)=1+x+\dots+x^9=\dfrac{1-x^{10}}{1-x}$. Полная ОПФ — $g(x)^n$. По E3 ответ равен $\tfrac{1}{5}\sum_{j=0}^{4}g(\omega^j)^n$, где $\omega=e^{2\pi i/5}$. Заметим, что $g(\omega^j)=\dfrac{1-\omega^{10j}}{1-\omega^j}=0$ при $j=1,2,3,4$ — так как $\omega^{10j}=1$, $\omega^j\ne 1$. При $j=0$ имеем $g(1)=10$. Итого ответ $\dfrac{10^n}{5}=2\cdot 10^{n-1}$.

2. Рекуррента $a_n=3a_{n-1}+a_{n-2}$, начальные условия $a_0=1$, $a_1=3$. По E4 ОПФ $A(x)=\dfrac{1}{1-3x-x^2}$. Корни характеристического $t^2-3t-1=0$ равны $\alpha,\beta=\dfrac{3\pm\sqrt{13}}{2}$. Имеем $\alpha+\beta=3$, $\alpha\beta=-1$. Тогда $a_n=\dfrac{\alpha^{n+1}-\beta^{n+1}}{\sqrt{13}}$.

3. По E1: $\sum_n\binom{2n}{n}x^n=\dfrac{1}{\sqrt{1-4x}}$. Свёртка двух копий равна $\dfrac{1}{1-4x}$. Коэффициент при $x^n$ равен $4^n$.

4. PGF одного кубика $g(x)=\dfrac{x+x^2+\dots+x^6}{6}=\dfrac{x(1-x^6)}{6(1-x)}$. По E3 вероятность равна $\tfrac{1}{5}\sum_{j=0}^{4}g(\omega^j)^n$. При $j=0$ имеем $g(1)=1$. При $j\ne 0$: $\omega^{5j}=1$, значит $\omega^{6j}=\omega^j$, и $g(\omega^j)=\dfrac{\omega^j(1-\omega^j)}{6(1-\omega^j)}=\dfrac{\omega^j}{6}$. Получаем
$$P=\frac{1}{5}+\frac{1}{5\cdot 6^n}\sum_{j=1}^{4}\omega^{jn}=\frac{1}{5}+\frac{c_n}{5\cdot 6^n},$$
где $c_n=-1$ при $5\nmid n$ и $c_n=4$ при $5\mid n$. Скорость сходимости — экспоненциальная, $O(6^{-n})$.

5. Один блок с отмеченной точкой — это пара (непустое множество, отмеченный элемент в нём). ЭПФ такого блока: $\hat C(x)=\sum_{n\ge 1}n\cdot\dfrac{x^n}{n!}=x\,e^x$. По E9 (упорядоченная последовательность блоков): $\hat F(x)=\dfrac{1}{1-x\,e^x}$.

6. По E7 с $\phi(t)=e^t$:
$$[x^n]T(x)=\frac{1}{n}\,[t^{n-1}]e^{nt}=\frac{1}{n}\cdot\frac{n^{n-1}}{(n-1)!}=\frac{n^{n-1}}{n!}.$$

7. По методу змеиного масла (E5): $S(x)=\sum_n S_n x^n=\sum_k\bigl(\sum_n\binom{n+k}{2k}(2x)^n\bigr)\cdot 2^{-k}$. Полагая $m=n+k$, внутренняя сумма равна
$$\sum_{m\ge 2k}\binom{m}{2k}(2x)^{m-k}=(2x)^{-k}\cdot\frac{(2x)^{2k}}{(1-2x)^{2k+1}}=\frac{(2x)^k}{(1-2x)^{2k+1}}.$$
Подставляя обратно,
$$S(x)=\frac{1}{1-2x}\sum_k\frac{x^k}{(1-2x)^{2k}}=\frac{1}{1-2x}\cdot\frac{1}{1-x/(1-2x)^2}=\frac{1-2x}{(1-x)(1-4x)}.$$
Разложение на простые дроби: $\dfrac{1/3}{1-x}+\dfrac{2/3}{1-4x}$, откуда $S_n=\dfrac{1+2\cdot 4^n}{3}$.

8. По E11 ОПФ числа разбиений с различными частями равна $\prod_{k\ge 1}(1+q^k)$. Поскольку $1+q^k=\dfrac{1-q^{2k}}{1-q^k}$, произведение телескопирует:
$$\prod_{k\ge 1}\frac{1-q^{2k}}{1-q^k}=\prod_{k\ge 1}\frac{1}{1-q^{2k-1}}.$$
Правая часть — ОПФ числа разбиений в нечётные части. Равенство производящих функций означает равенство коэффициентов при всех $q^n$.
