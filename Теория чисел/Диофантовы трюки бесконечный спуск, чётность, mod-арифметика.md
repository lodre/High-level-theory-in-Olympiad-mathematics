# Диофантовы трюки: бесконечный спуск, чётность, mod-арифметика

## Scope

Базовый арсенал теории чисел на олимпиадном уровне. Сюда входят способы доказывать отсутствие решений у диофантовых уравнений: препятствия по модулю, чётность, анализ $p$-адической оценки. Также методы строить или ловить бесконечную серию решений: спуск, прыжок Виета, структура Пелля. И техники подъёма решений со слоя на слой: Гензель и цепочка по китайской теореме.

## Определения

Пусть $p$ — простое, $n \in \mathbb{Z} \setminus \{0\}$. $p$-адическая оценка $v_p(n)$ — наибольшее $k$, при котором $p^k \mid n$. По соглашению $v_p(0) = +\infty$.

$\mathbb{Z}/n$ — кольцо вычетов по модулю $n$. $(\mathbb{Z}/n)^*$ — группа обратимых, имеет $\varphi(n)$ элементов.

Для $\gcd(a, n) = 1$ порядок $\mathrm{ord}_n(a)$ — наименьшее $d \ge 1$ с $a^d \equiv 1 \pmod n$. Этот порядок делит $\varphi(n)$.

Примитивная пифагорова тройка — $(x, y, z)$ с $x^2 + y^2 = z^2$ и $\gcd(x, y, z) = 1$. Каждая такая тройка с чётным $y$ параметризуется $x = m^2 - n^2$, $y = 2mn$, $z = m^2 + n^2$, где $m > n > 0$, $\gcd(m, n) = 1$ и числа $m, n$ разной чётности.

Совершенное число — $n$ с $\sigma(n) = 2n$, где $\sigma$ — сумма всех делителей. Число делителей обозначается $\tau(n)$.

## Записи — Основные

### E1. Бесконечный спуск (infinite descent)

Формулировка. Пусть утверждение $P(x_1, \dots, x_k)$ относится к наборам положительных целых $x_i$. Если из любого решения $\mathbf{x}$ строится решение $\mathbf{x}'$ со строго меньшей мерой $\mu(\mathbf{x}') < \mu(\mathbf{x})$, то решений нет.

Условия применимости. Мера $\mu$ принимает значения в $\mathbb{Z}_{\ge 0}$ и не может убывать бесконечно. Импликация «решение даёт меньшее решение» должна работать на всех решениях, а не только в типичном случае.

Пример, где спуск ломается. Уравнение $x^2 - 2 y^2 = 1$ имеет решения $(3, 2), (17, 12), \dots$. Если попытаться выписать спуск, то на каждом шаге получается бо́льшая, а не меньшая пара — направление спуска неверное. Здесь работает Пелль (запись E14), а не спуск.

Идея доказательства. Берём решение с минимальной $\mu$. Выводим меньшее. Противоречие с минимальностью.

Используется в. См. раздел «Использования».

### E2. Препятствие по модулю $n$

Формулировка. Если уравнение $F(x_1, \dots, x_k) = 0$ переводится в $\mathbb{Z}/n$ в уравнение $\bar F = 0$ без решений, то и в $\mathbb{Z}$ решений нет.

Условия применимости. Метод даёт только препятствие. Если уравнение разрешимо по всем модулям, выводить отсюда решения в $\mathbb{Z}$ нельзя.

Пример, где метод не работает. Уравнение Сельмера $3 x^3 + 4 y^3 + 5 z^3 = 0$ имеет нетривиальные решения по любому модулю $n$ и в $\mathbb{R}$, но в $\mathbb{Z}$ только $x = y = z = 0$. Это нарушение локально-глобального принципа.

Идея доказательства. Отображение $\mathbb{Z} \to \mathbb{Z}/n$ — гомоморфизм колец. Целое решение даёт решение по модулю $n$.

Используется в. См. раздел «Использования».

### E3. Квадраты по малым модулям

Формулировка. Полезные таблицы для $x \in \mathbb{Z}$:
$$x^2 \bmod 3 \in \{0, 1\}, \quad x^2 \bmod 4 \in \{0, 1\}, \quad x^2 \bmod 8 \in \{0, 1, 4\}, \quad x^2 \bmod 9 \in \{0, 1, 4, 7\}.$$
Прямые следствия: $a^2 + b^2 \not\equiv 3 \pmod 4$, а сумма трёх квадратов не может быть $\equiv 7 \pmod 8$. Для кубов аналогично: $x^3 \bmod 9 \in \{0, 1, -1\}$.

Условия применимости. Таблицы относятся к квадратам именно целых чисел. Для рациональных $x$ ничего этого без очистки знаменателя сказать нельзя.

Пример, где забывание условия ломает аргумент. Если про $x \in \mathbb{Q}$ написать «$x^2 \bmod 4 \in \{0, 1\}$», получится бессмыслица: $x = 1/2$ даёт $x^2 = 1/4$, и $\bmod 4$ не определено.

Идея доказательства. Перебор $x = 0, 1, \dots, n-1$.

Используется в. См. раздел «Использования».

### E4. $p$-адический баланс

Формулировка. Если $A = B$ в $\mathbb{Z}$ и обе части ненулевые, то $v_p(A) = v_p(B)$ для каждого простого $p$. Нарушение баланса — препятствие. Часто работает вместе с формулой Лежандра $v_p(n!) = \sum_{i \ge 1} \lfloor n / p^i \rfloor$.

Условия применимости. Обе части должны быть целыми и ненулевыми. Для рационального равенства $a/b = c/d$ работает $v_p(a) - v_p(b) = v_p(c) - v_p(d)$.

Пример, где условие важно. Из «$2 \cdot 6 = 3 \cdot 4$» при $p = 2$ имеем $v_2(\text{LHS}) = v_2(2) + v_2(6) = 2$, и $v_2(\text{RHS}) = v_2(3) + v_2(4) = 2$ — баланс. Но если бы кто-то писал $v_p$ от выражения, которое равно нулю в одной интерпретации и не равно в другой, баланс теряет смысл; типичный сбой — забыть проверить, что обе части ненулевые.

Идея доказательства. $v_p$ — гомоморфизм мультипликативной группы $\mathbb{Z} \setminus \{0\}$ в $\mathbb{Z}_{\ge 0}$.

Используется в. См. раздел «Использования».

### E5. Лемма Гензеля (Hensel)

Формулировка. Пусть $f \in \mathbb{Z}[x]$ и $p$ простое. Если $f(a) \equiv 0 \pmod p$ и $f'(a) \not\equiv 0 \pmod p$, то для каждого $k \ge 1$ существует единственный $a_k \in \mathbb{Z}/p^k$ с $a_k \equiv a \pmod p$ и $f(a_k) \equiv 0 \pmod{p^k}$.

Условия применимости. Условие $f'(a) \not\equiv 0 \pmod p$ (простой корень) обязательно. При его нарушении подъём может не существовать или быть неединственным.

Пример, где условие ломает результат. Возьмём $f(x) = x^2$, $p = 2$, $a = 0$. Тогда $f'(0) = 0 \equiv 0 \pmod 2$. По модулю $4$ корни $f$ — это $\{0, 2\}$, их два, единственности нет.

Идея доказательства. Шаг Ньютона: $a_{k+1} = a_k - f(a_k) / f'(a) \pmod{p^{k+1}}$. Знаменатель обратим по модулю $p$, значит и по модулю любой степени $p$.

Используется в. См. раздел «Использования».

### E6. Китайская теорема об остатках (КТО)

Формулировка. Пусть $n_1, \dots, n_k$ попарно взаимно просты. Для любых $a_1, \dots, a_k$ существует $x \in \mathbb{Z}$ с $x \equiv a_i \pmod{n_i}$ для всех $i$, и такой $x$ единствен по модулю $\prod n_i$.

Условия применимости. Попарная взаимная простота обязательна. Без неё нужна согласованность остатков на общих делителях.

Пример, где условие критично. Система $x \equiv 1 \pmod 2$, $x \equiv 0 \pmod 4$ не имеет решений, потому что $\gcd(2, 4) = 2$, а остатки $1$ и $0$ не согласованы по этому общему делителю.

Идея доказательства. Кольцевой изоморфизм $\mathbb{Z} / (n_1 \cdots n_k) \cong \prod_i \mathbb{Z} / n_i$.

Олимпиадная форма. Жадное построение последовательности $b_1, b_2, \dots$, на каждом шаге выбирающее новый член из остаточного класса, ещё не запрещённого предыдущими условиями. Пока запрещённых классов меньше, чем доступных, выбор существует.

Используется в. См. раздел «Использования».

### E7. Прыжок Виета (Vieta jumping)

Формулировка. Пусть уравнение симметрично по $a, b$ и при фиксированном $b$ записывается как $a^2 + \alpha(b) a + \beta(b) = 0$. Если $(a, b)$ — целое решение, то по теореме Виета второй корень $a' = -\alpha(b) - a$ — тоже целое решение. Обычно $|a'| < |a|$, что даёт спуск.

Условия применимости. Квадратичность по одной переменной критична. Положительность и целостность $a'$ требуют отдельной проверки.

Пример, где условие ломает метод. Уравнение $a^3 + b^3 = a b + 1$ кубическое по $a$, и второй корень не выражается линейной комбинацией. Прыжок не определён.

Идея доказательства. Спуск к минимальному решению по мере $|a| + |b|$. На минимуме разбирается отдельный случай.

Используется в. См. раздел «Использования».

## Записи — Стандартные

### E8. Лемма Туэ (Thue)

Формулировка. Пусть $m \ge 2$ и $\gcd(a, m) = 1$. Существуют целые $x, y$, не оба нулевые, с $|x|, |y| \le \sqrt m$ и $a y \equiv x \pmod m$.

Условия применимости. $\gcd(a, m) = 1$ нужно для нетривиальности утверждения о паре $(x, y)$.

Пример приложения. Если $m = p$ — нечётное простое и $a^2 \equiv -1 \pmod p$, то $x^2 + y^2 \equiv (a y)^2 + y^2 \equiv (a^2 + 1) y^2 \equiv 0 \pmod p$. Из $0 < x^2 + y^2 < 2 p$ выходит $x^2 + y^2 = p$. Это классическое доказательство теоремы Ферма о сумме двух квадратов.

Идея доказательства. Принцип Дирихле для $(\lfloor \sqrt m \rfloor + 1)^2 > m$ пар $(i, j) \in \{0, \dots, \lfloor \sqrt m \rfloor\}^2$ — две пары дают одинаковое $i - a j$ по модулю $m$.

Используется в. См. раздел «Использования».

### E9. Тождество Софи Жермен

Формулировка.
$$a^4 + 4 b^4 = (a^2 + 2 a b + 2 b^2)(a^2 - 2 a b + 2 b^2).$$

Условия применимости. Оба множителя нетривиальны при $a, b \ge 1$, кроме случая $a = b = 1$, где $a^4 + 4 b^4 = 5$ — простое.

Пример, где аккуратность важна. Чтобы применить к $n^4 + 4^n$ для нечётного $n = 2 m + 1$, представляем $4^n = 4 \cdot (2^m)^4$ и берём $b = 2^m$. Если $n$ чётное, такого представления нет, и тождество не помогает напрямую.

Идея доказательства. Прямое раскрытие, либо $(a^2 + 2 b^2)^2 - (2 a b)^2$.

Используется в. См. раздел «Использования».

---

### E10. Препятствие по порядку элемента

Формулировка. Пусть $\gcd(a, p) = 1$ и $d = \mathrm{ord}_p(a)$. Тогда $a^n \equiv 1 \pmod p$ влечёт $d \mid n$. По малой теореме Ферма $d \mid p - 1$.

Условия применимости. $\gcd(a, p) = 1$ обязательно — иначе порядок не определён.

Пример, где условие важно. Если $p \mid a$, то $a^n \equiv 0 \pmod p$ для любого $n \ge 1$, и говорить о порядке бессмысленно.

Идея доказательства. $a$ принадлежит циклической группе $(\mathbb{Z}/p)^*$ порядка $p - 1$. Порядок элемента делит порядок группы.

Используется в. См. раздел «Использования».

### E11. Мультипликативная чётность

Формулировка. Пусть $n \ge 2$. Запишем $k = n^{e(k)} m$ с $n \nmid m$. Положим $A = \{k \ge 1 : e(k) \text{ нечётно}\}$, $B = \{k \ge 1 : e(k) \text{ чётно}\}$. Тогда $A \sqcup B = \mathbb{Z}_{> 0}$ и $A = n \cdot B$, то есть умножение на $n$ — биекция $B \to A$.

Условия применимости. $n \ge 2$. При $n = 1$ функция $e$ не определена.

Пример. Для $n = 2$: $B = \{1, 3, 4, 5, 7, 9, 11, 12, 13, \dots\}$, $A = \{2, 6, 8, 10, 14, 18, \dots\}$. Каждое $a \in A$ — это удвоенное число из $B$.

Идея доказательства. Тождество $e(n k) = e(k) + 1$ меняет чётность.

Используется в. См. раздел «Использования».

---

### E12. Образ $p$-й степени по модулю $p^2$

Формулировка. Пусть $p$ — нечётное простое. Образ отображения $x \mapsto x^p$ на $(\mathbb{Z}/p^2)^*$ содержит ровно $p - 1$ элемент. Если включить ноль, в $\mathbb{Z}/p^2$ есть ровно $p$ классов вида $x^p$. Иными словами, $p$ классов по модулю $p$ склеиваются в один класс по модулю $p^2$ после возведения в $p$-ю степень.

Условия применимости. $p$ нечётное.

Пример, где условие важно. Для $p = 2$ структура $(\mathbb{Z}/4)^* = \{1, 3\}$ и $x^2 \bmod 4 \in \{0, 1\}$ — образ имеет всего $2$ класса в $\mathbb{Z}/4$, что не совпадает с формулой «$p = 2$ классов в $\mathbb{Z}/p^2 = \mathbb{Z}/4$». Поэтому в задачах с $p = 2$ нужен отдельный счёт по модулю $4$ или $8$.

Идея доказательства. Разложение $(x + k p)^p = x^p + p \cdot (k p) x^{p-1} + \binom{p}{2} (k p)^2 x^{p-2} + \dots$. Все слагаемые после первого делятся на $p^2$.

Используется в. См. раздел «Использования».

## Записи — Редкие

### E13. Теорема Лежандра о трёх квадратах

Формулировка. Натуральное $n$ представимо как $a^2 + b^2 + c^2$ тогда и только тогда, когда $n \ne 4^s (8 t + 7)$ для целых $s, t \ge 0$.

Условия применимости. На олимпиадах обычно используется лёгкая часть «только тогда», вытекающая из $x^2 \bmod 8 \in \{0, 1, 4\}$ и спуска по чётности. Полная теорема (включая обратное направление) требует теории тернарных квадратичных форм; в олимпиадных решениях ссылка на неё допустима, но риск, что её сочтут «слишком сильным фактом», всегда есть.

Используется в. См. раздел «Использования».

---

### E14. Серии Пелля

Формулировка. Пусть $D > 0$ — не полный квадрат и $(x_1, y_1)$ — наименьшее положительное решение уравнения $x^2 - D y^2 = 1$. Тогда все положительные решения задаются формулой $x_n + y_n \sqrt D = (x_1 + y_1 \sqrt D)^n$. Это даёт линейную рекуррентность с постоянными коэффициентами.

Условия применимости. $D$ — положительное не-квадратное целое. Для $x^2 - D y^2 = N$ при $|N| > 1$ серий может быть несколько, нужен полный список их представителей.

Используется в. См. раздел «Использования».

## Self-test

1. Решить в целых: $a^2 + b^2 = 4 c + 3$.
2. Доказать, что $x^4 + y^4 = z^2$ не имеет решений в положительных целых.
3. Доказать, что $n^4 + 4$ составное для всех целых $n > 1$.
4. Найти все простые $p > 3$, при которых $2 p^2 + 1$ — тоже простое.
5. Пусть $a, b \in \mathbb{Z}_{> 0}$ и $a b + 1 \mid a^2 + b^2$. Доказать, что $(a^2 + b^2) / (a b + 1)$ — точный квадрат.
6. Пусть $p$ — нечётное простое и $p \mid x^2 + 1$ для некоторого $x \in \mathbb{Z}$. Доказать, что $p \equiv 1 \pmod 4$.
7. Сколько решений у $x^2 \equiv x \pmod{10^n}$ в множестве $\{0, 1, \dots, 10^n - 1\}$?

## Решения

1. По модулю $4$ имеем $a^2, b^2 \in \{0, 1\}$, поэтому $a^2 + b^2 \in \{0, 1, 2\}$. Правая часть же $\equiv 3 \pmod 4$. Несовместно. Решений нет. (E3.)

2. Пусть $(x, y, z)$ — решение с минимальным $z > 0$. Можно считать $\gcd(x, y) = 1$, иначе делим уравнение и уменьшаем $z$. Тогда $(x^2, y^2, z)$ — примитивная пифагорова тройка. Пусть $y$ чётное. По параметризации $x^2 = m^2 - n^2$, $y^2 = 2 m n$, $z = m^2 + n^2$ с $\gcd(m, n) = 1$ и разной чётностью $m, n$. Из $x^2 + n^2 = m^2$ — снова примитивная тройка: $x = u^2 - v^2$, $n = 2 u v$, $m = u^2 + v^2$. Подставляем: $y^2 = 4 u v (u^2 + v^2)$, и $u, v, u^2 + v^2$ попарно взаимно просты. Значит каждый из них — точный квадрат: $u = a^2$, $v = b^2$, $u^2 + v^2 = c^2$. Получаем $a^4 + b^4 = c^2$ с $c \le m < z$ — противоречие минимальности. (E1.)

3. По тождеству Софи Жермен $n^4 + 4 = (n^2 - 2 n + 2)(n^2 + 2 n + 2)$. При $n \ge 2$: первый множитель $(n - 1)^2 + 1 \ge 2$, второй $\ge 10$. Оба больше единицы, число составное. (E9.)

4. При $p > 3$ имеем $p \not\equiv 0 \pmod 3$, значит $p^2 \equiv 1 \pmod 3$ и $2 p^2 + 1 \equiv 0 \pmod 3$. При этом $2 p^2 + 1 > 3$, так что $3$ — собственный делитель, а число составное. Таких $p$ нет. (E2.)

5. Пусть $k = (a^2 + b^2) / (a b + 1)$. Допустим, $k$ — не квадрат. Среди пар $(a, b)$ с этим $k$ возьмём пару $(A, B)$ с минимальной суммой $A + B$, считая $A \ge B \ge 1$. Уравнение $x^2 - k B x + (B^2 - k) = 0$ имеет корнями $A$ и $A' = k B - A$. По теореме Виета $A'$ — целое, и $A \cdot A' = B^2 - k$.

   Случай $A' < 0$. Подставим $A'$ в многочлен: $A'^2 - k B A' + B^2 - k = 0$. Все три слагаемых $A'^2, -k B A', B^2$ при $A' \le -1$ положительны, и в сумме дают что-то $\ge 1 + k + B^2 > k$. Это противоречит равенству нулю.

   Случай $A' = 0$. Тогда $k = B^2$ — квадрат, противоречит предположению.

   Случай $A' \ge 1$. Тогда $A' = (B^2 - k) / A < B^2 / A \le B$, и $(B, A')$ — пара с тем же $k$ и меньшей суммой. Противоречие минимальности.

   Остаётся $A = B$. Тогда $2 A^2 = k (A^2 + 1)$, откуда $k < 2$ и $k = 1 = 1^2$. (E7.)

6. Из $x^2 \equiv -1 \pmod p$ имеем $x^4 \equiv 1 \pmod p$. Пусть $d = \mathrm{ord}_p(x)$. Тогда $d \mid 4$. Случаи $d = 1, 2$ дали бы $x^2 \equiv 1 \pmod p$, а не $-1$ (здесь $p > 2$). Значит $d = 4$. По малой теореме Ферма $4 = d \mid p - 1$. (E10.)

7. Уравнение равносильно $x (x - 1) \equiv 0 \pmod{10^n}$. По КТО $\mathbb{Z} / 10^n \cong \mathbb{Z} / 2^n \times \mathbb{Z} / 5^n$. В каждом сомножителе $\gcd(x, x - 1) = 1$, поэтому $p^n \mid x (x - 1)$ влечёт $p^n \mid x$ или $p^n \mid x - 1$ — ровно два решения. Итого $2 \cdot 2 = 4$ решения. Проверка для $n = 1$: $\{0, 1, 5, 6\}$. (E5, E6.)

## Использования

E1. Бесконечный спуск: $x^4 + y^4 = z^2$ и $x^4 - y^4 = z^2$ (Ферма), $x^2 + y^2 = 3 z^2$, классическая задача «нет $n > 1$ с $n \mid 2^n - 1$» (спуск на наименьший простой делитель).

E2. Препятствие по модулю: IMC-2003-4, IMC-2012-DAY2-3 (через простой делитель $n! + 1$), IMC-2014-4 (нечётные совершенные числа через $\sigma \bmod 4$).

E3. Квадраты по малым модулям: IMC-2003-4 (как часть аргумента), множество классических «нет суммы двух/трёх квадратов $= n$», задача о делимости $n^2 \mid 2^n + 1$ (в комбинации с LTE).

E4. $p$-адический баланс: IMC-2012-DAY2-3 (оценка $v_p((2012 n)!)$ через формулу Лежандра), IMC-2014-4, IMC-2017-3 (контроль $\tau(a_k)$ через $v_p$), IMC-2025-4.

E5. Лемма Гензеля: IMC-2006-2 ($x^2 - x \equiv 0 \pmod{10^k}$, простой корень по $p = 2$ и $p = 5$).

E6. КТО: IMC-2006-2, IMC-2008-3 (построение $b_k$ с $p(a_i) \mid p(b_k)$), IMC-2013-DAY2-4 (жадный выбор без квадратных делителей).

E7. Прыжок Виета: IMO-1988-6 (легендарная Q6), IMO-2007-5, тройки Маркова, многочисленные USAMO и RMM задачи последних лет.

E8. Лемма Туэ: доказательство Ферма о сумме двух квадратов, IMC-задачи на представление простых формами $x^2 + d y^2$, IMO-2008-3.

E9. Софи Жермен: задача «$n^4 + 4^n$ составное при $n > 1$», региональные олимпиады с числами вида $X^4 + 4 Y^4$.

E10. Порядок элемента: IMO-2003-6, IMC-2024-10 (локальный анализ через подгруппу простого порядка), широкий класс задач на делимость $p \mid f(n)$.

E11. Мультипликативная чётность: IMC-2003-4 (разбиение $A = n \cdot B$ по чётности $v_n(k)$), родственные функциональные уравнения на $\mathbb{Z}_{> 0}$.

E12. Образ $p$-й степени по $p^2$: IMC-2010-4 (счёт чисел, не представимых $a x^n + b y^n$).

E13. Теорема Лежандра: задачи «можно ли представить $n$ суммой трёх квадратов» как готовое препятствие.

E14. Серии Пелля: IMC-2005-DAY2-6 (бесконечная серия $s^2 - 7 \Delta^2 t^2 = 4$ через степени $8 + 3 \sqrt 7$), IMC-2025-10 (асимптотика квадратных пар), классические задачи о квадратно-треугольных числах.
