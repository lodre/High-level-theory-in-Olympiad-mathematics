# Lifting the exponent (LTE)

## Scope
Семейство формул для $p$-adic valuation $v_p(x^n \pm y^n)$, $v_p(n!)$, $v_p\binom{n}{k}$, $v_p(\Phi_n(a))$, $v_p(u_n)$ (Lucas-последовательности), сводящих вычисление высокой степени простого $p$ к данным «на одном уровне» — $v_p(x \pm y)$ плюс $v_p(n)$. Ядро — odd-prime LTE, $p=2$ как отдельный случай, и Legendre/Kummer как комплементарный аппарат для биномиальных. На IMC/Putnam применяется в задачах вида «доказать $p^k \mid \dots$», «найти все $n$ с $n^2 \mid a^n+1$», «оценить рост знаменателей $\sum 1/k!$», а также как несущая лемма Zsigmondy и order-lifting.

## Записи — Core

### E1. LTE для нечётного $p$, разность $x^n - y^n$

Формулировка. Пусть $p$ — нечётное простое, $x,y \in \mathbb{Z}$, $p \mid x-y$, $p \nmid x$, $p \nmid y$, $n \in \mathbb{Z}_{\ge 1}$. Тогда
$$
v_p(x^n - y^n) = v_p(x - y) + v_p(n).
$$

Условия применимости. Все три условия критичны: $p$ нечётно (для $p=2$ см. E3), $p \mid x-y$ (иначе $v_p(x^n-y^n) = v_p(x^d - y^d)$, $d=\operatorname{ord}_p(x/y)$, см. E10), $p \nmid xy$ (иначе обе стороны теряют смысл — вычитаются нули по разным $p$-адическим шкалам).

Идея доказательства. Индукция по $v_p(n)$. База $p \nmid n$: $\frac{x^n - y^n}{x-y} = \sum_{k=0}^{n-1} x^{n-1-k} y^k \equiv n y^{n-1} \pmod{p}$ (т.к. $x \equiv y \pmod p$), и $p \nmid n y^{n-1}$. Шаг $n = pm$: пишем $x^n - y^n = (x^m)^p - (y^m)^p$ и сводим к случаю «показатель $p$»: $a^p - b^p = (a-b)(a^{p-1} + a^{p-2}b + \dots + b^{p-1})$, второй множитель $\equiv p a^{p-1} \pmod{p \cdot \text{lift}}$, что даёт ровно один добавочный $p$.

Варианты и обобщения.
- Можно записать в форме $v_p\!\left(\frac{x^n - y^n}{x - y}\right) = v_p(n)$ — базис цикломатической интерпретации.
- Версия с $x \equiv y \pmod{p^k}$, $k \ge 2$: $v_p(x^n - y^n) = k + v_p(n)$ при $k \ge 1$, $p \nmid xy$, $p$ нечётное. Это та же формула, $v_p(x-y) = k$.

Используется в. Базис всех записей. [IMC-2022-3] (mod $p$ Лукасом для central trinomial), [IMO-1990-3] (классика «найти все $n$: $n \mid 2^n + 1$»), [Putnam-1972-A6].

### E2. LTE для нечётного $p$, сумма $x^n + y^n$ (нечётное $n$)

Формулировка. Пусть $p$ — нечётное простое, $x,y \in \mathbb{Z}$, $p \mid x+y$, $p \nmid x$, $p \nmid y$, $n$ — нечётное натуральное. Тогда
$$
v_p(x^n + y^n) = v_p(x + y) + v_p(n).
$$

Условия применимости. $n$ обязательно нечётно: при чётном $n$ имеем $x^n + y^n$ без обращения в нуль mod $p$ (пример $x=2, y=1, p=3, n=2$: $4+1 = 5$, $v_3=0$, хотя $3 \nmid x+y=3$? нет, $3\mid 3$ — но $5$ не делится на $3$, формула давала бы $v_3(3) + v_3(2) = 1 + 0 = 1 \ne 0$; нечётность принципиальна). Остальные условия — как в E1.

Идея доказательства. Подстановка $y \to -y$ в E1: $v_p(x^n + y^n) = v_p(x^n - (-y)^n)$ при нечётном $n$, и $p \mid x - (-y) = x+y$.

Варианты и обобщения.
- Универсальная форма «E1+E2»: для нечётного $p$, $p \nmid xy$ и $p \mid x \mp y$ (выбираем знак): $v_p(x^n \mp y^n) = v_p(x \mp y) + v_p(n)$, причём в $-$ случае $n$ может быть любым, в $+$ — нечётным.

Используется в. [Putnam-1972-A6: $v_3(2^{3^n} + 1) = n+1$], [IMO-1990-3], стандартный шаблон для $a^n + b^n$ с малым простым делителем.

### E3. LTE для $p = 2$ (три случая)

Формулировка. Пусть $x, y$ — нечётные целые, $n \ge 1$.

Случай (a) — $n$ нечётно:
$$
v_2(x^n - y^n) = v_2(x - y).
$$

Случай (b) — $n$ чётно:
$$
v_2(x^n - y^n) = v_2(x - y) + v_2(x + y) + v_2(n) - 1.
$$

Случай (c) — $4 \mid x - y$, $n$ любое:
$$
v_2(x^n - y^n) = v_2(x - y) + v_2(n).
$$

Для суммы $x^n + y^n$: при $n$ нечётном $v_2(x^n + y^n) = v_2(x+y)$ (через $y \to -y$ в (a)); при $n$ чётном $x,y$ нечётны $\Rightarrow x^n + y^n \equiv 2 \pmod 4$, поэтому $v_2(x^n + y^n) = 1$ (формула «не лифтит»).

Условия применимости. $x,y$ оба нечётные (иначе свести делением общей степени двойки). Случай (b) корректно действует только при чётном $n$; случай (c) — это (b) при $4 \mid x-y$, тогда $x+y \equiv 2 \pmod 4$, $v_2(x+y) = 1$, и формула (b) превращается в $v_2(x-y) + 1 + v_2(n) - 1 = v_2(x-y) + v_2(n)$.

Идея доказательства. (a) тривиально из факторизации. (b): $x^n - y^n = (x^2)^{n/2} - (y^2)^{n/2}$, по индукции и тождеству $(x^2 - y^2) = (x-y)(x+y)$. Альтернативно — индукция по $v_2(n)$ + лемма $v_2((x^2 - y^2)) = v_2(x-y) + v_2(x+y)$.

Варианты и обобщения. Удобная мнемоника: «лишняя $-1$» возникает потому, что для чётного $n$ ровно одно из $x \pm y$ содержит «лишнюю» двойку, не отражённую в $v_2(n)$.

Используется в. Любые задачи с $2^k \| \dots$, особенно вокруг Mersenne / Fermat: $v_2(3^n - 1)$, $v_2(7^n - 5^n)$ и т.д. [IMO-2007-5 в shortlist].

### E4. Legendre's formula для $v_p(n!)$

Формулировка. Для простого $p$ и $n \in \mathbb{Z}_{\ge 0}$:
$$
v_p(n!) = \sum_{k \ge 1} \left\lfloor \frac{n}{p^k} \right\rfloor = \frac{n - s_p(n)}{p - 1},
$$
где $s_p(n)$ — сумма цифр $n$ в записи по основанию $p$.

Условия применимости. Любые $n \ge 0$, любое простое $p$. Сумма конечна, обрывается при $p^k > n$.

Идея доказательства. $\lfloor n/p^k \rfloor$ — количество кратных $p^k$ среди $1,\dots,n$; суммируя по $k$, считаем каждый $m$ ровно $v_p(m)$ раз. Вторая форма — следствие $\lfloor n/p^k \rfloor = (n - s_p(n)\bmod p^k\text{-tail})/(p-1)$, телескопически.

Варианты и обобщения.
- Для multinomial: $v_p\!\left(\frac{n!}{n_1! \cdots n_r!}\right) = \frac{1}{p-1}\!\left(\sum_i s_p(n_i) - s_p(n)\right)$ — базис E5.
- Формула Куммера для $v_p\binom{n}{k}$ — частный случай.

Используется в. [IMC-2019-7: рост $a_n$ — наименьшего $k$ с $n \mid k!$, через $v_p(k!) \ge v_p(n)$], [IMC-2023-10: оценка знаменателя $\sum_{k=0}^n 1/k!$ через $v_p$ всех $k!$, $k \le n$, дающее $g(n)^{1/n} \to e$], стандарт для оценок биномиальных коэффициентов.

### E5. Kummer's theorem для $v_p\binom{m+n}{n}$

Формулировка. Для простого $p$ и $m, n \ge 0$:
$$
v_p\!\binom{m+n}{n} = \#\{\text{переносов при сложении } m \text{ и } n \text{ в системе с основанием } p\}.
$$
Эквивалентно: $v_p\binom{m+n}{n} = \frac{s_p(m) + s_p(n) - s_p(m+n)}{p - 1}$.

Условия применимости. Любые $m,n\ge 0$, любое простое $p$.

Идея доказательства. По E4: $v_p\binom{m+n}{n} = \frac{(m+n-s_p(m+n)) - (m-s_p(m)) - (n-s_p(n))}{p-1}$. Разница сумм цифр и $s_p(m+n)$ ровно равна $(p-1) \cdot \#\text{carries}$ (каждый перенос «крадёт» $p$ из одного разряда и кладёт $1$ в следующий: чистая потеря в сумме цифр = $p-1$).

Варианты и обобщения.
- Lucas's theorem (комплементарно): $\binom{m+n}{n} \pmod p$ через произведение $\prod_i \binom{m_i+n_i}{n_i} \pmod p$ по цифрам.
- Granville-style: оценки $\binom{n}{k} \pmod{p^k}$ через расширенную Wolstenholme.

Используется в. Доказательства, что $\binom{2n}{n}$ имеет много простых делителей; [IMC-2022-3] (central trinomial coefficient через Lucas); классические задачи «найти $\gcd(\binom{n}{1},\dots,\binom{n}{n-1})$» (= $p$, если $n=p^k$, иначе $1$).

## Записи — Standard

### E6. Multiplicative order lifting

Формулировка. Пусть $p$ — нечётное простое, $p \nmid a$, $d = \operatorname{ord}_p(a)$, $c = v_p(a^d - 1) \ge 1$. Тогда для всех $k \ge 1$:
$$
\operatorname{ord}_{p^k}(a) = d \cdot p^{\max(0,\, k - c)}.
$$
В частности, $\operatorname{ord}_{p^k}(a) = d \cdot p^{k-1}$ при «генерическом» случае $c = 1$.

Для $p = 2$, $a$ нечётное, $a \ne \pm 1$: пусть $c_- = v_2(a - 1)$, $c_+ = v_2(a + 1)$ (одно из них $= 1$). Тогда $\operatorname{ord}_{2^k}(a) = 2^{\max(0, k - \max(c_-, c_+))}$ или $2^{\max(0, k - c_-)}$ — зависит от знака; формула удобна в виде «применить E3 к $a^n - 1$».

Условия применимости. $\gcd(a, p) = 1$. Случай $a \equiv 1 \pmod p$: $d = 1$, $c = v_p(a - 1)$, $\operatorname{ord}_{p^k}(a) = p^{\max(0,k-c)}$.

Идея доказательства. Из E1: $v_p(a^{nd} - 1) = v_p(a^d - 1) + v_p(n) = c + v_p(n)$. Минимальное $n$, при котором эта величина $\ge k$, — это $n = p^{\max(0, k-c)}$. Значит наименьший показатель — $d \cdot p^{\max(0, k-c)}$.

Варианты и обобщения. Для primitive root $g$ mod $p$: $g$ остаётся primitive root mod $p^k$ для всех $k$ тогда и только тогда, когда $v_p(g^{p-1} - 1) = 1$ — фейл означает Wieferich-style ситуацию (E11).

Используется в. Любая задача «найти наименьшее $n$ с $p^k \mid a^n - 1$»; конструкция элементов больших порядков; [Putnam-2009-A6 style].

### E7. $v_p(\Phi_n(a))$ через cyclotomic factorization

Формулировка. Пусть $a \in \mathbb{Z}$, $p$ — простое, $p \nmid a$, $d = \operatorname{ord}_p(a)$. Тогда
$$
v_p(\Phi_n(a)) = \begin{cases}
v_p(a^d - 1) - v_p(d), & n = d, \\
v_p(n/d) \cdot [\text{adjustment}], & n = d \cdot p^k, \; k \ge 1, \\
0, & n \ne d \cdot p^k.
\end{cases}
$$
Точнее (для нечётного $p$): если $n = d \cdot p^k$, $k \ge 1$, то $v_p(\Phi_n(a)) = 1$; если $n = d$, то $v_p(\Phi_d(a)) = v_p(a^d - 1)$; иначе $0$. Эквивалентная сводная формула:
$$
v_p(a^n - 1) = \sum_{m \mid n} v_p(\Phi_m(a)) = v_p(a^d - 1) + v_p(n/d) \quad [\text{при } d \mid n,\ p \nmid a].
$$

Условия применимости. $p$ нечётное, $p \nmid a$. Для $p = 2$ — отдельная серия, использующая E3.

Идея доказательства. $a^n - 1 = \prod_{m \mid n} \Phi_m(a)$. Применяем E1 к левой части и сравниваем с разложением. Только $m \in \{d, dp, dp^2, \dots\}$ дают $p \mid \Phi_m(a)$ — потому что $\Phi_m(a) \equiv 0 \pmod p$ требует $\operatorname{ord}_p(a) = m / p^j$ для некоторого $j \ge 0$.

Варианты и обобщения. Используется как мост между LTE и Zsygmondy: «новый» простой делитель $a^n - 1$ — это делитель $\Phi_n(a)$ (с возможной поправкой на единственный «ламповый» простой $p \mid n$).

Используется в. Доказательство Zsygmondy (E8); конструктивные нижние оценки для $\Phi_n(a)$; [IMO-Shortlist-2002-N4].

### E8. Zsygmondy / Bang theorem

Формулировка. Пусть $a > b \ge 1$ — взаимно простые целые, $n \ge 2$. Существует простое $p$, делящее $a^n - b^n$, но не делящее $a^k - b^k$ ни для одного $1 \le k < n$ (т.н. primitive prime divisor), кроме исключений:
- $n = 1$ (пусто);
- $n = 2$ и $a + b$ — степень двойки;
- $(a, b, n) = (2, 1, 6)$ (т.е. $2^6 - 1 = 63 = 7 \cdot 9$, оба делителя присутствуют ранее).

Bang's theorem — частный случай $b = 1$.

Условия применимости. $\gcd(a, b) = 1$. Без взаимно простоты теорема ложна.

Идея доказательства. Анализ $\Phi_n(a/b) \cdot b^{\varphi(n)}$ (целое число). По E7, простой $p \mid \Phi_n(a)$ либо primitive, либо $p \mid n$ — последний делитель встречается с очень малой кратностью. Оценка снизу $\Phi_n(a) > n$ (для большинства $n$) даёт существование «нелампового» простого. Случаи-исключения возникают, когда $\Phi_n(a)$ — степень единственного «лампового» простого.

Варианты и обобщения. Для Lucas-последовательностей $u_n$ — теорема Carmichael (1913): primitive prime divisor существует для всех $n > 12$.

Используется в. Доказательство, что $a^n - 1 \ne b^k$ для нетривиальных $a,b,n,k$ (Catalan-style), задачи на orders, [IMO-2003-6]: «существует $q$ простое такое, что $q \nmid n^p - p$ для всех $n$» — через Zsygmondy для $p$-й циклотоматической части $n^{p^2} - 1$.

### E9. Связка Legendre + Kummer для биномиальных оценок

Формулировка. Базовый pipeline: чтобы оценить $v_p(M)$ для $M$ — рационального выражения через факториалы и биномиалы, разложить $M$ как произведение $\prod \binom{n_i}{k_i}^{\epsilon_i} \cdot \prod m_j^{\eta_j}$ и применить:
- E4 к каждому факториалу;
- E5 к биномиальным;
- E1–E3 к степенным разностям.

Жёстких формул здесь нет — это техника. Ключ — переформулировать «странное» выражение в стандартный набор.

Условия применимости. Универсально для арифметических задач со смесью факториалов, биномиальных и степенных.

Идея доказательства. Линейность $v_p$ по умножению.

Используется в. [IMC-2008-DAY2-3: $2^{n-1} F_n$ через биномиальное разложение Бинэ — оценка $v_2$ суммы], [IMC-2025-10: $(a^2+a)(b^2+b)$ — анализ $v_p$ через факторизацию $ab(a+1)(b+1)$ по простым], почти любой Putnam Problem 6 с числовыми оценками.

## Записи — Exotic

### E10. Универсальная LTE через order: «E1 без условия $p \mid x - y$»

Формулировка. Пусть $p$ — нечётное простое, $p \nmid xy$, $d = \operatorname{ord}_p(x \bar y)$, где $\bar y = y^{-1} \bmod p$. Тогда:
$$
v_p(x^n - y^n) = \begin{cases}
v_p(x^d - y^d) + v_p(n / d), & d \mid n, \\
0, & d \nmid n.
\end{cases}
$$
Эквивалентно: $v_p(x^n - y^n) > 0 \iff d \mid n$, и при $d \mid n$ растёт по шагам $v_p$ от $n/d$.

Условия применимости. $p$ нечётное (для $p=2$ — формулы из E3 + анализ orders). $\gcd(x, p) = \gcd(y, p) = 1$.

Идея доказательства. Записать $x^n - y^n = y^n((x/y)^n - 1)$ в $\mathbb{Z}_p$ (или над $\mathbb{F}_p$ для условия $d \mid n$). Применить E1 к $(x/y)^d - 1$ и подставить $n = d \cdot m$.

Варианты и обобщения. Это формулировка «LTE без подготовки»: вместо того, чтобы переходить от $x \equiv y \pmod p$, сразу выбираем $d$, после чего работает E1.

Используется в. Удобно при оценке $a^n - 1$ для разных простых параллельно; неявно стоит за всеми current задачами на «найти все $n$ с $p^k \mid a^n + b^n$».

### E11. Wieferich-style и порча E1

Формулировка. Утверждение E1 показывает, что «генерически» $v_p(a^{p-1} - 1) = 1$ для $p \nmid a$ (по Малой Ферма). Wieferich-prime для основания $a$ — такое $p$, что
$$
v_p(a^{p-1} - 1) \ge 2.
$$
Для $a = 2$ известны лишь $p = 1093, 3511$. Их существование делает order lifting (E6) «кривым»: $\operatorname{ord}_{p^2}(a) = \operatorname{ord}_p(a)$ вместо $p \cdot \operatorname{ord}_p(a)$.

Условия применимости. Только теоретическая «контрольная точка»: подавляющая часть простых не Wieferich.

Идея доказательства. $a^{p-1} \equiv 1 \pmod{p}$ (Ферма), $v_p(a^{p-1} - 1) \ge 1$ автоматически; «лифт» до $\ge 2$ — редкое событие.

Используется в. На IMC напрямую не встречается, но даёт важную осторожность: «не предполагать, что $v_p(a^{p-1} - 1) = 1$»; контрпример к слишком общим LTE-формулировкам. Также Wieferich-pairs возникают в анализе уравнения Каталана.

### E12. Wall's law: LTE для Lucas-последовательностей

Формулировка. Пусть $u_n$ — Lucas-последовательность с параметрами $P, Q$ (т.е. $u_0 = 0, u_1 = 1, u_{n+1} = P u_n - Q u_{n-1}$), $D = P^2 - 4Q$ — дискриминант. Для нечётного простого $p$ с $p \nmid Q D$ положим $\alpha(p) := \min\{n \ge 1 : p \mid u_n\}$ — rank of apparition. Тогда $p \mid u_n \iff \alpha(p) \mid n$, и
$$
v_p(u_{n}) = v_p(u_{\alpha(p)}) + v_p(n / \alpha(p)) \quad \text{при } \alpha(p) \mid n.
$$
Для Fibonacci ($P=1, Q=-1$): $v_p(F_n) = v_p(F_{\alpha(p)}) + v_p(n/\alpha(p))$ при $\alpha(p) \mid n$. В частности $v_5(F_n) = v_5(n)$ (т.к. $\alpha(5) = 5$, $v_5(F_5) = v_5(5) = 1$).

Условия применимости. $p$ нечётное, $p \nmid QD$. Контрпример — Wall–Sun–Sun primes: $p$, для которых $v_p(F_{\alpha(p)}) \ge 2$ (ни одного известного на $p < 10^{17}$, но теоретически могут быть). На таких простых базовая формула остаётся, просто база $v_p(u_{\alpha(p)})$ — не обязательно $1$.

Идея доказательства. $u_n = (\phi^n - \psi^n)/(\phi - \psi)$, где $\phi, \psi$ — корни $x^2 - Px + Q$. В кольце $\mathbb{Z}_p[\sqrt{D}]$ (при $p \nmid D$) применяется E1 к $\phi^n - \psi^n$.

Варианты и обобщения. Аналог E10 для Lucas: «LTE без подготовки». Для companion-последовательности $v_n = \phi^n + \psi^n$ — формулы через E2-аналог.

Используется в. Задачи на делимость Fibonacci/Lucas в IMC и Putnam: $F_n \mid F_m \iff n \mid m$ (через Wall + $\alpha$); регулярная сборка контрпримеров; теория rank distribution.

## Self-test

1. Найти $v_3(2^{3^{100}} + 1)$.

2. Найти все натуральные $n$, при которых $n^2 \mid 2^n + 1$.

3. Доказать, что $7^{n} - 4^n$ делится на $3^k$ тогда и только тогда, когда $3^{k-1} \mid n$.

4. Сколько раз $7$ входит в простое разложение $\binom{1000}{300}$?

5. Доказать, что числа $F_k = 2^{2^k} + 1$ ($k = 0, 1, 2, \dots$) попарно взаимно просты.

6. Найти $v_2(3^{2^{50}} - 1)$.

7. Пусть $p$ — нечётное простое, $a, b$ — целые с $p \nmid ab$ и $p \nmid a - b$. Доказать, что $v_p(a^n - b^n) = v_p\!\left(a^{\operatorname{ord}_p(ab^{-1})} - b^{\operatorname{ord}_p(ab^{-1})}\right) + v_p(n / \operatorname{ord}_p(ab^{-1}))$ при условии, что $\operatorname{ord}_p(ab^{-1}) \mid n$.

## Решения

1. По E2: $3 \mid 2 + 1 = 3$, $3 \nmid 2 \cdot 1$, показатель $3^{100}$ нечётный. Тогда
$$v_3(2^{3^{100}} + 1) = v_3(2 + 1) + v_3(3^{100}) = 1 + 100 = 101.$$

2. Ответ: $n = 1$ и $n = 3$. Пусть $n > 1$, $p$ — наименьший простой делитель $n$. Из $n \mid 2^n + 1$ и порядка $\operatorname{ord}_p(2)$ делящего $\gcd(2n, p-1)$, при минимальности $p$ получаем $\operatorname{ord}_p(2) \mid 2$, т.е. $p = 3$. По E2: $v_3(2^n + 1) = v_3(3) + v_3(n) = 1 + v_3(n)$. Условие $n^2 \mid 2^n + 1 \Rightarrow 2 v_3(n) \le 1 + v_3(n) \Rightarrow v_3(n) \le 1$. Если $v_3(n) = 1$, $n = 3m$ с $\gcd(m, 3) = 1$. Если $m > 1$, у $m$ есть наименьший простой $q > 3$, и $\operatorname{ord}_q(2) \mid 2n = 6m$, что при $\gcd(\operatorname{ord}_q(2), m) = 1$ (по минимальности $q$) даёт $\operatorname{ord}_q(2) \mid 6$, т.е. $q \in \{3, 7\}$. Проверка $q=7$: $2^3 = 8 \equiv 1 \pmod 7$, нужно $7 \mid 2^n + 1$, но $2^n \pmod 7 \in \{2,4,1\}$, $+1 \in \{3,5,2\}$, ни один не $\equiv 0$. Значит $m = 1$, $n = 3$.

3. По E1: $3 \mid 7 - 4 = 3$, $3 \nmid 7 \cdot 4$, $v_3(7^n - 4^n) = v_3(7 - 4) + v_3(n) = 1 + v_3(n)$. Тогда $3^k \mid 7^n - 4^n \iff 1 + v_3(n) \ge k \iff v_3(n) \ge k-1 \iff 3^{k-1} \mid n$.

4. По E5 (Kummer): $v_7\binom{1000}{300} =$ число переносов при сложении $300$ и $700$ в системе по основанию $7$. $1000 = 2626_7$ (т.к. $7^3 = 343$, $1000 = 2 \cdot 343 + 314 = 2\cdot 343 + 6 \cdot 49 + 20 = 2\cdot 343 + 6 \cdot 49 + 2\cdot 7 + 6$). $300 = 606_7$ ($300 = 6\cdot 49 + 0 \cdot 7 + 6$, проверка $6\cdot 49 + 6 = 300$ ✓). $700 = 2020_7$ ($700 = 2\cdot 343 + 14 = 2\cdot 343 + 2\cdot 7 + 0$, проверка $686 + 14 = 700$ ✓). Сложение разрядно с $300 = 0606_7$, $700 = 2020_7$: $6+0=6$, $0+2=2$, $6+0=6$, $0+2=2$ — нет переносов. Ответ: $v_7\binom{1000}{300} = 0$. (Sanity: $\binom{1000}{300}$ не делится на $7$.)

5. $\gcd(F_k, F_l)$, $k < l$. Тождество $\prod_{j=0}^{l-1}(2^{2^j} + 1) = 2^{2^l} - 1$ даёт $F_k \mid 2^{2^l} - 1$. Тогда $F_l = 2^{2^l} + 1 = (2^{2^l} - 1) + 2 \equiv 2 \pmod{F_k}$. Поскольку $F_k$ нечётно, $\gcd(F_k, F_l) = \gcd(F_k, 2) = 1$. (LTE здесь не нужен — каноничный аргумент через факторизацию; формально согласуется с Zsygmondy E8: $\Phi_{2^{l+1}}(2) = F_l$ имеет primitive prime divisor.)

6. $3^{2^{50}} - 1$, $x = 3, y = 1$, $p = 2$. $x, y$ нечётные, $n = 2^{50}$ чётное, применяем E3(b):
$$v_2(3^{2^{50}} - 1) = v_2(3-1) + v_2(3+1) + v_2(2^{50}) - 1 = 1 + 2 + 50 - 1 = 52.$$

7. Это E10. Положим $d = \operatorname{ord}_p(ab^{-1})$, $b^{-1}$ — обратное в $\mathbb{F}_p$. Имеем $a^d \equiv b^d \pmod p$, т.е. $p \mid a^d - b^d$. Пусть $n = dm$. Тогда $a^n - b^n = (a^d)^m - (b^d)^m$, и для пары $X = a^d, Y = b^d$ верно $p \mid X - Y$, $p \nmid XY$ (т.к. $p \nmid ab$). По E1:
$$v_p(X^m - Y^m) = v_p(X - Y) + v_p(m) = v_p(a^d - b^d) + v_p(n/d).$$
