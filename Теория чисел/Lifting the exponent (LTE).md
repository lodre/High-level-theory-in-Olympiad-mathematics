# Lifting the exponent (LTE)

## Обзор

LTE — семейство формул, вычисляющих $p$-адическую оценку $v_p(x^n \pm y^n)$ при условии $p \mid x \mp y$.

Главная идея проста: при выполнении условий новые степени $p$ заходят в $x^n - y^n$ только через $n$, то есть
$v_p(x^n - y^n) = v_p(x - y) + v_p(n)$.

Подтема также включает формулы Лежандра и Куммера для $v_p(n!)$ и $v_p\binom{n}{k}$, подъём порядка по модулю $p^k$, разложение через круговые многочлены $\Phi_n$, теорему Жигмонди и аналог LTE для последовательностей Люка.

На IMC и Putnam LTE появляется в задачах вида «доказать $p^k \mid \dots$», «найти все $n$ с $n^2 \mid a^n + 1$», в оценках знаменателей частичных сумм $\sum 1/k!$, а также как несущая лемма доказательства Жигмонди и подъёма порядка.

## Определения

**$p$-адическая оценка** ($p$-adic valuation). $v_p(n)$ — показатель степени, с которой простое $p$ входит в разложение целого $n$.

По соглашению $v_p(0) = +\infty$. Базовые свойства: $v_p(ab) = v_p(a) + v_p(b)$ и $v_p(a + b) \ge \min(v_p(a), v_p(b))$, причём равенство при $v_p(a) \ne v_p(b)$.

**Сумма цифр** $s_p(n)$ — сумма цифр $n$ в записи по основанию $p$.

**Мультипликативный порядок** $\operatorname{ord}_m(a)$ для $\gcd(a, m) = 1$ — наименьшее $d \ge 1$ с $a^d \equiv 1 \pmod m$.

**Круговой многочлен** $\Phi_n(x)$ — единственный моничный неприводимый над $\mathbb{Q}$ многочлен, корни которого — первообразные корни $n$-й степени из единицы. Степень $\deg \Phi_n = \varphi(n)$. Базовое разложение:
$$x^n - 1 = \prod_{d \mid n} \Phi_d(x).$$

**Последовательность Люка** ($u_n$). Линейная рекуррентная последовательность с $u_0 = 0$, $u_1 = 1$ и $u_{n+1} = P u_n - Q u_{n-1}$. Характеристический многочлен $x^2 - P x + Q$, дискриминант $D = P^2 - 4Q$. При $P = 1$, $Q = -1$ получаются числа Фибоначчи $F_n$.

**Корневой индекс** (rank of apparition). $\alpha(p) := \min\{n \ge 1 : p \mid u_n\}$ — наименьший номер члена последовательности Люка, делящегося на $p$.

## Записи — Основные

### E1. LTE для нечётного $p$, разность

Формулировка. Пусть $p$ — нечётное простое, $x, y \in \mathbb{Z}$, $n \ge 1$, и выполнены три условия: $p \mid x - y$, $p \nmid x$, $p \nmid y$. Тогда
$$
v_p(x^n - y^n) = v_p(x - y) + v_p(n).
$$

Условия применимости.

Нечётность $p$ обязательна. Контрпример: $p = 2$, $x = 3$, $y = 1$, $n = 2$. Реально $v_2(8) = 3$, формула давала бы $v_2(2) + v_2(2) = 2$.

Условие $p \mid x - y$ обязательно. Контрпример: $p = 3$, $x = 2$, $y = 1$, $n = 1$. Реально $v_3(1) = 0$, формула давала бы $v_3(1) + v_3(1) = 0$ — случайное совпадение, но при $n = 3$: $v_3(7) = 0$, формула давала бы $0 + 1 = 1$.

Условие $p \nmid xy$ обязательно. Иначе обе стороны теряют смысл (например, $x = p$, $y = 0$).

Идея доказательства. Индукция по $v_p(n)$.

База $p \nmid n$: имеем $\frac{x^n - y^n}{x - y} = \sum_{k=0}^{n-1} x^{n-1-k} y^k$. Каждое слагаемое сравнимо с $y^{n-1}$ по модулю $p$, поэтому вся сумма сравнима с $n y^{n-1}$. Поскольку $p \nmid n y^{n-1}$, частное не делится на $p$.

Шаг $n = p m$: достаточно доказать вспомогательную лемму $v_p\!\bigl(\tfrac{a^p - b^p}{a - b}\bigr) = 1$ при $p \mid a - b$, $p \nmid a$. Подстановка $b = a - p t$ и разложение по биному дают ведущий по $p$ член $p a^{p-1} t$.

### E2. LTE для нечётного $p$, сумма (нечётное $n$)

Формулировка. Пусть $p$ — нечётное простое, $x, y \in \mathbb{Z}$, $n$ — нечётное натуральное, $p \mid x + y$, $p \nmid x$, $p \nmid y$. Тогда
$$
v_p(x^n + y^n) = v_p(x + y) + v_p(n).
$$

Условия применимости.

Нечётность $n$ обязательна. Контрпример: $p = 3$, $x = 2$, $y = 1$, $n = 2$. Имеем $3 \mid x + y$, но $x^2 + y^2 = 5$, $v_3(5) = 0$, формула давала бы $1$.

Остальные условия — как в E1.

Идея доказательства. Подстановка $y \to -y$ в E1: при нечётном $n$ имеем $x^n + y^n = x^n - (-y)^n$, и условие $p \mid x - (-y)$ — это и есть $p \mid x + y$.

### E3. LTE для $p = 2$

Формулировка. Пусть $x, y$ — нечётные целые, $n \ge 1$. Различают три случая.

Случай (a): $n$ нечётно.
$$
v_2(x^n - y^n) = v_2(x - y).
$$

Случай (b): $n$ чётно.
$$
v_2(x^n - y^n) = v_2(x - y) + v_2(x + y) + v_2(n) - 1.
$$

Случай (c): $4 \mid x - y$, $n$ любое.
$$
v_2(x^n - y^n) = v_2(x - y) + v_2(n).
$$

Случай (c) — следствие (b): при $4 \mid x - y$ имеем $v_2(x + y) = 1$, и формула (b) превращается в формулу (c).

Для суммы $x^n + y^n$ при нечётных $x, y$:
- при нечётном $n$ подстановкой $y \to -y$ получаем $v_2(x^n + y^n) = v_2(x + y)$;
- при чётном $n$ имеем $x^n + y^n \equiv 2 \pmod 4$, поэтому $v_2(x^n + y^n) = 1$.

Условия применимости.

Случай (b) — только при чётном $n$. Контрпример при $n = 1$: $x = 3$, $y = 1$. Реально $v_2(2) = 1$, формула (b) давала бы $1 + 2 + 0 - 1 = 2$.

Случай (a) — только при нечётном $n$. Контрпример при $n = 2$: $x = 3$, $y = 1$. Реально $v_2(8) = 3$, формула (a) давала бы $1$.

Идея доказательства.

Случай (a): из разложения $x^n - y^n = (x - y)(x^{n-1} + \dots + y^{n-1})$, второй множитель — сумма $n$ нечётных слагаемых, значит нечётен.

Случаи (b), (c): индукция по $v_2(n)$ с использованием тождества $x^2 - y^2 = (x - y)(x + y)$ и леммы $v_2(x + y) = 1$ при $x \equiv y \equiv 1 \pmod 4$ или $x \equiv y \equiv 3 \pmod 4$.

### E4. Формула Лежандра для $v_p(n!)$

Формулировка. Для простого $p$ и $n \ge 0$:
$$
v_p(n!) = \sum_{k \ge 1} \left\lfloor \frac{n}{p^k} \right\rfloor = \frac{n - s_p(n)}{p - 1}.
$$

Условия применимости. Любые $n \ge 0$, любое простое $p$. Сумма обрывается при $p^k > n$.

Идея доказательства. Значение $\lfloor n / p^k \rfloor$ равно количеству чисел от $1$ до $n$, делящихся на $p^k$. Суммируя по $k$, считаем каждое $m$ ровно $v_p(m)$ раз.

Вторая форма выводится подстановкой $n = \sum_k a_k p^k$ ($a_k$ — цифры $n$ в системе по основанию $p$) и телескопированием.

### E5. Теорема Куммера для $v_p\binom{m+n}{n}$

Формулировка. Для простого $p$ и $m, n \ge 0$:
$$
v_p\!\binom{m + n}{n} = \#\bigl\{\text{переносов при сложении $m$ и $n$ в системе по основанию } p\bigr\}.
$$
Эквивалентно:
$$
v_p\!\binom{m + n}{n} = \frac{s_p(m) + s_p(n) - s_p(m + n)}{p - 1}.
$$

Условия применимости. Любые $m, n \ge 0$, любое простое $p$.

Идея доказательства. По E4 имеем $v_p\binom{m+n}{n} = v_p((m+n)!) - v_p(m!) - v_p(n!)$. Подставляем формулу Лежандра во второй форме: получаем $\bigl(s_p(m) + s_p(n) - s_p(m+n)\bigr) / (p - 1)$. Каждый перенос при сложении уменьшает суммарную сумму цифр ровно на $p - 1$: одна цифра падает на $p$, соседняя растёт на $1$.

## Записи — Стандартные

### E6. Подъём порядка по модулю $p^k$

Формулировка. Пусть $p$ — нечётное простое, $p \nmid a$, $d = \operatorname{ord}_p(a)$, $c = v_p(a^d - 1) \ge 1$. Тогда для всех $k \ge 1$:
$$
\operatorname{ord}_{p^k}(a) = d \cdot p^{\max(0,\, k - c)}.
$$

Для $p = 2$ аналог получается заменой E1 на E3. Например, при $a \equiv 1 \pmod 4$ имеем $\operatorname{ord}_{2^k}(a) = 2^{\max(0,\, k - v_2(a - 1))}$.

Условия применимости. $\gcd(a, p) = 1$.

Контрпример к «типичному» случаю $c = 1$. При $a = 2$, $p = 1093$ выполнено $2^{1092} \equiv 1 \pmod{1093^2}$ (Wieferich-простое для основания $2$), поэтому $c \ge 2$ и $\operatorname{ord}_{1093^2}(2) = 1092$ вместо «ожидаемых» $1092 \cdot 1093$.

Идея доказательства. По E1 $v_p(a^{n d} - 1) = v_p(a^d - 1) + v_p(n) = c + v_p(n)$. Наименьшее $n$, при котором эта величина $\ge k$, равно $p^{\max(0,\, k - c)}$. Следовательно, наименьшее $N$ с $a^N \equiv 1 \pmod{p^k}$ — это $d \cdot p^{\max(0,\, k - c)}$.

### E7. $v_p(\Phi_n(a))$ через круговое разложение

Формулировка. Пусть $p$ — нечётное простое, $p \nmid a$, $d = \operatorname{ord}_p(a)$. Тогда:
- $v_p(\Phi_d(a)) = v_p(a^d - 1) \ge 1$;
- $v_p(\Phi_{d \cdot p^k}(a)) = 1$ для $k \ge 1$;
- $v_p(\Phi_n(a)) = 0$ для $n \ne d \cdot p^k$, $k \ge 0$.

Сводное тождество (эквивалентно E1):
$$
v_p(a^n - 1) = \sum_{m \mid n} v_p(\Phi_m(a)).
$$

Условия применимости. $p$ нечётное, $p \nmid a$. Для $p = 2$ — отдельная серия через E3.

Идея доказательства. Из базового разложения $a^n - 1 = \prod_{m \mid n} \Phi_m(a)$ получаем сводное тождество. Применяем E1 к левой части и сравниваем со вкладами правой. Условие $\Phi_m(a) \equiv 0 \pmod p$ требует, чтобы порядок $a$ по модулю $p$ был равен $m$, поделённому на степень $p$, то есть $m = d \cdot p^k$.

### E8. Теорема Жигмонди (Zsygmondy)

Формулировка. Пусть $a > b \ge 1$ — взаимно простые целые, $n \ge 2$. Тогда у числа $a^n - b^n$ существует простой делитель $p$, не делящий $a^k - b^k$ ни для одного $1 \le k < n$ (примитивный простой делитель, primitive prime divisor). Исключения:
- $n = 2$ и $a + b$ — степень двойки;
- $(a, b, n) = (2, 1, 6)$: число $2^6 - 1 = 63 = 7 \cdot 9$ не даёт новых простых.

Условия применимости. $\gcd(a, b) = 1$ обязательно. Без взаимной простоты теорема неверна.

Идея доказательства. Анализируется $\Phi_n(a / b) \cdot b^{\varphi(n)}$ — целое число. По E7 его простой делитель либо примитивен, либо делит $n$. Нижняя оценка $\Phi_n(a) > n$ при $a \ge 2$ и достаточно большом $n$ гарантирует существование примитивного делителя. Исключения возникают, когда $\Phi_n(a)$ полностью «съедается» делителем $n$.

### E9. Связка Лежандра и Куммера для биномиальных оценок

Формулировка. Универсальная схема. Для арифметического выражения $M$, составленного из факториалов, биномиальных и степенных разностей, оценка $v_p(M)$ разбивается на сумму вкладов:
- факториалы — через E4;
- биномиальные — через E5;
- степенные разности — через E1, E2 или E3.

Это методическая запись, а не теорема.

Условия применимости. Работает, когда $M$ удаётся представить в виде произведения факториалов, биномиальных и степенных разностей с явной структурой.

Идея доказательства. Линейность $v_p$ относительно умножения.

## Записи — Редкие

### E10. Общая LTE через порядок

Формулировка. Пусть $p$ — нечётное простое, $a, b \in \mathbb{Z}$, $p \nmid a b$, $d = \operatorname{ord}_p(a b^{-1})$ (порядок в $(\mathbb{Z}/p\mathbb{Z})^\times$). Тогда:
- $p \mid a^n - b^n \iff d \mid n$;
- при $d \mid n$ выполнено $v_p(a^n - b^n) = v_p(a^d - b^d) + v_p(n / d)$.

Условия применимости. $p$ нечётное, $\gcd(a b, p) = 1$. Для $p = 2$ — заменить E1 на E3.

Идея доказательства. При $d \mid n$, $n = d m$, записываем $a^n - b^n = (a^d)^m - (b^d)^m$. Для пары $(a^d, b^d)$ выполнены условия E1: $p \mid a^d - b^d$, $p \nmid a^d b^d$. Применяем E1 к этой паре.

### E11. Закон Уолла (Wall's law) для последовательностей Люка

Формулировка. Пусть $u_n$ — последовательность Люка с параметрами $P, Q$, $D = P^2 - 4Q$. Для нечётного простого $p$ с $p \nmid Q D$ и корневого индекса $\alpha = \alpha(p)$:
- $p \mid u_n \iff \alpha \mid n$;
- при $\alpha \mid n$ выполнено $v_p(u_n) = v_p(u_\alpha) + v_p(n / \alpha)$.

Для Фибоначчи: $v_5(F_n) = v_5(n)$ (так как $\alpha(5) = 5$ и $v_5(F_5) = 1$).

Условия применимости. $p$ нечётное, $p \nmid Q D$.

Контрпример к «$v_p(u_\alpha) = 1$»: гипотетические простые Уолла–Сун–Суна (Wall–Sun–Sun primes) — простые $p > 5$ с $v_p(F_{\alpha(p)}) \ge 2$. Ни одного не известно ($p < 10^{17}$), но если такое $p$ существует, база формулы будет больше единицы.

Идея доказательства. Бинетовская формула $u_n = (\phi^n - \psi^n)/(\phi - \psi)$, где $\phi, \psi$ — корни характеристического многочлена. В кольце $\mathbb{Z}_p[\sqrt{D}]$ при $p \nmid D$ применяется E1 к разности $\phi^n - \psi^n$.

## Использования

E1, E2 — несущая лемма практически любой задачи на «доказать $p^k \mid a^n \pm b^n$» или обратной «найти все $n$ с $p^k \mid a^n \pm b^n$». Также — база E6, E7, E8, E10.
Применения: IMC-2022-3 (через E5 и Лукаса), IMO-1990-3 (найти все $n > 1$ с $n^2 \mid 2^n + 1$ — E2 плюс порядок), Putnam-1972-A6 ($v_3(2^{3^n} + 1) = n + 1$).

E3 — все задачи с делимостью на степень двойки разностей вида $a^n - b^n$ при нечётных $a, b$. Особенно популярно вокруг чисел Мерсенна и Ферма.
Применение: IMC-2008-D2-3 (оценка $v_2$ для $\sum \binom{n}{2 k + 1} 5^k$ через формулу Бинэ для $F_n$).

E4 — стандартный инструмент для оценки роста факториалов и числа простых делителей биномиальных. База E5.
Применения: IMC-2019-7 (рост наименьшего $k$ с $n \mid k!$ через $v_p(k!) \ge v_p(n)$), IMC-2023-10 (рост знаменателя $\sum_{k=0}^n 1/k!$ — оценка $g(n)^{1/n} \to e$).

E5 — оценки $v_p$ биномиальных и доказательства вида «$\binom{2n}{n}$ имеет много простых делителей».
Применение: IMC-2022-3 (комплементарно через теорему Лукаса для остатка по модулю $p$).

E6 — задачи на нахождение наименьшего $n$ с $p^k \mid a^n - 1$ и конструкции элементов больших порядков.

E7 — мост к теореме Жигмонди (E8) и оценкам $\Phi_n(a)$ снизу и сверху.

E8 — диофантовы задачи: доказательства, что $a^n - 1 \ne b^k$ для нетривиальных параметров (стиль уравнения Каталана), оценки числа решений уравнений вида $a^n + 1 = c \cdot k^2$.
Применение: IMO-2003-6 (существование простого $q$ с $q \nmid n^p - p$ для всех $n$).

E9 — общая схема для арифметических оценок на IMC и Putnam.
Применение: IMC-2025-10 (анализ $(a^2 + a)(b^2 + b)$ через факторизацию $a b (a + 1)(b + 1)$ по простым).

E10 — удобная форма для оценки $a^n - 1$ при разных простых параллельно, без предварительного приведения к виду «$x \equiv y \pmod p$».

E11 — задачи на делимость чисел Фибоначчи и Люка, оценки $v_p(F_n)$.

## Задачи для проверки

1. Найти $v_3(2^{3^{100}} + 1)$.

2. Найти все натуральные $n$, при которых $n^2 \mid 2^n + 1$.

3. Доказать, что $3^k \mid 7^n - 4^n$ тогда и только тогда, когда $3^{k - 1} \mid n$.

4. Сколько раз $7$ входит в простое разложение $\binom{1000}{300}$?

5. Доказать, что для любого $n \ge 1$ выполнено $v_2(3^{2n} - 1) = 3 + v_2(n)$.

6. Доказать, что для любого $k \ge 2$ выполнено $\operatorname{ord}_{2^k}(5) = 2^{k - 2}$.

## Решения

1. По E2: $3 \mid 2 + 1$, $3 \nmid 2 \cdot 1$, показатель $3^{100}$ нечётный.
$$v_3(2^{3^{100}} + 1) = v_3(2 + 1) + v_3(3^{100}) = 1 + 100 = 101.$$

2. Ответ: $n \in \{1, 3\}$.

Случай $n = 1$ очевиден.

Пусть $n > 1$. Поскольку $2^n + 1$ нечётно, $n$ нечётно. Пусть $p$ — наименьший простой делитель $n$.

Тогда $p \mid 2^n + 1$, откуда $2^{2 n} \equiv 1 \pmod p$. Порядок $d = \operatorname{ord}_p(2)$ делит $2 n$, но не делит $n$ (иначе $2^n \equiv 1 \not\equiv -1$).

Значит $v_2(d) = v_2(2 n) = 1$. Каждый нечётный простой делитель $d$ должен делить $n$ и быть меньше $p$. По минимальности $p$ таких делителей нет, поэтому $d = 2$ и $p \mid 2^2 - 1 = 3$, то есть $p = 3$.

По E2: $v_3(2^n + 1) = 1 + v_3(n)$. Из $n^2 \mid 2^n + 1$ следует $2 v_3(n) \le 1 + v_3(n)$, то есть $v_3(n) \le 1$. Так как $3 \mid n$, имеем $v_3(n) = 1$, то есть $n = 3 m$ с нечётным $m$, $\gcd(m, 3) = 1$.

При $m > 1$ повторяем рассуждение для $q$ — наименьшего простого делителя $m$. Тогда $q > 3$, и $d_q = \operatorname{ord}_q(2)$ удовлетворяет: $v_2(d_q) = 1$, нечётные простые делители $d_q$ — только $3$, $d_q \mid 2 n = 6 m$, $d_q \nmid n$. Возможны $d_q \in \{2, 6\}$. При $d_q = 2$: $q = 3$, противоречие. При $d_q = 6$: $q \mid 2^6 - 1 = 63 = 9 \cdot 7$, значит $q = 7$. Но $7 \nmid 2^3 + 1 = 9$, противоречие.

Значит $m = 1$, $n = 3$.

3. По E1: $3 \mid 7 - 4$, $3 \nmid 7 \cdot 4$. Тогда $v_3(7^n - 4^n) = v_3(7 - 4) + v_3(n) = 1 + v_3(n)$.

Условие $3^k \mid 7^n - 4^n$ эквивалентно $1 + v_3(n) \ge k$, то есть $v_3(n) \ge k - 1$, то есть $3^{k - 1} \mid n$.

4. По E5 (Куммер): $v_7\binom{1000}{300}$ равно числу переносов при сложении $300 + 700$ в системе по основанию $7$.

Запись в семеричной: $1000 = 2626_7$, $300 = 606_7$, $700 = 2020_7$.

Сложение поразрядно справа налево: $6 + 0 = 6$, $0 + 2 = 2$, $6 + 0 = 6$, $0 + 2 = 2$. Переносов нет.

Ответ: $v_7\binom{1000}{300} = 0$.

5. Имеем $x = 3$, $y = 1$ — оба нечётные. Показатель $2 n$ чётный. Применяем E3 (b):
$$v_2(3^{2 n} - 1) = v_2(3 - 1) + v_2(3 + 1) + v_2(2 n) - 1 = 1 + 2 + (1 + v_2(n)) - 1 = 3 + v_2(n).$$

6. Так как $4 \mid 5 - 1$, применима E3 (c) для любого $n \ge 1$:
$$v_2(5^n - 1) = v_2(5 - 1) + v_2(n) = 2 + v_2(n).$$

Наименьшее $n$ с $2^k \mid 5^n - 1$ соответствует наименьшему $n$ с $v_2(n) \ge k - 2$, то есть $n = 2^{k - 2}$ при $k \ge 2$.

Значит $\operatorname{ord}_{2^k}(5) = 2^{k - 2}$.
