# Контурное интегрирование и вычеты для вещественных интегралов

## Scope
Перевод вещественных интегралов и сумм в контурные через подходящее аналитическое продолжение и выбор контура. Покрывает: рациональные и тригонометрические интегралы по $\mathbb{R}$ и $[0,2\pi]$, интегралы $\int_0^\infty x^{s-1}R(x)\,dx$ и $\int_0^\infty R(x)\ln x\,dx$ через keyhole, оценки через лемму Жордана (Jordan), Cauchy principal value через indented contour, суммирование рядов через $\pi\cot\pi z$, Mittag-Leffler разложения, аналитическое продолжение Гамма- и Бета-функций (Hankel, Pochhammer). Цель — дать набор шаблонов, в которых выбор контура диктуется типом сингулярности и асимптотикой подынтегральной функции.

## Записи — Core

### E1. Residue theorem (теорема о вычетах)

Формулировка. Пусть $\Omega \subset \mathbb{C}$ — открытое односвязное (или гомотопически тривиальное относительно $\gamma$) множество, $f$ голоморфна в $\Omega$ кроме изолированных особенностей $\{a_k\}$, $\gamma$ — спрямляемый замкнутый контур в $\Omega \setminus \{a_k\}$, не проходящий через особенности. Тогда
$$\oint_\gamma f(z)\,dz = 2\pi i \sum_k n(\gamma, a_k)\,\operatorname{Res}_{z=a_k} f,$$
где $n(\gamma, a_k)$ — winding number (индекс) контура относительно $a_k$.

Условия применимости. Особенностей конечное число внутри $\gamma$ (или ряд по особенностям сходится — версия для бесконечных контуров, см. E10). Контур не задевает $\{a_k\}$. Для Cauchy principal value полюса на контуре обрабатываются через E5.

Идея доказательства. Локально $f = $ (голоморфная) + (главная часть Лорана); главная часть интегрируется явно: $\oint (z-a)^{-n}\,dz = 2\pi i$ при $n=1$, $0$ иначе. Сумма по особенностям + теорема Коши для голоморфного остатка.

Используется в. [IMC-2004-6 (через cotangent partial fractions); базовый инструмент для всех записей ниже].

### E2. Способы вычисления вычетов (residue computation)

Формулировка. Для $f$ с изолированной особенностью в $a$:
- простой полюс: $\operatorname{Res}_{z=a} f = \lim_{z\to a}(z-a)f(z)$;
- полюс порядка $m$: $\operatorname{Res}_{z=a} f = \dfrac{1}{(m-1)!}\lim_{z\to a}\dfrac{d^{m-1}}{dz^{m-1}}\bigl[(z-a)^m f(z)\bigr]$;
- частное $f = g/h$ с простым нулём $h$ в $a$, $g(a)\neq 0$: $\operatorname{Res}_{z=a}\dfrac{g}{h} = \dfrac{g(a)}{h'(a)}$;
- через Laurent series: $\operatorname{Res}_{z=a} f = c_{-1}$ — коэффициент при $(z-a)^{-1}$;
- residue at infinity: $\operatorname{Res}_{z=\infty} f = -\operatorname{Res}_{w=0}\dfrac{1}{w^2}f(1/w)$, и сумма всех вычетов на $\widehat{\mathbb{C}}$ для рациональной функции равна $0$.

Условия применимости. Формула с производной — только для полюсов. Для существенных особенностей — только Laurent (либо специальный приём типа разложения экспоненты).

Идея. Прямая алгебра по определению $c_{-1}$. Формула с $h'$ — Тейлор $h(z)=h'(a)(z-a)+\dots$.

Варианты и обобщения.
- При $m \geq 3$ часто эффективнее напрямую разложить в Laurent, чем дифференцировать.
- Для рациональных $P/Q$ при подсчёте $\sum$ всех вычетов в $\mathbb{C}$: если $\deg Q - \deg P \geq 2$, то $\operatorname{Res}_\infty = 0$ и сумма конечных вычетов нулевая — мощный шорткат.

Используется в. Везде ниже.

### E3. Semicircle contour для $\int_{-\infty}^{\infty} R(x)\,dx$ (рациональные интегралы)

Формулировка. Пусть $R(z) = P(z)/Q(z)$ — рациональная, $\deg Q - \deg P \geq 2$, $Q$ не имеет нулей на $\mathbb{R}$. Тогда
$$\int_{-\infty}^{\infty} R(x)\,dx = 2\pi i \sum_{\operatorname{Im} a_k > 0} \operatorname{Res}_{z=a_k} R.$$
Симметрично — сумма по нижней полуплоскости с $-2\pi i$.

Условия применимости. $\deg Q \geq \deg P + 2$ (для исчезновения дуги). Никаких полюсов на $\mathbb{R}$ (иначе E5). Знак выбирается ориентацией.

Идея. Контур $\gamma_R = [-R, R] \cup \{Re^{i\theta}: \theta \in [0,\pi]\}$. На дуге $|R(z)| = O(1/R^2)$, длина $\pi R$, интеграл $\to 0$. Residue theorem на $\gamma_R$ с $R \to \infty$.

Варианты и обобщения.
- $\deg Q - \deg P = 1$: дуга не исчезает, нужно симметричное расширение $[-R, R]$ и Cauchy principal value (или $R(x)$ нечётная и $\int = 0$).
- При чётности $R$: $\int_0^\infty = \tfrac12 \int_{-\infty}^\infty$.

Используется в. Стандартный шаблон Putnam-style: $\int_{-\infty}^\infty \dfrac{dx}{1+x^4}$, $\int_{-\infty}^\infty \dfrac{x^2}{(1+x^2)^3}\,dx$ и т.д.

### E4. Jordan's lemma (лемма Жордана) и $\int R(x) e^{iax}\,dx$

Формулировка (Jordan's lemma). Пусть $f$ непрерывна на дуге $C_R = \{Re^{i\theta}: \theta \in [0,\pi]\}$ и $M_R := \max_{C_R}|f| \to 0$ при $R\to\infty$. Тогда для $a > 0$
$$\left|\int_{C_R} f(z) e^{iaz}\,dz\right| \leq \dfrac{\pi M_R}{a} \xrightarrow{R\to\infty} 0.$$
Следствие. Для $R$ рациональной с $\deg Q - \deg P \geq 1$, без нулей $Q$ на $\mathbb{R}$, и $a > 0$:
$$\int_{-\infty}^{\infty} R(x) e^{iax}\,dx = 2\pi i \sum_{\operatorname{Im} a_k > 0}\operatorname{Res}_{z=a_k}\bigl[R(z)e^{iaz}\bigr].$$
Тогда $\int R(x)\cos(ax)\,dx = \operatorname{Re}$, $\int R(x)\sin(ax)\,dx = \operatorname{Im}$.

Условия применимости. $a>0$ — замыкаем сверху; $a<0$ — снизу (со сменой знака). Достаточно $\deg Q - \deg P \geq 1$, не $\geq 2$ — это и есть мощь леммы Жордана. Если $f$ имеет полюс на $\mathbb{R}$, см. E5.

Идея. На $C_R$: $|e^{iaz}| = e^{-aR\sin\theta}$. Используется $\sin\theta \geq 2\theta/\pi$ на $[0,\pi/2]$ и симметрия. Интеграл $\int_0^{\pi/2} e^{-aR\sin\theta}\,d\theta \leq \int_0^{\pi/2} e^{-2aR\theta/\pi}\,d\theta \leq \pi/(2aR)$.

Варианты и обобщения.
- $\int_{-\infty}^\infty \dfrac{\sin x}{x}\,dx = \pi$ — комбинация E4 + E5 (полюс в $0$).
- Более сильные осцилляторные интегралы: $\int x^k e^{ix}/(x^2+1)$ и т.п.

Используется в. Стандартный приём для $\int \cos/\sin \cdot R$.

### E5. Indented contour и Cauchy principal value (объезд полюса на контуре)

Формулировка. Пусть $f$ голоморфна в окрестности $a \in \mathbb{R}$ кроме простого полюса в $a$ с вычетом $r$. Если $C_\varepsilon$ — полуокружность радиуса $\varepsilon$ вокруг $a$, обходимая против часовой стрелки на угол $\alpha$ ($\alpha = \pi$ для верхней половины), то
$$\lim_{\varepsilon \to 0}\int_{C_\varepsilon} f(z)\,dz = i\alpha\, r.$$
Для Cauchy principal value (главного значения) с простым полюсом на $\mathbb{R}$:
$$\mathrm{P.V.}\int_{-\infty}^\infty f(x)\,dx = 2\pi i\sum_{\operatorname{Im} a_k > 0}\operatorname{Res} f + \pi i \sum_{a_k \in \mathbb{R}}\operatorname{Res} f$$
(при выполнении условий E3/E4 на убывание).

Условия применимости. Полюс должен быть **простым** — для полюсов высших порядков предел $\int_{C_\varepsilon}$ расходится и P.V. в обычном смысле не существует. Угол $\alpha$ — фактический угол объезда (для четверть-окружности $\pi/2$).

Идея. Локально $f(z) = r/(z-a) + \text{(голоморф.)}$. Голоморфная часть даёт $0$ при $\varepsilon\to 0$. Главная часть: $\int_{C_\varepsilon} r/(z-a)\,dz = r \cdot i\alpha$ напрямую через $z = a + \varepsilon e^{i\theta}$.

Варианты и обобщения.
- Дирихле: $\int_0^\infty \dfrac{\sin x}{x}\,dx = \dfrac{\pi}{2}$ через $f(z) = e^{iz}/z$, контур $[-R,-\varepsilon] \cup C_\varepsilon \cup [\varepsilon, R] \cup C_R$.
- Принцип одинаков для логарифмических ветвей: при пересечении вырезной линии аргумент прыгает на $2\pi$.

Используется в. Все интегралы с осцилляцией и полюсом на $\mathbb{R}$ (Dirichlet, Fresnel, дисперсионные соотношения).

### E6. Unit circle substitution для $\int_0^{2\pi} R(\cos\theta, \sin\theta)\,d\theta$

Формулировка. Подстановка $z = e^{i\theta}$, $dz = iz\,d\theta$, $\cos\theta = (z+z^{-1})/2$, $\sin\theta = (z-z^{-1})/(2i)$ переводит
$$\int_0^{2\pi} R(\cos\theta,\sin\theta)\,d\theta = \oint_{|z|=1} R\!\left(\dfrac{z+z^{-1}}{2}, \dfrac{z-z^{-1}}{2i}\right)\dfrac{dz}{iz} = 2\pi \sum_{|a_k|<1}\operatorname{Res} \tilde R(z).$$

Условия применимости. $R$ — рациональная функция от $\cos,\sin$ без полюсов на $|z|=1$. Если на $|z|=1$ полюс — Cauchy principal value через E5 (но это редкость в задачах).

Идея. Прямая подстановка плюс residue theorem на $|z|=1$.

Варианты и обобщения.
- $\int_0^{2\pi} \dfrac{d\theta}{a + b\cos\theta} = \dfrac{2\pi}{\sqrt{a^2-b^2}}$ для $a>|b|$ — каноничный пример.
- Степени $\cos^n,\sin^n$ — биномиальная формула, потом residue at $z=0$.

Используется в. Putnam classics; интегралы Пуассона $\int_0^{2\pi}\dfrac{1-r^2}{1-2r\cos\theta+r^2}\,d\theta = 2\pi$.

## Записи — Standard

### E7. Keyhole contour (контур-«замочная скважина») для $\int_0^\infty x^{s-1} R(x)\,dx$ и $\int_0^\infty R(x)\ln x\,dx$

Формулировка. Для $R$ рациональной без полюсов на $[0,\infty)$, $\deg Q - \deg P \geq 1$, $0 < s < \deg Q - \deg P$:
$$\int_0^\infty x^{s-1} R(x)\,dx = \dfrac{2\pi i}{1 - e^{2\pi i s}}\sum_{a_k \neq 0}\operatorname{Res}_{z=a_k}\bigl[z^{s-1} R(z)\bigr],$$
где $z^{s-1} = e^{(s-1)\operatorname{Log} z}$ с разрезом по $[0,\infty)$, $\operatorname{Log} z = \ln|z| + i\arg z$, $\arg z \in (0,2\pi)$.

Для $\int_0^\infty R(x)\ln x\,dx$ берут $f(z) = R(z)(\ln z)^2$ или $R(z)\ln^2 z$ с тем же keyhole — после интегрирования квадратные члены $\ln$ сокращаются и остаётся $\int R\ln x$.

Условия применимости. $R$ не имеет полюсов на разрезе $[0,\infty)$. Дуги $|z|=R$ и $|z|=\varepsilon$ исчезают: первая при $\deg Q - \deg P > s$, вторая при $s > 0$.

Идея. Контур: окружность радиуса $R$ против часовой, сегмент чуть выше $[\varepsilon, R]$, окружность радиуса $\varepsilon$ по часовой, сегмент чуть ниже $[\varepsilon, R]$ обратно. На верхнем берегу $z^{s-1} = x^{s-1}$, на нижнем $z^{s-1} = x^{s-1} e^{2\pi i(s-1)} = x^{s-1} e^{2\pi i s}$. Разница $1 - e^{2\pi i s}$ умножается на $\int_0^\infty x^{s-1} R(x)\,dx$.

Варианты и обобщения.
- $\int_0^\infty \dfrac{x^{s-1}}{1+x}\,dx = \dfrac{\pi}{\sin\pi s}$ — каноничное Euler beta-reflection (см. E14 для альтернативного вывода).
- Логарифмические интегралы: $\int_0^\infty \dfrac{\ln x}{(1+x^2)^2}\,dx = -\dfrac{\pi}{4}$ через $(\ln z)^2/(1+z^2)^2$.

Используется в. Beta function evaluations, специальные значения.

### E8. Rectangle contour для $\int e^{ax}/\cosh x$, $\operatorname{sech}$, $\operatorname{csch}$ и периодических $f(z+T)=cf(z)$

Формулировка. Для функций с экспоненциальной периодичностью $f(z+ih) = e^{i\alpha} f(z)$ (например, $1/\cosh z$, $1/\sinh z$, $1/(1+e^z)$) контур — прямоугольник с вершинами $\pm R$ и $\pm R + ih$. Стороны $R \to \infty$ исчезают, верхняя сторона связана с нижней множителем, и residue theorem даёт:
$$\int_{-\infty}^\infty \dfrac{e^{ax}}{\cosh x}\,dx = \dfrac{\pi}{\cos(a\pi/2)}, \quad |a| < 1$$
(контур: высота $i\pi$, residue в $z = i\pi/2$).

Условия применимости. Подынтегральная функция убывает экспоненциально на $|x|\to\infty$ — вертикальные стороны прямоугольника $\to 0$. Высота прямоугольника — выбирается равной (полу)периоду.

Идея. Положить $f(z) = e^{az}/\cosh z$. На верхней стороне $z = x + i\pi$: $\cosh(x+i\pi) = -\cosh x$, $e^{a(x+i\pi)} = e^{ia\pi} e^{ax}$. Тогда $\int_{\text{нижн}} - e^{ia\pi}/(-1)\int_{\text{нижн}} = (1+e^{ia\pi})\int_{-\infty}^\infty\dfrac{e^{ax}}{\cosh x}\,dx = 2\pi i \cdot \operatorname{Res}_{z=i\pi/2}$.

Варианты и обобщения.
- $\int_0^\infty \dfrac{x}{\sinh x}\,dx = \dfrac{\pi^2}{4}$ через $z^2/\sinh z$ или через ряд.
- Интегралы Fresnel $\int_0^\infty \cos(x^2)\,dx = \int_0^\infty \sin(x^2)\,dx = \tfrac12\sqrt{\pi/2}$ — через сектор $\pi/4$ (E9).

Используется в. Fourier-обращение распределения вероятности $\operatorname{sech}^2$, Poisson kernel.

### E9. Sector / wedge contour для $\int_0^\infty R(x^n)\,dx$

Формулировка. Для $R$ голоморфной кроме полюсов, $z \to z^n$ симметрично относительно поворота на $2\pi/n$. Контур-сектор $[0,R]\cup C_R\cup [Re^{2\pi i/n}, 0]$. На наклонной стороне $f(z) = f(re^{2\pi i/n})$ — связано с исходным интегралом множителем $e^{2\pi i k/n}$ при подстановке $z = re^{2\pi i/n}$. Часто:
$$\int_0^\infty \dfrac{dx}{1+x^n} = \dfrac{\pi/n}{\sin(\pi/n)}, \qquad n \geq 2.$$

Условия применимости. $R$ имеет «n-кратную симметрию», т.е. $z \to ze^{2\pi i/n}$ даёт явную трансформацию. $R$ убывает достаточно быстро на дуге $C_R$.

Идея. На наклонной стороне $z = re^{2\pi i/n}$, $dz = e^{2\pi i/n}\,dr$, и $1 + z^n = 1 + r^n$ — то же что на $\mathbb{R}^+$. Дуга $C_R$ исчезает, residue в $z = e^{i\pi/n}$ — единственный полюс внутри сектора. Residue $= -e^{i\pi/n}/n$. Уравнение даёт ответ.

Варианты и обобщения.
- $\int_0^\infty \dfrac{x^{s-1}}{1+x^n}\,dx = \dfrac{\pi/n}{\sin(\pi s/n)}$ — обобщение через keyhole (E7) или сектор $2\pi/n$.
- Fresnel: сектор $\pi/4$ для $\int_0^\infty e^{ix^2}\,dx$.

Используется в. Gaussian-like интегралы, специальные значения Beta.

### E10. Cotangent residue trick: $\sum_{n=-\infty}^\infty f(n) = -\sum_{\text{полюсы } f} \operatorname{Res}\bigl[\pi\cot(\pi z) f(z)\bigr]$

Формулировка. Пусть $f$ мероморфна, без полюсов в $\mathbb{Z}$, и на квадратах $C_N = \{|x|,|y|\leq N+1/2\}$ выполнено $|f(z)| = O(|z|^{-2})$. Тогда $\oint_{C_N}\pi\cot(\pi z) f(z)\,dz \to 0$ (потому что $\pi\cot(\pi z)$ ограничена на $C_N$ равномерно по $N$), и residue theorem даёт
$$\sum_{n \in \mathbb{Z}} f(n) = -\sum_{\text{полюсы } a \text{ функции } f}\operatorname{Res}_{z=a}\bigl[\pi\cot(\pi z) f(z)\bigr].$$
Аналогично с $\pi/\sin(\pi z)$ — для альтернирующих сумм $\sum (-1)^n f(n) = -\sum\operatorname{Res}[\pi\csc(\pi z) f(z)]$.

Условия применимости. Скорость убывания $f$ на квадратах: $O(1/|z|^{1+\delta})$ достаточно для $\cot$, потому что $|\pi\cot(\pi z)|$ ограничена на $C_N$ константой не зависящей от $N$ (классическая лемма).

Идея. На $C_N$: $\cot(\pi z)$ ограничен (вершины — единственное где близко к полюсам, но $C_N$ их обходит на расстоянии $1/2$). Residue в $z=n$: $\operatorname{Res}_{z=n}\pi\cot(\pi z) = 1$, поэтому $\operatorname{Res}_{z=n}[\pi\cot\pi z \cdot f] = f(n)$.

Варианты и обобщения.
- $\zeta(2) = \pi^2/6$: применить к $f(z) = 1/z^2$, единственный полюс в $0$ кратности $2$, $\pi\cot(\pi z)/z^2 = (1/z - \pi^2 z/3 + O(z^3))/z^2$, residue в $0$ есть $-\pi^2/3$, значит $2\zeta(2) = \pi^2/3$, $\zeta(2) = \pi^2/6$.
- $\zeta(2k)$: $f(z) = 1/z^{2k}$, residue связан с числами Бернулли.
- $\sum 1/(n^2+a^2) = \pi\coth(\pi a)/a - 1/(2a^2)$ через $f(z) = 1/(z^2+a^2)$.

Используется в. [IMC-2018-10 (alternating $\sum(-1)^{a+b}/(a^2+b^2)$ — двумерная версия + theta-функция Якоби); классический инструмент для замкнутых форм рядов].

### E11. Mittag-Leffler partial fraction expansions (разложения в простые дроби)

Формулировка (Mittag-Leffler theorem). Если $f$ мероморфна с простыми полюсами в $\{a_n\}$ ($|a_n|\to\infty$) с вычетами $r_n$, и существует последовательность контуров $C_N \to \infty$, на которых $f$ ограничена ($\sup_{C_N}|f| \leq M$, $\operatorname{length}(C_N) = O(|C_N|)$), то
$$f(z) = f(0) + \sum_{n=1}^\infty r_n\!\left(\dfrac{1}{z-a_n} + \dfrac{1}{a_n}\right).$$
Каноничные разложения:
$$\pi\cot(\pi z) = \dfrac{1}{z} + \sum_{n=1}^\infty \dfrac{2z}{z^2-n^2}, \qquad \dfrac{\pi}{\sin(\pi z)} = \dfrac{1}{z} + \sum_{n=1}^\infty \dfrac{(-1)^n 2z}{z^2-n^2},$$
$$\pi\tan(\pi z) = \sum_{n=0}^\infty \dfrac{2z}{(n+1/2)^2 - z^2}, \qquad \dfrac{\pi^2}{\sin^2(\pi z)} = \sum_{n\in\mathbb{Z}} \dfrac{1}{(z-n)^2}.$$

Условия применимости. Главное — равномерная ограниченность $f$ на $C_N$. Проверка для $\cot$: на $C_N$ (квадраты с полу-целыми сторонами) $|\cot \pi z|$ ограничен.

Идея. Применить residue theorem к $f(w)/(w-z)$ по $C_N$. При $N\to\infty$ интеграл $\to 0$, residues внутри: $f(z)$ (от $1/(w-z)$) + $r_n/(a_n-z)$. Перестановка даёт ряд.

Варианты и обобщения.
- Дифференцирование тождества $\pi\cot(\pi z) = 1/z + \sum 2z/(z^2-n^2)$ даёт $\pi^2/\sin^2(\pi z)$.
- Hurwitz / Eisenstein формулы для $\sum 1/(z+n)^k$.
- Получение $\zeta(2k)$ через сравнение Тейлора $\pi\cot(\pi z)$.

Используется в. [IMC-2004-6 («$i\cot(iw/2)/2 = \sum 1/(w+2\pi i k)$, подстановка $w = \log z$»); прямая комбинация с E10 для замкнутых форм].

## Записи — Exotic

### E12. Branch cut и многолистные интегралы (вырезка по разрезу)

Формулировка / приём. Для интегралов, содержащих $\sqrt{x}$, $\log x$ или $x^a$ ($a \notin \mathbb{Z}$), выбор контура зависит от расположения разреза:
- разрез по $[0,\infty)$: keyhole (E7);
- разрез по $[-1,1]$: dumbbell контур (две петли вокруг точек ветвления, соединённые через $\infty$);
- разрез по $[a,b]$: контур-«гантель» с обходом интервала с двух сторон даёт интегралы вида $\int_a^b R(x)\sqrt{(x-a)(b-x)}\,dx$.

Каноничный пример. $\int_{-1}^1 \dfrac{dx}{\sqrt{1-x^2}(1+x^2)} = \pi/\sqrt{2}$: $\sqrt{1-z^2}$ имеет разрез $[-1,1]$, контур обходит разрез сверху и снизу (значения $\sqrt{}$ отличаются знаком), residue at $z=\pm i$ + residue at $\infty$.

Условия применимости. Аккуратное определение ветви $\sqrt{}$ или $\log$ — через явное $\arg z \in (\theta_0, \theta_0+2\pi)$. На каждой стороне разреза значение функции отличается фиксированным множителем.

Идея. Разница значений на двух берегах разреза = (множитель связи) $\cdot $ исходный интеграл. Residue theorem на замкнутом обходящем разрез контуре даёт уравнение.

Варианты и обобщения.
- Hankel contour (E13) — частный случай для $\Gamma$.
- Riemann surface техника: переход на двулистную поверхность для $\sqrt{P(z)}$ с $P$ степени $\leq 4$ — род $0$, интегралы вычислимы через $\arctan,\arcsin$.

Используется в. Класс эллиптических и алгебраических интегралов на отрезке.

### E13. Hankel's contour и аналитическое продолжение $\Gamma(s)$

Формулировка (Hankel). Контур $\mathcal{H}$ начинается в $-\infty$ выше отрицательной оси, обходит начало координат и возвращается в $-\infty$ ниже отрицательной оси. Тогда
$$\dfrac{1}{\Gamma(s)} = \dfrac{1}{2\pi i}\int_{\mathcal{H}} e^z z^{-s}\,dz, \qquad s \in \mathbb{C}.$$
Это определяет $1/\Gamma$ как entire function (целую) и сразу даёт reflection $\Gamma(s)\Gamma(1-s) = \pi/\sin\pi s$.

Условия применимости. Ветвь $z^{-s}$: $\arg z \in (-\pi, \pi]$, $z^{-s} = e^{-s \log z}$. Контур не пересекает разрез по $(-\infty, 0]$.

Идея. Стандартное определение $\Gamma(s) = \int_0^\infty t^{s-1} e^{-t}\,dt$ работает только при $\operatorname{Re} s > 0$. Hankel-контур обходит ветвление и даёт аналитическое продолжение во всю $\mathbb{C}$, причём показывает, что у $1/\Gamma$ нулей в полюсах $\Gamma$ (т.е. $s = 0,-1,-2,\dots$) и больше нигде.

Варианты и обобщения.
- Аналогичная формула для $\zeta(s)$: $\zeta(s) = \dfrac{\Gamma(1-s)}{2\pi i}\int_{\mathcal{H}}\dfrac{z^{s-1}}{e^{-z}-1}\,dz$ (Riemann's contour integral) — даёт продолжение $\zeta$ и функциональное уравнение.
- Reciprocal Bessel: подобная конструкция.

Используется в. Технический инструмент для аналитического продолжения; не часто встречается в IMC напрямую, но важен в Putnam-уровне задач на специальные функции.

### E14. Pochhammer контур и аналитическое продолжение Beta-функции

Формулировка. Pochhammer (или «двойной коммутатор») контур $\mathcal{P}$ — замкнутый путь $ABA^{-1}B^{-1}$, где $A$ — петля вокруг $0$, $B$ — петля вокруг $1$. Тогда
$$\int_{\mathcal{P}} z^{\alpha-1}(1-z)^{\beta-1}\,dz = \bigl(1 - e^{2\pi i\alpha}\bigr)\bigl(1 - e^{2\pi i\beta}\bigr)\,B(\alpha,\beta)$$
для всех $\alpha,\beta \in \mathbb{C}$, где $B(\alpha,\beta) = \int_0^1 t^{\alpha-1}(1-t)^{\beta-1}\,dt$ при $\operatorname{Re}\alpha,\operatorname{Re}\beta > 0$.

Условия применимости. Ветвь $z^{\alpha-1}(1-z)^{\beta-1}$ выбирается фиксированной начальной точкой; после прохождения коммутатора возвращается к исходному значению. Левая часть — голоморфная функция $\alpha,\beta$ всюду в $\mathbb{C}^2$.

Идея. На каждом проходе через точку ветвления $z = 0$ значение умножается на $e^{2\pi i\alpha}$, через $z=1$ — на $e^{2\pi i\beta}$. Композиция $A B A^{-1} B^{-1}$ оставляет значение неизменным. Деформацией контура к двойному обходу отрезка $[0,1]$ получаются 4 прохода с разными факторами; их комбинация и даёт $(1-e^{2\pi i\alpha})(1-e^{2\pi i\beta})\,B$.

Варианты и обобщения.
- Аналогично для гипергеометрической функции и интегралов Mellin-Barnes.
- Связь с Hankel: при $\beta \to \infty$ соответствующий predел Pochhammer-контура переходит в Hankel-контур для $\Gamma$.

Используется в. Технический приём для сложных задач с Beta/Гипергеометрией; см. Whittaker–Watson §12.43.

## Self-test

1. Вычислить $\displaystyle\int_{-\infty}^\infty \dfrac{dx}{(1+x^2)^n}$ для $n \geq 1$.

2. Найти $\displaystyle\int_0^\infty \dfrac{\cos x}{1+x^4}\,dx$.

3. Вычислить $\displaystyle\int_0^\infty \dfrac{\ln x}{(1+x^2)^2}\,dx$.

4. Найти $\displaystyle\int_0^{2\pi}\dfrac{d\theta}{(a+b\cos\theta)^2}$ при $a>|b|>0$.

5. Вычислить $\displaystyle\sum_{n=1}^\infty \dfrac{1}{n^2+a^2}$ при $a > 0$.

6. Найти $\displaystyle\int_0^\infty \dfrac{x^{s-1}}{1+x^3}\,dx$ для $0 < s < 3$.

7. Доказать $\displaystyle\int_{-\infty}^\infty\dfrac{e^{ax}}{1+e^x}\,dx = \dfrac{\pi}{\sin\pi a}$ для $0 < a < 1$.

## Решения

1. Замкнуть в верхнюю полуплоскость, единственный полюс $z=i$ кратности $n$. По формуле для residue кратного полюса: $\operatorname{Res}_{z=i}(z+i)^{-n}(z-i)^{-n} = \dfrac{1}{(n-1)!}\dfrac{d^{n-1}}{dz^{n-1}}(z+i)^{-n}\big|_{z=i}$. Получается $\dfrac{(2n-2)!}{4^{n-1}((n-1)!)^2}\cdot\dfrac{1}{(2i)^{2n-1}}$, итого $\int = \dfrac{\pi(2n-2)!}{4^{n-1}((n-1)!)^2}$.

2. $\int_0^\infty\dfrac{\cos x}{1+x^4}\,dx = \tfrac12\operatorname{Re}\int_{-\infty}^\infty\dfrac{e^{ix}}{1+x^4}\,dx$. Лемма Жордана: замкнуть сверху, полюса $e^{i\pi/4}$ и $e^{3i\pi/4}$. Сумма вычетов $\dfrac{e^{ie^{i\pi/4}}}{4e^{3i\pi/4}} + \dfrac{e^{ie^{3i\pi/4}}}{4e^{9i\pi/4}}$. После упрощения: $\dfrac{\pi}{2\sqrt{2}} e^{-1/\sqrt{2}}\bigl(\cos(1/\sqrt{2}) + \sin(1/\sqrt{2})\bigr)$.

3. Keyhole с $f(z) = (\log z)^2/(1+z^2)^2$, разрез по $[0,\infty)$. Полюса $\pm i$ кратности $2$. После вычитания верхнего и нижнего берегов получается $-4\pi i\int_0^\infty\dfrac{\log x}{(1+x^2)^2}\,dx + 4\pi^2\int_0^\infty\dfrac{dx}{(1+x^2)^2} = 2\pi i\sum\operatorname{Res}$. Вычеты считаются прямо, и в итоге $\int_0^\infty\dfrac{\ln x}{(1+x^2)^2}\,dx = -\pi/4$.

4. $z = e^{i\theta}$: $a + b\cos\theta = (bz^2 + 2az + b)/(2z) = b(z-z_1)(z-z_2)/(2z)$, где $z_{1,2} = (-a\pm\sqrt{a^2-b^2})/b$. Внутри $|z|=1$ только $z_1$ (т.к. $z_1 z_2 = 1$, $|z_1|<1<|z_2|$). $\int = \dfrac{4}{ib^2}\oint\dfrac{z\,dz}{(z-z_1)^2(z-z_2)^2}$ — полюс кратности 2 в $z_1$. Residue $= \dfrac{d}{dz}\bigl[z(z-z_2)^{-2}\bigr]_{z_1} = -(z_1+z_2)/(z_1-z_2)^3 = (2a/b)/(8(a^2-b^2)^{3/2}/b^3) = ab^2/(4(a^2-b^2)^{3/2})$. Итого $I = \dfrac{2\pi a}{(a^2-b^2)^{3/2}}$.

5. $\sum_{n=1}^\infty\dfrac{1}{n^2+a^2} = \tfrac12\bigl(\sum_{n\in\mathbb{Z}} - \tfrac{1}{a^2}\bigr)$. По E10: $\sum_{n\in\mathbb{Z}}\dfrac{1}{n^2+a^2} = -\operatorname{Res}_{z=ia}\dfrac{\pi\cot\pi z}{z^2+a^2} - \operatorname{Res}_{z=-ia}\dfrac{\pi\cot\pi z}{z^2+a^2} = -\dfrac{\pi\cot(\pi i a)}{2ia} - \dfrac{\pi\cot(-\pi i a)}{-2ia} = \dfrac{\pi\coth\pi a}{a}$. Итого: $\dfrac{\pi\coth\pi a}{2a} - \dfrac{1}{2a^2}$.

6. Keyhole с $f(z) = z^{s-1}/(1+z^3)$. Полюса $z = -1, e^{i\pi/3}, e^{-i\pi/3}$, точнее $e^{i\pi/3}, e^{i\pi}, e^{5i\pi/3}$ (нули $z^3+1$). Residue в $z_k$: $z_k^{s-1}/(3z_k^2) = z_k^{s-3}/3$. Сумма $\dfrac{1}{3}\bigl(e^{i\pi(s-3)/3} + e^{i\pi(s-3)} + e^{5i\pi(s-3)/3}\bigr)$. Делим на $1 - e^{2\pi i s}$ и на $2\pi i$, упрощаем: $\dfrac{\pi/3}{\sin(\pi s/3)}$ (это согласуется с E9 при $n=3$).

7. Прямоугольник с вершинами $\pm R$ и $\pm R + 2\pi i$. На верхней стороне $z = x+2\pi i$: $e^{a(x+2\pi i)}/(1+e^{x+2\pi i}) = e^{2\pi i a}\cdot e^{ax}/(1+e^x)$. Вертикальные стороны экспоненциально малы при $0<a<1$. Внутри прямоугольника один полюс $z = i\pi$, $\operatorname{Res} = -e^{ia\pi}$. Уравнение $(1-e^{2\pi i a})I = 2\pi i \cdot (-e^{ia\pi})$ даёт $I = \dfrac{-2\pi i\, e^{ia\pi}}{1-e^{2\pi ia}} = \dfrac{\pi}{\sin\pi a}$.
