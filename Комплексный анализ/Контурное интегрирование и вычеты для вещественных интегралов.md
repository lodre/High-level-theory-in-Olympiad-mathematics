# Контурное интегрирование и вычеты для вещественных интегралов

## Scope

Перевод вещественных интегралов и сумм в контурные через подходящее аналитическое продолжение. Покрывает шаблоны выбора контура под тип особенности и асимптотику: рациональные и тригонометрические интегралы, интегралы со степенью или логарифмом на полупрямой, оценки осциллирующих интегралов, главное значение, суммирование рядов через котангенс, продолжение специальных функций.

## Определения

**Голоморфная функция (holomorphic).** Функция $f\colon U \to \mathbb{C}$, дифференцируемая в комплексном смысле в каждой точке открытого $U$. Эквивалентно — представима локально сходящимся степенным рядом.

**Изолированная особенность точки $a$.** Точка, в окрестности которой $f$ голоморфна, кроме самой $a$. Три типа: устранимая (предел конечен), полюс кратности $m$ (разложение содержит конечное число членов с отрицательными степенями, $(z-a)^{-m}$ — самая отрицательная), существенная (бесконечно много отрицательных степеней).

**Ряд Лорана (Laurent series).** Разложение $f(z) = \sum_{n=-\infty}^{\infty} c_n (z-a)^n$ в кольце $r < |z-a| < R$. Главная часть — слагаемые с $n < 0$.

**Вычет (residue).** Коэффициент $c_{-1}$ в ряду Лорана $f$ в точке $a$. Обозначение $\operatorname{Res}_{z=a} f$.

**Точка ветвления и разрез (branch point, branch cut).** Точка, вокруг которой функция (например, $\sqrt{z}$ или $\log z$) не возвращается к исходному значению после полного обхода. Разрез — линия, по которой функция считается «отрезанной», чтобы сделать её однозначной.

**Главная ветвь логарифма.** $\operatorname{Log} z = \ln|z| + i\arg z$ с фиксированным выбором $\arg z$ в полуинтервале длины $2\pi$ (обычно $(-\pi, \pi]$ или $(0, 2\pi)$).

**Индекс контура (winding number).** Целое число $n(\gamma, a) = \dfrac{1}{2\pi i}\oint_\gamma \dfrac{dz}{z-a}$ — сколько раз $\gamma$ обходит точку $a$ против часовой стрелки.

**Главное значение Коши (Cauchy principal value).** Для функции с особенностью в $c \in (a,b)$: $\mathrm{V.p.}\!\int_a^b f = \lim_{\varepsilon \to 0^+}\bigl(\int_a^{c-\varepsilon} + \int_{c+\varepsilon}^b\bigr)$. Для несобственного на $\mathbb{R}$ — симметричный предел $\lim_{R\to\infty}\int_{-R}^R$.

## Записи — Основные

### E1. Теорема о вычетах (residue theorem)

Формулировка. Пусть $\Omega \subset \mathbb{C}$ открыто, $f$ голоморфна в $\Omega$ кроме изолированных особенностей $\{a_k\}$. Пусть $\gamma$ — спрямляемый замкнутый контур в $\Omega$, гомотопный точке внутри $\Omega$, не проходящий через $\{a_k\}$. Тогда
$$\oint_\gamma f(z)\,dz = 2\pi i \sum_k n(\gamma, a_k)\,\operatorname{Res}_{z=a_k} f.$$

Условия применимости.

Контур не задевает особенностей.
Пример поломки: $\oint_{|z|=1}\dfrac{dz}{z-1}$ — полюс лежит на самом контуре, интеграл не определён в обычном смысле.

Особенностей внутри конечное число (или бесконечно, но ряд сходится).
Пример поломки: $f(z) = 1/\sin(\pi z)$ имеет полюсы во всех целых; на квадрате со стороной $2N+1$ их $2N+1$ внутри, и сходимость суммы вычетов надо проверять отдельно.

Идея доказательства. Локально $f$ есть голоморфная часть плюс главная часть Лорана. Голоморфная часть даёт ноль по теореме Коши. От главной части выживает только член $c_{-1}/(z-a_k)$, и его интеграл равен $2\pi i\, c_{-1}$.

---

### E2. Вычисление вычетов

Формулы.

Простой полюс: $\operatorname{Res}_{z=a} f = \lim_{z\to a}(z-a)f(z)$.

Полюс кратности $m$: $\operatorname{Res}_{z=a} f = \dfrac{1}{(m-1)!}\lim_{z\to a}\dfrac{d^{m-1}}{dz^{m-1}}\bigl[(z-a)^m f(z)\bigr]$.

Частное с простым нулём знаменателя ($g(a) \ne 0$, $h(a) = 0$, $h'(a) \ne 0$): $\operatorname{Res}_{z=a}\dfrac{g}{h} = \dfrac{g(a)}{h'(a)}$.

Через ряд Лорана: разложить и взять $c_{-1}$.

Вычет на бесконечности: $\operatorname{Res}_{z=\infty} f = -\operatorname{Res}_{w=0}\dfrac{1}{w^2}f(1/w)$. Для рациональной функции сумма всех вычетов на $\widehat{\mathbb{C}}$ (включая бесконечность) равна нулю.

Условия применимости.

Формула с производной работает только для полюсов, не для существенных особенностей.
Пример поломки: $f(z) = e^{1/z}$ в нуле — никакая степень $z^m f(z)$ не голоморфна, нужен ряд Лорана: $c_{-1} = 1$.

Замечание. При $m \geq 3$ обычно быстрее разложить функцию в ряд напрямую, чем дифференцировать.

### E3. Полукруговой контур для $\int_{-\infty}^\infty R(x)\,dx$

Формулировка. Пусть $R = P/Q$ — рациональная, $\deg Q \geq \deg P + 2$, $Q$ не имеет нулей на $\mathbb{R}$. Тогда
$$\int_{-\infty}^{\infty} R(x)\,dx = 2\pi i \sum_{\operatorname{Im} a_k > 0} \operatorname{Res}_{z=a_k} R.$$

Условия применимости.

Степени отличаются хотя бы на 2.
Пример поломки: $\int_{-\infty}^\infty\dfrac{x}{1+x^2}\,dx$ — разность степеней 1, дуга радиуса $R$ не исчезает. Интеграл существует только как главное значение и равен нулю по симметрии.

Знаменатель не имеет нулей на $\mathbb{R}$.
Пример поломки: $\int_{-\infty}^\infty\dfrac{dx}{x^2-1}$ — полюс на оси, обычная формула неприменима, нужно главное значение и техника E5.

Идея доказательства. Контур — отрезок $[-R, R]$ и верхняя полуокружность. На полуокружности $|R(z)| = O(1/R^2)$, длина $\pi R$, интеграл стремится к нулю. По теореме о вычетах исходный интеграл равен сумме вычетов в верхней полуплоскости.

### E4. Лемма Жордана (Jordan) и $\int R(x) e^{iax}\,dx$

Формулировка леммы. Пусть $f$ непрерывна на верхней полуокружности $C_R$ радиуса $R$ и $M_R = \max_{C_R}|f| \to 0$ при $R \to \infty$. Тогда для $a > 0$
$$\left|\int_{C_R} f(z) e^{iaz}\,dz\right| \leq \dfrac{\pi M_R}{a}.$$

Следствие. Для рациональной $R$ с $\deg Q \geq \deg P + 1$, без нулей $Q$ на оси, и $a > 0$:
$$\int_{-\infty}^{\infty} R(x) e^{iax}\,dx = 2\pi i \sum_{\operatorname{Im} a_k > 0}\operatorname{Res}_{z=a_k}\bigl[R(z)e^{iaz}\bigr].$$

Условия применимости.

Знак $a$ определяет полуплоскость замыкания: $a>0$ — верх, $a<0$ — низ.
Пример поломки: для $\int e^{-ix}/(1+x^2)\,dx$ ($a=-1<0$) замыкание сверху даёт $|e^{iaz}| = e^{|a|y} \to \infty$ на верхней полуокружности — оценка леммы не работает. Замыкать надо снизу.

Идея доказательства. На $C_R$ имеем $|e^{iaz}| = e^{-aR\sin\theta}$. Используется неравенство $\sin\theta \geq 2\theta/\pi$ на $[0, \pi/2]$. Прямая оценка даёт $\pi M_R / a$.

---

### E5. Объезд полюса на контуре и главное значение

Формулировка. Пусть $f$ голоморфна в проколотой окрестности $a$ с простым полюсом и вычетом $r$. Пусть $C_\varepsilon$ — дуга радиуса $\varepsilon$ вокруг $a$, обходимая против часовой стрелки на угол $\alpha$. Тогда
$$\lim_{\varepsilon \to 0}\int_{C_\varepsilon} f(z)\,dz = i\alpha\, r.$$

Следствие для главного значения. Если у $f$ простые полюсы на $\mathbb{R}$ и $f$ удовлетворяет условиям E3 или E4:
$$\mathrm{V.p.}\!\int_{-\infty}^\infty f(x)\,dx = 2\pi i\sum_{\operatorname{Im} a_k > 0}\operatorname{Res}_{z=a_k} f + \pi i \sum_{a_k \in \mathbb{R}}\operatorname{Res}_{z=a_k} f.$$

Условия применимости.

Полюс на контуре должен быть простым.
Пример поломки: $f(z) = 1/z^2$, дуга вокруг нуля. Интеграл $\int_{C_\varepsilon} dz/z^2 = -2i\sin(\alpha)/\varepsilon$ — расходится при $\varepsilon \to 0$. Главное значение в обычном смысле не существует.

Идея доказательства. Локально $f(z) = r/(z-a) + g(z)$, где $g$ голоморфна. Голоморфная часть даёт ноль при $\varepsilon \to 0$ (длина дуги $\to 0$, $g$ ограничена). Главная часть: подстановка $z = a + \varepsilon e^{i\theta}$ даёт $\int r\,id\theta = i\alpha r$.

### E6. Подстановка $z = e^{i\theta}$ для $\int_0^{2\pi} R(\cos\theta, \sin\theta)\,d\theta$

Формулировка. Замена $z = e^{i\theta}$ даёт $dz = iz\,d\theta$, $\cos\theta = (z + z^{-1})/2$, $\sin\theta = (z - z^{-1})/(2i)$. Тогда
$$\int_0^{2\pi} R(\cos\theta,\sin\theta)\,d\theta = \oint_{|z|=1} R\!\left(\dfrac{z + z^{-1}}{2}, \dfrac{z - z^{-1}}{2i}\right)\dfrac{dz}{iz} = 2\pi\!\!\sum_{|a_k| < 1}\!\!\operatorname{Res} \tilde R(z).$$

Условия применимости.

После замены подынтегральная функция не должна иметь полюсов на $|z|=1$.
Пример поломки: $\int_0^{2\pi} d\theta/(1 - 2\cos\theta + 1) = \int_0^{2\pi} d\theta/(2 - 2\cos\theta)$ — расходится в $\theta = 0$, после замены получается полюс в $z=1$ на контуре.

Идея доказательства. Параметризация единичной окружности через $\theta$ и применение теоремы о вычетах.

## Записи — Стандартные

### E7. Контур-«замочная скважина» (keyhole) для $\int_0^\infty x^{s-1} R(x)\,dx$

Формулировка. Пусть $R$ рациональная, без полюсов на $[0, \infty)$, $0 < s < \deg Q - \deg P$. Берём ветвь $z^{s-1} = e^{(s-1)\operatorname{Log} z}$ с разрезом по $[0, \infty)$ и $\arg z \in (0, 2\pi)$. Тогда
$$\int_0^\infty x^{s-1} R(x)\,dx = \dfrac{2\pi i}{1 - e^{2\pi i s}}\sum_{a_k \ne 0}\operatorname{Res}_{z=a_k}\bigl[z^{s-1} R(z)\bigr].$$

Для интегралов с логарифмом $\int_0^\infty R(x) \ln x\,dx$ берётся $R(z) (\log z)^2$ — линейные по $\log$ члены после вычитания берегов дают нужный логарифмический интеграл.

Условия применимости.

Степень $s$ строго между нулём и разностью степеней.
Пример поломки: $s = 1$, $R = 1/(1+x)$ — разность степеней 1, верхняя оценка $s < 1$ нарушена, внешняя окружность не даёт нуля.

Идея доказательства. Контур: большая окружность $|z| = R$, малая окружность $|z| = \varepsilon$ и два берега разреза. На верхнем берегу $z^{s-1} = x^{s-1}$, на нижнем $z^{s-1} = x^{s-1} e^{2\pi i(s-1)}$. Окружности исчезают, остаётся разность берегов с множителем $1 - e^{2\pi i s}$.

---

### E8. Прямоугольный контур для функций с экспоненциальной периодичностью

Формулировка. Если $f$ удовлетворяет $f(z + ih) = c\cdot f(z)$ (например, $f = e^{az}/\cosh z$, период $i\pi$, $c = -e^{ia\pi}$), то контур — прямоугольник с вершинами $\pm R$ и $\pm R + ih$. Боковые стороны исчезают при $R \to \infty$. Верхняя сторона связана с нижней множителем $c$.

Классический пример:
$$\int_{-\infty}^\infty \dfrac{e^{ax}}{\cosh x}\,dx = \dfrac{\pi}{\cos(a\pi/2)}, \qquad |a| < 1.$$

Условия применимости.

Экспоненциальное убывание на $|x| \to \infty$.
Пример поломки: $f(z) = e^{ax}/(1 + e^x)$ при $a \notin (0, 1)$ — при $a \leq 0$ интеграл расходится при $x \to -\infty$, при $a \geq 1$ — при $x \to +\infty$.

Идея доказательства. Сумма по четырём сторонам прямоугольника равна $2\pi i \cdot \operatorname{Res}$. Связь верха и низа даёт линейное уравнение на интеграл.

---

### E9. Секторный контур для $\int_0^\infty R(x^n)\,dx$

Формулировка. Если подынтегральная функция симметрична относительно поворота на $2\pi/n$ (типичный случай $1 + z^n$ в знаменателе), берём контур-сектор с углом $2\pi/n$. Каноничный пример:
$$\int_0^\infty \dfrac{dx}{1 + x^n} = \dfrac{\pi/n}{\sin(\pi/n)}, \qquad n \geq 2.$$

Условия применимости.

Подынтегральная функция возвращается к себе с явным множителем на наклонной стороне.
Пример поломки: $\int_0^\infty \dfrac{dx}{1 + x^3 + x^5}$ — нет симметрии под поворот на $2\pi/n$, секторный контур не даёт уравнения.

Идея доказательства. На наклонной стороне $z = re^{2\pi i/n}$, и $z^n = r^n$ — знаменатель остаётся прежним. Дуга радиуса $R$ исчезает. Внутри сектора ровно один полюс $z = e^{i\pi/n}$.

---

### E10. Суммирование рядов через котангенс

Формулировка. Пусть $f$ мероморфна, без полюсов в целых точках, и $|f(z)| = O(|z|^{-2})$ при $|z| \to \infty$. Тогда
$$\sum_{n \in \mathbb{Z}} f(n) = -\sum_{\text{полюсы } a \text{ функции } f}\operatorname{Res}_{z=a}\bigl[\pi\cot(\pi z)\, f(z)\bigr].$$

Для альтернирующих сумм заменить $\pi\cot(\pi z)$ на $\pi/\sin(\pi z)$:
$$\sum_{n \in \mathbb{Z}}(-1)^n f(n) = -\sum_{\text{полюсы } a \text{ функции } f}\operatorname{Res}_{z=a}\bigl[\pi\csc(\pi z)\, f(z)\bigr].$$

Условия применимости.

Скорость убывания $f$ на квадратах должна побеждать линейный рост длины квадрата.
Пример поломки: $f(z) = 1/z$ — убывание $O(1/|z|)$ слишком слабое. Сумма $\sum 1/n$ расходится, и контурный интеграл не стремится к нулю.

Идея доказательства. На квадратах $C_N$ со сторонами в полу-целых точках $|\cot(\pi z)|$ ограничен константой не зависящей от $N$. Интеграл $\oint_{C_N} \pi\cot(\pi z) f(z)\,dz \to 0$. По теореме о вычетах сумма вычетов в целых точках (равная $\sum f(n)$, так как $\operatorname{Res}_{z=n}\pi\cot(\pi z) = 1$) равна минус сумме вычетов в полюсах $f$.

### E11. Разложения Миттаг-Леффлера (Mittag-Leffler)

Формулировка теоремы. Если $f$ мероморфна с простыми полюсами $\{a_n\}$ ($|a_n| \to \infty$), вычетами $r_n$, и $f$ равномерно ограничена на последовательности контуров $C_N$ растущего размера, то
$$f(z) = f(0) + \sum_{n=1}^\infty r_n\!\left(\dfrac{1}{z - a_n} + \dfrac{1}{a_n}\right).$$

Каноничные разложения:
$$\pi\cot(\pi z) = \dfrac{1}{z} + \sum_{n=1}^\infty \dfrac{2z}{z^2 - n^2},$$
$$\dfrac{\pi}{\sin(\pi z)} = \dfrac{1}{z} + \sum_{n=1}^\infty \dfrac{(-1)^n\, 2z}{z^2 - n^2},$$
$$\dfrac{\pi^2}{\sin^2(\pi z)} = \sum_{n \in \mathbb{Z}}\dfrac{1}{(z-n)^2}.$$

Условия применимости.

Равномерная ограниченность $f$ на $C_N$.
Пример поломки: для $f = \tan(z)$ нельзя брать прямоугольные контуры, на которых $|\tan(z)|$ растёт — нужны квадраты с полу-целыми сторонами или прямоугольники сдвинутые относительно полюсов.

Идея доказательства. Применить теорему о вычетах к $g(w) = f(w)/(w - z)$ на $C_N$. Вычет в $w = z$ даёт $f(z)$, вычет в $w = a_n$ даёт $r_n/(a_n - z)$. Интеграл по $C_N$ стремится к нулю.

## Записи — Редкие

### E12. Контур-«гантель» (dumbbell) для интегралов с разрезом на отрезке

Идея. Для интегралов $\int_a^b R(x)\sqrt{(x-a)(b-x)}\,dx$ берётся ветвь $\sqrt{(z-a)(b-z)}$ с разрезом по $[a, b]$. На двух берегах разреза значения различаются знаком. Контур охватывает отрезок (две почти-прямые линии сверху и снизу разреза, соединённые малыми окружностями вокруг $a$ и $b$). Замыкая большой окружностью на бесконечности и применяя теорему о вычетах, получаем интеграл через вычеты $R$ и вычет на бесконечности.

Пример. $\int_{-1}^1 \dfrac{dx}{(1 + x^2)\sqrt{1 - x^2}} = \dfrac{\pi}{\sqrt{2}}$.

Условия применимости. Разрез не должен задевать полюсы $R$ внутри.

---

### E13. Контур Ханкеля (Hankel) для $\Gamma$ и $B$

Идея. Контур начинается в $-\infty$ чуть выше отрицательной оси, обходит ноль и возвращается в $-\infty$ чуть ниже. Даёт аналитическое продолжение гамма-функции через интегральное представление:
$$\dfrac{1}{\Gamma(s)} = \dfrac{1}{2\pi i}\int_{\mathcal{H}} e^z z^{-s}\,dz, \qquad s \in \mathbb{C}.$$

Сразу следует, что $1/\Gamma$ — целая, и Γ имеет полюсы только в $0, -1, -2, \dots$.

Аналогичная конструкция для бета-функции — контур Похгаммера (Pochhammer) $A B A^{-1} B^{-1}$ вокруг точек $0$ и $1$ даёт продолжение $B(\alpha, \beta)$ на всю плоскость по обоим аргументам.

Замечание. Для $\zeta(s)$ аналогичный контурный интеграл $\zeta(s) = \dfrac{\Gamma(1-s)}{2\pi i}\int_{\mathcal{H}}\dfrac{z^{s-1}}{e^{-z}-1}\,dz$ даёт продолжение и функциональное уравнение.

## Self-test

1. Вычислить $\displaystyle\int_{-\infty}^\infty \dfrac{dx}{(1+x^2)^n}$ при $n \geq 1$.

2. Найти $\displaystyle\int_0^\infty \dfrac{\cos x}{1+x^4}\,dx$.

3. Вычислить $\displaystyle\int_0^\infty \dfrac{\ln x}{(1+x^2)^2}\,dx$.

4. Найти $\displaystyle\int_0^{2\pi}\dfrac{d\theta}{(a + b\cos\theta)^2}$ при $a > |b| > 0$.

5. Вычислить $\displaystyle\sum_{n=1}^\infty \dfrac{1}{n^2 + a^2}$ при $a > 0$.

6. Найти $\displaystyle\int_0^\infty \dfrac{x^{s-1}}{1 + x^3}\,dx$ при $0 < s < 3$.

7. Доказать $\displaystyle\int_{-\infty}^\infty\dfrac{e^{ax}}{1 + e^x}\,dx = \dfrac{\pi}{\sin\pi a}$ при $0 < a < 1$.

## Решения

1. Замыкаем в верхней полуплоскости. Единственный полюс $z = i$ кратности $n$. По формуле для вычета кратного полюса
$$\operatorname{Res}_{z=i}\dfrac{1}{(z+i)^n (z-i)^n} = \dfrac{1}{(n-1)!}\dfrac{d^{n-1}}{dz^{n-1}}\dfrac{1}{(z+i)^n}\bigg|_{z=i}.$$
После вычисления получается $\dfrac{(2n-2)!}{(2i)^{2n-1}\,4^{n-1}((n-1)!)^2}$. Итоговый интеграл равен $\dfrac{\pi}{4^{n-1}}\dbinom{2n-2}{n-1}$.

2. $\displaystyle\int_0^\infty\dfrac{\cos x}{1 + x^4}\,dx = \tfrac{1}{2}\operatorname{Re}\int_{-\infty}^\infty\dfrac{e^{ix}}{1 + x^4}\,dx$. Применяем E4: полюсы в верхней полуплоскости $e^{i\pi/4}$ и $e^{3i\pi/4}$. Складываем вычеты, упрощаем экспоненты через формулу Эйлера. Получаем $\dfrac{\pi}{2\sqrt{2}}\,e^{-1/\sqrt{2}}\bigl(\cos(1/\sqrt{2}) + \sin(1/\sqrt{2})\bigr)$.

3. Используем замочную скважину для $f(z) = (\log z)^2/(1 + z^2)^2$ с разрезом по $[0, \infty)$. На верхнем берегу $\log z = \ln x$, на нижнем $\log z = \ln x + 2\pi i$. Разность даёт $-4\pi i \int_0^\infty\dfrac{\ln x}{(1 + x^2)^2}\,dx + 4\pi^2\int_0^\infty\dfrac{dx}{(1 + x^2)^2}$. Внутри полюсы $\pm i$ кратности 2. Прямое вычисление вычетов и сравнение мнимых частей даёт $-\pi/4$.

4. Подстановка $z = e^{i\theta}$. Знаменатель $a + b\cos\theta = b(z - z_1)(z - z_2)/(2z)$, где $z_{1,2} = (-a \pm \sqrt{a^2 - b^2})/b$. Внутри $|z| = 1$ только $z_1$ (так как $z_1 z_2 = 1$). Интеграл сводится к $\dfrac{4}{ib^2}\oint \dfrac{z\,dz}{(z - z_1)^2 (z - z_2)^2}$. Вычет в полюсе кратности 2:
$$\dfrac{d}{dz}\dfrac{z}{(z - z_2)^2}\bigg|_{z_1} = -\dfrac{z_1 + z_2}{(z_1 - z_2)^3}.$$
Подставляя $z_1 + z_2 = -2a/b$, $z_1 - z_2 = 2\sqrt{a^2 - b^2}/b$, получаем $I = \dfrac{2\pi a}{(a^2 - b^2)^{3/2}}$.

5. По формуле E10 с $f(z) = 1/(z^2 + a^2)$:
$$\sum_{n \in \mathbb{Z}}\dfrac{1}{n^2 + a^2} = -\operatorname{Res}_{z = ia}\dfrac{\pi\cot\pi z}{z^2 + a^2} - \operatorname{Res}_{z = -ia} = \dfrac{\pi\coth\pi a}{a}.$$
Отделяя $n = 0$ и используя симметрию:
$$\sum_{n=1}^\infty\dfrac{1}{n^2 + a^2} = \dfrac{\pi\coth\pi a}{2a} - \dfrac{1}{2a^2}.$$

6. Замочная скважина для $f(z) = z^{s-1}/(1 + z^3)$. Полюсы — корни $z^3 = -1$: $z_k = e^{i\pi(2k+1)/3}$, $k = 0, 1, 2$. Вычет в $z_k$ равен $z_k^{s-1}/(3z_k^2) = z_k^{s-3}/3$. Сумма $z_k^{s-3} = -\bigl(e^{i\pi s/3} + e^{i\pi s} + e^{5i\pi s/3}\bigr)$. После деления на $1 - e^{2\pi i s} = -2i e^{i\pi s}\sin\pi s$ и упрощения через тождество $\sin\pi s = \sin(\pi s/3)(1 + 2\cos(2\pi s/3))$ получаем $\dfrac{\pi/3}{\sin(\pi s/3)}$.

7. Прямоугольник с вершинами $\pm R$ и $\pm R + 2\pi i$. На верхней стороне $z = x + 2\pi i$: знаменатель $1 + e^{x + 2\pi i} = 1 + e^x$, числитель $e^{a(x + 2\pi i)} = e^{2\pi i a}\,e^{ax}$. Боковые стороны экспоненциально малы при $0 < a < 1$. Внутри один полюс $z = i\pi$ с вычетом $-e^{ia\pi}$. Уравнение $(1 - e^{2\pi i a}) I = -2\pi i\, e^{ia\pi}$ даёт $I = \dfrac{\pi}{\sin\pi a}$.
