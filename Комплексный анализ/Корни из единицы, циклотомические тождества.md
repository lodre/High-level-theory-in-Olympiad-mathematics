# Корни из единицы, циклотомические тождества

## Scope
Алгебра и комбинаторика над $\mu_n = \{\zeta : \zeta^n = 1\}$: ортогональные соотношения характеров, фильтр корнями единицы, циклотомические многочлены $\Phi_n$ и их разложение, точечные тождества (произведения, суммы синусов/косинусов), факторизация $a^n - b^n$, кольца $\mathbb{Z}[\zeta_n]$ (особенно $\mathbb{Z}[i]$, $\mathbb{Z}[\omega]$ для $\omega = e^{2\pi i/3}$). Основной target — задачи, где «угадывается» $\zeta = e^{2\pi i / n}$ как способ разложить сумму, доказать иррациональность/неприводимость или свести функциональное уравнение к комбинаторике.

## Записи — Core

### E1. Ортогональность характеров $\mu_n$ (геометрическая сумма по корням единицы)

Формулировка. Пусть $\zeta = e^{2\pi i / n}$. Тогда для целого $a$
$$\sum_{k=0}^{n-1} \zeta^{ak} \;=\; n\cdot[\,n \mid a\,], \qquad \sum_{k=0}^{n-1} \zeta^{(a-b)k} \;=\; n\cdot[\,a\equiv b \pmod n\,].$$
Эквивалентно: $\{1,\zeta,\zeta^2,\ldots,\zeta^{n-1}\}$ образуют orthogonal basis в $\mathbb{C}^n$ относительно скалярного произведения $\langle u,v\rangle = \tfrac1n\sum u_k\overline{v_k}$.

Условия применимости. Ровно $\zeta^n = 1$ и $\zeta \ne 1$ для нетривиального случая. Над $\mathbb{F}_q$ работает аналогично, если $\gcd(n, q) = 1$ и $\mathbb{F}_q$ содержит примитивный корень степени $n$.

Идея доказательства. При $n \nmid a$: $S = \sum \zeta^{ak}$ — геометрическая прогрессия со знаменателем $\zeta^a \ne 1$, сумма $= (\zeta^{an} - 1)/(\zeta^a - 1) = 0$.

Используется в. [Атом для всех нижеследующих записей; IMC-1999-DAY2-2; IMC-2007-5; стандартный прием Putnam].

### E2. Фильтр корнями единицы (roots of unity filter)

Формулировка. Пусть $f(x) = \sum_{k\ge 0} a_k x^k$ — формальный или сходящийся ряд, $\zeta = e^{2\pi i/n}$, $0 \le r < n$. Тогда
$$\sum_{k\equiv r \pmod n} a_k x^k \;=\; \frac{1}{n}\sum_{j=0}^{n-1}\zeta^{-jr}\,f(\zeta^j x), \qquad \sum_{k\equiv r \pmod n} a_k \;=\; \frac{1}{n}\sum_{j=0}^{n-1}\zeta^{-jr}\,f(\zeta^j).$$

Условия применимости. Сходимость $f$ в нужных точках $\zeta^j x$. Для конечных многочленов работает безоговорочно. Часто применяется к $f(x) = (1+x)^N$, к производящим функциям комбинаторных задач, к $(x+x^2+\ldots+x^k)^n$ для задач про кости.

Идея доказательства. Подставить ряд для $f$ и поменять порядок суммирования; внутри получается E1.

Варианты и обобщения.
- **Двумерный фильтр**: для $\sum_{k \equiv r_1, m \equiv r_2}$ — двойная сумма по $\zeta_n^j \cdot \xi_m^l$.
- **Бином**: $\sum_{k\equiv r \bmod n}\binom{N}{k} = \frac{1}{n}\sum_{j=0}^{n-1}\zeta^{-jr}(1+\zeta^j)^N$. Для $n=3$, $r=0$: $\sum_k\binom{N}{3k} = \tfrac13(2^N + 2\cos(N\pi/3))$.
- **С весами $\zeta^{kr}$**: дает дискретное преобразование Фурье на $\mathbb{Z}/n\mathbb{Z}$.

Используется в. [IMC-1999-DAY2-2 (вероятность суммы $n$ кубиков $\equiv 0 \pmod 5$); регулярно в Putnam: B5/2003, A6/1985 — варианты бинома по mod].

### E3. Разложение $x^n - 1 = \prod_{d \mid n} \Phi_d(x)$ и Möbius inversion

Формулировка. $\Phi_n(x) = \prod_{\substack{1\le k\le n \\ \gcd(k,n)=1}}(x - e^{2\pi i k/n}) \in \mathbb{Z}[x]$ — циклотомический (cyclotomic) многочлен степени $\varphi(n)$. Тогда
$$x^n - 1 \;=\; \prod_{d \mid n} \Phi_d(x), \qquad \Phi_n(x) \;=\; \prod_{d \mid n}\bigl(x^d - 1\bigr)^{\mu(n/d)}.$$
Следствие: $\Phi_n(x) \in \mathbb{Z}[x]$ и моник.

Условия применимости. $\Phi_n$ определены для всех $n \ge 1$. Аналог над $\mathbb{F}_q$: $\Phi_n$ распадается на неприводимые степени $\operatorname{ord}_n(q)$ (порядок $q$ в $(\mathbb{Z}/n)^\times$).

Идея доказательства. Корни $x^n - 1$ — все $n$-е корни единицы; группируются по своему «истинному» порядку $d \mid n$. Möbius inversion стандартен.

Варианты и обобщения.
- **Степени**: $x^{2n} - 1 = (x^n - 1)(x^n + 1)$, и $x^n + 1 = \prod_{d \mid 2n,\, d \nmid n}\Phi_d(x)$.
- **Coefficients $\Phi_n$**: для $n$ с $\le 2$ нечётными простыми делителями коэффициенты $\in \{-1, 0, 1\}$; первый «непростой» коэффициент — у $\Phi_{105}$ (равен $-2$).
- **Значения**: $\Phi_n(1) = p$, если $n = p^k$ ($p$ простое, $k\ge 1$); $\Phi_n(1) = 1$ при $n$ с $\ge 2$ различными простыми делителями; $\Phi_1(1) = 0$. Аналогично $\Phi_n(-1)$.

Используется в. [IMC-2001-4 (неприводимость $\Phi_q$ для простого $q$ + оценка через корни); IMC-2010-DAY2-5 (cyclotomic shift kernel: $\gcd_{p \mid N}(x^N - 1)/(x^{N/p} - 1) = \Phi_N$); IMC-2007-5 (идеал в $R[X]$, генерируемый корнями единицы); IMC-2012-5 (substitution + cyclotomic для irreducibility)].

### E4. Точечные тождества: $\prod(x - \zeta^k)$, $\prod\sin(k\pi/n)$

Формулировка. Пусть $\zeta = e^{2\pi i/n}$. Базовые тождества:
$$\prod_{k=0}^{n-1}(x - \zeta^k) = x^n - 1, \qquad \prod_{k=1}^{n-1}(x - \zeta^k) = \frac{x^n - 1}{x - 1} = 1 + x + \cdots + x^{n-1}.$$
Подстановки:
$$\prod_{k=1}^{n-1}(1 - \zeta^k) = n; \qquad \prod_{k=1}^{n-1}\sin\!\frac{k\pi}{n} = \frac{n}{2^{n-1}}; \qquad \prod_{k=1}^{n-1} 2\sin\!\frac{k\pi}{n} = n.$$
Также $\prod_{k=0}^{n-1} 2\sin\!\bigl(\theta + \tfrac{k\pi}{n}\bigr) = 2\sin(n\theta)$ (формула Лагранжа для синусов).

Условия применимости. $n \ge 2$ (при $n=1$ произведение пустое). $|\zeta| = 1$, $\zeta \ne 1$ для нетривиальных множителей. Для $\sin$-формы используется $|1 - e^{i\theta}| = 2|\sin(\theta/2)|$ и аргумент $1 - \zeta^k = -e^{ik\pi/n}\cdot 2i\sin(k\pi/n)$.

Идея доказательства. Делим $x^n - 1$ на $x - 1$, подставляем $x = 1$. Для синусов берём модули и используем тригонометрию.

Варианты и обобщения.
- $\prod_{k=1}^{n-1}(z - \zeta^k) = (z^n - 1)/(z - 1)$ для любого $z \ne 1$.
- $\Phi_n(1) = p^{[n = p^k]}$ — частный случай (см. E3).
- $\sum_{k=1}^{n-1}\frac{1}{1 - \zeta^k} = \frac{n-1}{2}$ — выводится дифференцированием $\log$.
- $\prod_{k=1}^{n-1}(2 - 2\cos(2\pi k/n)) = n^2$.

Используется в. [IMC-2008-DAY2-1 (manipulation корней $g(x) = x^{2k} - x^k + 1$, корни — primitive $6k$-е); классика Putnam — вычисление сумм/произведений тригонометрических функций; instrumental для записи E5].

### E5. Извлечение коэффициентов и оценка многочленов через $\sum P(\zeta^k)$

Формулировка. Пусть $P(x) = \sum_{j=0}^{N} c_j x^j$, $\zeta = e^{2\pi i / n}$. Тогда
$$\frac{1}{n}\sum_{k=0}^{n-1} \zeta^{-rk}\,P(\zeta^k) \;=\; \sum_{j \equiv r \pmod n} c_j.$$
В частности:
- если $\deg P < n$ и $\sum_k P(\zeta^k) = 0$, то $c_0 = 0$ (нет «массы» на показателях, кратных $n$);
- среднее $\tfrac1n \sum |P(\zeta^k)|^2 = \sum_j |c_j|^2$ (Parseval на корнях единицы — точное равенство, дискретный аналог Gutzmer).

Условия применимости. Конечный многочлен (или ряд при $|x| \le 1$ с подходящей сходимостью). Для оценок $\max_{|z|=1}|P(z)|$ через средние по корням — используется только при $n > N$.

Идея доказательства. E1 + перестановка сумм. Для Parseval: $|P(\zeta^k)|^2 = P(\zeta^k)\overline{P(\zeta^k)} = \sum_{i,j} c_i\overline{c_j}\zeta^{(i-j)k}$, усреднение по $k$ убивает $i \ne j$.

Варианты и обобщения.
- **«Дискретный max-modulus»**: $\max_k |P(\zeta^k)| \ge \tfrac1n\sum_k |P(\zeta^k)| \ge |\tfrac1n\sum P(\zeta^k)| = |c_0|$ (при $n > N$).
- Если $P$ моник степени $n$ и $|P(z)| \le M$ на $|z|=1$, то $\bigl|\tfrac1n\sum P(\zeta^k)\bigr| = |c_0|$, а $\bigl|\tfrac1n\sum (P(\zeta^k) - \zeta^{kn})\bigr| = |c_0|$ — основа для аргументов «равенство в неравенстве ⇒ многочлен моном».
- Над $\mathbb{F}_p$: тот же фокус с Frobenius-сопряжёнными корнями.

Используется в. [IMC-2007-6 (averaging unit roots: $|P| \le 2$ на единичной окружности с $a_0 = 1$, форсирует $P(x) = 1 + x^n$ или подобный); IMC-2007-5 (идеал, порождённый corners-of-unity условиями)].

## Записи — Standard

### E6. Иррациональность $\Phi_p$ через Eisenstein на сдвиге $x \to x+1$

Формулировка. Для простого $p$ многочлен $\Phi_p(x) = 1 + x + \cdots + x^{p-1}$ неприводим (irreducible) над $\mathbb{Q}$. Доказательство: $\Phi_p(x+1) = \frac{(x+1)^p - 1}{x} = x^{p-1} + \binom{p}{1}x^{p-2} + \cdots + \binom{p}{p-1}$. Все коэффициенты, кроме старшего, делятся на $p$ (свойство $\binom{p}{k}$), свободный $= p$ не делится на $p^2$ — Eisenstein даёт неприводимость $\Phi_p(x+1)$, а значит и $\Phi_p(x)$.

Условия применимости. $p$ простое. Для $n = p^k$ работает аналогично с $\Phi_{p^k}(x+1)$. Для составных $n$ — нужна общая теорема (E7).

Идея доказательства. Биномиальные коэффициенты $\binom{p}{k}$, $1 \le k \le p-1$ кратны $p$ (числитель содержит $p$, знаменатель — нет). Eisenstein с простым $p$.

Варианты и обобщения.
- **$\Phi_{p^k}$**: $\Phi_{p^k}(x) = \Phi_p(x^{p^{k-1}})$, и $\Phi_{p^k}(x+1)$ снова Eisenstein.
- **$\Phi_n$ для общего $n$** — см. E7.

Используется в. [IMC-2001-4 (неприводимость $\Phi_q$ — ключевой шаг); классика всех NT-олимпиад; шаблон для irreducibility вообще].

### E7. Иррациональность $\Phi_n$ для всех $n$ (Dedekind / Landau)

Формулировка. $\Phi_n(x)$ неприводим над $\mathbb{Q}$ для любого $n \ge 1$.

Условия применимости. Все $n$.

Идея доказательства (Dedekind). Пусть $\zeta$ — примитивный $n$-й корень, $f \in \mathbb{Z}[x]$ — его минимальный многочлен. Достаточно показать: $\zeta^p$ тоже корень $f$ для всякого простого $p \nmid n$. Допустим противное: $f(\zeta^p) \ne 0$. Тогда $\zeta^p$ — корень $g$ из разложения $\Phi_n = fg$, т.е. $\zeta$ — корень $g(x^p)$. Значит $f \mid g(x^p)$ в $\mathbb{Z}[x]$. По модулю $p$: $g(x^p) \equiv g(x)^p \pmod p$ (Frobenius), значит $\bar f \mid \bar g^p$ в $\mathbb{F}_p[x]$, т.е. $\bar f$ и $\bar g$ имеют общий корень. Тогда $\bar f \bar g = \bar\Phi_n \mid x^n - 1$ имеет кратный корень в $\overline{\mathbb{F}_p}$ — противоречие, т.к. $\gcd(x^n-1, n x^{n-1}) = 1$ при $p \nmid n$.

Варианты и обобщения.
- Доказательство Landau / Schur — через resultant и Galois action.
- Над $\mathbb{F}_p$ ($p \nmid n$) $\Phi_n$ распадается на $\varphi(n)/\operatorname{ord}_n(p)$ неприводимых множителей одинаковой степени $\operatorname{ord}_n(p)$.

Используется в. [Фоновый факт для любых задач с $\mathbb{Q}(\zeta_n)$; IMC-2010-DAY2-5 (cyclotomic shift kernel — нужна точная степень $\Phi_N$); инструмент для нижних оценок степени минимального многочлена].

### E8. Циклотомическая факторизация $a^n - b^n$ + теорема Bang–Zsigmondy

Формулировка. $a^n - b^n = \prod_{d \mid n} \Phi_d(a, b)$, где $\Phi_d(a,b) = b^{\varphi(d)}\Phi_d(a/b)$ — однородный циклотомический многочлен. Теорема Zsigmondy: для целых $a > b \ge 1$ с $\gcd(a,b) = 1$ при $n \ge 1$ существует простое $p \mid a^n - b^n$, не делящее $a^k - b^k$ для $k < n$ — за исключением $(a, b, n) = (2, 1, 6)$ и $n = 2$, $a + b$ — степень двойки. Эквивалент: $\Phi_n(a, b)$ имеет «новый» простой делитель почти всегда.

Условия применимости. $a > b \ge 1$, $\gcd(a,b) = 1$. Для $b = 1$: $a > 1$. Исключения как выше.

Идея доказательства. Из E3 + индуктивный анализ. Для Zsigmondy: всякий простой делитель $\Phi_n(a)$ либо «примитивный» (порядок $a$ mod $p$ равен $n$), либо $p \mid n$; оценка $|\Phi_n(a)| > n$ при больших $n$ исключает второй случай.

Варианты и обобщения.
- **Lifting the Exponent (LTE)** для $v_p(a^n - b^n)$ — стандартное следствие при $p \nmid ab$, $p \mid a - b$.
- **Теорема Bang** ($b = 1$) — частный случай.
- $a^n + b^n = \prod_{d \mid 2n,\, d \nmid n}\Phi_d(a,b)$.

Используется в. [IMC-2008-DAY2-1 (анализ $g(x) = x^{2k} - x^k + 1 = \Phi_{6k}(x)/\Phi_{2k}(x)\cdot(\ldots)$ — корни как примитивные $6k$-е); классика NT олимпиад; обоснование «специальных» $n$ в задачах про $a^n - 1$].

### E9. Кольца $\mathbb{Z}[i]$ и $\mathbb{Z}[\zeta_n]$: нормы, факторизация, $\arctan$

Формулировка. $\mathbb{Z}[i] = \{a + bi : a, b \in \mathbb{Z}\}$ — Евклидов с нормой $N(a + bi) = a^2 + b^2$. Простые: рациональное $p \in \mathbb{Z}$ при $p \equiv 3 \pmod 4$; $p = \pi\bar\pi$ при $p \equiv 1 \pmod 4$ (с $\pi = a + bi$, $a^2 + b^2 = p$); $1 + i$ при $p = 2$. Аналогично $\mathbb{Z}[\omega] = \mathbb{Z}[e^{2\pi i / 3}]$ — Евклидов с нормой $N(a + b\omega) = a^2 - ab + b^2$.

Связь с $\arctan$ (Gauss / Stoermer): $\arctan(b/a) = \arg(a + bi)$. Тождества вида $\sum m_k \arctan(b_k/a_k) = m\pi/4 \pmod \pi$ ⇔ $\prod (a_k + b_k i)^{m_k} \in \mathbb{Z}\cdot(1+i)^?$; разложением Gauss-простыми сводится к комбинаторике показателей. Stoermer's theorem: всякое такое тождество (с натуральными $b_k$) сводится к комбинации «базовых» $\arctan(1/n)$ для $n$ — координат Gauss-простых.

Условия применимости. Для $\arctan$-сведения: $a > 0$ (иначе теряем главную ветвь). Для $\mathbb{Z}[\zeta_n]$ Евклидовость только при $n \in \{1,2,3,4,5,7,8,9,11,12,15,16,20,24\}$ — для остальных нужно работать с UFD/идеалами, что в IMC встречается реже.

Идея доказательства. Норма мультипликативна, $|N(a+bi)| < N(c+di)$ обеспечивает деление с остатком — стандартно.

Варианты и обобщения.
- **Машинная формула**: $\pi/4 = 4\arctan(1/5) - \arctan(1/239)$ — через $(5+i)^4 = (1+i)\cdot(239+i)\cdot u$ в $\mathbb{Z}[i]$.
- **Сумма двух квадратов** ($p = a^2 + b^2$ ⇔ $p \equiv 1 \pmod 4$ или $p = 2$).
- **$\mathbb{Z}[\omega]$ для задач с $a^2 + ab + b^2$ или $a^3 + b^3$**.

Используется в. [IMC-2015-4 (сводит арктангенсное тождество к разложению $16+i$ в $\mathbb{Z}[i]$); IMC-2019-5 (norm equation в $\mathbb{Z}[i]$ для $A^4 + 4A^2B^2 + 16B^4$)].

### E10. Манипуляции с $\omega = e^{2\pi i/3}$ ($1+\omega+\omega^2 = 0$)

Формулировка. Пусть $\omega = e^{2\pi i/3}$. Базовое: $\omega^3 = 1$, $1 + \omega + \omega^2 = 0$. Стандартные приёмы:
- $a^3 + b^3 + c^3 - 3abc = (a + b + c)(a + \omega b + \omega^2 c)(a + \omega^2 b + \omega c)$ — факторизация в $\mathbb{Z}[\omega]$.
- Для матриц $A, B \in M_n(\mathbb{R})$: $S = A + \omega B$, $\bar S = A + \omega^2 B$, $S\bar S = A^2 + B^2 + \omega B A + \omega^2 A B$ — позволяет перевести задачу о $\det$ в комплексное условие.
- Извлечение «трети» суммы: $\sum_{k\equiv 0 \bmod 3}\binom{n}{k} = \tfrac13(2^n + 2\cos(n\pi/3))$ (E2 с $n=3$).

Условия применимости. Везде, где есть симметрия порядка 3 или выражения вида $a^2 + ab + b^2 = (a - \omega b)(a - \omega^2 b)$.

Идея доказательства. $1 + \omega + \omega^2 = 0$ + симметричные функции корней.

Варианты и обобщения.
- $\zeta_4 = i$: $a^2 + b^2 = (a+ib)(a-ib)$, $a^4 + 4b^4 = (a^2 + 2b^2 + 2ab)(a^2 + 2b^2 - 2ab)$ (Sophie Germain).
- $\zeta_6 = -\omega^2$: $a^6 + b^6 = (a^2 + b^2)(a^4 - a^2 b^2 + b^4)$.

Используется в. [IMC-1997-3 ($S = A + \omega B$ форсирует $3 \mid n$ через $\det(S\bar S) = \omega^n \det(BA - AB) \in \mathbb{R}$); IMC-2024-1 ($|a|=|b|=1$, $a + b + ab \in \mathbb{R}$ — синус-факторизация близкого типа); классика Putnam].

## Записи — Exotic

### E11. Теорема Кронекера (Kronecker theorem) о алгебраических целых на единичной окружности

Формулировка. Пусть $\alpha$ — алгебраическое целое (algebraic integer), все сопряжённые которого по модулю $\le 1$. Тогда $\alpha = 0$ либо $\alpha$ — корень единицы.

Условия применимости. $\alpha \in \overline{\mathbb{Q}}$, целое над $\mathbb{Z}$ (минимальный многочлен моник, в $\mathbb{Z}[x]$). Все Galois-сопряжённые $\alpha_1 = \alpha, \alpha_2, \ldots, \alpha_d$ удовлетворяют $|\alpha_i| \le 1$.

Идея доказательства. Степени $\alpha^k$ — тоже алгебраические целые той же степени с $|\alpha_i^k| \le 1$. Их минимальные многочлены имеют ограниченные коэффициенты (по теореме Виета: $|e_j(\alpha_1^k,\ldots,\alpha_d^k)| \le \binom{d}{j}$). Следовательно, таких многочленов конечно много — найдутся $k < l$ с $\alpha^k$ и $\alpha^l$ сопряжёнными, откуда $\alpha^{l-k} = $ корень единицы (или $\alpha = 0$).

Варианты и обобщения.
- **Mahler measure**: $M(\alpha) = \prod \max(1, |\alpha_i|)$; $M(\alpha) = 1$ ⇔ Kronecker. Lehmer's problem — открытый вопрос про нижнюю границу $M(\alpha) > 1$.
- **Salem / Pisot numbers** — следующая «жёсткость» сразу за корнями единицы.

Используется в. [Аргументы об «отсутствии нетривиальных алгебраических целых модуля 1»; Miklós Schweitzer и алгебраические подзадачи IMC; полезен для контрпримеров и оценок снизу на степень минимального многочлена].

### E12. Гауссовы суммы (Gauss sums) для квадратичных характеров

Формулировка. Для нечётного простого $p$ положим $g = \sum_{k=1}^{p-1}\left(\tfrac{k}{p}\right)\zeta_p^k$, где $\left(\tfrac{\cdot}{p}\right)$ — символ Лежандра. Тогда
$$g^2 = \left(\tfrac{-1}{p}\right) p, \quad\text{т.е. } g = \begin{cases}\sqrt{p}, & p \equiv 1 \pmod 4,\\ i\sqrt{p}, & p \equiv 3 \pmod 4.\end{cases}$$
(Знак — теорема Гаусса, нетривиален.)

Условия применимости. $p$ нечётное простое. Аналог для общего модуля $n$ — обобщённые суммы Гаусса.

Идея доказательства. $g \cdot \bar g = \sum_{a, b}\left(\tfrac{ab}{p}\right)\zeta_p^{a-b} = $ диагональная сумма + cross-terms; используется E1.

Варианты и обобщения.
- **Квадратичная взаимность** через Гауссовы суммы (Eisenstein).
- **Gauss sums по любому характеру** $\chi$: $|g(\chi)|^2 = p$ для нетривиального $\chi$ mod $p$.
- $\sum_{k=0}^{p-1}\zeta_p^{k^2} = g + 1 - $ (выражается через $g$).

Используется в. [Нечасто в IMC напрямую, но возникает в задачах про $\sum \zeta^{k^2}$, в задачах с квадратичными вычетами, в Miklós Schweitzer; полезный «следующий слой» сверх корней единицы].

### E13. Substitution $Y = X(X+a)$ + cyclotomic для irreducibility

Формулировка. Если $g(Y) \in \mathbb{Q}[Y]$ неприводим и $h(X) = X(X+a)$ — биекция $\mathbb{C} \to \mathbb{C}$ с разветвлением (degree 2 cover), то $f(X) = g(h(X))$ может быть неприводим в $\mathbb{Q}[X]$. Стандартный пример: $f(X) = (X(X+a))^{2^n} + 1 = \Phi_{2^{n+1}}(X(X+a))$ — неприводим в $\mathbb{Q}[X]$.

Условия применимости. $g$ — циклотомический $\Phi_{2^{n+1}}(Y) = Y^{2^n} + 1$ (или общий $\Phi_m$ при $\gcd$-условиях). $a \in \mathbb{Q}$, надо отдельно проверять, что $X(X+a) - \beta$ неприводим над $\mathbb{Q}(\beta)$ для корня $\beta$ многочлена $g$ — тогда $f$ неприводим.

Идея доказательства. Если $f = uv$ нетривиально, то корни $f$ распадаются на 2 группы; но корни $f$ — это $\{X : X(X+a) = \zeta_{2^{n+1}}^{2j+1}\}$, и quadratic над circle of unity не дробится «правильно».

Варианты и обобщения.
- **Общий принцип**: если $\beta \in \overline{\mathbb{Q}}$ имеет степень $d$, и $X(X+a) - \beta$ степени 2 неприводим над $\mathbb{Q}(\beta)$, то $\deg_\mathbb{Q} \alpha = 2d$ для корня $\alpha$ — даёт $f$ неприводим.

Используется в. [IMC-2012-5 ($f(X) = X^{2^n}(X+a)^{2^n}+1$ — точно эта подстановка)].

## Self-test

1. Найти $\sum_{k=0}^{\lfloor n/3 \rfloor}\binom{n}{3k}$ в замкнутой форме.

2. Доказать, что $\displaystyle\prod_{k=1}^{n-1}\sin\frac{k\pi}{n} = \frac{n}{2^{n-1}}$.

3. При каких целых $n \ge 2$ многочлен $1 + x + x^2 + \cdots + x^{n-1}$ делится на $1 + x + x^2$?

4. Пусть $|a| = |b| = 1$, $a + b \in \mathbb{R}$. Описать множество таких пар $(a, b)$.

5. Пусть $f(x) = x^n + a_{n-1}x^{n-1} + \cdots + a_0 \in \mathbb{Z}[x]$ моник, и все корни $f$ лежат в замкнутом единичном круге $|z| \le 1$. Доказать, что каждый корень — либо ноль, либо корень из единицы.

## Решения

1. Применяем E2 к $f(x) = (1+x)^n$ с $n = 3$, $r = 0$ (фильтр):
$$\sum_{k\equiv 0 \pmod 3}\binom{n}{k} = \frac{1}{3}\bigl((1+1)^n + (1+\omega)^n + (1+\omega^2)^n\bigr).$$
$1 + \omega = -\omega^2 = e^{i\pi/3}$, $1 + \omega^2 = -\omega = e^{-i\pi/3}$, поэтому $(1+\omega)^n + (1+\omega^2)^n = 2\cos(n\pi/3)$. Ответ: $\tfrac13(2^n + 2\cos(n\pi/3))$.

2. По E4: $\prod_{k=1}^{n-1}(1 - \zeta^k) = n$ для $\zeta = e^{2\pi i/n}$. С другой стороны,
$$1 - e^{2\pi i k/n} = -e^{i\pi k/n}\cdot 2i\sin(k\pi/n) = -2i e^{i\pi k/n}\sin(k\pi/n).$$
Произведение модулей: $\prod_{k=1}^{n-1}|1 - \zeta^k| = 2^{n-1}\prod_{k=1}^{n-1}\sin(k\pi/n) = n$ (поскольку все $\sin > 0$). Делим — ответ.

3. $1 + x + \cdots + x^{n-1} = (x^n - 1)/(x - 1)$. $1 + x + x^2 = \Phi_3(x) = (x^3 - 1)/(x - 1)$. По E3, $\Phi_3(x) \mid (x^n - 1)$ ⇔ $3 \mid n$. Тогда $\Phi_3 \mid (x^n - 1)/(x-1)$ автоматически (т.к. $\Phi_3$ и $x-1$ взаимно просты как множители $x^n-1$). Ответ: $3 \mid n$.

4. $a + b \in \mathbb{R}$ ⇔ $\overline{a+b} = a+b$ ⇔ $\bar a + \bar b = a + b$ ⇔ $1/a + 1/b = a + b$ (т.к. $|a|=|b|=1$) ⇔ $a + b = (a + b)/(ab)$. Либо $a + b = 0$ (т.е. $b = -a$), либо $ab = 1$ (т.е. $b = \bar a$). Итого: $\{(a, -a) : |a|=1\} \cup \{(a, \bar a) : |a|=1\}$ — две одномерные семьи.

5. По E11 (Kronecker): $f$ моник в $\mathbb{Z}[x]$ ⇒ его корни — алгебраические целые. Если все корни в замкнутом единичном круге, теорема даёт: каждый корень либо $0$, либо корень единицы. (Альтернативно: сопряжённые корни тоже в круге, степени $\alpha^k$ — алгебраические целые с ограниченными элементарными симметрическими функциями ⇒ конечно много возможных минимальных многочленов ⇒ найдутся $k < l$ с $\alpha^l = \alpha^k$.)
