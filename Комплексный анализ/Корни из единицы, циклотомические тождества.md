# Корни из единицы, циклотомические тождества

## Scope

Подтема покрывает алгебру и комбинаторику над группой $\mu_n$ всех $n$-х корней единицы. Сюда входят суммы и произведения по $\mu_n$, циклотомические многочлены, факторизация $a^n - b^n$, а также кольца целых $\mathbb{Z}[\zeta_n]$ (особенно $\mathbb{Z}[i]$ и $\mathbb{Z}[\omega]$). Основной приём — заменить целочисленную или комбинаторную задачу на тождество в $\mathbb{C}$, где симметрия $\zeta \mapsto \zeta^k$ даёт сокращения.

## Определения

**Корень из единицы.** Число $\zeta \in \mathbb{C}$ называется $n$-м корнем из единицы, если $\zeta^n = 1$. Множество всех таких корней обозначаем $\mu_n$.

**Примитивный корень.** Корень $\zeta \in \mu_n$ примитивный (primitive), если его порядок в группе равен ровно $n$. Стандартный примитивный корень: $\zeta_n = e^{2\pi i / n}$.

**Функция Эйлера $\varphi(n)$.** Число целых $k \in [1, n]$ с $\gcd(k, n) = 1$. Совпадает с числом примитивных корней степени $n$.

**Функция Мёбиуса $\mu(n)$.** $\mu(1) = 1$; $\mu(n) = 0$ если $n$ делится на квадрат простого; $\mu(p_1 \cdots p_k) = (-1)^k$ для различных простых $p_i$.

**Циклотомический многочлен.**
$$\Phi_n(x) \;=\; \prod_{\substack{1 \le k \le n \\ \gcd(k, n) = 1}} \bigl(x - e^{2\pi i k / n}\bigr).$$
Это моник в $\mathbb{Z}[x]$ степени $\varphi(n)$.

**Гауссовы целые.** $\mathbb{Z}[i] = \{a + bi : a, b \in \mathbb{Z}\}$. Норма: $N(a + bi) = a^2 + b^2$.

**Кольцо $\mathbb{Z}[\omega]$.** Для $\omega = e^{2\pi i / 3}$: $\mathbb{Z}[\omega] = \{a + b\omega : a, b \in \mathbb{Z}\}$. Норма: $N(a + b\omega) = a^2 - ab + b^2$.

**Алгебраическое целое.** Число $\alpha \in \mathbb{C}$, являющееся корнем моника с целыми коэффициентами.

**Символ Лежандра $\bigl(\tfrac{a}{p}\bigr)$.** Для нечётного простого $p$: $+1$, если $a$ — ненулевой квадратичный вычет mod $p$; $-1$, если невычет; $0$, если $p \mid a$.

## Записи — Основные

### E1. Ортогональность характеров $\mu_n$

Формулировка. Пусть $\zeta = e^{2\pi i / n}$ и $a \in \mathbb{Z}$. Тогда
$$\sum_{k = 0}^{n - 1} \zeta^{a k} \;=\; \begin{cases} n, & n \mid a, \\ 0, & n \nmid a. \end{cases}$$

Условие применимости. В нетривиальном случае нужно $\zeta^a \ne 1$.

Пример нарушения. При $a = 0$ (или $n \mid a$) знаменатель геометрической прогрессии $\zeta^a - 1$ равен нулю, и формула $0$ ложна — сумма равна $n$.

Идея доказательства. При $\zeta^a \ne 1$ это геометрическая прогрессия со знаменателем $\zeta^a$ и суммой $(\zeta^{a n} - 1)/(\zeta^a - 1) = 0$.

---

### E2. Фильтр корнями единицы (roots of unity filter)

Формулировка. Пусть $f(x) = \sum_{k \ge 0} a_k x^k$ и $\zeta = e^{2\pi i / n}$, $0 \le r < n$. Тогда
$$\sum_{k \equiv r \,(n)} a_k \;=\; \frac{1}{n} \sum_{j = 0}^{n - 1} \zeta^{-j r} \, f(\zeta^j).$$

Условие применимости. Все суммы $f(\zeta^j)$ должны быть определены. Для многочлена выполняется автоматически.

Пример нарушения. Если $f(x) = \sum 1 \cdot x^k$ — расходящийся ряд, $f(\zeta^j)$ не имеет смысла, формула неприменима.

Идея доказательства. Подставить ряд для $f$ в правую часть, поменять порядок суммирования; внутренняя сумма по $j$ есть E1 с параметром $a = k - r$.

Используемое следствие для бинома. Для $f(x) = (1 + x)^N$:
$$\sum_{k \equiv r \,(n)} \binom{N}{k} \;=\; \frac{1}{n} \sum_{j = 0}^{n - 1} \zeta^{-j r} (1 + \zeta^j)^N.$$
В частности, при $n = 3$, $r = 0$: $\sum_k \binom{N}{3k} = \tfrac{1}{3}\bigl(2^N + 2 \cos(N \pi / 3)\bigr)$.

---

### E3. Разложение $x^n - 1 = \prod_{d \mid n} \Phi_d$ и обращение Мёбиуса

Формулировка.
$$x^n - 1 \;=\; \prod_{d \mid n} \Phi_d(x), \qquad \Phi_n(x) \;=\; \prod_{d \mid n} (x^d - 1)^{\mu(n / d)}.$$

Условие применимости. Тождества верны над любым полем характеристики $0$. Над $\mathbb{F}_p$ нужно $p \nmid n$; иначе $x^n - 1$ имеет кратные корни.

Пример нарушения. Над $\mathbb{F}_p$ при $p \mid n$ многочлен $x^p - 1 = (x - 1)^p$, и разложения на $\Phi_d$ в обычном смысле нет.

Идея доказательства. Корни $x^n - 1$ — все $n$-е корни единицы. Каждый имеет ровно один «истинный» порядок $d \mid n$ и попадает в $\Phi_d$. Обращение Мёбиуса применяется к логарифму обеих частей.

Полезные значения. $\Phi_n(1) = p$, если $n = p^k$ для простого $p$ и $k \ge 1$; $\Phi_n(1) = 1$, если у $n$ хотя бы два различных простых делителя; $\Phi_1(1) = 0$.

---

### E4. Точечные тождества: произведения по $\mu_n$

Формулировка. Для $\zeta = e^{2\pi i / n}$ и $n \ge 2$:
$$\prod_{k = 1}^{n - 1} (x - \zeta^k) \;=\; \frac{x^n - 1}{x - 1} \;=\; 1 + x + \cdots + x^{n - 1}.$$
Отсюда два следствия (подстановкой $x = 1$ и взятием модулей):
$$\prod_{k = 1}^{n - 1} (1 - \zeta^k) \;=\; n, \qquad \prod_{k = 1}^{n - 1} \sin \frac{k \pi}{n} \;=\; \frac{n}{2^{n - 1}}.$$

Условие применимости. $n \ge 2$ (при $n = 1$ произведение пустое).

Идея доказательства. Делим $x^n - 1$ на $x - 1$ и подставляем $x = 1$. Для синусов используем равенство $|1 - e^{i \theta}| = 2 |\sin(\theta / 2)|$ и положительность $\sin(k \pi / n)$ при $1 \le k \le n - 1$.

Формула Лагранжа. Обобщает $\prod \sin$:
$$\prod_{k = 0}^{n - 1} 2 \sin\!\Bigl(\theta + \frac{k \pi}{n}\Bigr) \;=\; 2 \sin(n \theta).$$

---

### E5. Извлечение коэффициентов через $\sum P(\zeta^k)$

Формулировка. Пусть $P(x) = \sum_{j = 0}^{N} c_j x^j$ и $\zeta = e^{2 \pi i / n}$ с $n > N$. Тогда для любого $0 \le r \le N$:
$$c_r \;=\; \frac{1}{n} \sum_{k = 0}^{n - 1} \zeta^{-r k} \, P(\zeta^k).$$
Дискретный аналог равенства Парсеваля:
$$\frac{1}{n} \sum_{k = 0}^{n - 1} |P(\zeta^k)|^2 \;=\; \sum_{j = 0}^{N} |c_j|^2.$$

Условие применимости. Для точного извлечения нужно $n > \deg P$. Без этого «алиасы» $j \equiv r \,(n)$ суммируются вместе.

Пример нарушения. $P(x) = 1 + x^3$, $n = 3$, $\zeta^3 = 1$, $r = 0$: формула даёт $\tfrac{1}{3} \sum (1 + 1) = 2$, а $c_0 = 1$. Лишняя единица — вклад $c_3$.

Идея доказательства. Подставить $P$ в правую часть и применить E1. Для Парсеваля: раскрыть $|P(\zeta^k)|^2 = \sum_{i, j} c_i \overline{c_j} \zeta^{(i - j) k}$ и усреднить по $k$.

Следствие для оценок. Если $P$ моник степени $n$ и $|P(z)| \le M$ на $|z| = 1$, то усреднение $\tfrac{1}{n} \sum P(\zeta^k) = c_0$ выдаёт точное значение свободного члена. Равенство $|c_0| = M$ форсирует $P(x) = c_0 + x^n$ или подобный «мономиальный» вид.

## Записи — Стандартные

### E6. Неприводимость $\Phi_p$ через Эйзенштейна на сдвиге $x \to x + 1$

Формулировка. Для простого $p$ многочлен $\Phi_p(x) = 1 + x + \cdots + x^{p - 1}$ неприводим над $\mathbb{Q}$.

Идея доказательства. Подстановка $x \to x + 1$ даёт
$$\Phi_p(x + 1) \;=\; \frac{(x + 1)^p - 1}{x} \;=\; x^{p - 1} + \binom{p}{1} x^{p - 2} + \cdots + \binom{p}{p - 1}.$$
Все коэффициенты, кроме старшего, делятся на $p$; свободный член равен $p$ и не делится на $p^2$. Эйзенштейн с простым $p$ даёт неприводимость $\Phi_p(x + 1)$, а значит и $\Phi_p(x)$.

Условие применимости. $p$ — простое.

Пример нарушения. Для составного $n = 4$ сдвинутый $\Phi_4(x + 1) = x^2 + 2x + 2$ ещё «случайно» удовлетворяет Эйзенштейну с $p = 2$. Для $n = 6$: $\Phi_6(x + 1) = x^2 + x + 1$ не удовлетворяет ни одному критерию Эйзенштейна напрямую. Нужна общая теорема (E7).

---

### E7. Неприводимость $\Phi_n$ для всех $n$ (Дедекинд)

Формулировка. $\Phi_n(x)$ неприводим над $\mathbb{Q}$ для любого $n \ge 1$.

Идея доказательства. Пусть $\zeta$ примитивный, $f \in \mathbb{Z}[x]$ его минимальный многочлен. Достаточно показать, что $\zeta^p$ — тоже корень $f$ для любого простого $p \nmid n$. Если нет, то $\zeta^p$ — корень другого множителя $g$ в разложении $\Phi_n = f g$. Тогда $\zeta$ — корень $g(x^p)$, откуда $f \mid g(x^p)$ в $\mathbb{Z}[x]$. По модулю $p$: $g(x^p) \equiv g(x)^p$ по теореме Ферма для $\mathbb{F}_p[x]$. Получаем $\bar f \mid \bar g^p$, то есть $\bar f$ и $\bar g$ имеют общий неприводимый делитель. Тогда $x^n - 1$ имеет кратный корень в $\overline{\mathbb{F}_p}$. Противоречие, поскольку $p \nmid n$ даёт $\gcd(x^n - 1, n x^{n - 1}) = 1$.

Условие применимости. Простое $p$, не делящее $n$, всегда существует — например, $p > n$.

---

### E8. Циклотомическая факторизация $a^n - b^n$ и теорема Жигмонди

Формулировка.
$$a^n - b^n \;=\; \prod_{d \mid n} \Phi_d(a, b), \qquad \Phi_d(a, b) \;=\; b^{\varphi(d)} \Phi_d(a / b).$$
Теорема Жигмонди (Zsigmondy). Пусть $a > b \ge 1$ целые, $\gcd(a, b) = 1$. Тогда существует простое $q \mid a^n - b^n$, не делящее $a^k - b^k$ ни для какого $1 \le k < n$. Исключения:
- $n = 1$;
- $n = 2$ и $a + b$ — степень двойки;
- $(a, b, n) = (2, 1, 6)$.

Условие применимости. $\gcd(a, b) = 1$.

Пример нарушения. При $a = b = 2$: $a^n - b^n = 0$ для всех $n$, никакого «нового простого» не появляется.

Идея доказательства. Однородная версия E3 даёт первое тождество. Для Жигмонди: всякий простой делитель $\Phi_n(a, b)$ либо примитивный (порядок $a / b$ mod $p$ равен $n$), либо делит $n$. Оценка $|\Phi_n(a, b)| > n$ при больших $n$ исключает второй случай вне списка исключений.

---

### E9. Кольцо Гауссовых целых $\mathbb{Z}[i]$: нормы и арктангенсы

Формулировка. $\mathbb{Z}[i]$ — евклидово кольцо с нормой $N(a + b i) = a^2 + b^2$. Простые элементы:
- $1 + i$ (с $N = 2$);
- рациональное простое $p$, если $p \equiv 3 \,(4)$;
- $\pi$ и $\bar\pi$ с $\pi \bar\pi = p$, если $p \equiv 1 \,(4)$.

Связь с арктангенсом: $\arctan(b / a) = \arg(a + b i)$. Тождество вида $\sum m_k \arctan(b_k / a_k) \equiv \theta \,(\pi)$ эквивалентно равенству $\prod (a_k + b_k i)^{m_k} = r \cdot e^{i \theta}$ для некоторого $r > 0$.

Условие применимости. Для арктангенс-сведения нужно $a > 0$ в каждом сомножителе. Иначе теряем главную ветвь $\arg$.

Пример нарушения. $\arctan(1 / (-1)) = -\pi / 4$, но $\arg(-1 + i) = 3\pi / 4$ — расхождение на $\pi$ ломает прямой перевод.

Идея доказательства. Норма мультипликативна. Деление с остатком: для $\alpha, \beta \in \mathbb{Z}[i]$ с $\beta \ne 0$ берём ближайший к $\alpha / \beta \in \mathbb{Q}(i)$ Гауссов целый — остаток имеет норму строго меньше $N(\beta)$.

Пример работы. Формула Машина $\pi / 4 = 4 \arctan(1 / 5) - \arctan(1 / 239)$ соответствует $(5 + i)^4 (239 - i) = 2^2 \cdot (1 + i)^?\cdot \ldots$ — проверяется разложением на Гауссовы простые.

### E10. Кубический корень из единицы $\omega = e^{2\pi i / 3}$

Формулировка. $\omega^3 = 1$, $1 + \omega + \omega^2 = 0$. Базовые тождества:
$$a^2 + a b + b^2 \;=\; (a - \omega b)(a - \omega^2 b);$$
$$a^3 + b^3 + c^3 - 3 a b c \;=\; (a + b + c)(a + \omega b + \omega^2 c)(a + \omega^2 b + \omega c).$$
Матричный приём: для $A, B \in M_n(\mathbb{R})$ положим $S = A + \omega B$. Тогда $\det(S) \cdot \overline{\det(S)} \in \mathbb{R}$, а сам $\det(S) \in \mathbb{Z}[\omega]$ — это часто форсирует делимость размерности $n$ на $3$.

Условие применимости. $A$ и $B$ должны быть вещественными для гарантии $\overline{S} = A + \omega^2 B$.

Пример нарушения. Если $B$ комплексная, то $\overline{S} \ne A + \omega^2 B$, и трюк не работает.

Идея доказательства. Тождества проверяются прямым раскрытием с $\omega^2 + \omega + 1 = 0$. Матричный приём: $\det(S \bar S) \in \mathbb{R}$, но $\det(S \bar S)$ выражается через $A, B$ с множителем $\omega^n$ — получаем $\omega^n \in \mathbb{R}$, то есть $3 \mid n$.

## Записи — Редкие

### E11. Теорема Кронекера

Формулировка. Пусть $\alpha$ — алгебраическое целое. Если все его сопряжённые $\alpha_1, \ldots, \alpha_d$ удовлетворяют $|\alpha_i| \le 1$, то $\alpha = 0$ или $\alpha$ — корень единицы.

Идея доказательства. Степени $\alpha^k$ — тоже алгебраические целые степени $\le d$ с сопряжёнными в единичном круге. Их симметрические функции ограничены биномиальными коэффициентами. Возможных минимальных многочленов конечно много. Найдутся $k < l$ с одинаковым минимальным многочленом, откуда $\alpha^l$ сопряжён с $\alpha^k$, и $\alpha^{l - k}$ — корень единицы.

Условие применимости. $\alpha$ должно быть именно целым, не просто алгебраическим.

Пример нарушения. $\alpha = 1/2$ — алгебраическое, $|\alpha| < 1$, но не корень единицы. Минимальный многочлен $2x - 1$ не моник, $\alpha$ не алгебраическое целое.

### E12. Гауссовы суммы для квадратичных характеров

Формулировка. Для нечётного простого $p$ и $\zeta_p = e^{2 \pi i / p}$:
$$g \;=\; \sum_{k = 1}^{p - 1} \Bigl(\frac{k}{p}\Bigr) \zeta_p^k, \qquad g^2 \;=\; \Bigl(\frac{-1}{p}\Bigr) p.$$
Теорема Гаусса о знаке: $g = \sqrt{p}$ при $p \equiv 1 \,(4)$ и $g = i \sqrt{p}$ при $p \equiv 3 \,(4)$.

Идея доказательства. $g \cdot \bar g = \sum_{a, b} \bigl(\tfrac{a b}{p}\bigr) \zeta_p^{a - b}$. Замена $b \to a b$ и применение E1 даёт $|g|^2 = p$. Знак — более тонкий аргумент через тэта-функцию или Дедекиндову сумму.

Условие применимости. $p$ — нечётное простое.

Пример нарушения. Для составного $n$ обобщённая сумма $\sum \chi(k) \zeta_n^k$ удовлетворяет $|g|^2 = n$ только для примитивных характеров; для непримитивных $g$ может обнуляться.

## Self-test

1. Найти $\displaystyle \sum_{k = 0}^{\lfloor n / 3 \rfloor} \binom{n}{3 k}$ в замкнутой форме.

2. Доказать, что $\displaystyle \prod_{k = 1}^{n - 1} \sin \frac{k \pi}{n} = \frac{n}{2^{n - 1}}$.

3. При каких целых $n \ge 2$ многочлен $1 + x + x^2 + \cdots + x^{n - 1}$ делится на $1 + x + x^2$?

4. Пусть $|a| = |b| = 1$ и $a + b \in \mathbb{R}$. Описать все такие пары $(a, b)$.

5. Пусть $f(x) \in \mathbb{Z}[x]$ — моник степени $n$, и все корни $f$ лежат в замкнутом единичном круге $|z| \le 1$. Доказать, что каждый корень $f$ — либо ноль, либо корень из единицы.

---

## Решения

1. По E2 с $n = 3$, $r = 0$ применённому к $f(x) = (1 + x)^N$:
$$\sum_{k \equiv 0 \,(3)} \binom{N}{k} \;=\; \frac{1}{3} \Bigl(2^N + (1 + \omega)^N + (1 + \omega^2)^N\Bigr).$$
Считаем $1 + \omega = e^{i \pi / 3}$ и $1 + \omega^2 = e^{-i \pi / 3}$. Тогда сумма двух последних степеней даёт $2 \cos(N \pi / 3)$. Итог:
$$\sum_{k \equiv 0 \,(3)} \binom{N}{k} \;=\; \tfrac{1}{3} \bigl(2^N + 2 \cos(N \pi / 3)\bigr).$$

2. По E4: $\prod_{k = 1}^{n - 1}(1 - \zeta^k) = n$. Раскладываем $1 - e^{2 \pi i k / n} = -2 i \, e^{i k \pi / n} \sin(k \pi / n)$. Берём модули:
$$\prod_{k = 1}^{n - 1} |1 - \zeta^k| \;=\; 2^{n - 1} \prod_{k = 1}^{n - 1} \sin(k \pi / n) \;=\; n.$$
Делим на $2^{n - 1}$.

3. $1 + x + x^2 = \Phi_3(x)$ и $1 + x + \cdots + x^{n - 1} = (x^n - 1) / (x - 1)$. По E3: $\Phi_3 \mid x^n - 1$ тогда и только тогда, когда $3 \mid n$. При $3 \mid n$ имеем $\Phi_3(1) = 3 \ne 0$, то есть $\Phi_3$ и $x - 1$ взаимно просты. Значит $\Phi_3 \mid (x^n - 1)/(x - 1)$. При $3 \nmid n$ деление невозможно. Ответ: $3 \mid n$.

4. Условие $a + b \in \mathbb{R}$ означает $\overline{a + b} = a + b$. Используя $\bar a = 1 / a$ и $\bar b = 1 / b$ (так как $|a| = |b| = 1$), получаем
$$\frac{1}{a} + \frac{1}{b} \;=\; a + b \;\Longleftrightarrow\; \frac{a + b}{a b} \;=\; a + b \;\Longleftrightarrow\; (a + b)(1 - a b) \;=\; 0.$$
Либо $b = -a$ (с $|a| = 1$), либо $a b = 1$, то есть $b = \bar a$. Описание: $\{(a, -a) : |a| = 1\} \cup \{(a, \bar a) : |a| = 1\}$.

5. Все корни $f$ лежат в $\mathbb{Z}[x]$-моник, значит они алгебраические целые. По условию все они в замкнутом единичном круге. Сопряжённые к корню $\alpha$ — тоже корни $f$ (или корни делителя минимального многочлена $\alpha$), то есть тоже в круге. По E11 каждый корень $\alpha$ либо нуль, либо корень единицы.
