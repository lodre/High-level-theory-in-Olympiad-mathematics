# Lagrange multipliers, условные экстремумы

## Scope
Условные экстремумы гладких функций на multimanifold $M = \{g_1 = \ldots = g_m = 0\} \subset \mathbb{R}^n$ и при смешанных ограничениях (равенства + неравенства): корректная формулировка Lagrange multiplier rule с constraint qualification, классификация критических точек через bordered Hessian, KKT и complementary slackness, обходы через касательное пространство и implicit function theorem, симметризационные и smoothing-аргументы для inequality proofs, экстремальные точки и Carathéodory для линейных функционалов на выпуклом компакте, а также вырожденный случай Fritz John и алгебраическое исключение множителя через resultant.

## Записи — Core

### E1. Lagrange multiplier theorem (теорема Лагранжа об условном экстремуме)

Формулировка. Пусть $f, g_1, \ldots, g_m : U \subset \mathbb{R}^n \to \mathbb{R}$ — функции класса $C^1$, $m < n$, $a \in M = \{g_1 = \ldots = g_m = 0\}$ — точка локального экстремума $f|_M$. Если в точке $a$ выполнено **LICQ** (linear independence constraint qualification — линейная независимость градиентов активных ограничений), то есть $\nabla g_1(a), \ldots, \nabla g_m(a)$ линейно независимы, то существуют $\lambda_1, \ldots, \lambda_m \in \mathbb{R}$ (множители Лагранжа), такие что
$$\nabla f(a) = \sum_{i=1}^{m} \lambda_i \nabla g_i(a).$$
Эквивалентная форма: $a$ — критическая точка функции Лагранжа $L(x, \lambda) = f(x) - \sum_i \lambda_i g_i(x)$ по всем переменным $(x, \lambda)$.

Условия применимости. LICQ — главное и часто опускаемое условие. Без него заключение может рухнуть. Канонический контрпример: $f(x, y) = x$, $g(x, y) = x^3 - y^2$. На $M = \{g = 0\}$ имеем $x = y^{2/3} \ge 0$, и $a = (0, 0)$ — глобальный минимум. Но $\nabla g(0, 0) = (0, 0)$, поэтому $\nabla f = (1, 0) \ne \lambda \cdot 0$ — представления нет. Точка $a$ — **abnormal extremal** (см. E9). Также LM даёт *необходимое*, не *достаточное* условие — критическими бывают и сёдла; нужен второй порядок (E3).

Идея доказательства. Через implicit function theorem (E7): после переноса осей $\nabla g_1, \ldots, \nabla g_m$ образуют первые $m$ стандартных базисных направлений; локально $M$ — график функции $\mathbb{R}^{n-m} \to \mathbb{R}^m$. Подставив параметризацию в $f$ и применив обычный критерий $\nabla(f \circ \varphi) = 0$, получаем, что $\nabla f(a)$ лежит в линейной оболочке $\nabla g_i(a)$ — нормальном пространстве к $T_a M$.

Варианты и обобщения.
- **Обобщение на банаховы пространства**: Lyusternik's theorem — то же утверждение для $C^1$-отображений с сюръективным дифференциалом ограничения.
- **Глобальная форма для convex задач**: для convex $f$ и $g$ LM даёт необходимое **и** достаточное условие через слабый теорему о slater (Slater's condition).
- **Слабая форма (E9)**: Fritz John без LICQ.

Используется в. Базовый «атом» подтемы — каждая остальная запись опирается. Прямые применения: задачи на extremum симметричных функций при $\sum x_i^k = c$ ([Putnam-2003-B5], классический «найти max $\prod x_i$ при $\sum x_i^2 = R^2$»). На IMC чаще как sub-tool: [IMC-2009-5, IMC-2014-DAY2-4, IMC-2015-5].

### E2. Compactness + boundary + LM workflow (рабочая схема)

Формулировка. Стандартная процедура поиска глобальных условных экстремумов гладкой $f$ на множестве $K \subset \mathbb{R}^n$, заданном уравнениями и неравенствами:
1. Доказать существование — обычно через **компактность** $K$ (closed + bounded в $\mathbb{R}^n$) и непрерывность $f$ (теорема Вейерштрасса).
2. Если $K$ некомпактно — отдельно проверить поведение на бесконечности или у несвязной границы (предельные значения, барьер).
3. Найти все критические точки во **внутренности** $K$ (для smooth manifold — через E1 / E5).
4. Найти все критические точки на **границе/грани** меньшей размерности — рекурсивно применяя ту же процедуру к нижестрочной структуре.
5. Сравнить значения $f$ во всех найденных точках.

Условия применимости. Шаг 1 принципиален: без априорного существования экстремума LM даёт лишь *кандидатов*, среди которых могут быть только сёдла. Если $K$ некомпактно — нужно либо ограничить через коэрцитивность ($f \to +\infty$ на бесконечности), либо проверить, что супремум не достигается. Типичная ловушка олимпиадных задач: исследовать только internal critical points и забыть о границе, где может лежать экстремум.

Идея доказательства. Existence — Вейерштрасс. Reduction to interior + boundary — индукция по размерности (или dimension of strata стратификации $K$). На каждой страте — гладкое многообразие, применим LM.

Варианты и обобщения.
- **Coercivity replacement compactness**: $f(x) \to +\infty$ при $\|x\| \to \infty$ + замкнутость $K$ $\Rightarrow$ минимум достигается даже на некомпактном.
- **Sublevel set compactness**: $\{f \le c\} \cap K$ компактно при некотором $c \ge \inf f$ — слабее coercivity.

Используется в. Любая задача с frase «найти максимум» на ограниченном множестве в $\mathbb{R}^n$ — Putnam стандарт. Особенно важно в [Putnam-2007-B6]: $f(x_1, \ldots, x_n) = $ симметричный функционал на симплексе, требуется проверка вершин и ребёр симплекса отдельно.

### E3. Bordered Hessian — second-order test (окаймлённый гессиан)

Формулировка. Пусть $a$ — критическая точка $f|_M$ (LM выполнено), $L(x) = f(x) - \sum_i \lambda_i g_i(x)$ — Lagrangian, $H_L = D^2_x L(a)$ — гессиан Lagrangian по $x$. **Окаймлённый гессиан** размера $(n + m) \times (n + m)$:
$$H_B = \begin{pmatrix} 0_{m \times m} & Dg(a) \\ Dg(a)^T & H_L \end{pmatrix},$$
где $Dg(a)$ — матрица $m \times n$ Якоби ограничений. Знаковые тесты для leading principal minors $H_B$:
- **Локальный минимум**: последние $n - m$ leading principal minors имеют знак $(-1)^m$.
- **Локальный максимум**: последние $n - m$ leading principal minors имеют чередующийся знак, начиная с $(-1)^{m+1}$.

Эквивалентно: $a$ — локальный min на $M$ $\Leftrightarrow$ $H_L|_{T_a M} \succ 0$ (положительно определена на касательном пространстве); локальный max $\Leftrightarrow$ $H_L|_{T_a M} \prec 0$.

Условия применимости. $f, g_i \in C^2$. LICQ (E1), иначе bordered Hessian вырождается. Если $H_L|_{T_a M}$ полуопределена с нетривиальным ядром — second-order тест не различает min/saddle, нужен третий порядок или прямой анализ. Equality cases в inequality proofs часто соответствуют именно полуопределённости.

Идея доказательства. Параметризовать $M$ локально через $\varphi: \mathbb{R}^{n-m} \to M$, посчитать гессиан $f \circ \varphi$ в нуле; через цепное правило 2-го порядка он равен ограничению $H_L$ на $T_a M$ (поправки от вторых производных $\varphi$ убиваются множителями $\lambda_i$ через $\nabla L \cdot D^2 \varphi = 0$, поскольку $\nabla L(a) = 0$ на $T_a M$). Знак leading principal minors $H_B$ — это знак leading principal minors редуцированной квадратичной формы.

Варианты и обобщения.
- **Polar form check**: вместо minors напрямую вычислить $v^T H_L v$ для $v \in T_a M$; быстрее когда $T_a M$ имеет явный базис.
- **Strict complementarity + BSI**: при KKT с активными неравенствами — bordered second-order condition только на $T_a M \cap \{\nabla g_i \cdot v = 0 : i \in I_+\}$, где $I_+$ — индексы активных неравенств с $\mu_i > 0$.

Используется в. Распознавание min/max после нахождения critical points через E1. На олимпиадах часто заменяется на прямой подсчёт $v^T H_L v$ для конкретного $v$ в касательном пространстве, плюс соображения симметрии (E4).

### E4. Symmetry reduction at critical points (симметризация условных критических точек)

Формулировка. Пусть $f, g_1, \ldots, g_m$ инвариантны относительно действия группы $G$ на $\mathbb{R}^n$ перестановками координат (или унитарным действием), $a$ — критическая точка $f|_M$. Тогда **G-орбита** точки $a$ целиком состоит из критических точек с тем же значением $f(a)$. Если group action транзитивно перемешивает координаты в подмножествах $S_1 \sqcup \ldots \sqcup S_k = \{1, \ldots, n\}$, то можно искать критическую точку с $x_i = c_j$ для всех $i \in S_j$ — то есть с не более чем $k$ различными значениями координат.

Усиленный вариант: если $f$ строго выпукла на множестве уровня constraint в направлениях ортогональных диагонали $\Delta = \{x_1 = \ldots = x_n\}$, то критическая точка лежит на $\Delta$. Аналогично для блочной симметрии.

Условия применимости. Симметрия должна сохранять и $f$, и **все** $g_i$ одновременно. Иначе действие не действует на $M$ и аргумент рушится. Для inequality proofs полезно: если конфигурация $(α, α, \ldots, α, β, β, \ldots, β)$ исчерпывает всех кандидатов, оптимизация сводится к 1-параметрической задаче.

Идея доказательства. $G$-инвариантность $f$ и $M$ $\Rightarrow$ $f \circ \sigma = f$, $\sigma(M) = M$ для всех $\sigma \in G$. Для $\sigma \in G$ образ критической точки — критическая точка с тем же значением. Если экстремум единственный (например, строгая выпуклость) — он $G$-инвариантен, то есть лежит в fixed point set действия.

Варианты и обобщения.
- **Покомпонентное smoothing**: вместо полной симметрии — попарная перестановочная инвариантность $\sigma_{ij}$. Если $f|_{x_i = x_j}$ всегда $\le f$ при сохранении constraint — критическая конфигурация выравнивает $x_i = x_j$.
- **Schur-convex / Schur-concave**: систематический способ проверки smoothing для симметричных $f$ при constraint $\sum x_i = s$ — оптимум на extremes ordering.
- **Block-diagonal action**: симметрия на блоках $\Rightarrow$ редукция к нескольким переменным.

Используется в. Любая симметричная inequality на $\mathbb{R}^n_+$ с constraint $\sum x_i = c$ или $\prod x_i = c$. Putnam-2003-B5, IMC-2015-5 (барицентрические координаты + симметрия). Стандартная защита от перебора: после E1 искать только симметричные конфигурации.

## Записи — Standard

### E5. Karush–Kuhn–Tucker conditions (условия KKT)

Формулировка. Пусть $f, g_i, h_j: \mathbb{R}^n \to \mathbb{R}$ — $C^1$, $a$ — локальный минимум задачи
$$\min f(x), \quad g_i(x) \le 0 \ (i = 1, \ldots, p), \quad h_j(x) = 0 \ (j = 1, \ldots, m).$$
Пусть в $a$ выполнено LICQ для активных ограничений (включая все $h_j$ и те $g_i$, для которых $g_i(a) = 0$). Тогда существуют $\mu_1, \ldots, \mu_p \ge 0$ и $\lambda_1, \ldots, \lambda_m \in \mathbb{R}$:
$$\nabla f(a) + \sum_{i=1}^{p} \mu_i \nabla g_i(a) + \sum_{j=1}^{m} \lambda_j \nabla h_j(a) = 0,$$
$$\mu_i g_i(a) = 0 \quad \forall i \quad (\text{complementary slackness}).$$
Complementary slackness разделяет случаи: либо $g_i(a) = 0$ (constraint активно, $\mu_i$ может быть $> 0$), либо $\mu_i = 0$ (constraint неактивно, $g_i(a) < 0$).

Условия применимости. Для max $f$ — условие на знак $\mu_i$ переворачивается: $\mu_i \le 0$ для $g_i \le 0$ или $\mu_i \ge 0$ для $-g_i \ge 0$. Без LICQ — abnormal случай (E9). Для **convex** задач (convex $f$, convex $g_i$, affine $h_j$) при Slater's condition (существует $x_0$ с $g_i(x_0) < 0$) KKT достаточны и необходимы для глобального минимума.

Идея доказательства. В $a$ касательный конус задачи (направления, в которых можно сделать допустимый шаг первого порядка) содержит все $v$ с $\nabla g_i(a) \cdot v \le 0$ (active $i$), $\nabla h_j(a) \cdot v = 0$. Локальный минимум $\Rightarrow$ $\nabla f(a) \cdot v \ge 0$ для всех таких $v$. Лемма Фаркаша даёт неотрицательное представление $\nabla f$ через $-\nabla g_i$ (active) и $\pm \nabla h_j$.

Варианты и обобщения.
- **Slater's condition**: для convex задач — affine $h_j$ + существование строго допустимой точки $\Rightarrow$ Strong duality + KKT для оптимума.
- **MFCQ** (Mangasarian–Fromovitz constraint qualification) — слабее LICQ, но даёт ограниченность множества множителей.
- **Tucker's theorem**: complementary slackness в линейном программировании имеет строгую форму — для каждой пары (constraint, multiplier) точно одно из них $> 0$.

Используется в. Задачи с $x_i \ge 0$ (стандартные неравенства из simplex/polytope), задачи с $\|x\| \le R$ (boundary активна $\Leftrightarrow$ optimum на сфере). На олимпиадах редко формальная KKT, чаще — анализ «активна/неактивна» руками. Полезно в Putnam-style конкурентых задачах с $\sum x_i = c$, $x_i \ge 0$.

### E6. Smoothing / equalization principle from Lagrange (принцип сглаживания)

Формулировка. Пусть $f: \mathbb{R}^n \to \mathbb{R}$ симметрична, $C^1$, и рассматривается экстремум на $M = \{\sum x_i = s\} \cap \{x_i \ge 0\}$. Из E1 на внутренности: $\partial_i f = \lambda$ const, то есть все частные производные равны. Если $\partial_i f(x) = \partial_j f(x) \Rightarrow x_i = x_j$ (например, $f = \sum \varphi(x_i)$ со строго монотонной $\varphi'$), то критическая точка имеет все равные координаты $x_i = s/n$.

Принцип сглаживания (двухточечный, Schur). Симметричная гладкая $f$ на компакте $\sum x_i = s$, $x_i \ge 0$ достигает экстремум либо в точке с равными координатами, либо на границе ($x_i = 0$ для некоторого $i$). Между двумя координатами $x_i, x_j$: при $x_i + x_j = $ const, направление $(1, -1)$ — касательное к $M$, и на нём $f$ — функция от $t = x_i - x_j$. Экстремум по $t$ дают: либо $t = 0$ (выравнивание), либо боковая граница ($x_i = 0$ или $x_j = 0$).

Условия применимости. Симметрия + гладкость + компактность области. Для $f$ — strict convexity или strict concavity по парной перемене обеспечивает однозначность. Если $f$ ни выпукла ни вогнута — принцип не отбраковывает промежуточных конфигураций; тогда возможны критические точки с координатами в *двух* разных значениях $\{a, b\}$ (но не более), потому что $\partial_i f = \lambda$ — уравнение на 1 переменной с обычно не более 2 решениями.

Идея доказательства. На внутренности $M$ Lagrange: $\partial_i f = \lambda$. Если $\varphi(x) = \partial_i f$ — функция только от $x_i$ (а не от соседних, что верно для $f = \sum \varphi(x_i)$), уравнение $\varphi(x_i) = \lambda$ имеет конечное число решений; координаты разбиваются на конечное число «уровней». Compactness + второй порядок отбирает min/max.

Варианты и обобщения.
- **n=2 reduction**: принцип сглаживания сводит n-мерную задачу к двумерной — вдоль каждого парного направления.
- **Karamata's inequality**: для convex $\varphi$, $\sum \varphi(x_i) \ge \sum \varphi(y_i)$ если $(x_i)$ majorizes $(y_i)$ — обоснованная форма smoothing.
- **SOS connection**: в inequality proofs через SOS равенство достигается на конкретных конфигурациях, что согласуется с LM.

Используется в. Доказательство AM–GM, Cauchy–Schwarz, Jensen на $\sum x_i = c$ — все формализуются через LM + smoothing. На IMC/Putnam — почти любая симметричная inequality. Schur-конкавность позволяет избежать формальных вычислений.

### E7. Tangent space formulation + implicit function alternative (касательное пространство и параметризация)

Формулировка. На $C^1$-multimanifold $M = \{g_1 = \ldots = g_m = 0\}$ при LICQ касательное пространство в $a$:
$$T_a M = \bigcap_{i=1}^{m} \ker dg_i(a) = \{v \in \mathbb{R}^n : \nabla g_i(a) \cdot v = 0 \ \forall i\}.$$
Точка $a$ — критическая для $f|_M$ $\Leftrightarrow$ $df(a)|_{T_a M} = 0$ $\Leftrightarrow$ $\nabla f(a) \perp T_a M$ $\Leftrightarrow$ $\nabla f(a) \in \mathrm{span}(\nabla g_i(a))$. Последнее эквивалентно записи Лагранжа.

Альтернатива через **implicit function theorem**: при LICQ с матрицей Якоби $Dg(a)$, имеющей базисный минор $m \times m$ (без потери общности — по первым $m$ переменным), локально $x_1, \ldots, x_m$ выражаются как $C^1$-функции от $x_{m+1}, \ldots, x_n$. Подставив параметризацию в $f$, сводим к безусловному экстремуму $n - m$ переменных. Часто проще, чем LM, если параметризация известна явно.

Условия применимости. Implicit function theorem требует невырожденности подматрицы Якоби — это и есть LICQ для конкретного выбора координат. Если базисный минор не очевиден — стандартный LM удобнее. Параметризация хороша для local анализа, но плохо «склеивается» глобально на топологически нетривиальных $M$ (тор, сфера).

Идея доказательства. Implicit function theorem + chain rule.

Варианты и обобщения.
- **Lagrange multipliers via projection**: $\nabla f$ ортогонально проектируется на $T_a M$; критическая точка — нулевая проекция. Эквивалентная формулировка, удобная численно.
- **Реализация constraint в координатах**: унитарным преобразованием можно превратить $M$ в координатную плоскость в окрестности; задача становится безусловной.

Используется в. Когда constraint простой (например, $x^2 + y^2 + z^2 = R^2$): параметризация в сферических координатах часто эффективнее, чем LM. На IMC параметризацию используют в [IMC-2002-DAY2-4] (срезы единичной сферы) и [IMC-2009-5] (аффинная интерполяция вдоль луча).

### E8. Extreme points + Carathéodory for linear objectives (теорема об экстремальной точке)

Формулировка. Пусть $K \subset \mathbb{R}^n$ — компактное выпуклое множество, $L: \mathbb{R}^n \to \mathbb{R}$ — линейный (или аффинный) функционал. Тогда:
$$\max_{x \in K} L(x) = \max_{x \in \mathrm{ext}(K)} L(x),$$
где $\mathrm{ext}(K)$ — множество **крайних точек** (extreme points) $K$, то есть точек, не представимых как нетривиальная convex combination двух других точек $K$.

**Carathéodory's theorem**: если $x \in \mathrm{conv}(S)$ для $S \subset \mathbb{R}^n$, то $x$ — convex combination не более $n + 1$ точек $S$. В частности, $K = \mathrm{conv}(\mathrm{ext}(K))$ для компактного выпуклого $K$ (теорема Krein–Milman в $\mathbb{R}^n$).

Условия применимости. Линейность $L$ критична: для нелинейного функционала максимум может лежать внутри $K$. Компактность $K$ нужна для существования max. Для polytope (конечное число extreme points = вершины) — задача сводится к перебору. Если $K$ — выпуклый body со сплошной границей (шар), все граничные точки крайние.

Идея доказательства. Для maximum: $a = \arg\max L$ — на границе $K$ (иначе можно сдвинуться вдоль $\nabla L \ne 0$). Множество $F = \{x \in K : L(x) = L(a)\}$ — face $K$, само компактное выпуклое. Индукция по размерности $F$: либо $F$ — точка (она extreme), либо $\mathrm{ext}(F) \subset \mathrm{ext}(K)$.

Варианты и обобщения.
- **Birkhoff–von Neumann**: extreme points doubly stochastic matrices — permutation matrices.
- **Каратеодори с фиксированной размерностью**: если $\dim(\mathrm{aff}(S)) = d$, то $d + 1$ точек достаточно (вместо $n + 1$).
- **Bauer's theorem**: непрерывная convex функция на компактном выпуклом множестве достигает max в крайней точке.

Используется в. Линейное программирование (вершины политопа), задачи о максимизации линейного функционала при линейных ограничениях. На олимпиадах: [IMC-2015-5] (барицентрические координаты — convex combination), задачи о геометрических конфигурациях с линейной целевой функцией. Альтернатива Lagrange при выпуклых constraints + линейной цели.

## Записи — Exotic

### E9. Fritz John conditions / abnormal extremals (вырожденный случай Лагранжа)

Формулировка. Пусть $a$ — локальный минимум $f$ при $g_i(x) \le 0$, $h_j(x) = 0$ (без всяких CQ). Тогда существуют $\lambda_0 \ge 0$, $\mu_i \ge 0$, $\lambda_j \in \mathbb{R}$, не все нули:
$$\lambda_0 \nabla f(a) + \sum_{i=1}^{p} \mu_i \nabla g_i(a) + \sum_{j=1}^{m} \lambda_j \nabla h_j(a) = 0,$$
$$\mu_i g_i(a) = 0 \quad \forall i.$$
Если $\lambda_0 > 0$ — обычный Lagrange (KKT после нормировки $\lambda_0 = 1$). Если $\lambda_0 = 0$ — **abnormal extremal**: $\nabla f$ не участвует, все информация в градиентах constraints, которые вырождены ($\Leftrightarrow$ LICQ нарушено).

Условия применимости. Универсально работает для $C^1$-задач без CQ. Платой служит потеря $\lambda_0$: уравнение может не «видеть» целевую функцию. Полезно для:
- задач с алгебраическими сингулярностями constraints (cusps, self-intersections),
- proof-of-concept существования множителей в общем случае,
- distinguishing geometrically degenerate optima.

Идея доказательства. Применение separation theorem (или леммы Фаркаша) к выпуклому конусу $\mathrm{cone}(\nabla f, -\nabla g_i \text{ active}, \pm\nabla h_j)$. Если 0 в относительной внутренности — обычный LM; если 0 на границе — abnormal с $\lambda_0 = 0$.

Варианты и обобщения.
- **Положительный множитель в abnormal**: иногда $\lambda_0 = 0$ даёт нетривиальную информацию (геометрия multimanifold), хоть и не задействует $f$.
- **Pontryagin maximum principle**: optimal control аналог Fritz John, $\lambda_0$ соответствует нормировке adjoint.

Используется в. Контрпример из E1: $f = x$, $g = x^3 - y^2$ в $(0, 0)$. Получаем $\nabla g = 0$, поэтому Lagrange $\nabla f = \lambda \nabla g$ невозможен. Fritz John даёт $\lambda_0 \cdot (1, 0) + \lambda_1 \cdot (0, 0) = 0$, отсюда $\lambda_0 = 0$ — abnormal extremal обнаружен. На олимпиадах редко: формальная Fritz John применяется лишь в задачах с явно вырожденными constraints.

### E10. Polynomial elimination of multiplier (исключение множителя через resultant)

Формулировка. Для алгебраической задачи $\nabla f = \lambda \nabla g$ с полиномиальными $f, g$ можно исключить $\lambda$ алгебраически:
- **Прямое попарное условие**: $\partial_i f \cdot \partial_j g - \partial_j f \cdot \partial_i g = 0$ для всех пар $i, j$. Получаем $\binom{n}{2}$ полиномиальных уравнений + само $g = c$.
- **Resultant trick**: $\lambda = \partial_1 f / \partial_1 g$, подставить в остальные уравнения, очистить знаменатели. Получается полиномиальная система от $x$.

Эквивалентная геометрическая формулировка: $\mathrm{rank}\begin{pmatrix} \nabla f(x) \\ \nabla g(x) \end{pmatrix} \le 1$ — условие коллинеарности, выражается через 2-минорные определители.

Условия применимости. $f, g$ — полиномы (или рациональные). Степени должны позволять резольвенту: для $\deg f = d_1$, $\deg g = d_2$, число решений системы по теореме Безу не превосходит $d_1 d_2 \cdot \ldots$ (точные оценки через Bernstein). Метод эффективен для $n = 2, 3$ или при дополнительной симметрии. В высоких размерностях — Gröbner basis.

Идея доказательства. Условие $\nabla f \parallel \nabla g$ — алгебраическое, эквивалентно занулению всех $2 \times 2$-миноров матрицы $\begin{pmatrix} \partial_i f \\ \partial_i g \end{pmatrix}_{i=1}^n$. Resultant исключает один параметр, оставляя систему меньшей сложности.

Варианты и обобщения.
- **Несколько constraints**: при $m$ ограничениях — условие $\mathrm{rank}\begin{pmatrix} \nabla f \\ \nabla g_1 \\ \vdots \\ \nabla g_m \end{pmatrix} \le m$; зануление всех $(m+1) \times (m+1)$-миноров.
- **Symbolic Gröbner basis approach**: автоматизирует исключение $\lambda$ для произвольного числа переменных.
- **Sylvester resultant** для $n = 2$: $\mathrm{Res}_y(\partial_x f - \lambda \partial_x g, g - c) = 0$ как полином от $\lambda, x$.

Используется в. Задачи с малой размерностью и алгебраическими constraints, где LM даёт высокую полиномиальную систему. Примеры: расстояние от точки до алгебраической кривой/поверхности, поиск критических точек на cubics. На IMC применяется неявно: например, [IMC-2025-1] (tangent line family covers — алгебраическое условие касания, эквивалентное LM с многочленом).

## Self-test

1. Найти максимум и минимум функции $f(x, y, z) = xyz$ на сфере $x^2 + y^2 + z^2 = 1$ при $x, y, z \ge 0$.

2. Пусть $A = (a_{ij}) \in \mathbb{R}^{n \times n}$ удовлетворяет $\sum_{i, j} a_{ij}^2 = 1$. Доказать, что $|\det A| \le n^{-n/2}$, равенство при $A = U/\sqrt{n}$, $U$ — ортогональная.

3. Пусть $A \in \mathbb{R}^{n \times n}$ симметрична. Доказать, что $\max_{\|x\| = 1} x^T A x = \lambda_{\max}(A)$ (наибольшее собственное значение), и максимум достигается на собственном векторе.

4. На кривой $\{(x, y) \in \mathbb{R}^2 : x^4 - y^2 = 0\}$ найти точку, ближайшую к $(0, 1)$. Объяснить, почему стандартный Lagrange не даёт всех критических точек.

5. Пусть $K \subset \mathbb{R}^3$ — выпуклый компактный многогранник с $V$ вершинами. Доказать, что для линейного функционала $L$ максимум $L|_K$ достигается в одной из $V$ вершин.

6. Найти максимум $\sum_{i < j} x_i x_j$ при $\sum x_i = s$, $x_i \in [0, 1]$ (фиксированы $n$, $s$). Описать, как ответ зависит от $s$.

7. Доказать, что для $a, b, c > 0$ с $a + b + c = 3$ верно $a^2 + b^2 + c^2 + abc \ge 4$.

## Решения

1. Внутренние критические точки на положительной части сферы: $\partial_x(xyz) = yz$, $\partial_y(xyz) = xz$, $\partial_z(xyz) = xy$, $\partial(x^2+y^2+z^2) = (2x, 2y, 2z)$. Lagrange: $yz = 2\lambda x$, $xz = 2\lambda y$, $xy = 2\lambda z$. Перемножив или поделив: $x^2 = y^2 = z^2$, то есть $x = y = z = 1/\sqrt{3}$ (положительность). Значение $f = 1/(3\sqrt{3})$. На границе ($x_i = 0$) — $f = 0$. Max $= 1/(3\sqrt{3})$, min $= 0$.

2. Lagrange на $\sum a_{ij}^2 = 1$: $\partial_{a_{ij}} \det A = C_{ij}$ (cofactor), $\partial_{a_{ij}} \sum a_{kl}^2 = 2 a_{ij}$. Условие: $C_{ij} = 2\lambda a_{ij}$, то есть $\mathrm{adj}(A) = 2\lambda A$. Используя $A \cdot \mathrm{adj}(A) = \det(A) \cdot I$: $A \cdot 2\lambda A = \det A \cdot I$, то есть $A^T A$ пропорциональна $I$ — $A$ ортогональна с точностью до скаляра. Из $\sum a_{ij}^2 = 1$: $A = U / \sqrt{n}$, $|\det A| = n^{-n/2}$.

3. Lagrange на $\|x\|^2 = 1$: $\nabla(x^T A x) = 2 A x$, $\nabla(\|x\|^2) = 2 x$. Условие: $A x = \lambda x$ — критические точки в собственных векторах. Значение $x^T A x = \lambda x^T x = \lambda$, то есть совпадает с собственным значением. Compact sphere $\Rightarrow$ extremum достигается; max — наибольшее собственное значение, достигается на соответствующем собственном векторе.

4. Кривая имеет cusp в $(0, 0)$: $\nabla(x^4 - y^2) = (4x^3, -2y) = (0, 0)$. Это **abnormal point** (E1, E9). Стандартный Lagrange там не работает. Анализ: $y = \pm x^2$, расстояние до $(0, 1)$ — $\sqrt{x^2 + (x^2 \mp 1)^2}$. Минимум при $\partial_x = 0$: получаем точку зависит от ветви. Cusp $(0, 0)$ даёт расстояние $1$, конкурирует с другими critical points на гладкой части.

5. По E8: $K = \mathrm{conv}(\mathrm{vertices})$ для polytope. $L(x)$ для $x = \sum \alpha_v v$ — convex combination — равно $\sum \alpha_v L(v) \le \max_v L(v)$. Достигается при $\alpha_{v^*} = 1$ для $v^*$ — максимизирующей вершины.

6. $f(x) = \sum_{i < j} x_i x_j = \frac{1}{2}((\sum x_i)^2 - \sum x_i^2) = \frac{1}{2}(s^2 - \sum x_i^2)$. Максимизация $f$ $\Leftrightarrow$ минимизация $\sum x_i^2$ при $\sum x_i = s$, $x_i \in [0, 1]$. Smoothing (E6): минимум суммы квадратов — все равны $s/n$, если это допустимо ($s/n \le 1$); иначе $k$ переменных равны $1$, остальные равны $(s - k)/(n - k)$ для оптимального $k = \lfloor s \rfloor$. Ответ: $f_{\max} = s^2/2 - \min(\sum x_i^2)$.

7. Compactness $a + b + c = 3$, $a, b, c > 0$ + boundary $abc = 0$: $a^2 + b^2 + c^2 + abc \to a^2 + b^2 \ge (a+b)^2/2 = 9/2 \ge 4$. На внутренности: Lagrange $\nabla(a^2+b^2+c^2+abc) = 2(a, b, c) + (bc, ac, ab) = \lambda(1, 1, 1)$. Симметрия (E4): $a = b = c = 1$, значение $3 + 1 = 4$. Симметризация показывает, что это и есть минимум. Альтернативно: $a^2 + b^2 + c^2 + abc - 4 = $ Schur-неравенство $t = 1$.
