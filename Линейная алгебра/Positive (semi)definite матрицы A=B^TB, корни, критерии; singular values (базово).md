# Positive (semi)definite матрицы: $A=B^TB$, корни, критерии; singular values (базово)

## Scope

Эрмитовы положительно (полу)определённые матрицы и их инварианты. Эквивалентные критерии положительности через спектр, представление Грама и миноры. Канонические разложения: Холецкий, квадратный корень, полярное, сингулярное. Минимаксные характеризации собственных и сингулярных чисел, перемежение, неравенства Вейля. Частичный порядок Лёвнера. Олимпиадные неравенства уровня IMC и Putnam: Адамара, Фишера, Минковского, Шура, Лёвнера–Хайнца.

## Определения

Везде ниже $A \in \mathbb{C}^{n \times n}$ эрмитова (Hermitian), то есть $A^* = A$. Над $\mathbb{R}$ это значит симметрична.

- **Положительно определённая** (positive definite, PD), запись $A \succ 0$: $x^* A x > 0$ для всех ненулевых $x \in \mathbb{C}^n$.
- **Положительно полуопределённая** (positive semidefinite, PSD), запись $A \succeq 0$: $x^* A x \ge 0$ для всех $x$.
- **Главный минор** (principal minor) порядка $k$: определитель подматрицы $A_{S, S}$ для произвольного $S \subseteq \{1, \dots, n\}$ с $|S| = k$.
- **Ведущий главный минор** (leading principal minor) $\Delta_k$: главный минор с $S = \{1, \dots, k\}$.
- **Сингулярные числа** (singular values) $\sigma_i(A) \ge 0$ матрицы $A \in \mathbb{C}^{m \times n}$: квадратные корни из собственных чисел $A^* A$ (равно собственных чисел $A A^*$). Упорядочены по убыванию.
- **Спектральная норма** $\|A\|_2 = \sigma_1(A) = \max_{\|x\| = 1} \|Ax\|$.
- **Фробениусова норма** $\|A\|_F = \sqrt{\operatorname{tr}(A^* A)} = \sqrt{\sum_i \sigma_i^2}$.
- **Матрица Грама** (Gram matrix) семейства $v_1, \dots, v_n$: $G_{ij} = \langle v_i, v_j \rangle$.
- **Порядок Лёвнера** (Loewner order): $A \succeq B$ означает $A - B \succeq 0$. Частичный порядок на эрмитовых матрицах.
- **Конгруэнтность** (congruence): преобразование $A \mapsto C^* A C$ при невырожденной $C$.

## Записи — Основные

### E1. Спектральная теорема (spectral theorem) и эквивалентные критерии PSD

**Формулировка.** Любая эрмитова $A$ диагонализуется унитарным преобразованием:
$$A = U \Lambda U^*, \quad U^* U = I, \quad \Lambda = \operatorname{diag}(\lambda_1, \dots, \lambda_n), \quad \lambda_i \in \mathbb{R}.$$
В вещественном случае $U$ ортогональна. Для $A$ эрмитовой следующие четыре условия эквивалентны:

1. $x^* A x \ge 0$ для всех $x \in \mathbb{C}^n$;
2. все собственные числа $\lambda_i \ge 0$;
3. $A = B^* B$ для некоторой $B$;
4. все главные миноры неотрицательны.

Для PD во всех пунктах строгие неравенства, и в (4) хватает ведущих главных миноров (E3).

**Условия применимости.** Эрмитовость необходима.
*Контрпример без эрмитовости:* $A = \begin{pmatrix} 0 & 1 \\ -1 & 0 \end{pmatrix}$. Собственные числа $\pm i$, но $x^T A x = 0$ для всех $x \in \mathbb{R}^2$. Условие (1) выполняется, (2) — нет.

**Идея доказательства.** Эквивалентность (1) $\Leftrightarrow$ (2) — диагонализация и замена координат $y = U^* x$: $x^* A x = \sum \lambda_i |y_i|^2$. (3) $\Rightarrow$ (1) — прямая подстановка $\|Bx\|^2 \ge 0$. (1) $\Rightarrow$ (3) — спектрально, $B = \Lambda^{1/2} U^*$.

---

### E2. Представление Грама: $A = B^* B$

**Формулировка.** Эрмитова $A$ положительно полуопределена тогда и только тогда, когда $A = B^* B$ для некоторой $B \in \mathbb{C}^{m \times n}$. Эквивалентно: $A_{ij} = \langle v_i, v_j \rangle$ для системы векторов $v_1, \dots, v_n$ (столбцов $B$).

Тогда $\operatorname{rank}(A) = \dim \operatorname{span}(v_1, \dots, v_n)$.

Определитель $\det A$ равен квадрату объёма параллелепипеда, натянутого на $\{v_i\}$. Минимальное $m$, при котором представление возможно, равно $\operatorname{rank}(A)$.

**Условия применимости.** Любое поле с инволюцией ($\mathbb{R}$ или $\mathbb{C}$). Для PD дополнительно $B$ имеет полный столбцовый ранг.

**Идея доказательства.** Через E1: положить $B = \Lambda^{1/2} U^*$, тогда $B^* B = U \Lambda U^* = A$. Обратно: $\|Bx\|^2 \ge 0$.

---

### E3. Критерий Сильвестра (Sylvester's criterion)

**Формулировка.** Для эрмитовой $A$:

- $A \succ 0$ тогда и только тогда, когда все ведущие главные миноры положительны: $\Delta_k > 0$ для $k = 1, \dots, n$.
- $A \succeq 0$ тогда и только тогда, когда все $2^n - 1$ главных миноров неотрицательны.

**Условия применимости.** Эрмитовость обязательна. Для PSD ведущих главных миноров не хватает.
*Контрпример (PSD-версия с ведущими минорами):* $A = \operatorname{diag}(0, -1)$. Оба ведущих минора равны нулю, но $A$ не PSD. Не хватает главного минора по индексу $\{2\}$: он равен $-1$.

**Идея доказательства.** Индукция по $n$ через дополнение Шура (E11):
$$\det A = \Delta_{n-1} \cdot (a_{nn} - b^* A_{n-1}^{-1} b).$$
Если $\Delta_{n-1} > 0$, остаётся проверить положительность скалярного множителя.

---

### E4. Разложение Холецкого (Cholesky decomposition)

**Формулировка.** Эрмитова $A$ положительно определена тогда и только тогда, когда существует единственная нижнетреугольная $L$ с $L_{ii} > 0$ и
$$A = L L^*.$$
Альтернативная форма $A = L D L^*$ с диагональной $D \succ 0$ и единичной нижнетреугольной $L$ — даёт алгоритм без извлечения корней.

**Условия применимости.** PD случай. Для PSD разложение существует, но единственности нет; нужна перестановка строк-столбцов (pivoted Cholesky).

**Идея доказательства.** Индукция. После выделения первой строки $\ell_{11} = \sqrt{a_{11}}$, $\ell_{i1} = a_{i1} / \ell_{11}$, остаток — дополнение Шура, которое снова PD.

Связь с критерием Сильвестра: $\ell_{kk}^2 = \Delta_k / \Delta_{k-1}$, поэтому $\det A = \prod \ell_{kk}^2 > 0$.

---

### E5. Квадратный корень (matrix square root) из PSD

**Формулировка.** Любая PSD-матрица $A$ имеет единственный PSD-корень. Через спектральное разложение:
$$A^{1/2} = U \Lambda^{1/2} U^*, \quad \Lambda^{1/2} = \operatorname{diag}(\sqrt{\lambda_i}).$$
Без ограничения положительности корней общего вида у PD-$A$ ровно $2^r$, где $r$ — число различных собственных чисел (выбор знака на каждом). При наличии кратных собственных чисел появляются непрерывные семейства корней.

**Условия применимости.** PSD.
*Контрпример без PSD:* $A = -I$. Вещественного корня нет: $X^2 = -I$ потребовало бы $\det(X)^2 = (-1)^n$, что невозможно при $n$ нечётном.

**Идея доказательства.** Существование — спектрально (E1). Единственность среди PSD: если $X \succeq 0$ и $X^2 = A$, то $X$ коммутирует с $A$ (так как $XA = X^3 = AX$), значит диагонализуется в том же базисе, и совпадает с $A^{1/2}$.

---

### E6. Полярное разложение (polar decomposition)

**Формулировка.** Любая $A \in \mathbb{C}^{m \times n}$ с $m \ge n$ представима как
$$A = U P, \quad U^* U = I_n, \quad P = (A^* A)^{1/2} \succeq 0.$$
Матрица $P$ всегда единственна. При полном столбцовом ранге $A$ матрица $U$ тоже единственна.

В квадратном вещественном случае $U$ ортогональна. Это матричный аналог записи комплексного числа $z = e^{i \theta} |z|$.

**Условия применимости.** Любая прямоугольная.

**Идея доказательства.** Через сингулярное разложение $A = U_1 \Sigma V^*$ (E7): положить $P = V \Sigma V^* = (A^* A)^{1/2}$, $U = U_1 V^*$.

---

### E7. Сингулярное разложение (singular value decomposition, SVD)

**Формулировка.** Любая $A \in \mathbb{C}^{m \times n}$ представима как
$$A = U \Sigma V^*,$$
где $U \in \mathbb{C}^{m \times m}$ и $V \in \mathbb{C}^{n \times n}$ унитарны, а $\Sigma \in \mathbb{R}^{m \times n}$ имеет на главной диагонали неотрицательные числа $\sigma_1 \ge \sigma_2 \ge \dots \ge 0$ и нули вне диагонали.

Сингулярные числа $\sigma_i$ совпадают с квадратными корнями из собственных чисел $A^* A$. Число ненулевых $\sigma_i$ равно $\operatorname{rank}(A)$.

**Условия применимости.** Любая матрица. Вещественный случай — с ортогональными $U, V$.

**Идея доказательства.** Спектральная теорема (E1) для эрмитовой PSD-матрицы $A^* A = V \Sigma^2 V^*$. Векторы $u_i = A v_i / \sigma_i$ для $\sigma_i > 0$ дополнить до ортонормированного базиса $U$.

**Варианты и обобщения.** Best rank-$k$ approximation (теорема Эккарта–Янга): $\min_{\operatorname{rank} B \le k} \|A - B\|_2 = \sigma_{k+1}$. Псевдообратная Мура–Пенроуза: $A^+ = V \Sigma^+ U^*$.

---

### E8. Минимаксная характеризация Куранта–Фишера (Courant–Fischer min-max)

**Формулировка.** Для эрмитовой $A$ с собственными числами $\lambda_1 \ge \dots \ge \lambda_n$:
$$\lambda_k(A) = \max_{\dim V = k} \min_{x \in V, \|x\| = 1} x^* A x = \min_{\dim V = n - k + 1} \max_{x \in V, \|x\| = 1} x^* A x.$$
Аналогично для сингулярных чисел произвольной $A$:
$$\sigma_k(A) = \max_{\dim V = k} \min_{x \in V, \|x\| = 1} \|A x\|.$$
В частности, $\sigma_1 = \|A\|_2$ и $\sigma_n = \min_{\|x\| = 1} \|Ax\|$ (для квадратной).

**Условия применимости.** Эрмитовость для собственных чисел.

**Идея доказательства.** Диагонализовать $A = U \Lambda U^*$, перейти к координатам $y = U^* x$, получить $\sum \lambda_i |y_i|^2$. Дальше — пересечение подпространств размерностей $k$ и $n - k + 1$ в $\mathbb{C}^n$ непусто по размерности.

---

## Записи — Стандартные

### E9. Перемежение Коши (Cauchy interlacing)

**Формулировка.** Пусть $A$ эрмитова с собственными числами $\lambda_1 \ge \dots \ge \lambda_n$, а $B$ — её главная подматрица размера $(n-1) \times (n-1)$ с собственными числами $\mu_1 \ge \dots \ge \mu_{n-1}$. Тогда
$$\lambda_1 \ge \mu_1 \ge \lambda_2 \ge \mu_2 \ge \dots \ge \mu_{n-1} \ge \lambda_n.$$
Для главной подматрицы размера $(n - k) \times (n - k)$: $\lambda_i \ge \mu_i \ge \lambda_{i + k}$.

**Условия применимости.** Подматрица должна быть главной, то есть выбраны одинаковые наборы строк и столбцов.
*Контрпример при произвольной подматрице:* убрав не строку и столбец с одинаковыми индексами, а только строку или столбец — получаем уже сингулярные числа, для которых отдельное неравенство.

**Идея доказательства.** Применить Куранта–Фишера (E8) к ограничению $A$ на подпространство $\mathbb{C}^{n-1} \subset \mathbb{C}^n$ (исключённая координата зануляется).

---

### E10. Неравенства Вейля (Weyl's inequalities)

**Формулировка.** Для эрмитовых $A, B$:
$$\lambda_{i+j-1}(A + B) \le \lambda_i(A) + \lambda_j(B), \quad i + j - 1 \le n,$$
$$\lambda_{i+j-n}(A + B) \ge \lambda_i(A) + \lambda_j(B), \quad i + j - n \ge 1.$$
В частности, $|\lambda_i(A) - \lambda_i(B)| \le \|A - B\|_2$ — ограничение возмущения спектра.

Для сингулярных чисел произвольных матриц аналогично: $\sigma_{i+j-1}(A + B) \le \sigma_i(A) + \sigma_j(B)$ и $\sigma_{i+j-1}(AB) \le \sigma_i(A) \sigma_j(B)$.

**Условия применимости.** Эрмитовость для версии с собственными числами.

**Идея доказательства.** Через Куранта–Фишера (E8): подобрать $i$-мерное подпространство, максимизирующее $A$, и $j$-мерное, максимизирующее $B$. Их пересечение имеет размерность хотя бы $i + j - n$.

---

### E11. Дополнение Шура и критерий положительности

**Формулировка.** Для блочной эрмитовой
$$M = \begin{pmatrix} A & B \\ B^* & D \end{pmatrix}$$
с $D \succ 0$ верно: $M \succ 0$ тогда и только тогда, когда $A - B D^{-1} B^* \succ 0$ (дополнение Шура).

Для PSD-версии: $M \succeq 0$ тогда и только тогда, когда $D \succeq 0$, образ $B$ лежит в образе $D$, и $A - B D^+ B^* \succeq 0$.

**Условия применимости.** Без $D \succ 0$ формула неприменима напрямую.
*Контрпример без условия на $D$:* $M = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$. $D = 0$, обратное не определено, $M$ не PSD ($\det = -1$).

**Идея доказательства.** Блочная конгруэнтность приводит $M$ к блочно-диагональной форме с блоками $A - B D^{-1} B^*$ и $D$. Конгруэнтность сохраняет сигнатуру (закон инерции Сильвестра).

---

### E12. Порядок Лёвнера: свойства и предостережения

**Формулировка.** На эрмитовых матрицах определён частичный порядок $A \succeq B \iff A - B \succeq 0$. Основные свойства:

1. *Конгруэнтная инвариантность.* $A \succeq B \Rightarrow C^* A C \succeq C^* B C$.
2. *Антитонность обратной.* $0 \prec A \preceq B \Rightarrow B^{-1} \preceq A^{-1}$.
3. *Монотонность корня.* $0 \preceq A \preceq B \Rightarrow A^{1/2} \preceq B^{1/2}$.
4. *Возведение в квадрат не монотонно.* Из $A \succeq B \succeq 0$ не следует $A^2 \succeq B^2$.
5. *Покомпонентный мажоризационный вывод.* $A \succeq B \succeq 0 \Rightarrow \lambda_i(A) \ge \lambda_i(B)$, $\operatorname{tr}(A) \ge \operatorname{tr}(B)$, $\det(A) \ge \det(B)$.

**Условия применимости.** Эрмитовость обоих операндов.
*Контрпример к монотонности квадрата:* $A = \begin{pmatrix} 2 & 1 \\ 1 & 1 \end{pmatrix}$, $B = \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix}$. Тогда $A - B = \begin{pmatrix} 1 & 1 \\ 1 & 1 \end{pmatrix} \succeq 0$, но $A^2 - B^2 = \begin{pmatrix} 4 & 3 \\ 3 & 2 \end{pmatrix}$ имеет определитель $-1$, поэтому не PSD.

**Идея доказательства.** (1)–(2) — прямая алгебра. (3) — частный случай Лёвнера–Хайнца (E15). (5) — из минимакса (E8).

---

### E13. Неравенство Минковского для определителей (Minkowski's determinant inequality)

**Формулировка.** Для PSD-матриц $A, B$ одного размера $n$:
$$\det(A + B)^{1/n} \ge \det(A)^{1/n} + \det(B)^{1/n}.$$
Равенство достигается тогда и только тогда, когда $A$ и $B$ пропорциональны или одна из них нулевая.

Эквивалентно: функция $A \mapsto \det(A)^{1/n}$ вогнута на конусе PSD-матриц.

**Условия применимости.** Обе матрицы PSD.
*Контрпример без PSD:* $A = \operatorname{diag}(1, -1)$, $B = \operatorname{diag}(-1, 1)$. $\det A = \det B = -1$, $\det(A + B) = 0$. Возведение $(-1)^{1/2}$ в вещественные числа не имеет смысла.

**Идея доказательства.** Конгруэнтностью свести к $A = I$, $B = D$ диагональной. Сводится к скалярному неравенству $\prod (1 + d_i)^{1/n} \ge 1 + \prod d_i^{1/n}$, которое доказывается через AM-GM.

## Записи — Редкие

### E14. Теорема Шура о произведении (Schur product theorem)

**Формулировка.** Произведение Адамара (поэлементное) $A \circ B = (a_{ij} b_{ij})$ двух PSD-матриц снова PSD.

**Идея доказательства.** Если $A = \sum x_k x_k^*$ и $B = \sum y_\ell y_\ell^*$ (спектральное разложение как сумма rank-1 PSD), то $A \circ B = \sum_{k, \ell} (x_k \circ y_\ell)(x_k \circ y_\ell)^*$ — снова сумма rank-1 PSD.

---

### E15. Неравенство Лёвнера–Хайнца (Loewner–Heinz inequality)

**Формулировка.** Если $A \succeq B \succeq 0$, то $A^t \succeq B^t$ для всех $t \in [0, 1]$.

**Условия применимости.** Параметр $t$ строго внутри $[0, 1]$.
*Контрпример для $t = 2$:* в E12 (пункт 4) приведена пара $A \succeq B \succeq 0$ с $A^2 \not\succeq B^2$.

**Идея доказательства.** Интегральное представление при $\alpha \in (0, 1)$:
$$t^\alpha = \frac{\sin(\pi \alpha)}{\pi} \int_0^\infty \frac{\lambda^{\alpha - 1} t}{\lambda + t} \, d\lambda.$$
Подынтегральная функция $\lambda t / (\lambda + t)$ операторно-монотонна (через антитонность обратной из E12.2). Интегрирование монотонность сохраняет.

---

### E16. Операторное геометрическое среднее $A \,\#\, B$

**Формулировка.** Для $A, B \succ 0$:
$$A \,\#\, B = A^{1/2} (A^{-1/2} B A^{-1/2})^{1/2} A^{1/2}.$$
Свойства: $A \,\#\, B = B \,\#\, A$, монотонность по обоим аргументам, среднее операторное AM–GM: $(A^{-1} + B^{-1})^{-1} \cdot 2 \preceq A \,\#\, B \preceq (A + B)/2$.

Эквивалентная характеризация: $A \,\#\, B$ — единственное PSD-решение уравнения $X A^{-1} X = B$.

## Self-test

1. Пусть $A \in M_n(\mathbb{R})$ симметрична и $A^3 = A$. Опиши все возможные собственные числа $A$. Докажи, что $A^2$ — ортогональная проекция.

2. Пусть $A, B \in M_n(\mathbb{R})$ симметричны и положительно полуопределены. Докажи, что все собственные числа произведения $AB$ вещественны и неотрицательны.

3. Пусть $A \in M_n(\mathbb{C})$ произвольная. Докажи равенство $\sum_{i, j} |a_{ij}|^2 = \sum_i \sigma_i(A)^2$.

4. Пусть $A, B \succeq 0$. Докажи $\operatorname{tr}(AB) \ge 0$. Когда достигается равенство?

5. Пусть $A \succeq 0$ с $a_{ii} = 1$ для всех $i$. Докажи $|a_{ij}| \le 1$ и $\det A \le 1$.

6. Пусть $H \in M_n(\mathbb{R})$ — матрица Адамара, то есть $h_{ij} = \pm 1$ и $H H^T = n I$. Найди $|\det H|$.

7. Пусть $A$ — симметричная $n \times n$ с $\operatorname{tr}(A) = 0$ и $\operatorname{tr}(A^2) = n$. Докажи, что хотя бы одно из собственных чисел по модулю не меньше единицы. Приведи пример, где равенство достигается.

8. Пусть $A, B \succ 0$ из $M_n(\mathbb{R})$. Докажи: $\det(A + B) \ge \det(A) + \det(B)$.

9. Пусть $u_1, \dots, u_k \in \mathbb{R}^n$ единичные с $\langle u_i, u_j \rangle = c$ для всех $i \ne j$. Найди диапазон допустимых $c$ при фиксированном $k$.

10. Пусть $A \in M_n(\mathbb{R})$ невырождена. Докажи, что существует единственная пара $(U, P)$, где $U$ ортогональна, $P$ симметрична положительно определена, и $A = UP$.

---

## Решения

1. $A$ симметрична, значит диагонализуется. Из $\lambda^3 = \lambda$: $\lambda \in \{-1, 0, 1\}$. Тогда у $A^2$ собственные числа $\lambda^2 \in \{0, 1\}$. Матрица $A^2$ симметрична и удовлетворяет $(A^2)^2 = A^4 = A^2$. Это и есть определение ортогональной проекции.

2. Положим $S = A^{1/2}$. Тогда $S B S$ симметрична и положительно полуопределена: $x^T (SBS) x = (Sx)^T B (Sx) \ge 0$. Матрицы $AB = S^2 B$ и $S B S$ подобны (через $S$), значит у них совпадает спектр. Следовательно, собственные числа $AB$ совпадают с собственными числами PSD-матрицы $SBS$, то есть вещественны и неотрицательны.

3. По определению фробениусовой нормы $\|A\|_F^2 = \operatorname{tr}(A^* A)$. След матрицы равен сумме собственных чисел: $\operatorname{tr}(A^* A) = \sum_i \lambda_i(A^* A) = \sum_i \sigma_i(A)^2$.

4. Положим $X = A^{1/2}$, $Y = B^{1/2}$. Тогда $\operatorname{tr}(AB) = \operatorname{tr}(X^2 Y^2) = \operatorname{tr}(X X Y Y)$. По циклическому свойству следа: $\operatorname{tr}(X X Y Y) = \operatorname{tr}(Y X X Y) = \operatorname{tr}((XY)^* (XY)) = \|XY\|_F^2 \ge 0$. Равенство $\iff XY = 0 \iff AB = X(XY)Y = 0$.

5. Главный $2 \times 2$ минор $\begin{pmatrix} 1 & a_{ij} \\ a_{ij} & 1 \end{pmatrix}$ неотрицателен по критерию из E3. Его определитель $1 - a_{ij}^2 \ge 0$, отсюда $|a_{ij}| \le 1$. Оценка $\det A \le 1$ — частный случай неравенства Адамара для PSD-матриц: $\det A \le \prod a_{ii} = 1$.

6. $|\det H|^2 = \det(H H^T) = \det(n I) = n^n$, поэтому $|\det H| = n^{n/2}$. Это в точности экстремум в неравенстве Адамара для матриц с элементами в $\{-1, +1\}$.

7. Пусть $M = \lambda_{\max}$ и $m = \lambda_{\min}$. Положим $K = \max(M, |m|)$. Все собственные числа лежат в $[-K, K]$, поэтому $\lambda_i^2 \le K^2$. Складывая, получаем $n = \sum \lambda_i^2 \le n K^2$, откуда $K \ge 1$.

   Пример равенства: $A = \operatorname{diag}(1, -1, 0, \dots, 0)$ при $n = 2$. При чётном $n$ можно взять $n/2$ единиц и $n/2$ минус-единиц.

8. Неравенство Минковского (E13) даёт $\det(A + B)^{1/n} \ge \det(A)^{1/n} + \det(B)^{1/n}$. Возведение в $n$-ю степень и тождество $(x + y)^n \ge x^n + y^n$ для $x, y \ge 0$ дают $\det(A + B) \ge \det A + \det B$.

9. Матрица Грама $G_{ij} = c$ при $i \ne j$, $G_{ii} = 1$. В блочной форме: $G = (1 - c) I + c \mathbf{1} \mathbf{1}^T$. Собственные числа: $1 + (k - 1) c$ (на векторе $\mathbf{1}$) и $1 - c$ (кратность $k - 1$, на ортогональном дополнении). Условие $G \succeq 0$: $1 + (k - 1) c \ge 0$ и $1 - c \ge 0$. Итог: $c \in \left[ -\frac{1}{k - 1}, \, 1 \right]$.

10. Существование: положить $P = (A^T A)^{1/2}$ (это PD, так как $A$ невырождена), $U = A P^{-1}$. Проверка: $U^T U = P^{-1} A^T A P^{-1} = P^{-1} P^2 P^{-1} = I$.

    Единственность: пусть $A = U_1 P_1 = U_2 P_2$ — два таких разложения. Тогда $U_2^{-1} U_1 = P_2 P_1^{-1}$. Левая часть ортогональна, правая — произведение двух PD-матриц. Единственная матрица, одновременно ортогональная и PD-симметричная, — это $I$. Значит, $U_1 = U_2$ и $P_1 = P_2$.
