# Eigenvalues, тождества со следом, spectral radius, ранговые аргументы

## Scope
Спектральные и ранговые техники для квадратных матриц над $\mathbb{R}$, $\mathbb{C}$ и произвольным полем. След (trace) как линейный функционал и как сумма собственных значений. Связь спектра, характеристического и минимального многочленов. Спектр произведения $AB$ и $BA$. Критерий нильпотентности через след степеней. Спектральный радиус. Ранг через миноры, ядро и образ, разложения в сумму матриц ранга один. Неравенства Сильвестра и Фробениуса для ранга. Идемпотенты. Триангуляризация Шура. Эрмитовы матрицы, отношение Рэлея, чередование. Структурные приёмы из теории Жордановой формы и теоремы Шоды.

## Определения

Везде ниже $A \in M_n(F)$, поле $F$ — обычно $\mathbb{C}$ или $\mathbb{R}$.

- **След** (trace): $\operatorname{tr}(A) = \sum_{i=1}^n A_{ii}$.
- **Характеристический многочлен** (characteristic polynomial): $\chi_A(\lambda) = \det(\lambda I - A)$. Степени $n$, нормирован.
- **Минимальный многочлен** (minimal polynomial): нормированный $\mu_A$ наименьшей степени с $\mu_A(A) = 0$. Делит $\chi_A$.
- **Собственное значение** (eigenvalue): корень $\chi_A$. Собственный вектор: $v \ne 0$, $Av = \lambda v$.
- **Спектр** $\operatorname{spec}(A)$: мультимножество корней $\chi_A$ с алгебраическими кратностями.
- **Алгебраическая кратность** $\lambda$: кратность корня в $\chi_A$. **Геометрическая** — $\dim \ker(A - \lambda I)$. Геометрическая $\le$ алгебраическая.
- **Спектральный радиус** (spectral radius): $\rho(A) = \max_{\lambda \in \operatorname{spec}(A)} |\lambda|$.
- **Ранг** (rank): $\operatorname{rank}(A) = \dim \operatorname{im}(A) = n - \dim \ker(A)$. Равен максимальному размеру невырожденного минора.
- **Нильпотентная** (nilpotent): $A^k = 0$ для некоторого $k \ge 1$. Эквивалентно: единственное собственное значение — ноль.
- **Идемпотент**: $A^2 = A$.
- **Эрмитова** (hermitian): $A^* = A$ (комплексное сопряжение и транспонирование). **Симметричная**: $A^T = A$ над $\mathbb{R}$. **Кососимметричная**: $A^T = -A$.
- **Нормальная**: $AA^* = A^*A$.
- **Подобие** (similarity): $B = S^{-1}AS$ при невырожденной $S$. Сохраняет $\chi_A$, $\mu_A$, спектр.

## Записи — Основные

### E1. Свойства следа: цикличность, сумма собственных значений, инвариантность

Формулировка. Для согласованных по размеру $A, B, C$:
$$\operatorname{tr}(AB) = \operatorname{tr}(BA), \qquad \operatorname{tr}(ABC) = \operatorname{tr}(BCA) = \operatorname{tr}(CAB).$$
След инвариантен под подобием: $\operatorname{tr}(S^{-1}AS) = \operatorname{tr}(A)$. Над алгебраически замкнутым полем
$$\operatorname{tr}(A) = \sum_{i=1}^n \lambda_i(A), \qquad \operatorname{tr}(A^k) = \sum_{i=1}^n \lambda_i^k$$
с учётом алгебраических кратностей.

Условия применимости. Цикличность — над любым кольцом. Формула $\operatorname{tr}(A^k) = \sum \lambda_i^k$ требует, чтобы $\chi_A$ полностью раскладывался — иначе работаем в алгебраическом замыкании.

Пример без условия. Над $\mathbb{R}$ матрица поворота $R_{\pi/2}$ имеет $\operatorname{tr}(R) = 0$, но вещественных собственных значений у неё нет — сумма «собственных значений» имеет смысл только в $\mathbb{C}$, где $\lambda = \pm i$.

Идея доказательства. $\operatorname{tr}(AB) = \sum_{i,j} A_{ij}B_{ji} = \operatorname{tr}(BA)$. Сумма собственных значений — коэффициент при $\lambda^{n-1}$ в $\chi_A$ со знаком, прямой пересчёт.

### E2. Спектр произведения: $AB$ и $BA$ имеют общие ненулевые собственные значения

Формулировка. Пусть $A$ размера $m \times n$, $B$ размера $n \times m$, $m \le n$. Тогда
$$\det(\lambda I_n - BA) = \lambda^{n-m}\det(\lambda I_m - AB).$$
Эквивалентно: ненулевая часть спектров $AB$ и $BA$ совпадает, кратности тоже. При $m = n$ характеристические многочлены полностью совпадают.

Условия применимости. Любое поле, любые согласованные размеры. Тождество $\det(I + AB) = \det(I + BA)$ — частный случай при $\lambda = -1$ (см. файл «Определители», запись E4).

Пример без условия. Жордановы блоки для нуля у $AB$ и $BA$ могут отличаться. Возьми $A = (1\ 0)$, $B = \begin{pmatrix}0\\0\end{pmatrix}$ — тогда $AB = 0 \in M_1$, $BA = 0 \in M_2$. Спектры совпадают на ненулевой части (она пуста), но размеры разные.

Идея доказательства. При обратимой $A$ имеем $BA = A^{-1}(AB)A$ — подобие. Общий случай — плотность обратимых матриц и непрерывность $\chi$.

### E3. Теорема Кэли–Гамильтона и образ спектра под многочленом

Формулировка. Кэли–Гамильтон: $\chi_A(A) = 0$. Минимальный многочлен $\mu_A$ делит $\chi_A$ и имеет те же корни (с возможно меньшими кратностями).

Образ спектра. Для $p \in F[x]$
$$\operatorname{spec}(p(A)) = \{p(\lambda) : \lambda \in \operatorname{spec}(A)\}$$
с учётом алгебраических кратностей. Для обратимой $A$ получаем $A^{-1}$ как многочлен от $A$ степени $\le n - 1$.

Условия применимости. Кэли–Гамильтон работает над любым коммутативным кольцом. Формула про спектр многочлена использует наличие $n$ корней — нужно алгебраическое замыкание.

Пример без условия. В некоммутативной алгебре кватернионов матрица $\begin{pmatrix} i & 0 \\ 0 & j \end{pmatrix}$ не удовлетворяет своему наивно записанному «характеристическому» многочлену с коэффициентами слева — некоммутативность ломает определитель.

Идея доказательства. Триангуляризация Шура (E9): $A = UTU^*$, тогда $p(A) = U p(T) U^*$, а $p(T)$ — верхнетреугольная с диагональю $p(\lambda_i)$.

### E4. Тождества Ньютона и критерий нильпотентности через след

Формулировка. Пусть $p_k = \operatorname{tr}(A^k)$, $e_k$ — коэффициенты $\chi_A(\lambda) = \sum (-1)^k e_k \lambda^{n-k}$. Тождества Ньютона:
$$p_k - e_1 p_{k-1} + e_2 p_{k-2} - \dots + (-1)^{k-1} k e_k = 0, \quad k = 1, \dots, n.$$
В характеристике ноль $p_1, \dots, p_n$ однозначно определяют $e_1, \dots, e_n$.

Критерий нильпотентности. Над полем характеристики ноль: $A$ нильпотентна тогда и только тогда, когда $\operatorname{tr}(A^k) = 0$ для всех $k = 1, \dots, n$.

Условия применимости. Тождества Ньютона работают всегда. Критерий нильпотентности через след — характеристика ноль (или $> n$).

Пример без условия. Над $\mathbb{F}_2$ матрица $I_2$ ненильпотентна, но $\operatorname{tr}(I_2^k) = 2 \equiv 0$ для всех $k$. Критерий ломается.

Идея доказательства. Тождества Ньютона — раскладка $\log\det(I - xA) = -\sum p_k x^k / k$. Нильпотентность: если все $p_k = 0$, то по Ньютону все $e_k = 0$, значит $\chi_A(\lambda) = \lambda^n$, и Кэли–Гамильтон даёт $A^n = 0$.

### E5. Ранг: миноры, ядро и образ, разложение в сумму матриц ранга один

Формулировка. Для $A \in M_{m,n}(F)$ эквивалентны:
1. $\operatorname{rank}(A) = r$.
2. Максимальный размер ненулевого минора $A$ равен $r$.
3. $A = \sum_{k=1}^r u_k v_k^T$ — минимальное представление в сумму матриц ранга один.
4. $\dim \operatorname{im}(A) = r$, то есть $\dim \ker(A) = n - r$.

В частности, $\operatorname{rank}(A) \le 1$ тогда и только тогда, когда $A = uv^T$, и тогда же когда все миноры $2 \times 2$ равны нулю.

Условия применимости. Любое поле.

Идея доказательства. Эквивалентность через гауссово исключение. Минорный критерий: если все миноры размера $r + 1$ нулевые, то любые $r + 1$ строк линейно зависимы.

---

### E6. Неравенства Сильвестра и Фробениуса для ранга

Формулировка. Сильвестр: для $A \in M_{m,n}$, $B \in M_{n,p}$
$$\operatorname{rank}(A) + \operatorname{rank}(B) - n \le \operatorname{rank}(AB) \le \min(\operatorname{rank}(A), \operatorname{rank}(B)).$$
Фробениус: для $A, B, C$ согласованных размеров
$$\operatorname{rank}(AB) + \operatorname{rank}(BC) \le \operatorname{rank}(B) + \operatorname{rank}(ABC).$$
При $B = I$ Фробениус даёт Сильвестра.

Условия применимости. Любое поле, любые согласованные размеры.

Идея доказательства. Сильвестр: $\operatorname{rank}(AB) = \dim A(\operatorname{im} B) = \operatorname{rank}(B) - \dim(\operatorname{im} B \cap \ker A) \ge \operatorname{rank}(B) - (n - \operatorname{rank}(A))$. Фробениус — применение Сильвестра к ограничению $A$ на $\operatorname{im}(BC) \subseteq \operatorname{im}(B)$.

Дополнение. Из неравенства Сильвестра при $AB = 0$ сразу получается $\operatorname{rank}(A) + \operatorname{rank}(B) \le n$.

## Записи — Стандартные

### E7. Идемпотент: $A^2 = A$ влечёт диагонализуемость и $\operatorname{rank}(A) = \operatorname{tr}(A)$

Формулировка. Пусть $A^2 = A$. Тогда:
1. $\operatorname{spec}(A) \subseteq \{0, 1\}$.
2. $A$ диагонализуема.
3. $F^n = \ker(A) \oplus \operatorname{im}(A)$.
4. $\operatorname{rank}(A) = \operatorname{tr}(A)$ — оба равны количеству единичных собственных значений.

Условия применимости. Любое поле для пунктов 1–3. Для $\operatorname{rank} = \operatorname{tr}$ удобна характеристика ноль; в характеристике $p$ trace берётся в $F$, и это всё ещё число единичных собственных значений по модулю $p$.

Пример без условия. Над $\mathbb{F}_2$: матрица $\begin{pmatrix} 1 & 1 \\ 0 & 0 \end{pmatrix}$ — идемпотент, $\operatorname{rank} = 1$, $\operatorname{tr} = 1$. Работает. Но при увеличении размера до $p \times p$ из $p$ копий — trace становится нулём в $\mathbb{F}_p$.

Идея доказательства. $\mu_A$ делит $x^2 - x = x(x-1)$ — простые корни, отсюда диагонализуемость. След равен сумме собственных значений, а они в $\{0,1\}$.

### E8. Полиномиальный аннулятор: ограничение спектра и противоречия по следу

Формулировка. Если $p(A) = 0$ для многочлена $p$, то $\mu_A$ делит $p$, и $\operatorname{spec}(A) \subseteq$ корней $p$.

Ограничения по следу. Если корни $p$ — $r_1, \dots, r_s$ с неизвестными кратностями $m_1, \dots, m_s$, $\sum m_i = n$, то
$$\operatorname{tr}(A^k) = \sum_{i=1}^s m_i r_i^k.$$
Система линейных уравнений по $m_i$ часто переопределена — отсюда противоречия в задачах «не существует такой $A$».

Условия применимости. Любое поле. Диагонализуемость по полиномиальному аннулятору требует простых корней $\mu_A$ — в характеристике $p$ может ломаться.

Пример без условия. Если $p$ имеет кратный корень, диагонализуемость не следует. $A = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}$ удовлетворяет $p(x) = x^2 = 0$, не диагонализуема.

Идея доказательства. Делимость $\mu_A \mid p$ — стандарт. Trace-уравнения — подсчёт по собственным значениям.

---

### E9. Триангуляризация Шура и одновременная триангуляризация

Формулировка. Над $\mathbb{C}$: для любой $A$ существует унитарная $U$ с $U^*AU = T$ — верхнетреугольной, на диагонали стоят собственные значения.

Одновременная триангуляризация. Семейство $\mathcal{F} \subseteq M_n(\mathbb{C})$ одновременно триангуляризуемо тогда и только тогда, когда у него есть общий собственный вектор, и это свойство наследуется на факторпространство. Достаточное условие — коммутативность всех пар $[A, B] = 0$.

Условия применимости. Триангуляризация Шура — над алгебраически замкнутым полем. Над $\mathbb{R}$ — квазитреугольная форма с блоками $1 \times 1$ и $2 \times 2$.

Пример без условия. Над $\mathbb{Q}$ матрица $\begin{pmatrix} 0 & -1 \\ 1 & 0 \end{pmatrix}$ — собственные значения $\pm i \notin \mathbb{Q}$, никакого треугольного представления нет.

Идея доказательства. Индукция по $n$. Берём собственный вектор $v_1$, продолжаем до ортонормированного базиса. Матрица $A$ в этом базисе блочно-треугольная, рекурсия на $(n-1) \times (n-1)$ блок.

### E10. Спектральный радиус и формула Гельфанда

Формулировка. Для любой матричной нормы $\|\cdot\|$ и $A \in M_n(\mathbb{C})$
$$\rho(A) = \lim_{k \to \infty} \|A^k\|^{1/k} = \inf_{k \ge 1} \|A^k\|^{1/k}.$$
Следствие: $A^k \to 0$ тогда и только тогда, когда $\rho(A) < 1$.

Условия применимости. Норма должна быть субмультипликативной ($\|AB\| \le \|A\|\|B\|$). Над $\mathbb{R}$ работает с тем же определением — собственные значения берутся в $\mathbb{C}$.

Пример без условия. Без субмультипликативности нижняя оценка $\|A^k\| \ge \rho(A)^k$ может ломаться. Если взять «странную» норму, то $\rho$ всё равно сохраняется, но связь с $\|A^k\|^{1/k}$ через равенство нужно отдельно проверять.

Идея доказательства. Через триангуляризацию Шура: $A = UTU^*$, оценить $\|T^k\|$, разложив $T = D + N$ на диагональную и строго треугольную (нильпотентную) части. Нильпотентная часть растёт полиномиально, экспонента диктуется $\rho$.

### E11. Вещественные матрицы: невещественные собственные значения идут парами

Формулировка. Пусть $A \in M_n(\mathbb{R})$. Если $\lambda \in \mathbb{C} \setminus \mathbb{R}$ — собственное значение с алгебраической кратностью $m$, то и $\bar\lambda$ — собственное значение с той же кратностью $m$. Количество невещественных собственных значений с кратностями — чётное.

Следствия. При нечётном $n$ есть хотя бы одно вещественное собственное значение. $\det(A)$ и $\operatorname{tr}(A)$ всегда вещественны.

Условия применимости. $A \in M_n(\mathbb{R})$. Над $\mathbb{Q}$ парность собственных значений тоже есть (по любому подполу $\mathbb{C}$, стабильному под сопряжением).

Пример без условия. Над $\mathbb{C}$ собственные значения произвольные — парности нет. Матрица $\operatorname{diag}(i)$ в $M_1(\mathbb{C})$ — единственное собственное значение $i$, не парное.

Идея доказательства. $\chi_A \in \mathbb{R}[x]$, его комплексные корни идут сопряжёнными парами.

---

### E12. Эрмитовы матрицы: вещественный спектр, Рэлей, Куранта–Фишера, чередование

Формулировка. Пусть $A = A^*$ ($A$ эрмитова). Тогда:
1. Все собственные значения вещественны, $A$ унитарно диагонализуема.
2. Рэлей (Rayleigh): $\lambda_1 \le \frac{x^*Ax}{x^*x} \le \lambda_n$ для $x \ne 0$, собственные значения отсортированы по возрастанию.
3. Куранта–Фишера (Courant–Fischer):
$$\lambda_k = \max_{\dim V = k} \min_{x \in V \setminus 0} \frac{x^*Ax}{x^*x}.$$
4. Чередование (Cauchy interlacing): для главного $(n-1)\times(n-1)$ минора $B$ с собственными значениями $\mu_1 \le \dots \le \mu_{n-1}$
$$\lambda_1 \le \mu_1 \le \lambda_2 \le \mu_2 \le \dots \le \mu_{n-1} \le \lambda_n.$$

Условия применимости. Эрмитовость. Без неё спектр может быть невещественным и $x^*Ax$ не контролирует собственные значения.

Пример без условия. Жорданов блок $\begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}$: $x^*Ax$ принимает любые значения, но единственное собственное значение — ноль.

Идея доказательства. $\lambda\langle v, v\rangle = \langle Av, v\rangle = \langle v, Av\rangle = \bar\lambda\langle v, v\rangle$, отсюда $\lambda \in \mathbb{R}$. Куранта–Фишера — компактность сферы и лагранжевы множители. Чередование — Куранта–Фишера на подпространстве.

## Записи — Редкие

### E13. Жорданова форма: ранги степеней кодируют структуру

Формулировка. Над $\mathbb{C}$ матрица $A$ подобна $\bigoplus J_{m_i}(\lambda_i)$ — прямой сумме жордановых блоков. Размеры блоков для каждого $\lambda$ однозначны и восстанавливаются по последовательности
$$d_k(\lambda) = \operatorname{rank}((A - \lambda I)^{k-1}) - 2\operatorname{rank}((A - \lambda I)^k) + \operatorname{rank}((A - \lambda I)^{k+1})$$
— это число блоков размера ровно $k$ для собственного значения $\lambda$.

Условия применимости. Алгебраически замкнутое поле. Над $\mathbb{R}$ — вещественная Жорданова форма с блоками $2 \times 2$ для пар $\lambda, \bar\lambda$.

Идея доказательства. Декомпозиция в обобщённые собственные подпространства $\ker(A - \lambda I)^n$. На каждом $A - \lambda I$ нильпотентна, выбираем циклический базис цепочками.

### E14. Теорема Шоды: матрица — коммутатор тогда и только тогда, когда $\operatorname{tr} = 0$

Формулировка. Над полем характеристики ноль $A \in M_n(F)$ представима как $A = BC - CB$ тогда и только тогда, когда $\operatorname{tr}(A) = 0$.

Условия применимости. Поле (Альберт–Мукенхуп распространили на произвольную характеристику для поля). Для общих коммутативных колец неверно.

Идея доказательства. Один из направлений тривиален: $\operatorname{tr}(BC - CB) = 0$. Обратное — сначала сопряжением приводим $A$ к матрице с нулевой диагональю, затем берём $B = \operatorname{diag}(1, 2, \dots, n)$, и $C$ восстанавливается из $(BC - CB)_{ij} = (i - j)C_{ij}$.

## Self-test

1. $A \in M_n(\mathbb{C})$, $\operatorname{tr}(A^k) = n$ для всех $k = 1, \dots, n$. Найди $\chi_A(\lambda)$.

2. $A, B \in M_n(\mathbb{C})$, $AB - BA = A$. Покажи, что $A$ нильпотентна.

3. $A \in M_n(\mathbb{R})$, $A^3 = A + I$. Покажи, что $\det(A) > 0$.

4. $A \in M_n(\mathbb{R})$ обратима, все $a_{ij} \ge 0$, и $A^{-1}$ имеет все элементы $\ge 0$. Что можно сказать о структуре $A$?

5. $A \in M_n(\mathbb{C})$, $A^k = I$ для некоторого $k \ge 1$. Покажи, что $|\operatorname{tr}(A)| \le n$, и равенство ровно тогда, когда $A = \zeta I$ для корня $\zeta^k = 1$.

6. $A \in M_n(\mathbb{C})$, $\operatorname{rank}(A - I) = 1$. Вырази $\det(A)$ через $\operatorname{tr}(A)$.

7. $A, B \in M_n(\mathbb{C})$, $AB = 0$. Докажи, что $\operatorname{rank}(A) + \operatorname{rank}(B) \le n$.

8. $A \in M_n(\mathbb{C})$, $A \ne 0$, $A^2 = 0$. Покажи, что $\operatorname{rank}(A) \le n/2$, и опиши форму $A$ в подходящем базисе.

9. $A \in M_n(\mathbb{R})$ кососимметрична. Покажи, что $\operatorname{rank}(A)$ чётный, а $\det(A) \ge 0$.

## Решения

1. Тождества Ньютона (E4) с $p_k = n$ дают $e_k = \binom{n}{k}$, то есть $\chi_A(\lambda) = (\lambda - 1)^n$. Замечание: это не значит $A = I$ — например, $A = I + N$ с нильпотентной $N$ даёт тот же характеристический многочлен.

2. $\operatorname{tr}(A) = \operatorname{tr}(AB - BA) = 0$. Дальше по индукции: $A^m B - B A^m = m A^m$ (умножь рекуррентно). Берём след: $0 = m\operatorname{tr}(A^m)$, отсюда $\operatorname{tr}(A^m) = 0$ для всех $m \ge 1$. По E4 $A$ нильпотентна.

3. $A^3 - A - I = 0$. Корни $p(x) = x^3 - x - 1$: один вещественный $\alpha \approx 1.32$ и пара комплексно сопряжённых $\beta, \bar\beta$ с $\alpha|\beta|^2 = 1$ по теореме Виета. $\det(A) = \alpha^a (\beta\bar\beta)^b = \alpha^{a - b}$, где $a + 2b = n$. Поскольку $\alpha > 0$, $\det(A) > 0$.

4. $A$ — мономиальная (произведение перестановочной матрицы на положительную диагональную). Идея: $i$-я строка $A$ ортогональна $j$-му столбцу $A^{-1}$ при $i \ne j$. Если в строке $A$ больше одной положительной позиции, то для ортогональности соответствующие позиции $A^{-1}$ должны иметь разные знаки — противоречит неотрицательности $A^{-1}$. Значит в каждой строке $A$ ровно один ненулевой элемент.

5. $A^k = I$. Минимальный многочлен делит $x^k - 1$ с простыми корнями, значит $A$ диагонализуема, $\operatorname{spec}(A) \subseteq \{\zeta : \zeta^k = 1\}$. По неравенству треугольника $|\operatorname{tr}(A)| = |\sum \zeta_i| \le n$. Равенство — все $\zeta_i$ сонаправлены, модули равны единице, значит совпадают.

6. $A - I = uv^T$, то есть $A = I + uv^T$. По лемме об определителе при возмущении ранга один (см. файл «Определители», E5): $\det(A) = 1 + v^T u$. С другой стороны $\operatorname{tr}(A) = n + v^T u$, поэтому $v^T u = \operatorname{tr}(A) - n$. Итог: $\det(A) = \operatorname{tr}(A) - (n - 1)$.

7. $AB = 0$, значит $\operatorname{im}(B) \subseteq \ker(A)$. Отсюда $\operatorname{rank}(B) \le \dim \ker(A) = n - \operatorname{rank}(A)$. Эквивалентно — Сильвестр для ранга (E6): $\operatorname{rank}(A) + \operatorname{rank}(B) - n \le \operatorname{rank}(AB) = 0$.

8. $A^2 = 0$ означает $\operatorname{im}(A) \subseteq \ker(A)$, откуда $r = \operatorname{rank}(A) \le n - r$, то есть $r \le n/2$. Возьмём базис $u_1, \dots, u_r$ образа, дополним до базиса ядра векторами $w_1, \dots, w_{n - 2r}$, и выберем $v_1, \dots, v_r$ с $Av_i = u_i$. В базисе $(u_1, \dots, u_r, w_1, \dots, w_{n-2r}, v_1, \dots, v_r)$ матрица $A$ имеет блочную форму $\begin{pmatrix} 0 & 0 & I_r \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{pmatrix}$.

9. $A^T = -A$, поэтому $\chi_A(\lambda) = \det(\lambda I - A) = \det(\lambda I + A^T) = (-1)^n \det(-\lambda I - A^T) = (-1)^n \chi_{-A^T}(-\lambda) = (-1)^n \chi_A(-\lambda)$. Это значит $\operatorname{spec}(A) = -\operatorname{spec}(A)$ — ненулевые собственные значения идут парами $\pm \lambda$. Дополнительно, вещественная $A$ даёт парность сопряжения (E11). Совместно — ненулевые собственные значения идут четвёрками $\pm i\mu$ (или парами, если $\mu \in \mathbb{R}$, но тут собственные значения чисто мнимые), и их количество чётно. $\det(A) = \prod(i\mu_k)(-i\mu_k) = \prod \mu_k^2 \ge 0$.
