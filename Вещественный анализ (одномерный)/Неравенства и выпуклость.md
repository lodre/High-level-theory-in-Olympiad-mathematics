# Неравенства и выпуклость: Cauchy–Schwarz, Jensen, AM–GM, rearrangement, Chebyshev + интегральные аналоги; convex / log-convex функции, supporting hyperplane

## Scope
Базовый арсенал классических неравенств (Cauchy–Schwarz, AM–GM, Jensen, Hölder, power mean, rearrangement, Chebyshev) и его выпуклый каркас: характеризация convex / log-convex функций через монотонность секущих, supporting line, three-chord lemma. Всё с интегральными аналогами и весовыми обобщениями. Дальше — рабочие тяжеловесы IMC: Karamata (majorization), Hardy, Carleman, Schur + SOS, tangent line trick, Hermite–Hadamard, Bohr–Mollerup, Young (как фундамент Hölder). Финал — экзотика для контрпримеров и тонких задач: Maclaurin / Newton inequalities, Steffensen, Bernoulli с дробной степенью.

## Записи — Core

### E1. AM-GM inequality (неравенство о средних) и weighted AM-GM

Формулировка. Для $x_1, \dots, x_n \geq 0$
$$\frac{x_1 + \dots + x_n}{n} \geq \sqrt[n]{x_1 \cdots x_n},$$
равенство $\Leftrightarrow$ все $x_i$ равны. Weighted: для $w_i \geq 0$, $\sum w_i = 1$,
$$\sum_{i=1}^n w_i x_i \geq \prod_{i=1}^n x_i^{w_i}.$$

Условия применимости. Все $x_i \geq 0$. Допустимы $x_i = 0$ (правая часть зануляется). Для нестрогой $\geq$ в weighted версии веса $w_i$ — произвольные неотрицательные с $\sum w_i = 1$.

Идея доказательства. Cauchy's forward-backward induction (степени двойки + спуск), либо прямо: $\ln$ переводит в Jensen для вогнутой $\ln$. Weighted: то же $\ln$-преобразование с дискретным Jensen.

Варианты и обобщения.
- Power-mean $M_r = (\sum w_i x_i^r)^{1/r}$ монотонно возрастает по $r \in \mathbb{R}$ (E4); AM-GM = частный случай $M_1 \geq M_0$.
- AM–GM–HM: $\frac{\sum x_i}{n} \geq \sqrt[n]{\prod x_i} \geq \frac{n}{\sum 1/x_i}$.
- Reverse AM–GM (Polya): для $x_i \in [m, M]$ оценка через ratio Kantorovich-style.
- Continuous (integral): $\exp\left(\frac{1}{b-a}\int_a^b \ln f\right) \leq \frac{1}{b-a}\int_a^b f$ — частный случай интегрального Jensen (E3).

Используется в. [IMC-1996-DAY2-6 — Carleman через weighted AM-GM с $c_n = (n+1)^n / n^{n-1}$], [IMC-2001-DAY2-6 — телескопическое произведение синусов через AM-GM на $g(\theta) = |\sin\theta||\sin 2\theta|^{1/2}$], [IMC-2022-1 — $f + 1/f \geq 2$ из AM-GM при $f(x)f(1-x) = 1$], типовая редукция $a + b + c \geq 3\sqrt[3]{abc}$.

### E2. Cauchy–Schwarz inequality (неравенство Коши–Буняковского–Шварца)

Формулировка. Дискретно:
$$\left(\sum_{i=1}^n a_i b_i\right)^2 \leq \left(\sum a_i^2\right)\left(\sum b_i^2\right).$$
Интегрально:
$$\left(\int_a^b f(x) g(x)\,dx\right)^2 \leq \int_a^b f^2 \int_a^b g^2.$$
Equality $\Leftrightarrow$ $a_i = \lambda b_i$ (соотв. $f = \lambda g$ п.в.).

Условия применимости. Любые $a_i, b_i \in \mathbb{R}$ или $\mathbb{C}$ (в комплексной форме $|\langle a, b\rangle|^2 \leq \|a\|^2 \|b\|^2$). Для интегральной формы — $f, g \in L^2$.

Идея доказательства. Дискриминант квадратичного полинома $P(t) = \sum (a_i + t b_i)^2 \geq 0$; либо Lagrange identity $\left(\sum a_i^2\right)\left(\sum b_i^2\right) - \left(\sum a_i b_i\right)^2 = \sum_{i<j}(a_i b_j - a_j b_i)^2$.

Варианты и обобщения.
- **Engel form (Titu's lemma)**: $\displaystyle\sum_{i=1}^n \frac{a_i^2}{b_i} \geq \frac{(\sum a_i)^2}{\sum b_i}$ при $b_i > 0$. Равносильно CS с $a_i / \sqrt{b_i}$ и $\sqrt{b_i}$ — ключевой workhorse.
- **Hölder** (E4) — обобщение на произвольные показатели.
- **Cauchy–Schwarz–Bunyakovsky variational form**: $\|x\| = \sup_{\|y\|=1} \langle x, y\rangle$; даёт способ оценивать линейные функционалы.
- **Polarization identity**: восстанавливает скалярное произведение из нормы.

Используется в. [IMC-2000-4 — двойная сумма $(\sum x_i/\sqrt{i})^2 = \sum x_i x_j / \sqrt{ij}$], [IMC-2016-3 — Cauchy–Schwarz на взвешенных суммах $b_i(a_i - b_i)/(a_i+b_i)$], [IMC-2018-1 — CS на рядах $\sum \sqrt{a/b}$], [IMC-2025-2 — вариационная задача $\int (f'')^2 \geq 15$ через CS + множители Лагранжа]; абсолютный workhorse, проверяется первой.

### E3. Jensen's inequality (неравенство Йенсена), discrete + integral

Формулировка. Для $f: I \to \mathbb{R}$ выпуклой и вероятностной меры $\mu$ на $I$:
$$f\!\left(\int x \,d\mu\right) \leq \int f(x)\,d\mu.$$
Дискретно: $f\!\left(\sum w_i x_i\right) \leq \sum w_i f(x_i)$ при $w_i \geq 0$, $\sum w_i = 1$. Для concave — обратное.

Условия применимости. $f$ выпукла на интервале, содержащем все $x_i$ (или $\mathrm{supp}\,\mu$); веса $w_i \geq 0$, $\sum w_i = 1$. Strict inequality при строгой выпуклости и не-вырожденной мере (не сосредоточена в одной точке).

Идея доказательства. Supporting line: для каждой $x_0 \in \mathrm{int}\,I$ существует $c$ с $f(x) \geq f(x_0) + c(x - x_0)$. Подставляем $x_0 = \int x\,d\mu$ и интегрируем — линейный член обнуляется.

Варианты и обобщения.
- **Чебышёва–Йенсена**: для wtd среднего; comparison разных средних $M_r$ (E4).
- **Reverse Jensen** (Lah-Ribarič / Edmundson): для $f$ выпуклой и $x_i \in [a, b]$ — оценка сверху линейной функцией концов.
- **Pečarić–Stolarski**: refinement через дополнительные supporting hyperplanes на подынтервалах.
- **Jensen–Steffensen**: для не-стандартных весов (могут быть отрицательны), но монотонно изменяются.

Используется в. [IMC-2004-3 — двусторонние оценки на $\sum x_i$ при $\sum \sin x_i = 1$: верх через хорду $2x/\pi \leq \sin x$, низ через Jensen $1/n \leq \sin(\bar x)$, ответ $\pi/2 - n\arcsin(1/n) \to \pi/2 - 1$], рутинная проверка convex/concave перед суммированием.

### E4. Hölder's inequality (Гёльдер) и Power-mean inequality (степенные средние)

Формулировка. Для $p, q > 1$ с $1/p + 1/q = 1$:
$$\sum_{i=1}^n |a_i b_i| \leq \left(\sum |a_i|^p\right)^{1/p}\left(\sum |b_i|^q\right)^{1/q},$$
интегрально $\int |fg| \leq \|f\|_p \|g\|_q$. Допускается $p = 1, q = \infty$.

Power mean: для $w_i \geq 0$, $\sum w_i = 1$, $x_i > 0$,
$$M_r(x; w) = \left(\sum w_i x_i^r\right)^{1/r}, \quad r \in \mathbb{R}\setminus\{0\},\quad M_0 = \prod x_i^{w_i}.$$
Тогда $r \mapsto M_r$ строго возрастает (если не все $x_i$ равны), $M_{-\infty} = \min$, $M_{\infty} = \max$.

Условия применимости. Hölder: $a_i, b_i \in \mathbb{R}$ либо $\mathbb{C}$; для интегральной формы $f \in L^p, g \in L^q$. Power mean: $x_i > 0$ (для $r \leq 0$); веса $w_i$ можно заменить нормализованной мерой.

Идея доказательства. Hölder из Young's $|ab| \leq |a|^p/p + |b|^q/q$ (E15) после нормализации $\|a\|_p = \|b\|_q = 1$. Power-mean monotonicity: $\log M_r = \frac{1}{r}\log \sum w_i e^{r \log x_i}$ — log-convex, плюс L'Hôpital для непрерывности в $r=0$. Альтернативно — Jensen с $f(t) = t^{r/s}$ для $r > s > 0$.

Варианты и обобщения.
- **Generalized Hölder**: $\sum a_i b_i c_i \leq \|a\|_p \|b\|_q \|c\|_r$, $1/p + 1/q + 1/r = 1$.
- **Reverse Hölder**: для $0 < p < 1$ (с конъюгатным $q < 0$): неравенство меняет знак.
- **Lyapunov inequality**: $\|f\|_q^{1/q} \leq \|f\|_p^{a/p} \|f\|_r^{b/r}$, $1/q = a/p + b/r$, $a+b=1$ — log-convexity функции $r \mapsto \log\|f\|_{1/r}$.
- **Continuous power mean**: $M_r(f) = \left(\frac{1}{|I|}\int f^r\right)^{1/r}$ — те же свойства.

Используется в. [IMC-1995-6 — строгая выпуклость $|t|^p$ при $p > 1$ позволяет исключить «угловые» решения], [IMC-1997-5 — норма $\|x+y\|_p \geq \|x\|_p$ через индекс-сплит знаков, контрпример при $p < 1$], основа для $L^p$-аргументов.

### E5. Convex function characterizations: secant slope, supporting line, three-chord (выпуклые функции — секущие, опорная прямая, three-chord lemma)

Формулировка. Для $f: I \to \mathbb{R}$ эквивалентны:
1. $f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y)$ при $\lambda \in [0,1]$.
2. **Secant slope monotonicity**: $a < b < c \Rightarrow \displaystyle\frac{f(b)-f(a)}{b-a} \leq \frac{f(c)-f(a)}{c-a} \leq \frac{f(c)-f(b)}{c-b}$ — three-chord lemma.
3. Для каждой $x_0 \in \mathrm{int}\,I$ существует $c$ (опорный наклон, в общем случае не единственный) с $f(x) \geq f(x_0) + c(x - x_0)$ — **supporting line**.
4. Если $f$ дифференцируема, $f'$ не убывает; если $f \in C^2$, $f'' \geq 0$.

Условия применимости. $I$ — интервал. Convex $\Rightarrow$ непрерывна на $\mathrm{int}\,I$, имеет односторонние производные $f'_\pm$ (монотонно возрастающие, совпадают вне счётного множества), Lipschitz на компактах внутри $I$.

Идея доказательства. (1)$\Leftrightarrow$(2): прямой пересчёт $\lambda$ через $a, b, c$. (2)$\Rightarrow$(3): любое значение между $f'_-(x_0)$ и $f'_+(x_0)$ — подходящий $c$. (3)$\Rightarrow$(1): из supporting line в точке $z = \lambda x + (1-\lambda)y$.

Варианты и обобщения.
- **Midpoint-convex + measurable $\Rightarrow$ convex** (Sierpiński); без измеримости можно построить дикий пример (пользуясь Hamel basis).
- **Strict convexity**: либо неравенство строгое, либо $f''>0$ (almost everywhere).
- **Subdifferential** $\partial f(x_0) = [f'_-(x_0), f'_+(x_0)]$ — все возможные опорные наклоны; характеризует точки негладкости.
- Многомерный аналог — supporting hyperplane theorem (опорная гиперплоскость для convex set).

Используется в. [IMC-1994-6 — $f''>0$ даёт монотонность $b_i / a_i = f'(x_i)$ через MVT, отсюда оценка числа целых точек на выпуклой кривой], [IMC-2020-5 — анализ ОДУ $f'' f \geq 2 (f')^2$ через выпуклость $1/f$], [IMC-2004-DAY2-2 — длина графика выпуклой функции монотонна по доминирующей выпуклой майоранте]; обязательная проверка перед применением Jensen / tangent line trick.

### E6. Tangent line trick (касательной прямой) + SOS reduction

Формулировка. Для симметричного неравенства вида $\sum_{i=1}^n f(x_i) \geq C$ при ограничении $\sum x_i = S$, ищем линейную нижнюю оценку $f(x) \geq \alpha x + \beta$, **касательную** к графику $f$ в предполагаемой точке равенства $x_0 = S/n$ (или другой экстремальной). Суммирование даёт $\sum f(x_i) \geq \alpha S + n\beta = C$. Доказательство касательной — обычно SOS-факторизация $f(x) - (\alpha x + \beta) = (x - x_0)^2 \cdot g(x)$ с $g \geq 0$ на нужном интервале.

Условия применимости. (а) Точка равенства известна / угадана из симметрии. (б) Касательная глобально остаётся снизу графика на всём допустимом множестве (для глобально выпуклой $f$ — автоматически по supporting line; в общем случае нужно проверить отдельно). (в) Для не-симметричных задач — комбинация нескольких касательных в разных подобластях.

Идея доказательства. Строим $f(x) - \alpha x - \beta$, факторизуем через $(x - x_0)$ — двойной корень даёт $(x-x_0)^2 g(x)$, проверяем $g \geq 0$. Альтернатива — проверка $f''(x_0) \geq 0$ + локальная выпуклость + globальное расширение.

Варианты и обобщения.
- **Several tangent lines / piecewise linear majorant**: при наличии нескольких точек равенства (например $k$ значений в одной точке, $n-k$ в другой) — комбинированная нижняя оценка.
- **n-1 EV theorem (Cirtoaje)**: для симметричных функций экстремум при фиксированной сумме достигается, когда максимум $n-1$ переменных равны.
- **Lagrange multipliers / KKT** — формальное обобщение, но касательная даёт чисто алгебраическое доказательство без учёта boundary cases.

Используется в. [IMC-1999-DAY2-3 — $x^3 - 3x/4 + 1/4 = (x+1)(x-1/2)^2 \geq 0$ при $x \geq -1$ даёт $\sum x_i \leq n/3$ при условии $\sum x_i^3 = -n/4$; равенство при $k$ значений $-1$ и $8k$ значений $1/2$].

### E7. Rearrangement inequality (неравенство о перестановке) + Chebyshev's sum inequality (неравенство Чебышёва о суммах)

Формулировка. Rearrangement: для $a_1 \leq \dots \leq a_n$ и $b_1 \leq \dots \leq b_n$ и любой перестановки $\sigma$:
$$\sum_{i=1}^n a_i b_{n+1-i} \leq \sum_{i=1}^n a_i b_{\sigma(i)} \leq \sum_{i=1}^n a_i b_i.$$
Chebyshev's sum (для same-ordered):
$$n \sum_{i=1}^n a_i b_i \geq \left(\sum a_i\right)\left(\sum b_i\right).$$
Интегральный Chebyshev: $f, g$ монотонны в одну сторону на $[a,b]$ $\Rightarrow$ $(b-a) \int_a^b fg \geq \int_a^b f \cdot \int_a^b g$; opposite-monotonicity — обратное.

Условия применимости. Rearrangement — для конечных последовательностей; сортировка обоих наборов — критична. Chebyshev требует sortedness (или совместной монотонности интегральных функций).

Идея доказательства. Rearrangement: bubble sort — каждая транспозиция «беспорядочной» пары $(a_i - a_j)(b_i - b_j) \geq 0$ увеличивает сумму. Chebyshev: $n \sum a_i b_i - (\sum a_i)(\sum b_i) = \frac{1}{2}\sum_{i,j}(a_i - a_j)(b_i - b_j) \geq 0$ при same-ordering.

Варианты и обобщения.
- **Hardy–Littlewood rearrangement** для функций: $\int f^* g^* \geq \int fg$ (decreasing rearrangement).
- **Riesz rearrangement** (трёхлинейный): $\int f^*(x) g^*(x-y) h^*(y)\,dx\,dy \geq \int fgh$ — основа изопериметрических неравенств.
- **Polya–Szegő**: убывающая перестановка не увеличивает $L^p$-норму градиента.
- **AM-GM из rearrangement**: применяя к $\log$-преобразованным переменным.

Используется в. Контроль смешанных сумм $\sum a_i b_{\pi(i)}$ в задачах на оптимизацию по перестановкам; integral Chebyshev для оценок типа $\int_0^1 f g \geq \int f \int g$ при монотонных $f, g$.

### E8. Minkowski's inequality (Минковский, треугольное неравенство в $L^p$)

Формулировка. Для $p \geq 1$:
$$\left(\sum (a_i + b_i)^p\right)^{1/p} \leq \left(\sum a_i^p\right)^{1/p} + \left(\sum b_i^p\right)^{1/p},$$
интегрально $\|f + g\|_p \leq \|f\|_p + \|g\|_p$. Для $0 < p < 1$ на $\geq 0$ — обратное.

Условия применимости. $p \geq 1$ — стандарт. Для $p < 1$ нужно неотрицательность. Случай равенства: $a, b$ положительно пропорциональны (для $p > 1$ строгого).

Идея доказательства. $\sum (a+b)^p = \sum a (a+b)^{p-1} + \sum b(a+b)^{p-1}$, к каждой сумме — Hölder с показателями $p, p/(p-1)$.

Варианты и обобщения.
- **Continuous Minkowski integral inequality**: $\left(\int \left(\int |f(x,y)|\,dy\right)^p dx\right)^{1/p} \leq \int \left(\int |f|^p dx\right)^{1/p} dy$.
- **Mixed-norm spaces**: позволяет переключать порядок интегрирования при $L^p$-нормах.

Используется в. [IMC-1997-5 — неравенство $\|x+y\|_p \geq \|x\|_p$ для $p \geq 1$ доказывается покомпонентно через index-split по знакам $x_i, y_i$, контрпример $\|x+y\|_p < \|x\|_p$ для $p < 1$]; основа $L^p$-геометрии.

### E9. Hardy's inequality (Харди), discrete + integral

Формулировка. Discrete: для $a_n > 0$, $p > 1$, и $A_n = \frac{1}{n}\sum_{k=1}^n a_k$,
$$\sum_{n=1}^\infty A_n^p \leq \left(\frac{p}{p-1}\right)^p \sum_{n=1}^\infty a_n^p.$$
Integral: для $f \geq 0$, $F(x) = \int_0^x f$,
$$\int_0^\infty \left(\frac{F(x)}{x}\right)^p dx \leq \left(\frac{p}{p-1}\right)^p \int_0^\infty f^p\,dx.$$
Константа $(p/(p-1))^p$ — sharp, не достигается на нетривиальных функциях.

Условия применимости. $p > 1$; неотрицательность $a_n, f$. При $p = 1$ Hardy fails (расходится тривиально), при $p < 1$ — обратное направление (Copson).

Идея доказательства. (Integral) IBP с весом $x^{1-p}$: $\int_0^\infty (F/x)^p dx = -\frac{1}{p-1}\int_0^\infty F^p \cdot d(x^{1-p})$, после IBP — Hölder. (Discrete) комбинация Abel summation + Hölder.

Варианты и обобщения.
- **Hardy–Littlewood maximal inequality**: $\|Mf\|_p \leq C_p \|f\|_p$, $C_p \sim p/(p-1)$ — основа ergodic theory.
- **Hilbert's inequality**: $\sum_{m,n} a_m b_n / (m+n) \leq \pi \|a\|_2 \|b\|_2$ — двойной аналог.
- **Hardy–Knopp**: $\int_0^\infty x^{-1} \exp\!\left(\frac{1}{x}\int_0^x \log f\right) dx \leq e \int_0^\infty f / x\,dx$ — мульт.-аддит. Hardy.
- **Carleman** (E10) — мультипликативный предел Hardy при $p \to \infty$.

Используется в. ### E10. Carleman's inequality (Карлеман)

Формулировка. Для $a_n \geq 0$:
$$\sum_{n=1}^\infty (a_1 a_2 \cdots a_n)^{1/n} \leq e \sum_{n=1}^\infty a_n,$$
константа $e$ — sharp (не улучшаема).

Условия применимости. $a_n \geq 0$; правая часть может быть бесконечной (тогда тривиально). При $\sum a_n < \infty$ — нетривиально.

Идея доказательства. Веса $c_n = (n+1)^n / n^{n-1}$ дают по weighted AM-GM
$$(a_1 \cdots a_n)^{1/n} = (a_1 c_1 \cdots a_n c_n)^{1/n} / (c_1 \cdots c_n)^{1/n} \leq \frac{a_1 c_1 + \dots + a_n c_n}{n \cdot (c_1\cdots c_n)^{1/n}}.$$
Подсчёт: $(c_1 \cdots c_n)^{1/n} = (n+1)$, отсюда RHS $= \frac{1}{n(n+1)} \sum a_k c_k$. Телескоп $\sum_{n \geq k} 1/(n(n+1)) = 1/k$ + неравенство $c_k / k = (1+1/k)^k < e$ дают итог.

Варианты и обобщения.
- **Knopp's inequality**: интегральный аналог, $\int_0^\infty \exp\!\left(\frac{1}{x}\int_0^x \log f\right) dx \leq e \int_0^\infty f$.
- **Carleman with refined constant**: на конечной сумме можно показать $\sum_{n=1}^N (\prod a_k)^{1/n} \leq e (1 - O(1/\log N)) \sum a_n$.
- **Pólya's proof** через выбор $c_n$ — образец «как угадать веса для AM-GM».

Используется в. [IMC-1996-DAY2-6 — задача в точности про Carleman; стандартный pattern для оценки сумм геометрических средних].

### E11. Karamata's majorization inequality (Караматы / majorization)

Формулировка. Пусть $(x_1, \dots, x_n)$ и $(y_1, \dots, y_n)$ — упорядоченные по убыванию, и $(x) \succ (y)$ (majorize), то есть
$$\sum_{i=1}^k x_i \geq \sum_{i=1}^k y_i \text{ для } k = 1, \dots, n-1, \quad \sum_{i=1}^n x_i = \sum_{i=1}^n y_i.$$
Тогда для любой выпуклой $f: I \to \mathbb{R}$ (где $I$ содержит все $x_i, y_i$):
$$\sum_{i=1}^n f(x_i) \geq \sum_{i=1}^n f(y_i).$$

Условия применимости. Sort обязателен (decreasing). Majorization — частичный порядок: $x$ «более spread out», $y$ «более усреднён». Strict при строго convex и не-тривиальной majorization.

Идея доказательства. Abel summation + supporting line $f(x_i) \geq f(y_i) + c_i(x_i - y_i)$ с подходящим $c_i = f'(y_i)$ или из subdifferential, плюс non-decreasing $c_i$ (так как $y$ упорядочены) и условия majorization. Telescoping даёт результат.

Варианты и обобщения.
- **Hardy–Littlewood–Pólya theorem**: $(x) \succ (y)$ $\Leftrightarrow$ $y = Px$ для двойно-стохастической матрицы $P$; $\Leftrightarrow$ $y$ — выпуклая комбинация перестановок $x$.
- **Schur-convex functions** $\Phi: \mathbb{R}^n \to \mathbb{R}$: $\Phi(x) \geq \Phi(y)$ при $x \succ y$. Эквивалентно $\Phi$ симметрична + $(x_i - x_j)(\partial_i \Phi - \partial_j \Phi) \geq 0$ (Schur–Ostrowski).
- **Continuous majorization** для функций $f^*, g^*$ (decreasing rearrangements): $f \prec g$ $\Leftrightarrow$ $\int_0^t f^* \leq \int_0^t g^*$.

Используется в. Сильный инструмент для $\sum f(x_i)$ vs $\sum f(y_i)$, когда наборы связаны сохранением суммы и majorization. Часто скрыт в задачах с условиями $\sum x_i = \mathrm{const}$ и поиском экстремума симметричной выпуклой функции.

### E12. Schur's inequality (Шура) + SOS (sum-of-squares) decomposition

Формулировка (Schur). Для $x, y, z \geq 0$ и $t \geq 0$:
$$x^t (x-y)(x-z) + y^t (y-z)(y-x) + z^t (z-x)(z-y) \geq 0,$$
равенство при $x=y=z$ или двух равных и третьем = 0. Чаще всего $t = 1$ или $t = 2$.

Условия применимости. $x, y, z \geq 0$. Для $t < 0$ или произвольных вещественных — может нарушаться. WLOG $x \geq y \geq z$.

Идея доказательства. WLOG $x \geq y \geq z$: первые два слагаемых дают $(x-y)[x^t(x-z) - y^t(y-z)] \geq 0$ (обе скобки $\geq 0$), третье $z^t(x-z)(y-z) \geq 0$.

SOS общая идея: симметричная форма $P(x,y,z)$ представима как $\sum_{i<j}(x_i - x_j)^2 Q_{ij}(x,y,z)$ с $Q_{ij} \geq 0$ на нужном множестве. Тогда $P \geq 0$ автоматически. Schur при $t = 1$:
$$\sum x(x-y)(x-z) = \tfrac{1}{2}\!\left[(x-y)^2(x+y-z) + \dots\right] \text{ (циклически)},$$
что неотрицательно при $x+y \geq z$ и т.д. (треугольные условия).

Варианты и обобщения.
- **SOS на симметрических полиномах**: проверка положительности через разложение в сумму $\binom{n}{2}$ квадратов с коэффициентами, явно зависящими от $e_k$ (elementary sym).
- **Schur–Cauchy form** (взвешенный Schur): для $t = 0$ редуцируется к $\binom{x}{2} + \binom{y}{2} + \binom{z}{2} \geq xy + yz + zx$ при $x = y = z$.
- **Muirhead's inequality** (см. E16): тоже доказывается через SOS при доминировании показателей.

Используется в. [IMC-2013-4 — Schur-like оценка $(n+1)A^2 B + (n-2)B^2 - A^4 - (2n-2)AC \geq 0$ перегруппировкой в $\sum_{i<j}(x_i - x_j)^2 (\dots) \geq 0$], стандартная техника разложения симметричных оценок.

### E13. Log-convex functions (логарифмически выпуклые) + Bohr–Mollerup theorem

Формулировка. $f: I \to (0,\infty)$ — log-convex, если $\log f$ выпукла, эквивалентно
$$f(\lambda x + (1-\lambda) y) \leq f(x)^\lambda f(y)^{1-\lambda}.$$
Закрыто относительно сумм (!) и произведений, но не разностей. Log-convex $\Rightarrow$ convex.

**Bohr–Mollerup theorem**: $\Gamma(x)$ — единственная функция $f: (0,\infty) \to (0,\infty)$ с
1. $f(1) = 1$,
2. $f(x+1) = x f(x)$,
3. $\log f$ выпукла на $(0, \infty)$.

Условия применимости. $f > 0$ — критично (иначе $\log$ не определён). Log-convexity — строго более сильное, чем convex. Sum-closure (если $f, g$ log-convex, то $f + g$ log-convex) — нетривиальное и часто полезное свойство.

Идея доказательства. Log-convexity суммы: для $z = \lambda x + (1-\lambda)y$ применяем Hölder $(f(x) + g(x))^\lambda (f(y) + g(y))^{1-\lambda} \geq f(x)^\lambda f(y)^{1-\lambda} + g(x)^\lambda g(y)^{1-\lambda}$. Bohr–Mollerup: log-convexity + функциональное уравнение $f(x+1) = x f(x)$ полностью определяют $f$ на $(0,1)$ через сравнение с $f(n) = (n-1)!$.

Варианты и обобщения.
- **Holland-Mitrinović**: log-convex + ratio test для сходимости $\sum f(n)$.
- **Completely monotonic functions** ($(-1)^n f^{(n)} \geq 0$) — log-convex и абсолютно монотонные классы для $\Gamma$, $\log\Gamma$, polygamma.
- **Hadamard's three-line theorem** — комплексный аналог log-convexity для $\sup_t |F(\sigma + it)|$ как функции $\sigma$.

Используется в. [IMC-2020-5 — анализ ОДУ $f f'' \geq 2(f')^2$, эквивалентного $(\log f)'' \geq (f'/f)^2$ или $(1/f)'' \leq 0$ (т.е. $1/f$ вогнута и положительна) ⇒ $1/f$ постоянна или линейна на $\mathbb{R}$], характеристика $\Gamma$ для Putnam-style задач на функциональные уравнения.

### E14. Hermite–Hadamard inequality (Эрмита–Адамара)

Формулировка. $f: [a,b] \to \mathbb{R}$ выпуклая:
$$f\!\left(\frac{a+b}{2}\right) \leq \frac{1}{b-a}\int_a^b f(x)\,dx \leq \frac{f(a) + f(b)}{2}.$$
Concave — обратные знаки.

Условия применимости. $f$ выпукла на $[a,b]$ (достаточно midpoint-convex + измеримости). Без выпуклости не работает в общем виде.

Идея доказательства. Левая часть — Jensen с равномерной мерой на $[a,b]$; правая — supporting line снизу + tangent в точках $a,b$ и линеаризация (или прямо: $f(x) \leq \frac{b-x}{b-a}f(a) + \frac{x-a}{b-a}f(b)$ на хорде, интегрируя).

Варианты и обобщения.
- **Bullen's refinement**: $\frac{f(a)+f(b)}{2} - \frac{1}{b-a}\int f \geq \frac{1}{b-a}\int f - f((a+b)/2)$ (нижняя половина оценки больше верхней).
- **Fejér's inequality**: weighted version с симметричной плотностью.
- **Hadamard inequality (multi-dim)** для выпуклых тел: средние значения над концентрическими сферами/шарами.
- **Hermite–Hadamard–Iyengar**: refinement через значения $f$ на дополнительных точках.

Используется в. Workhorse для оценки $\int_a^b f$ через значения в концах и середине; быстрое доказательство $\sin x \geq 2x/\pi$ на $[0, \pi/2]$, $\log x \leq x - 1$, классические неравенства Putnam-уровня.

### E15. Young's inequality (Юнга)

Формулировка. Для $a, b \geq 0$ и $p, q > 1$ с $1/p + 1/q = 1$:
$$ab \leq \frac{a^p}{p} + \frac{b^q}{q}, \quad \text{равенство при } a^p = b^q.$$
Усиленная (Young-1912) форма: для строго возрастающей $\varphi: [0,\infty) \to [0,\infty)$, $\varphi(0) = 0$,
$$ab \leq \int_0^a \varphi(t)\,dt + \int_0^b \varphi^{-1}(t)\,dt,$$
равенство при $b = \varphi(a)$.

Условия применимости. $a, b \geq 0$. Для усиленной формы $\varphi$ — гомеоморфизм $[0,\infty) \to [0,\infty)$ (например, $\varphi(t) = t^{p-1}$ даёт стандартную форму).

Идея доказательства. (Стандартная) Concavity of $\log$: $\log(a^p/p + b^q/q) \geq (1/p) \log a^p + (1/q) \log b^q = \log(ab)$. (Geometric) Площадь прямоугольника $[0,a] \times [0,b]$ ≤ сумма площадей под графиком $\varphi$ и над ним (т.е. под $\varphi^{-1}$), с равенством когда $b = \varphi(a)$.

Варианты и обобщения.
- **Young's convolution inequality**: $\|f * g\|_r \leq \|f\|_p \|g\|_q$ при $1/p + 1/q = 1 + 1/r$. Оптимальная константа Beckner-Brascamp-Lieb.
- **Fenchel–Young inequality**: $\langle x, y\rangle \leq f(x) + f^*(y)$ для convex $f$ и сопряжённой $f^*$ (Legendre transform).
- $\varepsilon$-Young: $ab \leq \varepsilon a^p / p + \varepsilon^{-q/p} b^q / q$ — маленькое $\varepsilon$ для абсорбции.

Используется в. Фундамент Hölder (E4) и Minkowski (E8); в чистом виде — для оценок типа $f g \leq \alpha f^p + \beta g^q$ с подходящими $\alpha, \beta$ (классический трюк сжатия одного слагаемого в другое с малым весом).

### E16. Maclaurin's inequality (Маклорена) и Newton's inequalities (Ньютона)

Формулировка. Для $x_1, \dots, x_n > 0$ обозначим $\sigma_k = \sum_{i_1 < \dots < i_k} x_{i_1}\cdots x_{i_k}$ (elementary symmetric) и $S_k = \sigma_k / \binom{n}{k}$ (нормализованные). Тогда

**Maclaurin**: $S_1 \geq \sqrt{S_2} \geq \sqrt[3]{S_3} \geq \dots \geq \sqrt[n]{S_n}$.

**Newton's inequalities**: $S_k^2 \geq S_{k-1} S_{k+1}$ при $k = 1, \dots, n-1$ (log-concavity последовательности $\log S_k$).

Условия применимости. $x_i > 0$ (Newton's обобщается на вещественные, если все корни полинома $\prod(t - x_i)$ вещественны). Equality в Maclaurin $\Leftrightarrow$ все $x_i$ равны.

Идея доказательства. Newton's: производная $\prod (t - x_i)$ имеет вещественные корни (Rolle между корнями), и применение Newton's identity связывает $\sigma_k$ полинома и его производных. Maclaurin следует из Newton's: $S_k^2 \geq S_{k-1} S_{k+1}$ цепочкой даёт $\sqrt[k]{S_k}$ убывает.

Варианты и обобщения.
- **Power mean monotonicity** (E4) — родственная цепочка для степенных средних, включающая в крайностях $S_1, \sqrt[n]{S_n}$.
- **Muirhead's inequality**: для показателей $(\alpha) \succ (\beta)$ (majorization) симметрическая сумма $\sum_{\sigma} x^{\alpha_{\sigma(1)}} \cdots x^{\alpha_{\sigma(n)}} \geq$ той же для $\beta$.
- **Schur–Newton**: связь Newton's inequalities с $\log$-concavity полиномов с вещественными корнями.

Используется в. Не появляется напрямую в IMC, но даёт мост между AM-GM и более тонкими симметрическими неравенствами; полезно для polynomial root inequalities (Putnam-уровень).

### E17. Steffensen's inequality (Штеффенсена)

Формулировка. Пусть $f: [a,b] \to \mathbb{R}$ убывает, $g: [a,b] \to [0,1]$ интегрируема, $\lambda = \int_a^b g$. Тогда
$$\int_{b-\lambda}^b f(x)\,dx \leq \int_a^b f(x) g(x)\,dx \leq \int_a^{a+\lambda} f(x)\,dx.$$

Условия применимости. $f$ убывает (для возрастающей — обратные знаки), $g \in [0,1]$ почти всюду. $\lambda \in [0, b-a]$.

Идея доказательства. Правая часть: $\int_a^{a+\lambda} f - \int_a^b fg = \int_a^{a+\lambda} f(1-g) - \int_{a+\lambda}^b fg \geq f(a+\lambda)\left[\int_a^{a+\lambda}(1-g) - \int_{a+\lambda}^b g\right] = 0$ (внутри скобок: $\lambda - \int g + \int_a^{a+\lambda} g = 0$ по определению $\lambda$). Левая часть симметрично.

Варианты и обобщения.
- **Steffensen–Hayashi**: при $g \in [0, M]$ заменяем $\lambda$ на $\lambda / M$ и нормализуем.
- **Бесконечная версия** для $[a, \infty)$ при подходящих условиях интегрируемости.
- **Дискретный аналог**: для убывающей $a_n$ и $0 \leq b_n \leq 1$, $\sum b_n = K$: $\sum_{n > N-K} a_n \leq \sum a_n b_n \leq \sum_{n \leq K} a_n$.

Используется в. Тонкий инструмент для оценок интегралов через «массу» сечения; редко в IMC, но ценен для контрпримеров к ослаблениям Cauchy–Schwarz и в Putnam-задачах с интегральными ограничениями.

### E18. Bernoulli's inequality (Бернулли) с дробной степенью

Формулировка. Для $x > -1$:
$$(1+x)^r \geq 1 + rx, \quad r \geq 1 \text{ или } r \leq 0,$$
$$(1+x)^r \leq 1 + rx, \quad 0 \leq r \leq 1.$$
Равенство при $x = 0$ (или $r \in \{0, 1\}$).

Условия применимости. $x > -1$ (обязательно). Для целых $r \geq 1$ — стандартная форма. Дробная — через выпуклость.

Идея доказательства. Касательная в $x=0$ к графику $f(x) = (1+x)^r$: $f'(0) = r$. Convexity $f''(x) = r(r-1)(1+x)^{r-2}$ даёт направление: $f$ выпукла при $r \geq 1$ или $r \leq 0$, вогнута при $0 \leq r \leq 1$. Касательная всегда снизу (выпуклая) или сверху (вогнутая).

Варианты и обобщения.
- **Aczél's inequality**: $(\sqrt{a_0^2 - \sum a_i^2})(\sqrt{b_0^2 - \sum b_i^2}) \leq a_0 b_0 - \sum a_i b_i$ при $a_0^2 \geq \sum a_i^2$ — гиперболический аналог CS.
- **Mitrinović–Pečarić refinements**: $(1+x)^r - 1 - rx \geq c_r x^2$ с $c_r$, явно зависящим от $r$ — более сильная локальная оценка.

Используется в. Тонкие оценки Putnam-уровня; быстрый способ убедиться, что $(1+1/n)^n < e$, что $(1+x)^{1/n} \approx 1 + x/n - O(x^2)$ для асимптотики; вспомогательная оценка для итеративных задач.

## Self-test

1. Непрерывная $f: [0,1] \to (0,\infty)$. Доказать $\displaystyle\left(\int_0^1 f(x)\,dx\right)\left(\int_0^1 \frac{1}{f(x)}\,dx\right) \geq 1$.

2. Положительные $a_1, \dots, a_n$. Доказать $\displaystyle\frac{a_1^2}{a_2} + \frac{a_2^2}{a_3} + \dots + \frac{a_{n-1}^2}{a_n} + \frac{a_n^2}{a_1} \geq a_1 + a_2 + \dots + a_n$.

3. Последовательность $a_n > 0$ с $\sum_{n=1}^\infty a_n < \infty$. Доказать $\displaystyle\sum_{n=1}^\infty (a_1 a_2 \cdots a_n)^{1/n} < \infty$, причём с явной оценкой через $\sum a_n$ и константу, не зависящую от $n$.

4. $f: [0,1] \to \mathbb{R}$ непрерывна, вогнута, $f(0) = f(1) = 0$, $\max_{[0,1]} f = M$ (достигается в некоторой $c \in (0,1)$). Доказать $\displaystyle\int_0^1 f(x)\,dx \geq \frac{M}{2}$ и показать, что оценка достижима в пределе.

5. Для $a, b, c \geq 0$ доказать $\displaystyle a^3 + b^3 + c^3 + abc \geq \tfrac{1}{2}(a+b+c)(a^2+b^2+c^2)/\text{(уточнить, см. решение)}$. (Версия Schur при $t = 1$.) Точнее: доказать $a^3 + b^3 + c^3 + abc \geq ab(a+b) + bc(b+c) + ca(c+a) - 3abc$, что эквивалентно $\sum a(a-b)(a-c) \geq 0$.

## Решения

1. По Cauchy–Schwarz применённому к $\sqrt{f}$ и $1/\sqrt{f}$:
$$1 = \left(\int_0^1 \sqrt{f} \cdot \frac{1}{\sqrt{f}}\,dx\right)^2 \leq \left(\int_0^1 f\right)\left(\int_0^1 \frac{1}{f}\right).$$

2. Engel form (Titu's lemma) для CS: $\displaystyle\sum_{i=1}^n \frac{a_i^2}{a_{i+1}} \geq \frac{(\sum a_i)^2}{\sum a_{i+1}} = \frac{(\sum a_i)^2}{\sum a_i} = \sum a_i$, где индексы $a_{n+1} = a_1$ (циклически).

3. Это и есть Carleman's inequality (E10):
$$\sum_{n=1}^\infty (a_1 \cdots a_n)^{1/n} \leq e \sum_{n=1}^\infty a_n.$$
Доказательство — weighted AM-GM с весами $c_n = (n+1)^n / n^{n-1}$, $(c_1\cdots c_n)^{1/n} = n+1$, и телескопирование $\sum_{n \geq k} 1/(n(n+1)) = 1/k$, плюс $c_k / k = (1+1/k)^k < e$.

4. Concavity $\Rightarrow$ график над хордой:
   - На $[0, c]$: $f(x) \geq Mx/c$ (хорда от $(0,0)$ до $(c, M)$).
   - На $[c, 1]$: $f(x) \geq M(1-x)/(1-c)$ (хорда от $(c, M)$ до $(1, 0)$).

   Интегрируем: $\int_0^c f \geq Mc/2$, $\int_c^1 f \geq M(1-c)/2$, сумма $= M/2$.

   Достижимость: $f$, близкая к ломаной с вершиной $(c, M)$ — $\int_0^1 f \to M/2$ при «треугольном» профиле.

5. Schur (E12) при $t = 1$: $\sum_{\text{cyc}} a(a-b)(a-c) \geq 0$. Раскроем: $\sum a^3 - \sum a^2 b - \sum a^2 c + 3abc \geq 0$, что после $\sum a^2 b + \sum a^2 c = ab(a+b) + bc(b+c) + ca(c+a)$ переписывается как
$$a^3 + b^3 + c^3 + 3abc \geq ab(a+b) + bc(b+c) + ca(c+a).$$
WLOG $a \geq b \geq c$: первое слагаемое $a(a-b)(a-c) \geq 0$, $c(c-a)(c-b) \geq 0$ (произведение двух $\leq 0$), $b(b-a)(b-c) \leq 0$, и группировка $a(a-b)(a-c) + b(b-a)(b-c) = (a-b)[a(a-c) - b(b-c)] \geq 0$ так как $a \geq b$ и $a(a-c) \geq b(b-c)$.
