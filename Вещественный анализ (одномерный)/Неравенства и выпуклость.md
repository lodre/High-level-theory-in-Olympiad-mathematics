# Неравенства и выпуклость

## Scope
Базовые классические неравенства (AM–GM, Cauchy–Schwarz, Jensen, Hölder, степенные средние, перестановочное, Чебышёва) и их интегральные аналоги. Каркас — выпуклые и логарифмически выпуклые функции, опорная прямая. Дальше — рабочие тяжеловесы IMC: майоризация (Karamata), Hardy, Carleman, Schur, приём касательной, неравенство Эрмита–Адамара, теорема Бора–Моллерупа, неравенство Юнга. Финал — редкие инструменты: неравенства Маклорена и Ньютона, неравенство Штеффенсена.

## Определения

**Выпуклая функция** $f: I \to \mathbb{R}$ на интервале $I$:
$$f(\lambda x + (1-\lambda) y) \le \lambda f(x) + (1-\lambda) f(y),\quad \lambda \in [0,1].$$
Если неравенство строгое при $x \ne y$ и $\lambda \in (0,1)$ — строго выпукла. Вогнутая — с обратным знаком.

**Опорная прямая (supporting line)** в точке $x_0 \in \mathrm{int}\, I$ — прямая $y = f(x_0) + c(x - x_0)$, лежащая снизу графика выпуклой функции на всём $I$. Существует всегда; при гладкости $c = f'(x_0)$.

**Логарифмически выпуклая (log-convex) функция** $f: I \to (0, \infty)$ — функция, для которой $\log f$ выпукла. Эквивалентно
$$f(\lambda x + (1-\lambda)y) \le f(x)^\lambda f(y)^{1-\lambda}.$$
Логарифмически выпуклая всегда выпукла; обратное неверно.

**Степенное среднее (power mean)** порядка $r \in \mathbb{R}$ для $x_1, \dots, x_n > 0$ с весами $w_i \ge 0$, $\sum w_i = 1$:
$$M_r(x; w) = \left(\sum w_i x_i^r\right)^{1/r}\ (r \ne 0),\quad M_0 = \prod x_i^{w_i}.$$
Предельные случаи: $M_{-\infty} = \min x_i$, $M_{\infty} = \max x_i$.

**Сопряжённые показатели (conjugate exponents)** — пары $p, q$ с $1/p + 1/q = 1$, $p, q > 1$ (или $p = 1$, $q = \infty$).

**Майоризация (majorization).** Для упорядоченных по убыванию наборов $(x_i)$ и $(y_i)$ длины $n$ пишут $(x) \succ (y)$, если
$$\sum_{i=1}^k x_i \ge \sum_{i=1}^k y_i\ \text{для}\ k = 1, \dots, n-1,\quad \sum_{i=1}^n x_i = \sum_{i=1}^n y_i.$$
Содержательно — $x$ «более рассеян», $y$ «более усреднён».

**Симметрическое среднее.** Для $x_1, \dots, x_n > 0$: $\sigma_k = \sum_{i_1 < \dots < i_k} x_{i_1} \cdots x_{i_k}$ — элементарный симметрический многочлен, $S_k = \sigma_k / \binom{n}{k}$ — нормализованный.

## Записи — Основные

### E1. Неравенство о средних арифметическом и геометрическом (AM–GM inequality)

Формулировка. Для $x_1, \dots, x_n \ge 0$:
$$\frac{x_1 + \dots + x_n}{n} \ge \sqrt[n]{x_1 \cdots x_n}.$$
Равенство — только при $x_1 = \dots = x_n$.

Взвешенный вариант: при $w_i \ge 0$, $\sum w_i = 1$:
$$\sum w_i x_i \ge \prod x_i^{w_i}.$$

Условия применимости. Все $x_i \ge 0$.

Контрпример при отрицательных значениях: $n = 2$, $x_1 = 1$, $x_2 = -4$. Тогда среднее арифметическое $= -3/2$, а корень $\sqrt{-4}$ не определён в вещественных числах.

Идея доказательства. Логарифмирование и неравенство Йенсена для вогнутой функции $\ln$. Альтернатива — прямая индукция «вперёд-назад» Коши: сначала для степеней двойки, затем спуск.

Варианты.
- Неравенство о средних AM–GM–HM: $\frac{\sum x_i}{n} \ge \sqrt[n]{\prod x_i} \ge \frac{n}{\sum 1/x_i}$.
- Интегральная форма: $\exp\!\left(\frac{1}{b-a}\int_a^b \ln f\right) \le \frac{1}{b-a}\int_a^b f$.

### E2. Неравенство Коши–Буняковского–Шварца (Cauchy–Schwarz inequality)

Формулировка. Дискретно:
$$\left(\sum_{i=1}^n a_i b_i\right)^2 \le \left(\sum a_i^2\right)\left(\sum b_i^2\right).$$
Интегрально:
$$\left(\int_a^b f g\right)^2 \le \int_a^b f^2 \cdot \int_a^b g^2.$$
Равенство — при пропорциональности $a_i = \lambda b_i$ (соответственно $f = \lambda g$ почти всюду).

Условия применимости. Любые вещественные или комплексные значения. Для интегральной формы — $f, g \in L^2$.

Контрпример при $f \notin L^2$: $f(x) = 1/\sqrt{x}$ на $(0, 1)$, $g \equiv 1$. Левая часть $(\int_0^1 1/\sqrt{x}\,dx)^2 = 4$, правая часть $\int_0^1 1/x\,dx = \infty$ — формально выполнено, но неравенство содержательно теряет смысл.

Идея доказательства. Квадратичный многочлен $P(t) = \sum (a_i + t b_i)^2 \ge 0$ имеет неотрицательный дискриминант. Альтернатива — тождество Лагранжа:
$$\left(\sum a_i^2\right)\left(\sum b_i^2\right) - \left(\sum a_i b_i\right)^2 = \sum_{i<j} (a_i b_j - a_j b_i)^2.$$

Варианты.
- Форма Энгеля (Titu's lemma): $\sum a_i^2 / b_i \ge (\sum a_i)^2 / \sum b_i$ при $b_i > 0$. Главный рабочий вариант на олимпиадах.

---

### E3. Неравенство Йенсена (Jensen's inequality)

Формулировка. Для выпуклой $f$ на интервале $I$, точек $x_i \in I$ и весов $w_i \ge 0$ с $\sum w_i = 1$:
$$f\!\left(\sum w_i x_i\right) \le \sum w_i f(x_i).$$
Интегральная версия — для вероятностной меры $\mu$ на $I$:
$$f\!\left(\int x\,d\mu\right) \le \int f\,d\mu.$$
Для вогнутой — обратный знак.

Условия применимости. $f$ выпукла на $I$, все $x_i \in I$.

Контрпример при вогнутой функции вместо выпуклой: $f(x) = -x^2$, $x_1 = -1$, $x_2 = 1$, $w_i = 1/2$. Левая часть $= f(0) = 0$, правая $= -1$. Неравенство $0 \le -1$ ложно.

Идея доказательства. Берём опорную прямую к графику $f$ в точке $x_0 = \sum w_i x_i$: $f(x) \ge f(x_0) + c(x - x_0)$. Усреднение с весами $w_i$ зануляет линейную часть и оставляет $\sum w_i f(x_i) \ge f(x_0)$.

### E4. Неравенство Гёльдера и степенное среднее (Hölder + Power mean inequality)

Формулировка. Гёльдер: при сопряжённых $p, q$
$$\sum |a_i b_i| \le \left(\sum |a_i|^p\right)^{1/p}\left(\sum |b_i|^q\right)^{1/q},$$
интегрально $\int |fg| \le \|f\|_p \|g\|_q$.

Степенное среднее $M_r(x; w)$ строго возрастает по $r \in \mathbb{R}$, если не все $x_i$ равны. В частности, $M_0 \le M_1$ — это AM–GM.

Условия применимости. $p, q > 1$ с $1/p + 1/q = 1$ (или $p = 1, q = \infty$). Степенное среднее — для $x_i > 0$.

Контрпример к Гёльдеру без сопряжённости: $p = q = 4$ (тогда $1/p + 1/q = 1/2 \ne 1$), $a = b = (1, 1, \dots, 1)$ длины $n$. Левая часть $= n$, правая $= n^{1/4} \cdot n^{1/4} = \sqrt{n}$. При $n > 1$ имеем $n > \sqrt n$ — нарушение.

Идея доказательства. Гёльдер выводится из неравенства Юнга (E15) после нормализации $\|a\|_p = \|b\|_q = 1$. Монотонность степенного среднего — это Йенсен для $t \mapsto t^{r/s}$ при $r > s > 0$.

Варианты.
- Неравенство Ляпунова: $r \mapsto \log \|f\|_{1/r}$ выпукло. Даёт интерполяцию $\|f\|_q$ через $\|f\|_p$ и $\|f\|_r$.

### E5. Характеризации выпуклости: монотонность секущих и опорная прямая

Формулировка. Для $f: I \to \mathbb{R}$ эквивалентны:
1. Неравенство хорды (определение).
2. Монотонность секущих (three-chord lemma): при $a < b < c$
$$\frac{f(b) - f(a)}{b - a} \le \frac{f(c) - f(a)}{c - a} \le \frac{f(c) - f(b)}{c - b}.$$
3. Существование опорной прямой в каждой внутренней точке.
4. Производная $f'$ не убывает (при дифференцируемости); $f'' \ge 0$ (при $f \in C^2$).

Условия применимости. $I$ — интервал (не произвольное множество).

Следствие. Выпуклая функция непрерывна во внутренних точках $I$, имеет односторонние производные $f'_-(x) \le f'_+(x)$, обе монотонно не убывают. На границе $I$ функция может разрываться.

Контрпример к попытке распространить на несвязное множество: $f \equiv 0$ на $[0,1]$ и $f \equiv -1$ на $[2,3]$ — выпукла на каждом куске отдельно, но не выпукла на $[0,1] \cup [2,3]$ (если воспринимать как функцию на объединении): хорда между точками из разных кусков не определяется неравенством на интервале, и склейка ломается.

Идея доказательства. (1) $\Leftrightarrow$ (2) — прямой перевод $\lambda$ через тройку точек. (2) $\Rightarrow$ (3): любой наклон между $f'_-(x_0)$ и $f'_+(x_0)$ задаёт опорную прямую. (3) $\Rightarrow$ (1): берём опорную прямую в точке $z = \lambda x + (1-\lambda) y$ и подставляем $x$, $y$.

### E6. Приём касательной + разложение в сумму квадратов

Формулировка. Симметричное неравенство $\sum_i f(x_i) \ge C$ при условии $\sum x_i = S$ доказывается так. Берём предполагаемую точку равенства $x_0 = S/n$ и ищем линейную нижнюю оценку $f(x) \ge \alpha x + \beta$, касательную к графику в $x_0$. Тогда
$$\sum f(x_i) \ge \alpha \sum x_i + n \beta = \alpha S + n \beta = C.$$
Касательная доказывается через разложение в сумму квадратов (sum of squares, SOS):
$$f(x) - \alpha x - \beta = (x - x_0)^2 \cdot g(x),\quad g(x) \ge 0\ \text{на допустимой области.}$$

Условия применимости. Точка равенства известна или угадывается из симметрии. Касательная глобально лежит снизу графика на всём допустимом множестве.

Контрпример к глобальности касательной: $f(x) = x^3$, $x_0 = 1$, касательная $y = 3x - 2$. На $x = -2$: $f(-2) = -8$, $3 \cdot (-2) - 2 = -8$. Совпадение. На $x = -3$: $f = -27$, касательная $= -11$. Касательная выше графика — приём здесь не работает на $x \le -2$. Нужно отдельно отслеживать диапазон $x_i$.

Идея доказательства. Подгонка $\alpha = f'(x_0)$, $\beta = f(x_0) - \alpha x_0$. Затем разложение разности через двойной корень $(x - x_0)^2$ и проверка положительности остатка.

---

### E7. Перестановочное неравенство и неравенство Чебышёва о суммах

Формулировка. Перестановочное (rearrangement inequality): для $a_1 \le \dots \le a_n$ и $b_1 \le \dots \le b_n$ и любой перестановки $\sigma$:
$$\sum a_i b_{n+1-i} \le \sum a_i b_{\sigma(i)} \le \sum a_i b_i.$$
Чебышёв (для одинаково упорядоченных $a_i, b_i$):
$$n \sum a_i b_i \ge \left(\sum a_i\right)\left(\sum b_i\right).$$
Интегральная версия Чебышёва: $f, g$ монотонны в одну сторону на $[a,b]$ влечёт
$$(b - a) \int_a^b f g \ge \int_a^b f \cdot \int_a^b g.$$

Условия применимости. Сортировка обоих наборов; для интегральной версии — одинаковая монотонность.

Контрпример без сортировки: $a = (1, 2)$, $b = (2, 1)$. Тогда $\sum a_i b_i = 4$, а $\frac{1}{n}\sum a_i \sum b_i = 9/2$. Имеем $4 < 4.5$ — Чебышёв нарушен.

Идея доказательства. Тождество
$$n \sum a_i b_i - \sum a_i \sum b_i = \frac{1}{2} \sum_{i,j} (a_i - a_j)(b_i - b_j),$$
правая часть неотрицательна при одинаковой сортировке.

## Записи — Стандартные

### E8. Неравенство Минковского (Minkowski's inequality)

Формулировка. Для $p \ge 1$:
$$\left(\sum (a_i + b_i)^p\right)^{1/p} \le \left(\sum a_i^p\right)^{1/p} + \left(\sum b_i^p\right)^{1/p}.$$
Интегральная форма — $\|f + g\|_p \le \|f\|_p + \|g\|_p$. Это треугольное неравенство нормы $L^p$.

Условия применимости. $p \ge 1$.

Контрпример при $p < 1$: $p = 1/2$, $a = (1, 0)$, $b = (0, 1)$. Левая часть $= (1 + 1)^2 = 4$. Правая $= 1 + 1 = 2$. Прямое неравенство нарушено, выполнено обратное.

Идея доказательства. Разложить $\sum (a + b)^p = \sum a (a+b)^{p-1} + \sum b (a+b)^{p-1}$, к каждому слагаемому применить Гёльдер с показателями $p$ и $p/(p-1)$.

---

### E9. Неравенство Харди (Hardy's inequality)

Формулировка. Дискретно: для $a_n > 0$, $p > 1$, $A_n = \frac{1}{n}\sum_{k=1}^n a_k$:
$$\sum_{n=1}^\infty A_n^p \le \left(\frac{p}{p-1}\right)^p \sum_{n=1}^\infty a_n^p.$$
Интегрально: для $f \ge 0$ и $F(x) = \int_0^x f$
$$\int_0^\infty \left(\frac{F(x)}{x}\right)^p dx \le \left(\frac{p}{p-1}\right)^p \int_0^\infty f^p.$$
Константа $(p/(p-1))^p$ точна.

Условия применимости. $p > 1$.

Контрпример при $p = 1$: $a_n = 1/n$. Тогда $A_n \sim (\ln n)/n$, и $\sum A_n = \infty$, но и $\sum a_n = \infty$ — ограниченной константы $C$ с $\sum A_n \le C \sum a_n$ не существует.

Идея доказательства. Интегрирование по частям с весом $x^{1-p}$ переводит $\int (F/x)^p$ в $\int F^{p-1} f / x^{p-1} \cdot p/(p-1)$. Затем Гёльдер.

Варианты.
- Неравенство Гильберта: $\sum_{m,n \ge 1} a_m b_n / (m + n) \le \pi \|a\|_2 \|b\|_2$. Двойной аналог Харди.

---

### E10. Неравенство Карлемана (Carleman's inequality)

Формулировка. Для $a_n \ge 0$:
$$\sum_{n=1}^\infty (a_1 a_2 \cdots a_n)^{1/n} \le e \sum_{n=1}^\infty a_n.$$
Константа $e$ точна.

Условия применимости. $a_n \ge 0$; при $\sum a_n = \infty$ обе стороны бесконечны и неравенство тривиально.

Идея доказательства. Подбираем веса $c_n = (n+1)^n / n^{n-1}$. По взвешенному AM–GM
$$(a_1 \cdots a_n)^{1/n} = \frac{(c_1 a_1 \cdots c_n a_n)^{1/n}}{(c_1 \cdots c_n)^{1/n}} \le \frac{c_1 a_1 + \dots + c_n a_n}{n \cdot (c_1 \cdots c_n)^{1/n}}.$$
Прямой подсчёт: $(c_1 \cdots c_n)^{1/n} = n + 1$. Затем суммирование по $n$, телескоп $\sum_{n \ge k} 1/(n(n+1)) = 1/k$ и оценка $c_k/k = (1 + 1/k)^k < e$.

---

### E11. Неравенство Караматы (Karamata's majorization inequality)

Формулировка. Пусть $x_1 \ge \dots \ge x_n$ и $y_1 \ge \dots \ge y_n$ — упорядоченные по убыванию, причём $(x) \succ (y)$ (мажоризирует). Тогда для любой выпуклой функции $f$ на интервале, содержащем все $x_i, y_i$:
$$\sum_{i=1}^n f(x_i) \ge \sum_{i=1}^n f(y_i).$$

Условия применимости. Обе последовательности отсортированы по убыванию.

Контрпример без сортировки: возьмём $x = (3, 0, 0)$, $y = (1, 1, 1)$. Без сортировки $y$ как $(1,1,1)$ всё равно отсортирован, но если бы мы взяли $y = (0, 2, 1)$ — мажоризация определяется через отсортированные версии, неотсортированный набор без переупорядочивания даст неверный ответ при проверке кумулятивных сумм.

Идея доказательства. В каждой точке $y_i$ берём опорный наклон $c_i$ из субдифференциала $\partial f(y_i)$. Последовательность $c_i$ не возрастает (так как $f'$ не убывает, а $y_i$ убывают). Преобразование Абеля
$$\sum (f(x_i) - f(y_i)) \ge \sum c_i (x_i - y_i)$$
вместе с условиями мажоризации даёт результат.

Следствие (теорема Харди–Литлвуда–Полиа). $(x) \succ (y)$ эквивалентно тому, что $y = P x$ для дважды стохастической матрицы $P$. Эквивалентно: $y$ — выпуклая комбинация перестановок $x$.

### E12. Неравенство Шура и метод суммы квадратов (Schur's inequality + SOS)

Формулировка. Шур: для $x, y, z \ge 0$ и $t \ge 0$:
$$x^t (x - y)(x - z) + y^t (y - z)(y - x) + z^t (z - x)(z - y) \ge 0.$$
Наиболее частые случаи $t = 1, 2$.

Метод суммы квадратов: симметричную форму $P(x_1, \dots, x_n)$ пытаются представить как
$$P = \sum_{i < j} (x_i - x_j)^2 Q_{ij},$$
где $Q_{ij} \ge 0$ на допустимом множестве. Из такого разложения сразу $P \ge 0$.

Условия применимости. Для Шура — $x, y, z \ge 0$.

Контрпример при отрицательных значениях: $t = 1$, $x = -1$, $y = z = 0$. Получаем $(-1)(-1)(-1) + 0 + 0 = -1 < 0$. Условие $x \ge 0$ обязательно.

Идея доказательства. Без ограничения общности $x \ge y \ge z$. Группируем первые два слагаемых:
$$x^t (x - y)(x - z) + y^t (y - z)(y - x) = (x - y) \cdot [x^t (x - z) - y^t (y - z)] \ge 0,$$
обе скобки неотрицательны. Третье слагаемое тоже неотрицательно.

### E13. Логарифмически выпуклые функции и теорема Бора–Моллерупа (Bohr–Mollerup theorem)

Формулировка. Логарифмически выпуклая функция (см. определения). Замкнутость относительно суммы: если $f, g$ логарифмически выпуклы, то $f + g$ тоже.

Теорема Бора–Моллерупа. Гамма-функция $\Gamma(x)$ — единственная функция $f: (0, \infty) \to (0, \infty)$ с тремя свойствами:
1. $f(1) = 1$;
2. $f(x + 1) = x f(x)$;
3. $\log f$ выпукла на $(0, \infty)$.

Условия применимости. $f > 0$ во всех точках области.

Контрпример к замене выпуклости на логарифмическую выпуклость: $f(x) = e^{-x^2}$ на $\mathbb{R}$. Тут $\log f(x) = -x^2$ — вогнута, значит $f$ не логарифмически выпукла. При этом $f$ сама выпукла на $(-\infty, -1/\sqrt{2}] \cup [1/\sqrt{2}, \infty)$.

Идея доказательства. Логарифмическая выпуклость суммы — через Гёльдер:
$$f(x)^\lambda f(y)^{1-\lambda} + g(x)^\lambda g(y)^{1-\lambda} \le (f(x) + g(x))^\lambda (f(y) + g(y))^{1-\lambda}.$$
Теорема Бора–Моллерупа — функциональное уравнение и логарифмическая выпуклость полностью фиксируют $f$ на $(0, 1)$ через сравнение с $f(n) = (n-1)!$.

### E14. Неравенство Эрмита–Адамара (Hermite–Hadamard inequality)

Формулировка. Для выпуклой $f: [a, b] \to \mathbb{R}$:
$$f\!\left(\frac{a + b}{2}\right) \le \frac{1}{b - a} \int_a^b f(x)\,dx \le \frac{f(a) + f(b)}{2}.$$
Для вогнутой — обратные знаки.

Условия применимости. $f$ выпукла на $[a, b]$.

Контрпример без выпуклости: $f(x) = \sin(\pi x)$ на $[0, 1]$. Тогда $f(1/2) = 1$, среднее $\frac{1}{1} \int_0^1 \sin(\pi x)\,dx = 2/\pi \approx 0{,}64$. Левая часть $1 > 0{,}64$ — прямое неравенство нарушено.

Идея доказательства. Левая часть — Йенсен для интегрального среднего с равномерной мерой. Правая часть — хорда между $(a, f(a))$ и $(b, f(b))$ лежит сверху над выпуклым графиком; интегрирование уравнения хорды даёт $(f(a) + f(b))/2$.

### E15. Неравенство Юнга (Young's inequality)

Формулировка. Для $a, b \ge 0$ и сопряжённых $p, q$:
$$ab \le \frac{a^p}{p} + \frac{b^q}{q}.$$
Равенство — при $a^p = b^q$.

Условия применимости. Сопряжённость $1/p + 1/q = 1$, $p, q > 1$.

Контрпример при $1/p + 1/q > 1$: $p = q = 3/2$. Тогда $a = b = 2$. Левая часть $= 4$. Правая $= 2 \cdot 2^{3/2}/(3/2) = (4/3) \cdot 2\sqrt{2} \approx 3{,}77$. Прямое неравенство нарушено.

Идея доказательства. Выпуклость $\exp$: $ab = \exp((1/p) \cdot p \log a + (1/q) \cdot q \log b)$. По Йенсену для $\exp$
$$ab \le \frac{1}{p} e^{p \log a} + \frac{1}{q} e^{q \log b} = \frac{a^p}{p} + \frac{b^q}{q}.$$

Замечание. Неравенство Юнга — фундамент для Гёльдера (E4) и через него для Минковского (E8). Иногда применяется в виде $\varepsilon$-Юнга: $ab \le \varepsilon a^p / p + \varepsilon^{-q/p} b^q / q$, чтобы абсорбировать одно слагаемое.

### E16. Неравенства Маклорена и Ньютона (Maclaurin + Newton inequalities)

Формулировка. Для $x_1, \dots, x_n > 0$ и нормализованных симметрических средних $S_k$:
$$S_1 \ge \sqrt{S_2} \ge \sqrt[3]{S_3} \ge \dots \ge \sqrt[n]{S_n}\quad \text{(Маклорен)},$$
$$S_k^2 \ge S_{k-1} S_{k+1},\ k = 1, \dots, n-1\quad \text{(Ньютон)}.$$
Эквивалентно: последовательность $\log S_k$ вогнута.

Условия применимости. $x_i > 0$. Ньютон обобщается на любые $x_i \in \mathbb{R}$, если все они вещественны как корни одного многочлена.

Идея доказательства. Многочлен $\prod_i (t - x_i)$ имеет вещественные корни. По теореме Ролля его производные тоже имеют только вещественные корни. Связь $S_k$ исходного многочлена с $S_{k-1}, S_{k+1}$ производных даёт неравенства Ньютона. Маклорен следует из цепочки $S_k^2 \ge S_{k-1} S_{k+1}$.

### E17. Неравенство Штеффенсена (Steffensen's inequality)

Формулировка. Пусть $f: [a, b] \to \mathbb{R}$ убывает, $g: [a, b] \to [0, 1]$ интегрируема, $\lambda = \int_a^b g$. Тогда
$$\int_{b - \lambda}^b f(x)\,dx \le \int_a^b f(x) g(x)\,dx \le \int_a^{a + \lambda} f(x)\,dx.$$

Условия применимости. $f$ монотонна; $g(x) \in [0, 1]$ почти всюду.

Идея доказательства. Для правой части:
$$\int_a^{a+\lambda} f - \int_a^b f g = \int_a^{a+\lambda} f (1 - g) - \int_{a + \lambda}^b f g.$$
Первый интеграл $\ge f(a + \lambda) \int_a^{a+\lambda} (1 - g)$, второй $\le f(a + \lambda) \int_{a + \lambda}^b g$. Сумма этих оценок равна нулю по определению $\lambda$.

## Self-test

1. Непрерывная функция $f: [0, 1] \to (0, \infty)$. Доказать
$$\left(\int_0^1 f(x)\,dx\right)\left(\int_0^1 \frac{1}{f(x)}\,dx\right) \ge 1.$$

2. Положительные $a_1, \dots, a_n$. Доказать
$$\frac{a_1^2}{a_2} + \frac{a_2^2}{a_3} + \dots + \frac{a_{n-1}^2}{a_n} + \frac{a_n^2}{a_1} \ge a_1 + a_2 + \dots + a_n.$$

3. Последовательность $a_n > 0$ с $\sum_{n = 1}^\infty a_n < \infty$. Доказать, что
$$\sum_{n = 1}^\infty (a_1 a_2 \cdots a_n)^{1/n} < \infty.$$
Дать явную оценку с константой, не зависящей от $n$.

4. Непрерывная вогнутая $f: [0, 1] \to \mathbb{R}$ с $f(0) = f(1) = 0$ и $\max_{[0,1]} f = M$ (достигается в $c \in (0, 1)$). Доказать
$$\int_0^1 f(x)\,dx \ge \frac{M}{2}.$$

## Решения

1. По Коши–Шварцу для $\sqrt{f}$ и $1/\sqrt{f}$:
$$1 = \left(\int_0^1 \sqrt{f} \cdot \frac{1}{\sqrt{f}}\right)^2 \le \int_0^1 f \cdot \int_0^1 \frac{1}{f}.$$

2. По форме Энгеля для Коши–Шварца с циклическими индексами ($a_{n+1} = a_1$):
$$\sum_{i=1}^n \frac{a_i^2}{a_{i+1}} \ge \frac{(\sum a_i)^2}{\sum a_{i+1}} = \frac{(\sum a_i)^2}{\sum a_i} = \sum a_i.$$

3. Это неравенство Карлемана (запись E10):
$$\sum_{n=1}^\infty (a_1 \cdots a_n)^{1/n} \le e \sum_{n=1}^\infty a_n.$$
Доказательство — взвешенный AM–GM с весами $c_n = (n+1)^n / n^{n-1}$ и телескопом $\sum_{n \ge k} 1/(n(n+1)) = 1/k$.

4. Вогнутость даёт, что график лежит над хордами.

На отрезке $[0, c]$ хорда соединяет $(0, 0)$ и $(c, M)$: имеем $f(x) \ge M x / c$.

На отрезке $[c, 1]$ хорда соединяет $(c, M)$ и $(1, 0)$: имеем $f(x) \ge M (1 - x) / (1 - c)$.

Интегрируем по частям:
$$\int_0^c \frac{M x}{c}\,dx = \frac{M c}{2},\quad \int_c^1 \frac{M (1 - x)}{1 - c}\,dx = \frac{M (1 - c)}{2}.$$
Сумма равна $M / 2$. Оценка достижима в пределе на треугольных профилях со всё более острой вершиной в $c$.
