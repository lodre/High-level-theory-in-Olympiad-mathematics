# Семейство теорем о среднем (Rolle / Lagrange / Cauchy / IVT) + Taylor, дифференцирование по параметру

## Scope
Теоремы о среднем для производной и интеграла, их продвинутые расширения (Флетт, Помпею), формула Тейлора во всех формах остаточного члена. Олимпиадная техника подбора вспомогательной функции, итерированное Ролля для счёта нулей, неравенство Гронуолла, правило Лейбница и приём Фейнмана.

## Определения

**Дифференцируемость в точке.** $f$ дифференцируема в $a$, если существует $f'(a) = \lim_{x\to a}\frac{f(x)-f(a)}{x-a}$. Эквивалентно: $f(a+h) = f(a) + f'(a)h + o(h)$ при $h \to 0$.

**Класс $C^n([a,b])$.** Функции, у которых производные до порядка $n$ существуют и непрерывны на $[a,b]$. $C^0$ — непрерывные. $C^\infty$ — бесконечно гладкие. $C^\omega$ — аналитические.

**Кратность нуля.** Точка $a$ — нуль кратности $k$ для $f \in C^k$, если $f(a) = f'(a) = \cdots = f^{(k-1)}(a) = 0$ и $f^{(k)}(a) \neq 0$. Эквивалентно: $f(x) = (x-a)^k g(x)$ с $g(a) \neq 0$.

**Интегрируемость по Риману.** $f: [a,b] \to \mathbb{R}$ интегрируема по Риману, если для любого разбиения с мелкостью $\to 0$ интегральные суммы сходятся к общему пределу при любом выборе узлов.

**Большое $O$ и малое $o$.** $f(x) = O(g(x))$ при $x \to a$ означает $|f(x)| \leq C|g(x)|$ в окрестности $a$. $f(x) = o(g(x))$ — отношение $f/g \to 0$.

**Выпуклость.** $f$ выпукла на $I$, если $f(tx + (1-t)y) \leq tf(x) + (1-t)f(y)$ для всех $x, y \in I$, $t \in [0,1]$. Для $f \in C^2$ эквивалентно $f'' \geq 0$.

## Основные

### E1. Теоремы о среднем: Ролля, Лагранжа, Коши (Rolle / Lagrange / Cauchy MVT)

Формулировка. Пусть $f, g \in C([a,b])$ и дифференцируемы на $(a,b)$.
- **Ролля**: $f(a) = f(b) \Rightarrow \exists\,\xi \in (a,b): f'(\xi) = 0$.
- **Лагранжа**: $\exists\,\xi: f(b) - f(a) = f'(\xi)(b-a)$.
- **Коши**: $\exists\,\xi: (f(b) - f(a))\,g'(\xi) = (g(b) - g(a))\,f'(\xi)$.

Условия применимости.

*Вещественнозначность.* Для вектор-функций теорема Ролля ломается: $f(t) = (\cos t, \sin t)$ на $[0, 2\pi]$, $f(0) = f(2\pi)$, но $f'(t) = (-\sin t, \cos t)$ всюду ненулевая. Доступный аналог — неравенство $\|f(b) - f(a)\| \leq (b-a)\sup\|f'\|$.

*Непрерывность на замкнутом, дифференцируемость на открытом.* Замена одного на другое ломает: $f(x) = x$ на $[0,1)$ с $f(1) = 0$ — нет точки с $f'(\xi) = 0$.

Идея доказательства. Ролля — Вейерштрасс даёт экстремум, Ферма даёт нулевую производную. Лагранжа — Ролля для $h(x) = f(x) - \frac{f(b)-f(a)}{b-a}(x-a)$. Коши — Ролля для $h(x) = (f(b)-f(a))(g(x)-g(a)) - (g(b)-g(a))(f(x)-f(a))$.

### E2. Формула Тейлора (Taylor's formula)

Формулировка. Пусть $f \in C^n([a,b])$ и $f^{(n+1)}$ существует на $(a,b)$. Тогда
$$f(x) = \sum_{k=0}^{n} \frac{f^{(k)}(a)}{k!}(x-a)^k + R_n(x).$$
Остаток $R_n$ имеет четыре стандартные формы:
- **Лагранжа** (Lagrange): $R_n = \frac{f^{(n+1)}(\xi)}{(n+1)!}(x-a)^{n+1}$, $\xi \in (a, x)$.
- **Коши** (Cauchy): $R_n = \frac{f^{(n+1)}(\xi)}{n!}(x-\xi)^n(x-a)$.
- **Интегральная** (Integral): $R_n = \frac{1}{n!}\int_a^x (x-t)^n f^{(n+1)}(t)\,dt$, требует $f^{(n+1)} \in L^1$.
- **Пеано** (Peano): $R_n = o((x-a)^n)$, $x \to a$, требует только существования $f^{(n)}(a)$.

Условия применимости.

*Существование $f^{(n+1)}$ для форм Лагранжа и Коши.* Пример без условия: $f(x) = x^2|x|$ имеет $f''(0) = 0$, но $f'''(0)$ не существует.

*Гладкость на компакте для интегральной формы.* Удобна для оценок: $|R_n| \leq \sup|f^{(n+1)}| \cdot \frac{|x-a|^{n+1}}{(n+1)!}$.

Идея доказательства. Лагранжа — итерация теоремы Коши на $g_n(x) = R_n$ и $h(x) = (x-a)^{n+1}$, всего $n+1$ раз. Интегральная — индукция по $n$ через интегрирование по частям.

Какая форма когда. Лагранжа — для простых оценок и неравенств. Интегральная — для тонких оценок через $L^p$-нормы и моментов. Пеано — для асимптотик в окрестности.

### E3. Теоремы о среднем для интеграла (Mean value theorems for integrals)

Формулировка.

*Первая.* $f \in C([a,b])$, $g \geq 0$ интегрируема на $[a,b]$. Тогда $\exists\,\xi \in [a,b]$:
$$\int_a^b f(x) g(x)\,dx = f(\xi) \int_a^b g(x)\,dx.$$

*Вторая (Бонне, Bonnet).* $f$ монотонна, $g$ интегрируема. Тогда $\exists\,\xi \in [a,b]$:
$$\int_a^b f g = f(a)\int_a^\xi g + f(b)\int_\xi^b g.$$
Частный случай: $f$ убывает к $0$ — $\int_a^b f g = f(a) \int_a^\xi g$.

Условия применимости.

*Непрерывность $f$ в первой теореме.* Пример без условия: $f = \chi_{[0, 1/2]}$, $g \equiv 1$ на $[0,1]$. Среднее значение $1/2$, но $f$ нигде не равна $1/2$.

*Монотонность $f$ во второй.* Бонне — интегральный аналог преобразования Абеля. Через него любая оценка $\sup|\int_a^x g|$ переносится на $\int f g$ при монотонной $f$.

Идея доказательства. Первая — теорема о промежуточном значении для непрерывной $f$ применительно к $\int fg / \int g \in [\min f, \max f]$. Вторая — интегрирование по частям с $G(x) = \int_a^x g$ + первая теорема.

### E4. Правило Лопиталя (L'Hôpital's rule)

Формулировка. $f, g$ дифференцируемы в проколотой окрестности $a$, $g'(x) \neq 0$. Если $\lim_{x \to a} f/g$ имеет вид $\frac{0}{0}$ или $\frac{*}{\infty}$ и $\lim f'/g' = L$, то $\lim f/g = L$.

Условия применимости.

*$g'(x) \neq 0$ на проколотой окрестности.* Не «почти всюду», а именно всюду. Иначе $f'/g'$ не определено в нужных точках.

*Существование предела $f'/g'$.* Обратное направление неверно. Пример: $f(x) = x + \sin x$, $g(x) = x$ при $x \to \infty$. Имеем $f/g \to 1$, но $f'/g' = 1 + \cos x$ не имеет предела.

*Тип неопределённости $*/\infty$.* На $f$ ничего не требуется, нужно только $g \to \infty$. Это менее известная, но более сильная форма.

Идея доказательства. Случай $0/0$ — теорема Коши даёт $f(x)/g(x) = f'(\xi)/g'(\xi)$ с $\xi \in (a, x)$. Случай $*/\infty$ — Коши между двумя точками и $\varepsilon$-аргумент с растущим знаменателем.

### E5. Теорема Дарбу (Darboux's theorem)

Формулировка. Пусть $f$ дифференцируема на $[a, b]$. Тогда $f'$ принимает все промежуточные значения между $f'(a)$ и $f'(b)$, даже если $f'$ разрывна.

Условия применимости.

*Только дифференцируемость, непрерывность $f'$ не нужна.* Пример: $f(x) = x^2 \sin(1/x)$ на $\mathbb{R} \setminus \{0\}$ с $f(0) = 0$. Производная $f'(x) = 2x \sin(1/x) - \cos(1/x)$ при $x \neq 0$, $f'(0) = 0$. Производная разрывна в $0$, но теорема Дарбу выполняется.

*Производная не может иметь разрывов первого рода.* Это сильнее, чем теорема о промежуточном значении для непрерывных функций.

Идея доказательства. Свести к случаю $f'(a) < 0 < f'(b)$ и показать, что $f$ имеет внутреннюю точку минимума. По теореме Ферма в этой точке $f'(\xi) = 0$.

### E6. Дифференцирование по параметру: правило Лейбница (Leibniz integral rule)

Формулировка. $f(x, t)$ непрерывна на прямоугольнике, $\partial_t f$ непрерывна, $a(t)$ и $b(t)$ дифференцируемы. Тогда
$$\frac{d}{dt}\int_{a(t)}^{b(t)} f(x, t)\,dx = f(b(t), t)\,b'(t) - f(a(t), t)\,a'(t) + \int_{a(t)}^{b(t)} \partial_t f(x, t)\,dx.$$

Для несобственных интегралов: $\int_a^\infty f(x, t)\,dx$ дифференцируется почленно, если $\int |\partial_t f|$ равномерно сходится по $t$, либо при наличии интегрируемой мажоранты (теорема о доминированной сходимости).

Условия применимости.

*Гладкость по параметру на компакте.* В конечном случае достаточно $\partial_t f \in C$.

*Мажоранта для несобственного интеграла.* Пример без неё: $I(t) = \int_0^\infty e^{-tx} \sin x \,dx = \frac{1}{1+t^2}$ при $t > 0$. Формальное дифференцирование при $t = 0$ даёт $\int_0^\infty (-x \sin x)\,dx$, который не сходится.

Идея доказательства. Конечный случай — основная теорема анализа и цепное правило плюс равномерная непрерывность $\partial_t f$ на компакте. Несобственный — теорема о доминированной сходимости.

Приём Фейнмана (Feynman trick). Вводится параметр в интеграл, который от него явно не зависит. После дифференцирования получается простой интеграл; обратное интегрирование с одной начальной точкой даёт ответ.

## Стандартные

### E7. Метод вспомогательной функции (auxiliary function technique)

Формулировка (как метод). Чтобы доказать $\exists\,\xi: \Phi(\xi, f(\xi), f'(\xi), \ldots) = 0$, строят вспомогательную $g(x)$ такую, что:
1. $g(a) = g(b)$ или $g(a) \cdot g(b) < 0$,
2. $g'(\xi) = 0 \Leftrightarrow \Phi(\xi, \ldots) = 0$.

Дальше — теорема Ролля или о промежуточном значении.

Шаблоны $\Phi \to g$.

| Уравнение $\Phi = 0$ | Вспомогательная $g$ |
|---|---|
| $f' - \lambda f$ | $g = f \cdot e^{-\lambda x}$ |
| $f' + \lambda f$ | $g = f \cdot e^{\lambda x}$ |
| $f' - f/x$ | $g = f(x)/x$, $0 \notin [a,b]$ |
| $f' + f^2 + 1$ | $g = \arctan f(x) + x$ |
| $f^{(n+1)} - f$ | $g = (f + f' + \cdots + f^{(n)}) e^{-x}$ |
| $f - x f'$ | $g = f(x)/x$ |
| $f'' - 5f' + 6f$ | $g = e^{-2x}(f' - 3f)$ (факторизация $(D-2)(D-3)$) |

Условия применимости.

*Возможность представить $\Phi$ как $g'(x) \cdot h(x)$ с ненулевой $h$.* Если $\Phi$ — линейная комбинация $f$ и $f'$ с переменными коэффициентами, спасает интегрирующий множитель $e^{\int p}$.

Идея метода. Прямая проверка: $g'(\xi) = 0 \Leftrightarrow \Phi(\xi) = 0$. Дальше — Ролля для подобранной $g$.

### E8. Итерированное Ролля и счёт нулей (iterated Rolle)

Формулировка. Пусть $f \in C^n([a,b])$ имеет $n+1$ нулей на $[a,b]$ (с учётом кратности). Тогда:
- $f'$ имеет $\geq n$ нулей внутри,
- $f''$ имеет $\geq n - 1$ нулей,
- $f^{(n)}$ имеет хотя бы один внутренний ноль.

Условия применимости.

*Учёт кратности.* Если $a$ — нуль кратности $k$ для $f$, то $a$ автоматически нуль для $f', f'', \ldots, f^{(k-1)}$. Без учёта кратности оценка слабее.

*Существование $f^{(n)}$ на $(a, b)$.* Для последнего шага достаточно дифференцируемости $f^{(n-1)}$ внутри.

Идея доказательства. Между каждой парой соседних нулей $f$ есть нуль $f'$ по теореме Ролля. Итерация даёт цепочку.

Вариант с экспонентой. Между нулями $f$ есть нуль $f' + \lambda f$ — применить итерированное Ролля к $g(x) = f(x) e^{\lambda x}$. Полезно для оценок числа корней многочленов вида $\sum c_k e^{\alpha_k x}$.

---

### E9. Интерполяция Эрмита плюс итерированное Ролля (Hermite interpolation + iterated Rolle)

Формулировка. Пусть $f \in C^{n+1}([a, b])$ и заданы узлы $x_0, \ldots, x_n \in [a, b]$ (возможно с кратностями). Интерполянт Эрмита $H$ совпадает с $f$ в узлах вместе с производными нужных порядков. Остаток:
$$f(x) - H(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!} \prod_{k=0}^{n} (x - x_k),$$
где $\xi$ зависит от $x$ и лежит в выпуклой оболочке $\{x_0, \ldots, x_n, x\}$.

Условия применимости.

*Гладкость $f^{(n+1)}$.* На всём отрезке, содержащем узлы.

*Кратные узлы.* Если узел $x_i$ повторяется $m_i$ раз, интерполянт согласует значения $f, f', \ldots, f^{(m_i - 1)}$ в нём.

Идея доказательства. К $\phi(x) = f(x) - H(x) - C \prod(x - x_k)$ с подобранной $C$ применяется итерированное Ролля $n+1$ раз. Это даёт $\xi$ с нужным значением $\phi^{(n+1)}(\xi) = 0$, что и есть формула для остатка.

Связь со стандартной формулой Тейлора. Если все узлы совпадают ($x_0 = \cdots = x_n = a$), формула превращается в Тейлор с остатком Лагранжа.

### E10. Неравенство Гронуолла (Gronwall's inequality)

Формулировка.

*Интегральная форма.* $u, \alpha \geq 0$ непрерывны на $[a, b]$, $C \geq 0$. Если
$$u(t) \leq C + \int_a^t \alpha(s) u(s)\,ds,$$
то $u(t) \leq C \cdot \exp\bigl(\int_a^t \alpha(s)\,ds\bigr)$.

*Дифференциальная форма.* $u'(t) \leq \alpha(t) u(t) + \beta(t)$ с $u(a) \leq u_0$ даёт
$$u(t) \leq u_0 \cdot e^{\int_a^t \alpha} + \int_a^t \beta(s) e^{\int_s^t \alpha}\,ds.$$

Условия применимости.

*Знак $\alpha \geq 0$ для интегральной формы.* Пример без условия: $\alpha \equiv -1$, $u \equiv 1$ удовлетворяет $1 \leq 1 + \int_0^t (-1) \cdot 1\,ds = 1 - t$ только при $t \leq 0$. При $t > 0$ оценка $u \leq e^{-t}$ ложна.

*Знак $\alpha$ не важен для дифференциальной формы.* Там работает интегрирующий множитель $e^{-\int \alpha}$, и знак включается в показатель.

Идея доказательства. Дифференциальная — умножить на $e^{-\int \alpha}$ и проинтегрировать. Интегральная — положить $v(t) = C + \int \alpha u$, тогда $v' = \alpha u \leq \alpha v$, и применить дифференциальную форму к $v$.

Дискретный аналог. $u_n \leq C + \sum_{k < n} \alpha_k u_k$ даёт $u_n \leq C \cdot \prod_{k < n}(1 + \alpha_k) \leq C \cdot e^{\sum \alpha_k}$.

## Редкие

### E11. Теорема Флетта (Flett's MVT)

Формулировка. $f$ дифференцируема на $[a, b]$ и $f'(a) = f'(b)$. Тогда $\exists\,\xi \in (a, b)$:
$$f(\xi) - f(a) = (\xi - a) f'(\xi).$$
Геометрически: касательная в точке $\xi$ проходит через $(a, f(a))$.

Условия применимости. Дифференцируемость на замкнутом отрезке (на концах — односторонняя). Условие $f'(a) = f'(b)$ — аналог гипотезы Ролля для производных.

Идея доказательства. Без потери общности $f'(a) = f'(b) = 0$ (иначе вычесть линейную). Функция $g(x) = (f(x) - f(a))/(x - a)$ при $x > a$ и $g(a) = 0$ имеет внутренний экстремум $\xi$, что даёт $g'(\xi) = 0$ — это и есть формула.

---

### E12. Теорема Помпею (Pompeiu's MVT)

Формулировка. $f \in C([a, b])$, дифференцируема на $(a, b)$, $0 \notin [a, b]$. Тогда $\exists\,\xi \in (a, b)$:
$$\frac{b \cdot f(a) - a \cdot f(b)}{b - a} = f(\xi) - \xi \cdot f'(\xi).$$

Условия применимости.

*$0 \notin [a, b]$ строго.* При $0 \in [a, b]$ замена координат $t \mapsto 1/t$ вырождается.

Идея доказательства. Применить теорему Лагранжа к $g(t) = t \cdot f(1/t)$ на отрезке $[1/b, 1/a]$. Производная $g'(\eta) = f(1/\eta) - (1/\eta) f'(1/\eta)$. Подставить $\xi = 1/\eta$.

---

### E13. Покрытия Витали для дифференцируемости (Vitali covering)

Формулировка. Из любого покрытия Витали множества $E \subset [a, b]$ интервалами с длинами $\to 0$ можно выделить счётный непересекающийся поднабор, покрывающий $E$ почти всюду по мере Лебега.

Олимпиадное применение. Дано счётное семейство интервалов $\{I_n\}$ с $\sum |I_n| < L$. Тогда дополнение в $[0, L]$ непусто — есть точка, не лежащая ни в одном $I_n$. Этот трюк используется для нахождения точек, где интересующее свойство выполнено (например, где функция дифференцируема с заданной производной).

Идея доказательства. Жадный алгоритм: выбираем интервал почти максимальной длины, удаляем все пересекающиеся, повторяем. Полученный непересекающийся набор покрывает $E$ с фактором 5.

## Self-test

1. $f \in C^2([0, 1])$, $f(0) = f(1) = 0$, и существует $c \in (0, 1)$ с $f(c) = 1$. Докажи, что $\exists\,\xi \in (0, 1)$: $f''(\xi) \leq -8$.

2. $f: [0, +\infty) \to \mathbb{R}$ непрерывна, дифференцируема на $(0, \infty)$, $f(0) = 0$, $|f'(x)| \leq |f(x)|$ при $x > 0$. Докажи, что $f \equiv 0$.

3. Найди $\displaystyle\int_0^\infty \frac{e^{-ax} - e^{-bx}}{x}\,dx$ для $a, b > 0$.

4. $f \in C^1([1, 2])$, $f'(1) = f'(2)$. Докажи, что найдётся $\xi \in (1, 2)$, в котором касательная к графику $f$ проходит через точку $(1, f(1))$.

5. $f \in C^2(\mathbb{R})$, $f$ и $f''$ ограничены. Докажи неравенство Ландау–Адамара:
$$(\sup_{\mathbb{R}} |f'|)^2 \leq 2 \cdot \sup_{\mathbb{R}} |f| \cdot \sup_{\mathbb{R}} |f''|.$$

## Решения

1. Тейлор в точке $c$:
$$0 = f(0) = 1 - c \cdot f'(c) + \frac{c^2}{2} f''(\xi_1), \quad \xi_1 \in (0, c),$$
$$0 = f(1) = 1 + (1-c) \cdot f'(c) + \frac{(1-c)^2}{2} f''(\xi_2), \quad \xi_2 \in (c, 1).$$
Умножим первое на $(1-c)$, второе на $c$ и сложим — линейный по $f'(c)$ член занулится:
$$0 = 1 + \frac{c(1-c)}{2} \bigl[c \cdot f''(\xi_1) + (1-c) \cdot f''(\xi_2)\bigr].$$
Правая скобка — выпуклая комбинация значений $f''$. Минимум не превосходит её:
$$\min(f''(\xi_1), f''(\xi_2)) \leq -\frac{2}{c(1-c)} \leq -8,$$
поскольку $c(1-c) \leq 1/4$ на $[0, 1]$. Равенство достигается на $f(x) = 4x(1-x)$.

2. Применяю Гронуолла. На любом $[0, T]$:
$$|f(x)| = \left|\int_0^x f'(s)\,ds\right| \leq \int_0^x |f(s)|\,ds.$$
Это интегральная форма с $C = 0$, $\alpha \equiv 1$. Заключение: $|f(x)| \leq 0$, то есть $f \equiv 0$.

3. Дифференцирование по параметру. Положу $I(a) = \int_0^\infty \frac{e^{-ax} - e^{-bx}}{x}\,dx$. Подынтегральная функция при $x \to 0$ имеет конечный предел $b - a$, при $x \to \infty$ экспоненциально мала — интеграл сходится.

   Дифференцирую по $a$:
$$I'(a) = -\int_0^\infty e^{-ax}\,dx = -\frac{1}{a}.$$
Мажоранта $e^{-\varepsilon x}$ при $a \geq \varepsilon > 0$ оправдывает обмен.

   Интегрирую: $I(a) = -\ln a + C(b)$. Из $I(b) = 0$ получаю $C(b) = \ln b$. Ответ: $\ln(b/a)$.

4. Прямое применение теоремы Флетта. Условия выполнены, отсюда $\exists\,\xi \in (1, 2)$: $f(\xi) - f(1) = (\xi - 1) f'(\xi)$. Это в точности означает, что касательная $y = f(\xi) + f'(\xi)(x - \xi)$ в точке $x = 1$ проходит через $(1, f(1))$.

5. Обозначу $A = \sup |f|$, $B = \sup |f''|$. Применяю Тейлор второго порядка в обе стороны от точки $x$ с шагом $h > 0$:
$$f(x + h) = f(x) + h f'(x) + \frac{h^2}{2} f''(\xi_+), \quad \xi_+ \in (x, x+h),$$
$$f(x - h) = f(x) - h f'(x) + \frac{h^2}{2} f''(\xi_-), \quad \xi_- \in (x-h, x).$$
Вычитаю второе из первого:
$$f(x+h) - f(x-h) = 2h f'(x) + \frac{h^2}{2}\bigl[f''(\xi_+) - f''(\xi_-)\bigr].$$
Оцениваю модули: $2h |f'(x)| \leq |f(x+h)| + |f(x-h)| + h^2 B \leq 2A + h^2 B$. Делю на $2h$:
$$|f'(x)| \leq \frac{A}{h} + \frac{Bh}{2}.$$
Минимизирую правую часть по $h > 0$. Производная $-A/h^2 + B/2 = 0$ даёт $h^* = \sqrt{2A/B}$. Значение в минимуме:
$$|f'(x)| \leq \sqrt{AB/2} + \sqrt{AB/2} = \sqrt{2AB}.$$
Возведя в квадрат, получаю $|f'(x)|^2 \leq 2AB$, что и требовалось. Это неравенство Адамара. Аналогичная оценка на полупрямой даёт константу $4$ вместо $2$ (односторонний Тейлор слабее).
