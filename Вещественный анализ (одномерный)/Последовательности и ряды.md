# Последовательности и ряды: тесты сходимости, Stolz–Cesàro, Abel summation, телескопирование, перестановки, асимптотики

## Scope
Тонкие критерии сходимости числовых последовательностей и рядов, вычисление и оценка частных сумм через summation by parts и телескопирование, перестановки и перегруппировки условно сходящихся рядов, точные асимптотики ($n!$, $H_n$, $\binom{2n}{n}$) через Stirling, Wallis, Euler–Maclaurin, а также связанные с ними классические неравенства (Carleman, Hardy) и summability methods.

## Записи — Core

### E1. Stolz–Cesàro theorem (теорема Штольца–Чезаро)

Формулировка ($\frac{\infty}{\infty}$). Пусть $b_n \uparrow +\infty$ строго монотонно. Если существует $\lim_{n\to\infty} \frac{a_{n+1}-a_n}{b_{n+1}-b_n} = L \in \overline{\mathbb{R}}$, то $\lim \frac{a_n}{b_n} = L$. Аналог $\frac{0}{0}$: $a_n,b_n \to 0$, $b_n$ строго убывает к $0$ — заключение то же.

Условия применимости. Строгая монотонность $b_n$ критична. Обратная импликация в общем случае ложна (как и l'Hôpital). Не работает, если предел разностей колеблется.

Идея доказательства. Для $\frac{\infty}{\infty}$: $a_n - a_m = \sum_{k=m}^{n-1}(a_{k+1}-a_k)$, $b_n - b_m = \sum (b_{k+1}-b_k) > 0$; зажатие отношения через $\inf$/$\sup$ на хвосте даёт $\frac{a_n - a_m}{b_n - b_m} \to L$, далее устремляем $m$ медленнее $n$.

Варианты и обобщения.
- **Cesàro mean**: $a_n \to L \Rightarrow \frac{1}{n}\sum_{k=1}^n a_k \to L$ (частный случай при $b_n = n$).
- **Геометрическая версия**: $a_n > 0$, $\frac{a_{n+1}}{a_n} \to L \Rightarrow \sqrt[n]{a_n} \to L$. Отсюда $\sqrt[n]{n!}/n \to 1/e$.
- **Степенная**: если $a_n \to L$, то $\sqrt[n]{a_1 \cdots a_n} \to L$.

Используется в. [IMC-1996-4, IMC-1994-DAY2-6, Putnam-1966-B3, Putnam-1995-A4, многие задачи на $\lim \sqrt[n]{a_n}$].

### E5. Abel summation by parts (преобразование Абеля)

Формулировка. Для конечных сумм с $A_k = \sum_{i=1}^k a_i$:
$$\sum_{k=1}^{n} a_k b_k = A_n b_n - \sum_{k=1}^{n-1} A_k (b_{k+1} - b_k).$$
Дискретный аналог интегрирования по частям.

Условия применимости. Универсально для любых $a_k, b_k$. Полезно когда $b_k$ монотонна (тогда разности одного знака) и/или $A_k$ ограничена.

Идея доказательства. Подстановка $a_k = A_k - A_{k-1}$ и сдвиг индекса.

Варианты и обобщения.
- **Dirichlet's test**: если $A_n$ ограничена и $b_n \downarrow 0$, то $\sum a_n b_n$ сходится.
- **Abel's test**: если $\sum a_n$ сходится и $b_n$ монотонна и ограничена, то $\sum a_n b_n$ сходится.
- **Leibniz** — частный случай Dirichlet с $a_n = (-1)^n$.
- **Equivalence**: для перевода $\sum$ в интеграл от ступенчатой функции.

Используется в. [IMC-1995-DAY2-2, IMC-1996-4, IMC-1997-DAY2-3, Putnam-2002-A6, Putnam-2010-B4]; универсальный инструмент для оценки $\sum a_n b_n$ через $\sup |A_n|$.

### E6. Abel's theorem on power series (теорема Абеля о граничном поведении)

Формулировка. Пусть $\sum a_n$ сходится к $S$. Тогда $f(x) = \sum a_n x^n$ непрерывна в $x=1$ слева: $\lim_{x \to 1^-} f(x) = S$.

Условия применимости. Достаточна обычная сходимость $\sum a_n$, не требуется абсолютная. Радиальный предел; для угловых пределов внутри Stolz angle есть аналог.

Идея доказательства. Через Abel summation по частям: $f(x) - S_N = \sum_{n>N}(S_n - S)(x^n - x^{n+1}) - (S_N - S)x^{N+1}$, далее $\varepsilon$-аргумент.

Варианты и обобщения.
- **Abel's limit theorem** для рядов Дирихле.
- Используется для вычисления значений типа $\sum (-1)^{n-1}/n = \ln 2$, $\sum (-1)^n/(2n+1) = \pi/4$ через подстановку $x \to 1^-$ в известное разложение.
- Не имеет обратной без Tauberian условия (см. E7).

Используется в. Putnam-1990-B2, Putnam-2014-B2; стандартный приём для замкнутых выражений сумм через известные разложения.

### E11. Stirling's formula с явными границами Robbins

Формулировка. Для любого $n\geq 1$:
$$n! = \sqrt{2\pi n}\,\Bigl(\frac{n}{e}\Bigr)^n e^{r_n}, \qquad \frac{1}{12n+1} < r_n < \frac{1}{12n}.$$
Отсюда $n! \sim \sqrt{2\pi n}\,(n/e)^n$ с относительной ошибкой $\Theta(1/n)$.

Условия применимости. Универсально. Двусторонние границы Robbins полезны, когда нужна точная нижняя/верхняя оценка, а не только асимптотика (типичный приём в IMC: после применения Stirling появляется $\sqrt{n}$, и нужно показать строгое неравенство).

Идея доказательства. Положить $d_n = \ln n! - (n+1/2)\ln n + n$. Показать $d_n - d_{n+1} > 0$ и оценить разность через $\frac{1}{12n} - \frac{1}{12(n+1)}$ — телескоп даёт сходимость $d_n \to d_\infty$. Константа $\sqrt{2\pi}$ извлекается из Wallis (см. E12).

Варианты и обобщения.
- **Полная Euler–Maclaurin асимптотика**: $\ln n! = (n+1/2)\ln n - n + \frac{1}{2}\ln(2\pi) + \sum_{k\geq 1} \frac{B_{2k}}{2k(2k-1)n^{2k-1}}$ (расходящийся ряд, но дающий оптимальную аппроксимацию при правильном обрезании).
- $\Gamma(s)$ — гамма-функция, та же асимптотика.

Используется в. Любая задача с $\binom{2n}{n}$, факториалами в знаменателях, скоростью сходимости. Putnam-2017-B3, IMC-2000-A1.

### E12. Wallis product (формула Уоллиса)

Формулировка.
$$\frac{\pi}{2} = \prod_{n=1}^{\infty} \frac{(2n)^2}{(2n-1)(2n+1)} = \lim_{n\to\infty} \frac{1}{2n+1}\Bigl(\frac{(2n)!!}{(2n-1)!!}\Bigr)^2.$$
Эквивалентно: $\binom{2n}{n} \sim \frac{4^n}{\sqrt{\pi n}}$.

Условия применимости. Прямое следствие $W_n = \int_0^{\pi/2} \sin^n x\,dx$ и рекурсии $W_n = \frac{n-1}{n} W_{n-2}$ + монотонности $W_{n-1} \geq W_n \geq W_{n+1}$.

Идея доказательства. Монотонность и точное вычисление $W_{2n}, W_{2n+1}$ через двойной факториал; зажатие $W_{2n+1}/W_{2n} \to 1$ даёт сходимость.

Варианты и обобщения.
- Связь с $\Gamma$: $\sqrt{\pi} = \Gamma(1/2)$, формула удвоения.
- Используется для вычисления константы $\sqrt{2\pi}$ в Stirling: сравнение $\binom{2n}{n}$ через факториал и через Wallis даёт $\sqrt{2\pi}$.
- **Stirling's series для центрального биномиального**: $\binom{2n}{n} = \frac{4^n}{\sqrt{\pi n}}\bigl(1 - \frac{1}{8n} + \frac{1}{128n^2} + O(1/n^3)\bigr)$.

Используется в. IMC-1996-DAY2-5 (Riemann sum для $\int_0^\infty t/(1+t^2)^2$); Putnam-1970-B6, задачи на random walk возвратность.

### E16. Sums as Riemann integrals (сумма как Riemann sum)

Формулировка. Если $f$ Riemann-интегрируема на $[0,1]$, то $\frac{1}{n}\sum_{k=1}^{n} f(k/n) \to \int_0^1 f$. Более тонкие версии: для $f$ с интегрируемой производной погрешность $O(1/n)$; для $f \in C^2$ — $\frac{1}{n}\sum f(k/n) = \int_0^1 f + \frac{f(1)-f(0)}{2n} + O(1/n^2)$ (частный случай Euler–Maclaurin).

Условия применимости. Riemann-интегрируемость; для асимптотики выше — гладкость. На неравномерных сетках мелкость $\max(x_{k+1}-x_k) \to 0$ заменяет $1/n$.

Идея доказательства. Стандартное определение интеграла Римана через равномерные суммы; для асимптотики — Euler–Maclaurin с одним членом.

Варианты и обобщения.
- **Неравномерная сетка** $h = 1/\sqrt{x}$: $\sum f(n/\sqrt{x})/\sqrt{x} \to \int_0^\infty f$ — мелкость зависит от параметра.
- Замена сумм по решётке двумерной на $\iint$ при подходящих условиях (multi-dim).
- Если $f$ только полунепрерывна — нужно следить за односторонними пределами.

Используется в. IMC-1996-DAY2-5 ($\sum nx/(n^2+x)^2 \to 1/2$ через сетку $h = 1/\sqrt{x}$); Putnam-1970-B6, Putnam-2014-A2; универсальный приём для $\sum f(k)/n^\alpha$.

## Записи — Standard

### E2. Cauchy condensation test (тест Коши о сгущении)

Формулировка. Если $a_n \geq 0$ монотонно невозрастает, то $\sum_{n=1}^\infty a_n$ и $\sum_{k=0}^\infty 2^k a_{2^k}$ сходятся или расходятся одновременно.

Условия применимости. Монотонность необходима. Без неё: $\sum a_{2^k}$ может вести себя как угодно относительно $\sum a_n$.

Идея доказательства. Зажатие частными суммами: $\sum_{n=2^{k}}^{2^{k+1}-1} a_n$ оценивается сверху $2^k a_{2^k}$ и снизу $2^k a_{2^{k+1}}$ из монотонности.

Варианты и обобщения.
- **Schlömilch**: вместо $2^k$ можно брать любую $n_k$ с $\frac{n_{k+1}}{n_k}$ ограниченной сверху и снизу.
- Сразу даёт сходимость $\sum 1/(n \log^p n)$ при $p>1$ и расходимость при $p \leq 1$ — каскадно для $\sum 1/(n \log n \log\log n \cdots \log_k^p n)$.

Используется в. [Putnam-2008-A4, многочисленные задачи на ряды Бертрана; стандартный приём при ряде с $\log$].

### E3. Kummer–Raabe–Gauss tests (Куммер, Раабе, Гаусс)

Формулировка (Kummer). $a_n>0$, $\{p_n\}>0$ — некоторая выбранная последовательность. Положим $K_n = p_n \frac{a_n}{a_{n+1}} - p_{n+1}$. Тогда:

- $\liminf K_n > 0 \Rightarrow \sum a_n$ сходится;
- $\limsup K_n < 0$ и $\sum 1/p_n$ расходится $\Rightarrow \sum a_n$ расходится.

Частные случаи.
- **Raabe**: $p_n = n$, проверяем $R_n = n\bigl(\frac{a_n}{a_{n+1}} - 1\bigr)$. Сходимость при $\liminf R_n > 1$, расходимость при $\limsup R_n < 1$.
- **Bertrand**: $p_n = n \ln n$. Применяется когда Raabe даёт ровно $1$.
- **Gauss**: если $\frac{a_n}{a_{n+1}} = 1 + \frac{\mu}{n} + O(1/n^{1+\delta})$, то сходимость $\Leftrightarrow \mu > 1$ (включает граничный случай $\mu=1$ — расходится).

Условия применимости. $a_n > 0$ строго. Тесты лестничные: каждый следующий разрешает граничный случай предыдущего. Когда ratio test даёт $1$ — переходим на Raabe; если Raabe даёт $1$ — на Bertrand или Gauss.

Идея доказательства. Kummer: если $K_n \geq c > 0$, то $a_{n+1} \leq \frac{1}{c}(p_n a_n - p_{n+1} a_{n+1})$, телескоп даёт $\sum a_{n+1} \leq \frac{p_1 a_1}{c}$.

Используется в. Гипергеометрические ряды, ряды с рациональным $a_n/a_{n+1}$ (Putnam-1992-B4, типовой приём в задачах с факториалами).

### E4. Olivier theorem (теорема Оливье)

Формулировка. Если $a_n \geq 0$, $a_n$ невозрастает и $\sum a_n < \infty$, то $n a_n \to 0$.

Условия применимости. Монотонность критична: контрпример $a_n = 1/n$ при $n=k^2$, $a_n = 2^{-n}$ иначе — ряд сходится, но $k^2 a_{k^2} \to 1$.

Идея доказательства. Из монотонности $\sum_{k=n+1}^{2n} a_k \geq n a_{2n}$, и эта сумма $\to 0$ как хвост сходящегося ряда. Значит $2n a_{2n} \to 0$; для нечётных аналогично.

Варианты и обобщения.
- **Pringsheim**: для расходимости $\sum a_n$ с $a_n \downarrow 0$ всё равно может быть $n a_n \not\to 0$ — контрпримеры строятся через сгущение.
- Усиление: $\sum_{k \leq n} k a_k = o(n)$ при тех же условиях.
- Без монотонности эквивалент: статистическая сходимость $n a_n \to 0$ по плотности.

Используется в. Доказательство, что $\sum 1/n$ расходится без интегрального теста; задачи, где из расходимости/сходимости нужно извлечь скорость.

### E8. Riemann rearrangement theorem (теорема Римана о перестановке)

Формулировка. Если $\sum a_n$ сходится условно (но не абсолютно), то для любого $L \in \mathbb{R} \cup \{\pm\infty\}$ существует перестановка $\sigma$ со $\sum a_{\sigma(n)} = L$. Более того, можно сделать частные суммы перестановки осциллирующими между любыми $\liminf < \limsup$.

Условия применимости. Только условная сходимость. Абсолютная сходимость даёт инвариантность под перестановкой (сразу из Tonelli).

Идея доказательства. Положительная и отрицательная части $\sum a_n^+ = +\infty$, $\sum a_n^- = -\infty$. Жадный алгоритм: пока сумма ниже цели — берём положительные слагаемые в порядке индексов; затем переключаемся на отрицательные; и т.д.

Варианты и обобщения.
- **Lévy–Steinitz**: в $\mathbb{R}^d$ множество сумм перестановок $\sum v_n$ — это аффинное подпространство.
- **Sierpiński**: ограниченные перестановки (только $\sigma$ с $|\sigma(n)-n|$ ограничено) сохраняют сумму.
- **Block-permutation**: перестановка с блоками длины $\leq f(n)$ сохраняет сумму, если $f$ растёт «не слишком быстро».

Используется в. [IMC-1997-2 — переставление с блоками длины $2^{n-1}$ сохраняет; контрпример с $a_n = (-1)^{n+1}/\sqrt{n}$ показывает, что для блоков большей длины нет].

### E9. Mertens' theorem on Cauchy product (теорема Мертенса о произведении рядов)

Формулировка. Пусть $\sum a_n = A$ сходится абсолютно, $\sum b_n = B$ сходится (хотя бы условно). Тогда Cauchy product $c_n = \sum_{k=0}^n a_k b_{n-k}$ даёт сходящийся ряд $\sum c_n = AB$.

Условия применимости. Достаточно абсолютной сходимости одного из рядов. Если оба только условно — можно расходиться: $a_n = b_n = (-1)^n/\sqrt{n+1}$ — оба сходятся, $c_n \not\to 0$.

Идея доказательства. $\sum_{n\leq N} c_n - A_N B_N$ оценивается через $\sup |B_n|$ и хвост абсолютно сходящегося $\sum |a_n|$.

Варианты и обобщения.
- **Abel's theorem на произведения**: если все три ряда $\sum a_n, \sum b_n, \sum c_n$ сходятся, то $\sum c_n = (\sum a_n)(\sum b_n)$ — без условий абсолютной сходимости.
- Cauchy product экспоненциальных рядов даёт $e^{x+y} = e^x e^y$ (формальное доказательство тождества).

Используется в. Putnam-2004-A2; стандартное обоснование при работе с генерирующими функциями и тождествами вида $\bigl(\sum 1/n!\bigr)^2 = \sum 2^n/n!$.

### E13. Euler–Maclaurin formula (формула Эйлера–Маклорена)

Формулировка. Для $f \in C^{2m}([a,b])$, $a,b\in\mathbb{Z}$:
$$\sum_{k=a}^{b} f(k) = \int_a^b f(x)\,dx + \frac{f(a)+f(b)}{2} + \sum_{k=1}^{m} \frac{B_{2k}}{(2k)!}\bigl[f^{(2k-1)}(b) - f^{(2k-1)}(a)\bigr] + R_m,$$
где $|R_m| \leq \frac{2\zeta(2m)}{(2\pi)^{2m}} \int_a^b |f^{(2m)}(x)|\,dx$, $B_{2k}$ — числа Бернулли.

Условия применимости. Гладкость $f$ нужна на всём $[a,b]$. В асимптотических разложениях ряд по $m$ обычно расходится, но обрезанный на оптимальном $m$ даёт экспоненциальную точность.

Идея доказательства. Интегрирование по частям с периодическими функциями Bernoulli $\widetilde{B}_k(x) = B_k(\{x\})$, индукция по $m$.

Варианты и обобщения.
- **Гармонические числа**: $H_n = \ln n + \gamma + \frac{1}{2n} - \sum_{k\geq 1} \frac{B_{2k}}{2k\,n^{2k}}$.
- **Stirling**: применение к $f(x)=\ln x$ даёт полное асимптотическое разложение $\ln n!$.
- **Abel–Plana formula** — комплексный аналог.

Используется в. Тонкая асимптотика $H_n - \ln n$, $\zeta(s) - \frac{1}{s-1}$ при $s\to 1$; Putnam-1979-B5; задачи где нужны члены ниже главного.

### E14. Carleman's inequality (неравенство Карлемана)

Формулировка. Для $a_n > 0$, $\sum a_n < \infty$:
$$\sum_{n=1}^{\infty} (a_1 a_2 \cdots a_n)^{1/n} \leq e \sum_{n=1}^{\infty} a_n,$$
причём константа $e$ — наилучшая возможная.

Условия применимости. Положительность строгая. Граничный случай — равенство недостижимо при нетривиальной $a$.

Идея доказательства. Подбор веса $c_n = (n+1)^n/n^{n-1}$. AM–GM: $(c_1 a_1 \cdots c_n a_n)^{1/n} \leq \frac{1}{n}\sum c_k a_k$. Перегруппировка и телескоп $\sum_{m \geq n} \frac{1}{m(m+1)} = \frac{1}{n}$ дают $\sum (a_1\cdots a_n)^{1/n} \leq e \sum a_n$. Точность $e$: $\prod_{k=1}^n (1+1/k)^k \to e^n$.

Варианты и обобщения.
- **Hardy's inequality** (E15) — Carleman есть его предельный случай при $p\to\infty$.
- Весовая Carleman: $\sum w_n (a_1 \cdots a_n)^{1/n} \leq C(w) \sum a_n$ для подходящих $w_n$.
- Mixed means inequality (Hardy–Littlewood–Polya).

Используется в. IMC-1996-DAY2-6 (явное доказательство с $c_n$); Putnam-2003-B6; задачи с произведениями частных сумм.

### E15. Hardy's inequality (неравенство Харди)

Формулировка. Для $p > 1$ и $a_n \geq 0$:
$$\sum_{n=1}^{\infty} \Bigl(\frac{1}{n}\sum_{k=1}^{n} a_k\Bigr)^p \leq \Bigl(\frac{p}{p-1}\Bigr)^p \sum_{n=1}^{\infty} a_n^p,$$
константа $\bigl(\frac{p}{p-1}\bigr)^p$ оптимальна.

Условия применимости. $p>1$ строго. При $p=1$ оценки нет (берём $a_n = \delta_{n=1}$). Для интегрального аналога: $\int_0^\infty \bigl(\frac{1}{x}\int_0^x f\bigr)^p dx \leq \bigl(\frac{p}{p-1}\bigr)^p \int_0^\infty f^p$.

Идея доказательства. Положим $A_n = \frac{1}{n}\sum_{k\leq n} a_k$. Через Hölder и Abel summation: $\sum n A_n^p - \sum (n-1) A_{n-1}^p \leq \frac{p}{p-1}(\sum a_n^p)^{1/p}(\sum A_n^p)^{1-1/p}$. Деление и предел.

Варианты и обобщения.
- **Carleman** (E14) при $p \to \infty$.
- **Copson, Pólya, Hilbert** — родственные неравенства.
- Двусторонние оценки на $\sum (\frac{1}{n}\sum a_k)^p$ через $\sum a_n^p$ — Mathieu и Knopp.

Используется в. Оценки $\ell^p$-норм усреднений; Putnam-1995-B5; теоретические задачи с моментами.

### E17. Weyl equidistribution (равномерное распределение mod 1 по Вейлю)

Формулировка. Последовательность $(x_n) \subset [0,1)$ равномерно распределена тогда и только тогда, когда для каждого $h \in \mathbb{Z}\setminus\{0\}$:
$$\frac{1}{N}\sum_{n=1}^{N} e^{2\pi i h x_n} \to 0.$$
Следствие: если $\alpha$ иррационально, то $\{n\alpha\}$ равномерно распределено в $[0,1)$.

Условия применимости. Эквивалентность — точная (через теорему Вейерштрасса о полиномиальной аппроксимации). Достаточно проверить для тригонометрических полиномов.

Идея доказательства. ($\Rightarrow$) Очевидно для тригонометрических. ($\Leftarrow$) Аппроксимация индикатора интервала тригонометрическим полиномом сверху и снизу + предельный переход.

Варианты и обобщения.
- **Weyl на полиномиальных последовательностях**: $\{p(n)\}$ равномерно распределено для полинома $p$ хотя бы с одним иррациональным коэффициентом старшей степени.
- **Erdős–Turán** оценка discrepancy.
- $\{\log n / \log b\}$ не равномерно распределено для иррационального $\log b$ (Benford), но $\{n \log b\}$ — да.

Используется в. IMC-1997-DAY2-3 ($\sin(\log n)$ не сходится к нулю через равномерное распределение $\log n / \pi \mod 1$); Putnam-2014-A6; задачи с $\{n\alpha\}$ и оценками частных сумм через discrepancy.

## Записи — Exotic

### E7. Tauberian theorems (теоремы Таубера и Литтлвуда)

Формулировка (Tauber, 1897). Пусть $f(x) = \sum a_n x^n$, $\lim_{x\to 1^-} f(x) = S$ и $n a_n \to 0$. Тогда $\sum a_n = S$.

Формулировка (Littlewood, 1911). Если $\lim_{x\to 1^-} f(x) = S$ и $a_n = O(1/n)$, то $\sum a_n = S$.

Hardy–Littlewood. Если $a_n \geq 0$ и $\sum a_n x^n \sim \frac{C}{1-x}$ при $x\to 1^-$, то $\sum_{k\leq n} a_k \sim Cn$.

Условия применимости. Tauber: «маленький» односторонний рост коэффициентов. Littlewood и Hardy–Littlewood — глубокие, доказательство нетривиально (Karamata-style argument).

Идея доказательства (Karamata для Hardy–Littlewood). Аппроксимация индикатора $\mathbf{1}_{[0,1]}$ многочленами Бернштейна по $x^n$ и использование $\sum a_n x^n \sim C/(1-x)$.

Варианты и обобщения.
- Wiener Tauberian theorem (через преобразование Фурье).
- Karamata's theorem для регулярно меняющихся функций.

Используется в. Подсчёт $\pi(n) \sim n/\ln n$ из эквивалентов $\zeta$-функции; обоснование переходов между Abel-, Cesàro- и обычной суммируемостью в задачах.

### E10. Silverman–Toeplitz theorem (теорема о регулярной матричной суммируемости)

Формулировка. Бесконечная матрица $(t_{nk})_{n,k \geq 0}$ задаёт regular summation (т.е. сохраняет любые сходящиеся последовательности с пределом) тогда и только тогда, когда:
1. $\sup_n \sum_k |t_{nk}| < \infty$,
2. $\lim_n t_{nk} = 0$ для каждого $k$,
3. $\lim_n \sum_k t_{nk} = 1$.

Условия применимости. Все три условия независимы. Cesàro-суммирование $t_{nk} = 1/n$ для $k\leq n$ удовлетворяет всем трём.

Идея доказательства. ($\Leftarrow$) Стандартный $\varepsilon$-аргумент с разделением на конечный начальный блок и хвост. ($\Rightarrow$) Применить к специальным последовательностям ($e_k$, постоянная $1$, единичные импульсы).

Варианты и обобщения.
- **Hausdorff means**, **Borel summation**, **Abel summation** — частные случаи.
- Toeplitz lemma: если $\sum_k |t_{nk}| < C$, $t_{nk} \to 0$ покoординатно, $\sum_k t_{nk} \to 1$, и $x_k \to L$, то $\sum_k t_{nk} x_k \to L$ — рабочая формулировка для решения задач.

Используется в. Доказательство Stolz–Cesàro; Putnam-2010-B4 (взвешенное среднее); задачи о пределах вида $\lim_n \sum_k w_{nk} f(k/n)$ — это Toeplitz lemma + узнавание Riemann sum.

### E18. Engel-like / greedy infinite expansion (жадное разложение)

Формулировка. Любое $\theta \in (1, \infty)$ единственным образом представимо в виде сходящегося произведения
$$\theta = \prod_{k=1}^{\infty} \Bigl(1 + \frac{1}{n_k}\Bigr),$$
где $n_1 \leq n_2 \leq \ldots$ выбираются жадно: $n_k$ — минимальное натуральное с $\prod_{j\leq k}(1+1/n_j) \leq \theta$.

Условия применимости. $\theta > 1$. Для $\theta < 1$ — аналогичный жадный алгоритм с $1 - 1/n_k$.

Идея доказательства. Сходимость: $\theta_k = \theta / \prod_{j\leq k}(1+1/n_j) > 1$, и $1+1/n_{k+1} \leq \theta_k < 1+1/(n_{k+1}-1)$ даёт $n_{k+1} \geq n_k$ и $n_{k+1} \geq n_k^2$ (после оценки). Единственность — индукцией на жадности.

Варианты и обобщения.
- **Engel expansion** (классический): $x = \sum 1/(a_1 a_2 \cdots a_k)$, $a_1 \leq a_2 \leq \ldots$.
- **Lüroth expansion**, **Sylvester–Fibonacci** для дробей.
- Связь с рациональностью: $\theta$ рационально $\Leftrightarrow$ хвост обрывается на $n_{k+1} = n_k^2$.

Используется в. [IMC-1997-4]; задачи на представления чисел и характеризацию рациональности через алгоритмы.

## Self-test

1. Найди $\displaystyle\lim_{n\to\infty} \Bigl(\frac{(2n)!}{n!\,n^n}\Bigr)^{1/n}$.

2. Вычисли $\displaystyle\lim_{n\to\infty} n\sin(2\pi e\cdot n!)$.

3. Пусть $a_n > 0$ и $\sum a_n = \infty$. Покажи, что найдётся такая монотонно невозрастающая $\varepsilon_n \to 0$, что $\sum a_n \varepsilon_n = \infty$, и найдётся $\delta_n \to 0$, что $\sum a_n \delta_n < \infty$.

4. Пусть $a_n = \int_0^1 x^n \sqrt{1-x^2}\,dx$. Найди $\lim_{n\to\infty} n^{3/2} a_n$.

5. Покажи, что для иррационального $\alpha$ последовательность $b_n = \cos(2\pi n \alpha) + \cos(2\pi n^2 \alpha)$ удовлетворяет $\frac{1}{N}\sum_{n=1}^N b_n \to 0$.

## Решения

1. По Stirling (E11): $(2n)! \sim \sqrt{4\pi n}\,(2n/e)^{2n}$, $n! \sim \sqrt{2\pi n}\,(n/e)^n$. Отношение $(2n)!/(n!\,n^n) = \sqrt 2\cdot (2n)^{2n} e^{-n} / (n^n \cdot n^n) = \sqrt 2\cdot 4^n e^{-n}\cdot n^{2n}/n^{2n} = \sqrt 2\cdot (4/e)^n$. Корень $n$-й: $\sqrt 2^{1/n}\cdot 4/e \to 4/e$.

2. Запишем $e\cdot n! = \sum_{k=0}^\infty n!/k! = N_n + r_n$, где $N_n = \sum_{k=0}^n n!/k! \in\mathbb{Z}$, $r_n = \sum_{k>n} n!/k! = \frac{1}{n+1} + \frac{1}{(n+1)(n+2)} + \ldots = \frac{1}{n+1} + O(1/n^2) \sim \frac{1}{n}$. Тогда $\sin(2\pi e\,n!) = \sin(2\pi r_n) \sim 2\pi r_n \sim 2\pi/n$. **Ответ**: $2\pi$.

3. Это теорема **Abel–Dini**. Возьмём $S_n = \sum_{k\leq n} a_k \to \infty$.

   *Расходимость*: $\varepsilon_n = 1/S_{n-1}$ (монотонно убывает к $0$). Из $\ln(S_n/S_{n-1}) = \ln(1+a_n/S_{n-1}) \leq a_n/S_{n-1}$ имеем $\sum_{k\leq N} a_k/S_{k-1} \geq \ln S_N \to \infty$.

   *Сходимость*: $\delta_n = 1/S_n$ (тоже $\to 0$). $a_n/S_n^2 = (S_n - S_{n-1})/S_n^2 \leq (S_n - S_{n-1})/(S_n S_{n-1}) = 1/S_{n-1} - 1/S_n$ — телескоп. Значит $\sum a_n \cdot \delta_n^2 = \sum a_n/S_n^2 < \infty$. Если нужен именно $\delta_n$ (а не $\delta_n^2$) с $\sum a_n\delta_n < \infty$, берём $\delta_n = 1/S_n^2$ (или $1/(S_n \ln S_n)$). **Готово.**

4. $a_n = \int_0^1 x^n \sqrt{1-x^2}\,dx$. Подстановка $x = \cos\theta$: $a_n = \int_0^{\pi/2}\cos^n\theta\,\sin^2\theta\,d\theta = W_n - W_{n+2} = W_n - \frac{n+1}{n+2}W_n = \frac{W_n}{n+2}$. Из Wallis (E12) $W_n \sim \sqrt{\pi/(2n)}$, значит $a_n \sim \sqrt{\pi/(2n)}/n = \sqrt{\pi/2}\cdot n^{-3/2}$. **Ответ**: $\sqrt{\pi/2}$.

5. По Weyl (E17) $\{n\alpha\}$ равномерно распределено (иррациональное $\alpha$); $\{n^2 \alpha\}$ — тоже (полиномиальная Weyl). Для равномерно распределённых $\frac{1}{N}\sum f(\{x_n\}) \to \int_0^1 f$ для Riemann-интегрируемых $f$. $\int_0^1 \cos(2\pi t)dt = 0$, поэтому каждая компонента $\to 0$. Сумма $\to 0$.
