# Степенные ряды и равномерная сходимость: перестановка $\lim$, $\int$, $\sum$; аналитические функции

## Scope
Точная аппаратура для перестановки $\lim$, $\int$ и $\sum$: критерии равномерной сходимости (Weierstrass M-test, Dini, Arzelà–Ascoli), переноса предела внутрь суммы (Tannery, DCT для счётной меры, Fubini для двойных рядов). Степенные ряды: Cauchy–Hadamard, почленное дифференцирование/интегрирование на компактах внутри disc of convergence, граничное поведение (Abel, Pringsheim, Hadamard gap). Real-analytic функции: identity theorem, Bernstein-критерий, Borel — формальная серия не определяет $C^\infty$-функцию однозначно. Аппроксимационная теория: Weierstrass через Bernstein polynomials, Stone–Weierstrass, Müntz–Szász. Нестандартные приёмы: factorization дифференциального оператора через вещественные корни, повторное интегрирование generating function.

## Записи — Core

### E1. Cauchy–Hadamard formula (формула Коши–Адамара)

Формулировка. Для $f(x) = \sum_{n\geq 0} a_n (x-x_0)^n$ радиус сходимости
$$R = \frac{1}{\limsup_{n\to\infty} \sqrt[n]{|a_n|}}.$$
Ряд абсолютно сходится при $|x-x_0|<R$, расходится при $|x-x_0|>R$, на границе $|x-x_0|=R$ — поведение зависит от ряда. На любом компакте $K \subset (x_0-R, x_0+R)$ сходимость равномерная (и абсолютная).

Условия применимости. Универсально для любых $a_n \in \mathbb{C}$. $\limsup$ обязателен, не $\lim$ — иначе теряются ряды с нерегулярным ростом коэффициентов. Версия с ratio: $R = \lim |a_n/a_{n+1}|$ работает, только если этот предел существует.

Идея доказательства. Comparison с геометрической прогрессией: для $|x|<R$ и $\rho \in (|x|, R)$ имеем $|a_n| \leq \rho^{-n}$ для $n$ больших, значит $|a_n x^n| \leq (|x|/\rho)^n$ — мажоранта.

Варианты и обобщения.
- **Pringsheim**: если $a_n \geq 0$ и $R<\infty$, то $x_0+R$ — особая точка $f$ (см. E11).
- **Hadamard gap**: лакунарный ряд имеет естественную границу на $|x|=R$ (E10).
- Для рядов Дирихле $\sum a_n n^{-s}$ — аналог: abscissa of convergence через $\limsup \log|a_n|/\log n$.

Используется в. [IMC-2010-2 — радиус $1$ для $\sum x^{4k+4}/[(4k+1)(4k+2)(4k+3)(4k+4)]$]; типовая первая проверка перед любой манипуляцией с power series.

### E2. Termwise differentiation and integration of power series (почленное дифференцирование и интегрирование степенного ряда)

Формулировка. Если $f(x) = \sum a_n (x-x_0)^n$ имеет радиус сходимости $R>0$, то $f \in C^\infty(x_0-R, x_0+R)$, причём
$$f'(x) = \sum_{n\geq 1} n a_n (x-x_0)^{n-1}, \qquad \int_{x_0}^x f(t)\,dt = \sum_{n\geq 0} \frac{a_n}{n+1}(x-x_0)^{n+1},$$
и оба ряда имеют тот же радиус $R$. Соответственно $a_n = f^{(n)}(x_0)/n!$.

Условия применимости. Только внутри disc of convergence. На границе формально продифференцированный ряд может расходиться, даже если исходный сходится; почленное интегрирование часто даёт ряд с большим «запасом» сходимости (полезно для вычислений, см. E3 и IMC-2010-2).

Идея доказательства. Равномерная сходимость на любом компакте $|x-x_0|\leq r<R$ + классическая теорема о перестановке $d/dx$ и $\sum$ при равномерной сходимости производных. Радиус сохраняется, поскольку $\sqrt[n]{n} \to 1$.

Варианты и обобщения.
- Композиция: если $g$ аналитична в $f(x_0)$ и $f$ — в $x_0$, то $g \circ f$ аналитична с радиусом, ограниченным снизу через расстояние от $f(x_0)$ до особенностей $g$.
- Cauchy product: $(\sum a_n x^n)(\sum b_n x^n) = \sum c_n x^n$, $c_n = \sum_{k=0}^n a_k b_{n-k}$, радиус $\geq \min(R_a, R_b)$.
- Inversion: если $f(0)=0, f'(0)\neq 0$, то $f^{-1}$ аналитична в $0$ — Lagrange inversion formula даёт коэффициенты явно.

Используется в. [IMC-2010-2: четырёхкратное интегрирование $1/(1-x^4)$ от $0$ до $1$ + Abel'ев предел; IMC-2013-5: переход $\sum a_n^2$ vs $\sum a_n^4$ через специально подобранные ряды]; универсальный приём для замкнутого вычисления сумм.

### E3. Weierstrass M-test (мажорантный признак Вейерштрасса)

Формулировка. Пусть $f_n: X \to \mathbb{R}$ и $|f_n(x)| \leq M_n$ для всех $x \in X$, $\sum M_n < \infty$. Тогда $\sum f_n$ сходится абсолютно и равномерно на $X$.

Условия применимости. Мажоранта $M_n$ должна быть числовой (не зависит от $x$). Если такой нет — пробуем Dirichlet/Abel test для рядов функций (равномерные аналоги E5 предыдущего файла).

Идея доказательства. Cauchy criterion: $|\sum_{k=m}^n f_k(x)| \leq \sum_{k=m}^n M_k \to 0$ равномерно по $x$.

Варианты и обобщения.
- **Dirichlet test (uniform)**: если $\sum a_n(x)$ имеет равномерно ограниченные частные суммы, $b_n(x) \downarrow 0$ равномерно, то $\sum a_n b_n$ сходится равномерно.
- **Abel test (uniform)**: $\sum a_n$ равномерно сходится, $b_n(x)$ равномерно ограничена и монотонна по $n$ — $\sum a_n b_n$ равномерно сходится.
- M-test даёт абсолютную сходимость; для условной нужны Dirichlet/Abel.

Используется в. Стандартный workhorse при доказательстве регулярности функций, заданных рядом. [IMC-2017-6 (косвенно через DCT-обоснование), IMC-2002-DAY2-3 (мажоранта $k^n e^{-k}$ для перестановки $\sum$ и индукции)].

### E6. Tannery's theorem / DCT for series (теорема Таннери, dominated convergence для рядов)

Формулировка. Пусть $a_n^{(k)} \to a_n$ при $k\to\infty$ для каждого $n$, и существует $M_n \geq 0$ с $|a_n^{(k)}|\leq M_n$ для всех $k$, $\sum M_n < \infty$. Тогда
$$\lim_{k\to\infty} \sum_{n=0}^\infty a_n^{(k)} = \sum_{n=0}^\infty a_n.$$
Это в точности dominated convergence theorem (DCT) для меры подсчёта на $\mathbb{N}$.

Условия применимости. Мажоранта $M_n$ — общая для всех $k$. Без неё контрпример: $a_n^{(k)} = \mathbf{1}_{n=k}$ — поточечно к $0$, сумма каждого равна $1$.

Идея доказательства. Хвост $\sum_{n>N} M_n < \varepsilon$; начальный отрезок $\sum_{n\leq N} a_n^{(k)} \to \sum_{n\leq N} a_n$ при $k\to\infty$ как конечная сумма.

Варианты и обобщения.
- **Monotone convergence (MCT) для рядов**: $a_n^{(k)} \uparrow a_n$ — без условия мажоранты, но с монотонностью.
- **Fatou для рядов**: $\sum \liminf a_n^{(k)} \leq \liminf \sum a_n^{(k)}$ при $a_n^{(k)} \geq 0$.
- Если ряд параметризован непрерывно ($t$ вместо $k$) — то же утверждение для предела по $t$.

Используется в. [IMC-2002-DAY2-3 (предел $\lim_{k\to\infty} \sum k^n/k!$ через мажоранту)]; типовое обоснование переноса предела внутрь $\sum$ в задачах генерирующих функций.

### E7. Fubini / Tonelli for double series (теорема Фубини для двойных рядов)

Формулировка. Если $\sum_m \sum_n |a_{m,n}| < \infty$ (или $a_{m,n}\geq 0$ — Tonelli), то
$$\sum_{m=0}^\infty \sum_{n=0}^\infty a_{m,n} = \sum_{n=0}^\infty \sum_{m=0}^\infty a_{m,n} = \sum_{(m,n)\in\mathbb{N}^2} a_{m,n}$$
(последняя сумма — по любой биекции $\mathbb{N}^2 \leftrightarrow \mathbb{N}$).

Условия применимости. Абсолютная сходимость одного из повторных рядов. Без неё контрпример: $a_{m,n} = 1$ при $n=m$, $-1$ при $n=m+1$, $0$ иначе. Один повторный ряд $=1$, другой $=0$.

Идея доказательства. Tonelli для счётной меры: монотонная сходимость; затем разложение $a_{m,n} = a_{m,n}^+ - a_{m,n}^-$ для общего случая.

Варианты и обобщения.
- **Cauchy product как частный случай**: для $\sum a_n x^n \cdot \sum b_n x^n$ — суммирование по диагоналям.
- **Reindexing absolutely convergent series**: любая перестановка / перегруппировка сохраняет сумму.
- **Multiplication of formal power series**: алгебраически Fubini не нужен, но при подстановке $x \in \mathbb{R}$ — нужна абсолютная сходимость.

Используется в. Манипуляции с двойными рядами в задачах на $\zeta$-функцию, на тождества типа $\sum_n \sum_k \binom{n}{k}/n! = \ldots$; стандартное обоснование в задачах с генерирующими функциями.

### E8. Abel's theorem on power series + boundary behavior

Формулировка. Если $\sum a_n$ сходится, то $\lim_{x\to 1^-} \sum a_n x^n = \sum a_n$. Применение к почленно проинтегрированному ряду: если $\int$ от $0$ до $1$ исходного ряда «не лезет» внутрь, то после интегрирования радиус остаётся $\geq 1$, и Abel даёт замкнутую формулу через значение на границе.

Условия применимости. Только сходимость числового ряда $\sum a_n$ (не абсолютная). Двусторонняя версия: $\lim_{x\to(-1)^+}$ при сходимости $\sum (-1)^n a_n$ — аналогично.

Идея доказательства. См. подробно E6 в файле «Последовательности и ряды».

Варианты и обобщения.
- **Tauberian обращение** требует условий типа $a_n = O(1/n)$ (Littlewood) или $n a_n \to 0$ (Tauber) — см. E7 предыдущего файла.
- **Многократное интегрирование степенного ряда**: $f(x) = \sum a_n x^n$ радиуса $1$, $F^{(k)}(0) = 0$ для $k=0,\ldots,m-1$, $F^{(m)} = f$ — даёт $F(1) = \sum a_n / [(n+1)\cdots(n+m)]$ через Abel.

Используется в. [IMC-2010-2: $F^{(4)} = 1/(1-x^4)$, четырёхкратное интегрирование от $0$ до $1$ через partial fractions $1/(1-x^4) = \frac{1}{4}\bigl(\frac{1}{1-x} + \frac{1}{1+x} + \frac{2}{1+x^2}\bigr)$, ответ $\frac{\ln 2}{4} - \frac{\pi}{24}$].

### E12. Weierstrass approximation theorem via Bernstein polynomials (теорема Вейерштрасса, конструктивно через Бернштейновы многочлены)

Формулировка. Для $f \in C([0,1])$ и $B_n(f)(x) = \sum_{k=0}^{n} \binom{n}{k} x^k(1-x)^{n-k} f(k/n)$ имеет место $\|B_n(f) - f\|_\infty \to 0$. На отрезке $[a,b]$ — линейная замена. На любом компакте $K\subset\mathbb{R}$ полиномы плотны в $C(K)$ в равномерной норме.

Условия применимости. Только непрерывность $f$ на компакте. Bernstein-аппроксимация сохраняет монотонность, выпуклость и положительность $f$ — поэтому конструкция полезна, когда нужны полиномы с дополнительной структурой.

Идея доказательства. Вероятностная: $B_n(f)(x) = \mathbb{E}[f(S_n/n)]$, где $S_n \sim \text{Bin}(n,x)$. Из Чебышёва $\Pr(|S_n/n - x|>\delta) \leq 1/(4n\delta^2)$. Разделить $|f(S_n/n) - f(x)|$ по событиям «$|S_n/n - x|<\delta$» (равномерная непрерывность $f$) и «$|S_n/n - x|\geq\delta$» (малая вероятность $\times 2\|f\|_\infty$).

Варианты и обобщения.
- **Скорость**: $\|B_n(f)-f\|_\infty = O(\omega(f, 1/\sqrt n))$, где $\omega$ — модуль непрерывности. Для $f\in C^2$: $O(1/n)$.
- **Stone–Weierstrass** (E13) — обобщение на произвольные алгебры.
- **Müntz–Szász** (E14) — какие подмножества степеней $\{x^{\lambda_k}\}$ плотны.
- **Korovkin's theorem**: достаточно проверить аппроксимацию $1, x, x^2$ для положительного линейного оператора.

Используется в. [IMC-1995-DAY2-4: Weierstrass + проекция на нечётные функции $q(x) = (p(x)-p(-x))/2$]; стандартная техника для сведения непрерывного к полиномиальному.

## Записи — Standard

### E4. Dini's theorem (теорема Дини)

Формулировка. Пусть $K$ компакт, $f_n: K \to \mathbb{R}$ непрерывны, $f_n \to f$ поточечно, $f$ непрерывна, и $f_n$ монотонна по $n$ в каждой точке (например, $f_n \uparrow f$). Тогда $f_n \to f$ равномерно на $K$.

Условия применимости. **Все три**: компактность $K$, непрерывность всех $f_n$ И предельной $f$, монотонность по $n$. Без любого из условий теорема ложна. Контрпример к компактности: $f_n(x)=x^n$ на $[0,1)$ — нет равномерной сходимости. Без непрерывности $f$: $f_n = x^n$ на $[0,1]$ — предел разрывен.

Идея доказательства. По противоречию: если нет равномерной сходимости, есть $x_n \in K$ и $\varepsilon>0$ с $f(x_n)-f_n(x_n)\geq\varepsilon$. Подпоследовательность $x_{n_k}\to x^*$. Из непрерывности $f$ и $f_{n_0}$: для фиксированного $n_0$ при больших $k$ имеем $f(x_{n_k})-f_{n_0}(x_{n_k})$ близко к $f(x^*)-f_{n_0}(x^*)$, что мало по выбору $n_0$ — противоречие с монотонностью.

Варианты и обобщения.
- **Dini для рядов**: если $\sum f_n$ — ряд непрерывных $f_n \geq 0$ с непрерывной суммой $S$ на компакте, то сходимость равномерная.
- **Polya version**: для монотонной последовательности дискретных мер, сходящейся слабо к непрерывной мере, сходимость распределений равномерна.

Используется в. Перенос регулярности с поточечной сходимости на равномерную в задачах с монотонными аппроксимациями (Bernstein polynomials с положительностью; convex / concave приближения). Putnam-style: «доказать, что сходимость равномерная» — почти всегда либо M-test, либо Dini.

### E5. Arzelà–Ascoli theorem (теорема Арцела–Асколи)

Формулировка. Семейство $\mathcal{F} \subset C(K)$ на компакте $K$ относительно компактно (предкомпактно в равномерной топологии) тогда и только тогда, когда оно равномерно ограничено и равностепенно непрерывно (equicontinuous): $\forall\varepsilon>0\,\exists\delta>0\,\forall f\in\mathcal{F}\,\forall x,y\in K: |x-y|<\delta\Rightarrow |f(x)-f(y)|<\varepsilon$.

Условия применимости. Компактность $K$ обязательна. Эквивалентно: из любой последовательности $f_n \in \mathcal{F}$ можно выделить равномерно сходящуюся подпоследовательность.

Идея доказательства. ($\Rightarrow$) Тривиально из равномерной сходимости. ($\Leftarrow$) Диагональный аргумент: на счётном плотном множестве $\{q_k\}\subset K$ выбираем подпоследовательность, сходящуюся в каждом $q_k$; equicontinuity + плотность дают равномерную сходимость на $K$.

Варианты и обобщения.
- **Точечная компактность + equicontinuity**: достаточно поточечной ограниченности (если $K$ компактен, то equicontinuity её и так даёт).
- **Lipschitz / $\alpha$-Hölder с общей константой $\Rightarrow$ equicontinuous**.
- **Aubin–Lions** — функциональный аналог в Sobolev-spaces.
- $C^1$-семейство с общей оценкой $\|f'\|_\infty \leq M$ автоматически equicontinuous (Lipschitz с константой $M$).

Используется в. Олимпиадный шаблон: «семейство решений ОДУ имеет сгущение» (через равномерные оценки производных). Putnam-1998-B4 (existence of subsequence with prescribed limit). Базовый инструмент для существования экстремалей в вариационных задачах.

### E9. Identity theorem for real analytic functions (теорема единственности)

Формулировка. Пусть $f, g: I \to \mathbb{R}$ real analytic на связном интервале $I$. Если множество $\{x \in I: f(x)=g(x)\}$ имеет точку накопления в $I$, то $f \equiv g$ на $I$.

Условия применимости. **Real analyticity** обязательна (т.е. $f$ локально равна сумме своего Taylor series, не просто $C^\infty$). Без неё контрпример: $f(x) = e^{-1/x^2}$ ($f\equiv 0$ при $x\leq 0$) и $g\equiv 0$ совпадают на $(-\infty, 0]$, но различны на $(0,\infty)$.

Идея доказательства. Множество $A = \{x: f^{(n)}(x) = g^{(n)}(x)\,\forall n\}$ замкнуто (пересечение замкнутых) и открыто (если $x_0 \in A$, Taylor series $f-g$ в $x_0$ нулевой $\Rightarrow$ $f-g \equiv 0$ в окрестности). Точка накопления нулей $f-g$ даёт $f^{(n)}(x^*)=g^{(n)}(x^*)$ для всех $n$ через iterated Rolle, значит $A\neq\emptyset$.

Варианты и обобщения.
- **Аналитическое продолжение**: значения $f$ на любом подинтервале однозначно определяют $f$ на всём $I$.
- **Strong unique continuation**: даже из «бесконечного порядка зануления» в одной точке следует $f\equiv 0$.
- В $\mathbb{C}$ ослабляется: достаточно accumulation point, аналог Riemann mapping и т.д.

Используется в. Доказательство тождеств для вычисленных через интегрирование функций (расширение тождества с интервала на всю область определения). Putnam-style: «$f(1/n) = g(1/n)$ для всех $n$, $f,g$ analytic $\Rightarrow$ $f\equiv g$».

### E13. Stone–Weierstrass theorem (теорема Стоуна–Вейерштрасса)

Формулировка. Пусть $K$ компакт, $\mathcal{A} \subset C(K, \mathbb{R})$ — подалгебра, содержащая константы и разделяющая точки ($\forall x\neq y \in K\,\exists f\in\mathcal{A}: f(x)\neq f(y)$). Тогда $\mathcal{A}$ плотна в $C(K)$ в равномерной норме.

Условия применимости. **Алгебра** (замкнута относительно сложения, умножения, скалярного умножения), **разделение точек** и **константы** — все три необходимы. Без констант: $\{xp(x): p\in\mathbb{R}[x]\}$ на $[0,1]$ — алгебра, разделяет точки, но плотна только в $\{f: f(0)=0\}$.

Для комплексных $\mathcal{A} \subset C(K, \mathbb{C})$ требуется дополнительно замкнутость относительно сопряжения ($f\in\mathcal{A}\Rightarrow \bar{f}\in\mathcal{A}$). Без сопряжения теорема ложна: дисковая алгебра $A(\overline{D})$ — алгебра аналитических функций, разделяет, содержит константы, но не плотна в $C(\overline{D})$.

Идея доказательства. Шаг 1: $\mathcal{A}$ замкнута относительно $|\cdot|$ через равномерную аппроксимацию $|t| \approx p(t^2)$ (Weierstrass на $[0,1]$). Шаг 2: $\max(f,g) = (f+g+|f-g|)/2 \in \overline{\mathcal{A}}$. Шаг 3: для каждых $x,y \in K$ и $\varepsilon$ строим $f_{x,y}\in\mathcal{A}$ с предписанными значениями в $x,y$; через $\min$ и $\max$ собираем функцию, $\varepsilon$-близкую к произвольной $g \in C(K)$.

Варианты и обобщения.
- **Тригонометрические полиномы** на окружности (комплексная версия с сопряжением).
- **Нечётные полиномы** на $[-1,1]$ — плотны в $\{f \text{ непрерывная нечётная}\}$ (используется в IMC-1995-DAY2-4).
- **Lattice-version (Kakutani–Krein)**: $\mathcal{A}$ — векторная решётка с константами, разделяющая точки $\Rightarrow$ плотна.

Используется в. Обоснование плотности тригонометрических полиномов в $C(\mathbb{T})$, плотности чётных/нечётных полиномов в соответствующих подпространствах. [IMC-1995-DAY2-4 — сужение Weierstrass до нечётных].

### E16. Generating function trick for moments / Bell-style identities

Формулировка (схема). Если $a_n = \sum_{k=0}^\infty f(k) k^n / k!$ для подходящего веса $f(k)$, то $\sum_n a_n t^n / n!$ часто имеет замкнутую форму через $e^{kt}$:
$$\sum_{n=0}^\infty \frac{t^n}{n!} \sum_{k=0}^\infty \frac{f(k) k^n}{k!} = \sum_{k=0}^\infty \frac{f(k)}{k!} e^{kt}.$$
Перестановка $\sum$ по Fubini (E7) при выполнении условий абсолютной сходимости.

Условия применимости. Абсолютная сходимость обоих рядов в окрестности $t=0$ (мажоранта $f(k) k^n/k! \cdot t^n/n!$ просуммирована $\sum_k$ должна быть конечной для $|t|$ малых).

Идея доказательства. Перестановка $\sum_n \sum_k = \sum_k \sum_n$ по Fubini + узнавание $\sum_n (kt)^n/n! = e^{kt}$.

Варианты и обобщения.
- **Bell numbers**: $B_n = \frac{1}{e}\sum_{k=0}^\infty k^n / k!$, EGF $\sum B_n t^n/n! = e^{e^t - 1}$.
- **Touchard polynomials**: $T_n(x) = \sum_{k=0}^\infty x^k k^n / k! \cdot e^{-x}$.
- **Stirling numbers второго рода**: $k^n = \sum_{j} S(n,j) k(k-1)\cdots(k-j+1)$ — даёт явные формулы для $a_n$.
- **Pochhammer / hypergeometric** идентичности — расширение в ту же сторону.

Используется в. [IMC-2002-DAY2-3: $a_n = \sum k^n/k!$, $b_n = \sum (-1)^k k^n/k!$ — индукция из бинома $(k+1)^{n+1} = \sum \binom{n}{m} k^m$ + перестановка $\sum$ даёт $a_{n+1} = \sum \binom{n}{m} a_m$, рекурсия Bell numbers].

### E10. Hadamard gap theorem (Hadamard'ова теорема о лакунарных рядах)

Формулировка. Пусть $f(x) = \sum_{k=0}^\infty a_k x^{n_k}$ — лакунарный (gap) ряд: показатели удовлетворяют $n_{k+1}/n_k \geq \lambda > 1$ для всех $k$. Если радиус сходимости $R<\infty$, то окружность $|x|=R$ — естественная граница: $f$ нельзя аналитически продолжить ни в какую точку этой окружности.

Условия применимости. Условие Hadamard $n_{k+1}/n_k \geq \lambda > 1$ — сильное лакунарное. Уточнение Fabry: достаточно $n_k/k \to \infty$. На прямой $\mathbb{R}$ это означает: ряд расходится «экстремально нерегулярно» в каждой граничной точке.

Идея доказательства (idea Mordell / Hadamard). Если бы $f$ продолжалась через $x_0=R$, то в окрестности $x_0$ имела бы Taylor expansion с бóльшим радиусом. Сравнение коэффициентов через интегрирование Cauchy + лакунарность даёт противоречие — большие пропуски в разложении не могут «исчезнуть» при сдвиге центра.

Варианты и обобщения.
- **Fabry gap theorem**: $n_k/k \to \infty$ ⇒ естественная граница.
- **Pólya gap theorem**: для лакунарных рядов minimum modulus $\min_{|x|=r}|f(x)|$ имеет «правильное» поведение.
- **Wiener Tauberian theorem** даёт обратные результаты: при определённых условиях лакунарность не препятствует продолжению.

Используется в. Конструкция явных $C^\infty$-функций без аналитичности; контрпримеры в задачах вида «существует ли аналитическая $f$ с заданным свойством». Классика Theorem-Building, не «решает» олимпиадную задачу напрямую, но известна как редкое мощное orудие.

### E11. Pringsheim's theorem (теорема Прингсхайма о положительных коэффициентах)

Формулировка. Если $f(x) = \sum a_n x^n$, $a_n \geq 0$, имеет конечный радиус сходимости $R>0$, то точка $x=R$ является особой точкой $f$ — $f$ не может быть аналитически продолжена в окрестность $R$.

Условия применимости. Положительность коэффициентов. Без неё контрпример: $\sum (-1)^n x^n = 1/(1+x)$ имеет радиус $1$, но особенность только в $x=-1$.

Идея доказательства. Если $f$ продолжается в окрестность $R$, то в точке $\rho<R$ Taylor expansion $f(\rho+t) = \sum f^{(n)}(\rho)/n!\,t^n$ имеет радиус $>R-\rho$. Из положительности $a_n$ получаем $f^{(n)}(\rho)\geq 0$, значит $f(\rho+t)\geq f(\rho)$ для $t>0$. Сходимость в $t = R-\rho+\varepsilon$ даёт оценку коэффициентов исходного ряда, противоречащую радиусу $R$.

Варианты и обобщения.
- Усиление: единственная особая точка может быть $x=R$, но для $\sum a_n x^n$ с $a_n\geq 0$ всегда особая в $R$.
- **Vivanti–Pringsheim** для рядов Дирихле: аналог для $\sigma$ — abscissa of convergence.
- Для Riemann zeta: $\zeta(s) = \sum 1/n^s$ имеет полюс в $s=1$ — каноничный пример.

Используется в. Доказательство расходимости/особенности в задачах с генерирующими функциями (если все коэффициенты неотрицательны, особая точка обязательно на положительной полуоси). Обоснование выбора $R$ как точки оценки в задачах вида «найти асимптотику $a_n$ через сингулярность $f$».

### E14. Müntz–Szász theorem (теорема Мюнца–Саса)

Формулировка. Пусть $0 < \lambda_1 < \lambda_2 < \ldots$ Линейная оболочка $\{1, x^{\lambda_1}, x^{\lambda_2}, \ldots\}$ плотна в $C([0,1])$ тогда и только тогда, когда
$$\sum_{k=1}^\infty \frac{1}{\lambda_k} = \infty.$$
В $L^2([0,1])$ — то же условие.

Условия применимости. Чрезвычайно тонкое: $\{x^{n^2}\}$ — линейная оболочка не плотна, поскольку $\sum 1/n^2 < \infty$. Зато $\{x^p: p \text{ простое}\}$ — плотна, так как $\sum 1/p$ расходится.

Идея доказательства. ($\Rightarrow$) Аппроксимация $x = \sum c_k x^{\lambda_k}$ невозможна, если $\sum 1/\lambda_k <\infty$, через Hadamard's gap theorem или прямой комплексный анализ. ($\Leftarrow$) Конструктивно через биортогональные системы или Blaschke-products в $H^2$.

Варианты и обобщения.
- **Müntz для $L^p$** ($1\leq p<\infty$): то же условие.
- **С отрицательными $\lambda_k$**: на $[1,\infty)$ — двойственная задача.
- **Блейаску-доказательство** через Blaschke products в полуплоскости.

Используется в. Putnam-2000-B3 (плотность подпространства полиномов специального вида). Сложный инструмент, редко применяется напрямую, но важен как граничное утверждение.

### E15. Differential operator factorization via real roots (факторизация дифференциального оператора по вещественным корням)

Формулировка (IMC-2006-6 setup). Условие на функцию $f$:
$$\bigl(\text{любая линейная комбинация } a_0 f + a_1 f' + \ldots + a_n f^{(n)} \text{ имеет нуль на } \mathbb{R}\bigr)$$
эквивалентно тому, что характеристический полином $A(x) = a_0 + a_1 x + \ldots + a_n x^n$ имеет все вещественные корни. В случае комплексных корней — контрпример $f(x) = e^{ux}\sin(vx) + \varepsilon x^k$.

Условия применимости. Идея: оператор $L = \sum a_k D^k$ с $D=d/dx$ факторизуется в $\mathbb{R}[D]$ как $\prod (D-c_i)$ при вещественных корнях. Каждый множитель $(D-c)$ — оператор первого порядка, ядро $e^{cx}$.

Идея доказательства. Прямая часть: индукция по $n$. Если $A(x) = (x-c)B(x)$, то $L = (D-c)L_B$. Применяем $e^{-cx} f \cdot (\text{приём вспомогательной функции, см. E6 предыдущего})$: множитель $e^{-cx}$ переводит $(D-c)$ в $D$, после чего у $L_B(e^{-cx}f)$ есть нуль (или $L_B(e^{-cx}f)$ обращается в ноль на бесконечности по Rolle), и индукция даёт нуль у $L_B f \cdot e^{cx}$ — после деления нуль у $f$.

Обратная часть: для $A = (x^2 - 2ux + u^2 + v^2) C(x)$ функция $f(x) = e^{ux}\sin(vx)$ удовлетворяет $L_{\text{квадратичная часть}} f = 0$, добавление $\varepsilon x^k$ ломает зануление, не давая нулей.

Условия применимости. $f \in C^n(\mathbb{R})$, специфика — вещественные/комплексные корни характеристического полинома. Олимпиадно используется как «классификация» допустимых $f$.

Варианты и обобщения.
- **Интегрирующий множитель $e^{\alpha x}$** для дифференциальных уравнений первого порядка.
- **Polya–Szegő**: для $L = D^n + \ldots$ с вещественными корнями характеристический полином $\Rightarrow$ оценки числа нулей решений (см. также E7 предыдущего файла).
- Связь с teorema де Lounesto: число вещественных нулей решения $L f = 0$ — $\leq$ число вещественных корней $A$.

Используется в. [IMC-2006-6]; нестандартный приём для задач с произвольными производными (когда нужно из условия «у любой комбинации производных есть нуль» извлечь структуру оператора).

### E17. Borel's theorem on $C^\infty$ realizability (теорема Бореля)

Формулировка. Для любой последовательности $(c_n)_{n\geq 0} \subset \mathbb{R}$ существует $f \in C^\infty(\mathbb{R})$ с $f^{(n)}(0) = c_n$ для всех $n$. Никаких ограничений на рост $|c_n|$ нет.

Условия применимости. Универсально. Демонстрирует резкое различие $C^\infty$ и real analytic: формальный Taylor series в общем случае не сходится нигде, кроме $x=0$, но $f$ всё равно $C^\infty$.

Идея доказательства. $f(x) = \sum_{n=0}^\infty \frac{c_n}{n!} x^n \chi(\lambda_n x)$, где $\chi$ — гладкая cutoff с $\chi=1$ около $0$, $\chi=0$ вне $[-1,1]$, и $\lambda_n$ выбираются так быстро растущими, что каждое слагаемое и все его производные дают мажоранту $\leq 2^{-n}$ при $n>k$. Тогда ряд и все ряды производных сходятся равномерно.

Варианты и обобщения.
- **Whitney extension theorem**: множество $C^\infty$-струй на замкнутом множестве реализуется как ограничение $C^\infty$-функции.
- **В $\mathbb{R}^d$**: то же утверждение для multi-index.
- **Quasi-analytic классы (Denjoy–Carleman)**: для каких ограничений на $|c_n|$ функция определяется однозначно. Класс real analytic — экстремальный quasi-analytic.

Используется в. Конструкция $C^\infty$-функций с предписанным локальным поведением (плоские функции, bump functions). Demonstrating: «не каждый формальный ряд сходится, но реализуется как $C^\infty$» — используется в задачах на построение контрпримеров и в обосновании, что $C^\infty$ — слабое условие.

## Self-test

1. Вычисли $\displaystyle\sum_{k=0}^\infty \frac{1}{(4k+1)(4k+2)(4k+3)(4k+4)}$.

2. Пусть $f \in C([0,1])$ нечётна на $[-1,1]$ (продолжая $f(-x)=-f(x)$). Покажи, что $f$ можно равномерно приблизить нечётными многочленами.

3. Пусть $f: [0,1) \to \mathbb{R}$, $f(x) = \sum_{n=0}^\infty a_n x^n$, $a_n \geq 0$, и $\lim_{x\to 1^-} f(x) = L < \infty$. Докажи, что $\sum a_n = L$.

4. Существует ли $f \in C^\infty(\mathbb{R})$ с $f^{(n)}(0) = (n!)^2$ для всех $n \geq 0$? Если да — какова $f$ (укажи one из конструкций)?

5. Пусть $a_n^{(k)} = \binom{k}{n}/k^n$ для $0\leq n\leq k$, $a_n^{(k)} = 0$ для $n>k$. Найди $\lim_{k\to\infty} \sum_{n=0}^\infty a_n^{(k)} x^n$ для $x \in \mathbb{R}$.

## Решения

1. По E2 + E8 (IMC-2010-2). Положим $F(x) = \sum_{k\geq 0} x^{4k+4}/[(4k+1)(4k+2)(4k+3)(4k+4)]$. Радиус $R=1$ (по Cauchy–Hadamard, $\sqrt[n]{1/n^4}\to 1$). Четыре раза дифференцируя: $F^{(4)}(x) = \sum x^{4k} = 1/(1-x^4)$ на $|x|<1$. Также $F(0) = F'(0) = F''(0) = F'''(0) = 0$.

   Partial fractions: $\frac{1}{1-x^4} = \frac{1}{4}\bigl(\frac{1}{1-x} + \frac{1}{1+x} + \frac{2}{1+x^2}\bigr)$. Интегрируем $4$ раза от $0$ до $x$:
   - 1 раз: $\frac{1}{4}\bigl(-\ln(1-x) + \ln(1+x) + 2\arctan x\bigr)$.
   - 4 раза подряд (с правильными константами интегрирования $=0$ на каждом шаге).

   На границе $x=1$ ряд $\sum 1/[(4k+1)\cdots(4k+4)]$ сходится (мажоранта $1/k^4$), значит по Abel'евой теореме $F(1) = \lim_{x\to 1^-} F(x)$. Прямое вычисление повторного интеграла даёт **ответ $\frac{\ln 2}{4} - \frac{\pi}{24}$**.

2. По E12 + E13. Weierstrass: $\exists$ полиномы $p_n \to f$ равномерно на $[-1,1]$. Положим $q_n(x) = (p_n(x) - p_n(-x))/2$ — нечётный полином. Тогда
$$\|f - q_n\|_\infty = \sup\bigl|f(x) - \tfrac{p_n(x)-p_n(-x)}{2}\bigr| = \sup\bigl|\tfrac{(f(x)-p_n(x)) - (f(-x)-p_n(-x))}{2}\bigr| \leq \|f-p_n\|_\infty \to 0.$$
Это в точности приём IMC-1995-DAY2-4. Альтернатива: Stone–Weierstrass для алгебры нечётных полиномов на $[-1,1]$ требует чуть более тонкой работы (нечётные полиномы — модуль над чётными), проще через прямую проекцию.

3. По E6 (Tannery / MCT). Поскольку $a_n \geq 0$, частные суммы $S_N(x) = \sum_{n\leq N} a_n x^n$ монотонно растут по $x \in [0,1)$ и по $N$. Для фиксированного $N$: $\sum_{n\leq N} a_n = \lim_{x\to 1^-} S_N(x) \leq \lim_{x\to 1^-} f(x) = L$. Значит $\sum a_n \leq L$. Обратно, $f(x) \leq \sum a_n$ для $x \in [0,1)$ (положительность), значит $L \leq \sum a_n$. **Готово**.

   Это специальный случай Abel'евой теоремы (E8) при $a_n \geq 0$, не требующий отдельного Tauberian условия благодаря MCT.

4. По E17 (Borel). **Да**, существует — например, $f(x) = \sum_{n=0}^\infty (n!)^2 \cdot \frac{x^n}{n!} \cdot \chi(\lambda_n x) = \sum_n n! \cdot x^n \chi(\lambda_n x)$, где $\chi \in C_c^\infty(\mathbb{R})$, $\chi\equiv 1$ около $0$, и $\lambda_n$ растут достаточно быстро (например, $\lambda_n = (n!)^2 \cdot 2^n$). Каждое слагаемое — $C^\infty$ с компактным носителем и ограниченными производными $\Rightarrow$ ряд и все ряды производных сходятся равномерно. Прямой подсчёт: $f^{(n)}(0)$ — конечная сумма, поскольку только слагаемые с индексом $\leq n$ влияют (выше — обнуляются производные ниже порядка $n$ или $\chi$ убирает вклад).

   Замечание: формальный Taylor series $\sum (n!)^2 x^n/n! = \sum n! x^n$ имеет радиус $0$ (по Cauchy–Hadamard $\sqrt[n]{n!}\to\infty$), значит $f$ заведомо не аналитична в $0$.

5. По E6 + узнавание $e^{x}$. $a_n^{(k)} = \binom{k}{n}/k^n = \frac{k(k-1)\cdots(k-n+1)}{n!\,k^n}$. Для фиксированного $n$: $a_n^{(k)} = \frac{1}{n!} \prod_{j=0}^{n-1}(1 - j/k) \to 1/n!$ при $k\to\infty$.

   Мажоранта: $a_n^{(k)} \leq 1/n!$ для всех $k \geq n$ (произведение членов $\leq 1$); $a_n^{(k)} = 0$ для $n>k$. Значит $|a_n^{(k)} x^n| \leq |x|^n/n!$, $\sum |x|^n/n! = e^{|x|} < \infty$. По Tannery (E6): $\sum_n a_n^{(k)} x^n \to \sum_n x^n/n! = e^x$. **Ответ**: $e^x$ для всех $x \in \mathbb{R}$.

   Узнаваемая формулировка: $\sum_n \binom{k}{n}(x/k)^n = (1+x/k)^k \to e^x$ — компактное доказательство второго замечательного предела через перестановку $\lim_k$ и $\sum_n$.
