# Порядки, теорема Lagrange, циклические группы, структура подгрупп

## Контекст

Подтема покрывает работу с порядком элемента, теоремой Лагранжа и её сильными усилениями (Коши, Фробениус), классификацией подгрупп циклической группы и структурой подгрупп через индекс и пересечение. Типичные сюжеты на IMC и Putnam — оценка или вычисление порядка элемента, доказательство цикличности группы, выделение нормальной подгруппы малого индекса, контрпримеры в некоммутативных группах.

## Определения

Порядок элемента $g$ — наименьшее $n \ge 1$ с $g^n = e$. Если такого $n$ нет, порядок бесконечен. Обозначение: $\operatorname{ord}(g)$.

Порядок группы — мощность $|G|$.

Индекс подгруппы $H \le G$ — число левых смежных классов (косет, coset). Обозначение: $[G:H]$.

Циклическая группа — группа, порождённая одним элементом: $G = \langle g\rangle$.

Нормальная подгруппа: $H \trianglelefteq G$, если $gHg^{-1} = H$ для всех $g \in G$.

Экспонента группы — наименьшее $n \ge 1$ с $g^n = e$ для всех $g \in G$. Равносильно: НОК порядков всех элементов.

$p$-группа — конечная группа, порядок которой есть степень простого числа $p$.

Силовская $p$-подгруппа — подгруппа максимального возможного $p$-порядка в $G$.

Функция Эйлера $\varphi(n)$ — число натуральных $k \le n$, взаимно простых с $n$.

## Записи - Основные

### E1. Теорема Лагранжа

Формулировка. Пусть $G$ — конечная группа, $H \le G$. Тогда $|H|$ делит $|G|$, и $|G| = [G:H] \cdot |H|$. В частности, $\operatorname{ord}(g)$ делит $|G|$, и $g^{|G|} = e$ для всех $g$.

Условия применимости. Группа конечна.

Идея доказательства. Левые смежные классы $H$ разбивают $G$ на куски одинаковой мощности $|H|$. Биекция между $H$ и $gH$ задаётся $h \mapsto gh$.

Обратное неверно. В группе $A_4$ порядка $12$ нет подгруппы порядка $6$, хотя $6$ делит $12$.

---

### E2. Теорема Коши

Формулировка. Если $p$ простое и $p$ делит $|G|$, то в $G$ есть элемент порядка $p$.

Условия применимости. Группа конечна. Простота $p$ обязательна.

Пример поломки без простоты. В $\mathbb Z/2 \times \mathbb Z/2$ порядок группы $4$, но элемента порядка $4$ нет — все нетривиальные элементы имеют порядок $2$.

Идея доказательства (Маккей). Группа $\mathbb Z/p$ действует на множестве наборов $(g_1, \dots, g_p)$ с $g_1 \cdots g_p = e$ циклическим сдвигом. Число неподвижных точек делится на $p$. Среди них тривиальный набор $(e, \dots, e)$. Значит есть и нетривиальный набор $(g, g, \dots, g)$ с $g^p = e$ и $g \ne e$.

---

### E3. Формула порядка степени

Формулировка. Пусть $\operatorname{ord}(g) = n < \infty$. Тогда
$$\operatorname{ord}(g^k) = \frac{n}{\gcd(k,n)}.$$
В частности, $g^k$ имеет тот же порядок $n$ ровно при $\gcd(k,n) = 1$.

Условия применимости. Элемент конечного порядка.

Идея доказательства. $g^{km} = e$ равносильно $n \mid km$, то есть $n / \gcd(k,n)$ делит $m$.

---

### E4. Структура подгрупп циклической группы

Формулировка. Пусть $G = \langle g\rangle$ — циклическая группа порядка $n$. Тогда:

1. Любая подгруппа $H \le G$ циклическая.
2. Для каждого делителя $d$ числа $n$ существует ровно одна подгруппа порядка $d$, а именно $\langle g^{n/d}\rangle$.
3. Биекция $\{d : d \mid n\} \leftrightarrow \{H \le G\}$.

Условия применимости. Группа циклическая. Для $\mathbb Z$ подгруппы — это $m\mathbb Z$.

Идея доказательства. Возьмём наименьшее положительное $k$ с $g^k \in H$. Деление с остатком по $k$ показывает, что любой элемент $H$ есть степень $g^k$.

---

### E5. Теорема Фробениуса о числе решений $x^n = e$

Формулировка. Пусть $G$ — конечная группа, $n$ делит $|G|$. Тогда число решений уравнения $x^n = e$ в $G$ делится на $n$.

Условия применимости. Группа конечна. Делимость $n \mid |G|$ обязательна для сильной формы.

Пример поломки без $n \mid |G|$. В $G = \mathbb Z/5$ при $n = 3$ уравнение $x^3 = e$ имеет ровно одно решение $x = e$. Число $1$ не делится на $3$. Слабая форма: число решений делится на $\gcd(n, |G|)$ всегда.

Идея доказательства. Индукция по $|G|$ с использованием теоремы о двойных смежных классах. Современное доказательство (Брусянская–Клячко) использует характеры.

---

### E6. Критерий цикличности через число решений $x^d = e$

Формулировка. Конечная группа $G$ циклическая тогда и только тогда, когда для каждого делителя $d$ числа $|G|$ уравнение $x^d = e$ имеет не более $d$ решений.

Равносильная форма. $G$ циклическая тогда и только тогда, когда для каждого $d$ в $G$ не более одной подгруппы порядка $d$.

Условия применимости. Группа конечна.

Идея доказательства. Пусть $\psi(d)$ — число элементов порядка $d$. Если есть хоть один такой элемент, циклическая подгруппа, им порождённая, содержит ровно $\varphi(d)$ элементов порядка $d$. По предположению других элементов порядка $d$ нет, так что $\psi(d) \in \{0, \varphi(d)\}$. Сравнение $\sum_{d \mid n}\psi(d) = n = \sum_{d \mid n}\varphi(d)$ даёт $\psi(d) = \varphi(d)$ для всех $d$, в частности $\psi(n) > 0$.

---

### E7. Число элементов данного порядка в циклической группе

Формулировка. В циклической группе $\mathbb Z/n$ ровно $\varphi(d)$ элементов порядка $d$ для каждого делителя $d$ числа $n$. Следствие — тождество $\sum_{d \mid n}\varphi(d) = n$.

Условия применимости. Группа циклическая.

Идея доказательства. Элементы порядка $d$ — это образующие единственной подгруппы порядка $d$ (см. E4). Их ровно $\varphi(d)$.

---

### E8. Порядок произведения в абелевой группе

Формулировка. Пусть $G$ абелева, $\operatorname{ord}(a) = m$, $\operatorname{ord}(b) = n$.

1. Если $\gcd(m, n) = 1$, то $\operatorname{ord}(ab) = mn$.
2. В общем случае в $\langle a, b\rangle$ существует элемент порядка $\operatorname{lcm}(m, n)$.

Условия применимости. Коммутативность критична.

Пример поломки без коммутативности. В $S_3$ возьмём $a = (1\,2)$ порядка $2$ и $b = (1\,2\,3)$ порядка $3$. Тогда $ab = (1\,3)$ — снова порядка $2$, а не $6$.

Идея доказательства. (1) При коммутативности $(ab)^k = a^k b^k$, и пересечение $\langle a\rangle \cap \langle b\rangle$ тривиально из-за взаимной простоты порядков. (2) Раскладываем $m$ и $n$ по простым: для каждого простого $p$ берём из $a$ или $b$ ту степень, у которой $p$-часть порядка больше.

## Записи - Стандартные

### E9. Подгруппа индекса, равного наименьшему простому делителю, нормальна

Формулировка. Пусть $H \le G$, $[G:H] = p$, где $p$ — наименьший простой делитель $|G|$. Тогда $H \trianglelefteq G$.

Условия применимости. Группа конечна. Индекс должен быть именно наименьшим простым делителем порядка.

Пример поломки без условия «наименьший». В $S_3$ подгруппа $\langle (1\,2)\rangle$ имеет индекс $3$, но не нормальна: $(1\,3)(1\,2)(1\,3)^{-1} = (2\,3)$. Наименьший простой делитель $|S_3| = 6$ — это $2$, а не $3$.

Идея доказательства. Действие $G$ на левых смежных классах $G/H$ задаёт гомоморфизм $\varphi \colon G \to S_p$. Ядро $K \le H$ нормально, $[G:K]$ делит $p!$. Все простые делители $[G:K]$ делят $|G|$. В разложении $p!$ нет простых делителей $\ge p$ кроме $p$. Значит $[G:K] = p$, откуда $K = H$.

---

### E10. Изоморфизм $\mathbb Z/m \times \mathbb Z/n \cong \mathbb Z/(mn)$ при $\gcd(m, n) = 1$

Формулировка. Прямое произведение двух циклических групп взаимно простых порядков снова циклическое.

Условия применимости. Взаимная простота $\gcd(m, n) = 1$ обязательна.

Пример поломки без взаимной простоты. $\mathbb Z/2 \times \mathbb Z/2$ не изоморфна $\mathbb Z/4$: в первой все нетривиальные элементы порядка $2$, во второй есть элементы порядка $4$.

Идея доказательства. Элемент $(1, 1)$ имеет порядок $\operatorname{lcm}(m, n) = mn$. Мощности групп совпадают.

---

### E11. Формула $|HK| = |H| \cdot |K| / |H \cap K|$

Формулировка. Для подгрупп $H, K \le G$ конечной группы
$$|HK| = \frac{|H| \cdot |K|}{|H \cap K|},$$
где $HK = \{hk : h \in H, k \in K\}$ — множество, не обязательно подгруппа.

Условия применимости. Подгруппы конечны.

Идея доказательства. Каждый элемент $g \in HK$ представим ровно $|H \cap K|$ способами как $hk$. Если $h_1 k_1 = h_2 k_2$, то $h_1^{-1} h_2 = k_1 k_2^{-1} \in H \cap K$.

---

### E12. Индекс пересечения подгрупп; теорема Пуанкаре

Формулировка. Для подгрупп $H, K \le G$
$$[G : H \cap K] \le [G : H] \cdot [G : K].$$
Следствие (Пуанкаре). Пересечение конечного числа подгрупп конечного индекса имеет конечный индекс.

Условия применимости. Любая группа, не обязательно конечная.

Идея доказательства. Отображение $g(H \cap K) \mapsto (gH, gK)$ инъективно из $G/(H \cap K)$ в $G/H \times G/K$.

---

### E13. Группа порядка $p^2$ абелева

Формулировка. Любая группа порядка $p^2$ (где $p$ простое) абелева. С точностью до изоморфизма таких групп ровно две: $\mathbb Z/p^2$ и $\mathbb Z/p \times \mathbb Z/p$.

Условия применимости. $|G| = p^2$, $p$ простое.

Идея доказательства. У $p$-группы центр $Z(G)$ нетривиален (через уравнение классов). Если $|Z(G)| = p$, то $G/Z(G)$ имеет порядок $p$ и значит циклическая. Лемма: если $G/Z(G)$ циклическая, то $G$ абелева, тогда $Z(G) = G$ — противоречие. Значит $Z(G) = G$ сразу.

---

### E14. Группы порядка $pq$

Формулировка. Пусть $|G| = pq$, $p < q$ простые. Тогда:

1. Если $p$ не делит $q - 1$, то $G$ циклическая, $G \cong \mathbb Z/(pq)$.
2. Если $p$ делит $q - 1$, то есть ровно две группы: циклическая и неабелева $\mathbb Z/q \rtimes \mathbb Z/p$ (полупрямое произведение).

Условия применимости. Оба делителя простые, различные.

Идея доказательства. По третьей теореме Силова число $q$-силовских подгрупп сравнимо с $1$ по модулю $q$ и делит $p$. Поскольку $p < q$, остаётся только $1$. Значит $q$-силовская нормальна. Аналогично для $p$-силовской при условии $p \nmid q-1$. Когда обе силовские нормальны, $G$ — прямое произведение, то есть циклическая.

---

### E15. Группа с $g^2 = e$ всюду — элементарная абелева 2-группа

Формулировка. Если в группе $G$ выполнено $g^2 = e$ для всех $g$, то $G$ абелева. Если дополнительно $G$ конечна, то $G \cong (\mathbb Z/2)^k$ для некоторого $k$.

Условия применимости. Условие $g^2 = e$ обязательно. Для аналога с $g^p = e$ при $p > 2$ группа может быть неабелевой.

Пример поломки при $p > 2$. Группа Гейзенберга над $\mathbb F_p$ ($p$ нечётное) состоит из верхнетреугольных матриц $3\times 3$ с единицами на диагонали. Все её элементы имеют порядок $p$, но она не абелева.

Идея доказательства. $g^{-1} = g$, поэтому $(ab)^{-1} = b^{-1} a^{-1} = ba$. С другой стороны $(ab)^{-1} = ab$. Значит $G$ абелева, превращается в векторное пространство над $\mathbb F_2$. Выбор базиса даёт $(\mathbb Z/2)^k$.

## Записи - Редкие

### E16. Экспонента абелевой группы равна максимальному порядку

Формулировка. В конечной абелевой группе $G$ существует элемент, порядок которого равен экспоненте $G$.

Условия применимости. Группа конечная и абелева.

Пример поломки без коммутативности. В $A_4$ экспонента равна $6$ (НОК порядков $1, 2, 3$), но элементов порядка $6$ нет — максимальный порядок $3$.

Идея доказательства. Прямое следствие пункта (2) из E8: для любых двух элементов $a, b$ найдётся $c$ с порядком $\operatorname{lcm}(\operatorname{ord} a, \operatorname{ord} b)$. Индукция по числу элементов даёт элемент порядка $\operatorname{lcm}$ всех порядков, то есть экспоненты.

## Самопроверка

1. Пусть $G$ — конечная группа, $a, b \in G$ коммутируют, $\operatorname{ord}(a) = m$, $\operatorname{ord}(b) = n$, $\gcd(m, n) = 1$ и $a^k = b^\ell$ для каких-то $k, \ell$. Докажи, что $a^k = b^\ell = e$.

2. Существует ли нециклическая группа порядка $77$? Группа порядка $35$?

3. Может ли в конечной группе быть ровно две подгруппы порядка $5$?

4. Пусть для каждого делителя $d$ числа $|G|$ в конечной группе $G$ есть ровно одна подгруппа порядка $d$. Докажи, что $G$ циклическая.

5. Пусть $H \le G$ — подгруппа индекса $3$ в конечной группе $G$. Обязана ли $H$ быть нормальной? Если нет, приведи минимальный контрпример.

6. Найди все конечные абелевы группы, в которых число элементов порядка $2$ равно $3$.

7. Пусть $G$ — конечная группа, и каждый элемент $G$ имеет порядок $1$ или $p$, где $p$ — фиксированное простое. Докажи, что $|G|$ есть степень $p$.

8. Пусть $H, K \le G$, $|G| = 120$, $|H| = 24$, $|K| = 20$. Найди $|H \cap K|$.

9. Пусть $G$ конечна, $a, b \in G$ коммутируют, $\operatorname{ord}(a) = 12$, $\operatorname{ord}(b) = 18$. Найди $\operatorname{ord}(ab)$.

## Решения

1. Множество $\langle a\rangle \cap \langle b\rangle$ — подгруппа порядка, делящего одновременно $m$ и $n$. Из $\gcd(m, n) = 1$ оно тривиально. Значит $a^k = b^\ell$ может быть только нейтральным.

2. Применима E14. Для $|G| = 77 = 7 \cdot 11$: $7 \nmid 10$, любая группа порядка $77$ циклическая. Для $|G| = 35 = 5 \cdot 7$: $5 \nmid 6$, та же история. Нециклических нет.

3. Не может. Две различные подгруппы порядка $5$ пересекаются тривиально (пересечение делит $5$ и не равно $5$). Каждая даёт $4$ элемента порядка $5$, всего $8$ таких элементов. Число решений $x^5 = e$ равно $9$. Если $5 \mid |G|$, по Фробениусу (E5) это число делится на $5$ — противоречие. Если $5 \nmid |G|$, элементов порядка $5$ вообще нет — снова противоречие.

4. Раз для каждого $d \mid n$ ровно одна подгруппа $H_d$ порядка $d$, то решений $x^d = e$ ровно $|H_d| = d$. Применима E6, $G$ циклическая.

5. Не обязана. Минимальный контрпример: $G = S_3$, $H = \langle (1\,2)\rangle$. Порядок $H$ равен $2$, индекс $3$. Подгруппа не нормальна: $(1\,3)(1\,2)(1\,3)^{-1} = (2\,3)$.

6. Элементы порядка не больше $2$ образуют подгруппу $G[2]$ в абелевой группе. Все её элементы удовлетворяют $g^2 = e$, поэтому $G[2] \cong (\mathbb Z/2)^k$ (см. E15). Условие «ровно $3$ элемента порядка $2$» даёт $|G[2]| = 4$, то есть $k = 2$. В разложении $G$ по элементарным делителям ровно две компоненты с чётным порядком. Иными словами, $2$-силовская подгруппа изоморфна $\mathbb Z/2^a \times \mathbb Z/2^b$ с $a, b \ge 1$, а нечётная часть произвольна.

7. Если простое $q \ne p$ делит $|G|$, то по теореме Коши (E2) в $G$ есть элемент порядка $q$, чего нет по условию. Значит у $|G|$ нет простых делителей, отличных от $p$, то есть $|G| = p^k$.

8. По формуле E11: $|HK| = 24 \cdot 20 / |H \cap K| \le |G| = 120$, откуда $|H \cap K| \ge 4$. С другой стороны, $|H \cap K|$ делит $\gcd(24, 20) = 4$. Значит $|H \cap K| = 4$.

9. Раскладываем по простым: $12 = 2^2 \cdot 3$, $18 = 2 \cdot 3^2$. В абелевой группе если $v_p(\operatorname{ord} a) \ne v_p(\operatorname{ord} b)$, то $v_p(\operatorname{ord}(ab)) = \max(v_p(\operatorname{ord} a), v_p(\operatorname{ord} b))$ (проектируем на $p$-силовскую часть). Для $p = 2$: $2$ против $1$, максимум $2$. Для $p = 3$: $1$ против $2$, максимум $2$. Итого $\operatorname{ord}(ab) = 4 \cdot 9 = 36$.

## Использования

E1 (теорема Лагранжа). Скелет любой задачи на делимость порядка, например в IMC-2005-6 как стартовый шаг.

E2 (теорема Коши). IMC-2005-6 — извлечение элементов заданного простого порядка. Дальше — классификация групп порядка $pq$ (E14).

E3 (формула порядка степени). IMC-2001-2 — переход к нужной степени для устранения лишних множителей.

E4 (подгруппы циклической). IMC-1996-DAY2-3 — пример нециклической не конечно порождённой подгруппы $\mathbb Z[1/2]$ в $\mathrm{GL}_2(\mathbb Q)$. IMC-2025-7 — замыкание относительно удвоения и среднего.

E5 (теорема Фробениуса). Доказательства существования подгрупп заданных порядков; задачи с нижними оценками на число элементов фиксированного порядка.

E6 (критерий цикличности). Главное применение — цикличность $\mathbb F_q^\times$ для конечного поля. Putnam-style задачи о первообразных корнях.

E7 (число элементов порядка $d$). Тождество $\sum_{d \mid n} \varphi(d) = n$ всплывает в комбинаторике через лемму Бёрнсайда; подсчёт примитивных элементов.

E8 (порядок произведения). IMC-2001-2 — в абелевой группе через тождество Безу получается элемент нужного порядка.

E9 (наименьший простой делитель). Быстрое выделение нормальной подгруппы; стандартная техника в задачах со структурой $S_n$, $A_n$.

E10 (китайская теорема). Классификация конечных абелевых групп; разложение $(\mathbb Z/n)^\times$ по простым.

E11 (формула $|HK|$). Доказательства теорем Силова; задача 8 из самопроверки; IMC-2008-5 — подсчёт автоморфизмов через структуру полупрямого произведения.

E12 (индекс пересечения). Технический инструмент для перехода к нормальной подгруппе.

E13 (порядок $p^2$). Малые порядки в Putnam-стиле; Putnam-1969-B2 как структура небольших групп.

E14 (порядок $pq$). Сразу классифицирует группы малого порядка; контрпример $|G| = 6$ ($S_3$ против $\mathbb Z/6$).

E15 (булева группа). IMC-2003-DAY2-4 — система троек Штейнера как $(\mathbb Z/2)^k$. Putnam-style задачи на «булевы группы».

E16 (экспонента абелевой). IMC-2001-2 опирается ровно на это — существование универсального элемента.
