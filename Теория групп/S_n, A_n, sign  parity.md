# S_n, A_n, sign / parity

## Scope
Подтема покрывает арсенал, связанный с группами перестановок $S_n$ и $A_n$: знак (sign / parity) как гомоморфизм, cycle decomposition, классы сопряжённости, генераторы, нормальные подгруппы и простота $A_n$. Главные сценарии IMC/Putnam — представление произвольных биекций через перестановки специального вида, доказательства генерации $S_n$/$A_n$, использование знака как «инварианта чётности», эксплуатация простоты $A_n$ для $n\ge 5$, оценки на максимальный порядок элемента, чётностные аргументы в комбинаторных играх.

## Записи - Core

### E1. Sign homomorphism (знак / parity) и его корректная определённость

Формулировка. Существует единственный гомоморфизм $\operatorname{sgn}\colon S_n \to \{\pm 1\}$ такой, что $\operatorname{sgn}(\tau) = -1$ для любой транспозиции $\tau$. Эквивалентно, $\operatorname{sgn}(\sigma) = (-1)^{\#\{(i,j)\colon i<j,\,\sigma(i)>\sigma(j)\}}$ (число inversions). Ядро — alternating group $A_n = \ker\operatorname{sgn}$, $[S_n:A_n]=2$ при $n\ge 2$.

Условия применимости. $n\ge 2$. Для $n=1$ группа тривиальна.

Идея доказательства. Корректная определённость через многочлен Vandermonde $V(x_1,\dots,x_n) = \prod_{i<j}(x_i-x_j)$: $\sigma$ переставляет факторы, $V \mapsto \pm V$, знак — глобальная константа на $S_n$. Альтернативно — через cycle decomposition (E4).

Варианты и обобщения. Sign даёт точную последовательность $1 \to A_n \to S_n \to \mathbb Z/2 \to 1$. Любой групповой гомоморфизм $S_n \to \mathbb C^\times$ при $n\ge 5$ — либо тривиален, либо $\operatorname{sgn}$ (см. E13).

Используется в. Любой определитель через формулу Лейбница $\det A = \sum_\sigma \operatorname{sgn}(\sigma)\prod a_{i,\sigma(i)}$; Putnam-2005-B6 (тождество $\sum_\sigma \operatorname{sgn}(\sigma)/(\nu(\sigma)+1)=(-1)^{n+1}n/(n+1)$); парности в комбинаторных играх (IMC-2012-3).

### E2. Cycle decomposition и cycle type как полный инвариант сопряжённости в $S_n$

Формулировка. Каждая $\sigma \in S_n$ единственным способом (с точностью до порядка) разлагается в произведение непересекающихся циклов. Cycle type $\lambda(\sigma) = (\lambda_1\ge\lambda_2\ge\dots)$ — partition of $n$. Две перестановки $\sigma,\tau\in S_n$ сопряжены $\iff \lambda(\sigma)=\lambda(\tau)$. Число классов сопряжённости в $S_n$ равно $p(n)$ — числу разбиений $n$.

Условия применимости. Любое $n$.

Идея доказательства. Орбиты $\langle\sigma\rangle$-действия на $\{1,\dots,n\}$ дают цикли. Сопряжение — переименование меток: $\rho\sigma\rho^{-1}$ имеет те же длины циклов (см. E3).

Варианты и обобщения. В $A_n$ класс может расщепиться (E8). Для бесконечного множества — каждое биективное отображение разлагается в непересекающиеся циклы и/или $\mathbb Z$-орбиты.

Используется в. IMC-1997-DAY2-5 (явное использование орбит); подсчёт классов сопряжённости; индукция по cycle type.

### E3. Conjugation formula: $\rho(a_1\,a_2\,\dots\,a_k)\rho^{-1}=(\rho(a_1)\,\rho(a_2)\,\dots\,\rho(a_k))$

Формулировка. Для любого $\rho\in S_n$ и любого цикла $(a_1,\dots,a_k)$:
$$\rho\,(a_1\,a_2\,\cdots\,a_k)\,\rho^{-1} = (\rho(a_1)\,\rho(a_2)\,\cdots\,\rho(a_k)).$$
Соответственно, сопряжение — это переименование меток циклов.

Условия применимости. Любая $\rho\in S_n$, любой цикл (не обязательно полный).

Идея доказательства. Прямая проверка: $\rho\sigma\rho^{-1}$ переводит $\rho(a_i)$ в $\rho(a_{i+1})$.

Используется в. Конструкция явных сопряжений; проверка нормальности подгрупп; IMC-1998-2 (порождение $S_n$ заданными элементами).

### E4. Sign через cycle structure: $\operatorname{sgn}(\sigma) = (-1)^{n - c(\sigma)} = \prod_i (-1)^{\lambda_i - 1}$

Формулировка. Если $\sigma\in S_n$ имеет cycle type $\lambda = (\lambda_1,\dots,\lambda_m)$ (с учётом fixed points как 1-циклов), $c(\sigma) = m$ — число циклов. Тогда:
$$\operatorname{sgn}(\sigma) = (-1)^{n-c(\sigma)} = \prod_{i=1}^{m}(-1)^{\lambda_i - 1}.$$
В частности, $k$-цикл имеет $\operatorname{sgn} = (-1)^{k-1}$. Минимальное число транспозиций, в произведение которых разлагается $\sigma$, равно $n-c(\sigma)$.

Условия применимости. Любое $n\ge 1$.

Идея доказательства. $k$-цикл $(a_1\,\dots\,a_k) = (a_1\,a_2)(a_2\,a_3)\cdots(a_{k-1}\,a_k)$ — $k-1$ транспозиций.

Варианты и обобщения. Лемма Cayley: число факторизаций $n$-цикла в произведение $n-1$ транспозиций равно $n^{n-2}$ (matrix-tree connection).

Используется в. Быстрое вычисление знака в задачах с конкретной перестановкой; формула Leibniz для $\det$; алгоритм минимальной сортировки массива транспозициями.

### E5. Generators of $S_n$: транспозиции, adjacent transpositions, $(1\,2)$ + $n$-cycle

Формулировка. Группа $S_n$ порождается:
1. Всеми транспозициями.
2. Adjacent transpositions $(1\,2),(2\,3),\dots,(n-1\,n)$.
3. Любой транспозицией $(i\,j)$ и любым $n$-циклом $\sigma$. В частности, $S_n = \langle (1\,2),\,(1\,2\,\dots\,n)\rangle$.
4. Любой транспозицией двух соседних элементов в $n$-цикле и самим $n$-циклом.

Условия применимости. $n\ge 2$. В пункте 3 для не-соседних пар $(i,j)$ в цикле — критерий: $\gcd(j-i, n) = 1$.

Идея доказательства. (1) очевидно из E2/E4. (2) $(i\,j) = (i\,i{+}1)(i{+}1\,i{+}2)\cdots (j-1\,j)\cdots(i\,i{+}1)$. (3) $\sigma^k(1\,2)\sigma^{-k} = (1+k\bmod n,\,2+k\bmod n)$ — пробегаются все adjacent transpositions при $\gcd(\cdot,n)=1$.

Варианты и обобщения. $S_n$ Coxeter-presentable: $\langle s_1,\dots,s_{n-1}\mid s_i^2=1,\ (s_is_{i+1})^3=1,\ s_is_j=s_js_i\,(|i-j|>1)\rangle$. $S_p$ ($p$ простое) порождается *любой* транспозицией и *любым* $p$-циклом — отсюда задачи о генерации в $\mathbb F_p$-полиномах.

Используется в. IMC-2009-DAY2-4 (полиномы по модулю $x^p-x$ как $S_p$ через циклический сдвиг $x{+}1$ и транспозицию через $x^{p-2}+\dots+1$); IMC-1998-2.

### E6. Generators of $A_n$: 3-cycles, $(1\,2\,3)$ + $n$-cycle / $(n-1)$-cycle

Формулировка. Группа $A_n$ ($n\ge 3$) порождается:
1. Всеми 3-циклами.
2. 3-циклами вида $(1\,2\,k)$, $k=3,\dots,n$.
3. Парой генераторов: $(1\,2\,3)$ и $(2\,3\,\dots\,n)$ при нечётном $n$, $(1\,2\,3)$ и $(1\,2\,3\,\dots\,n)$ при чётном $n$ (т.е. cycle нужной чётности).

Условия применимости. $n\ge 3$.

Идея доказательства. Любая чётная перестановка — произведение чётного числа транспозиций; пара $(a\,b)(c\,d)$ записывается через 3-циклы: при общем элементе $(a\,b)(a\,c)=(a\,c\,b)$, при разных — $(a\,b)(c\,d)=(a\,b\,c)(b\,c\,d)$.

Варианты и обобщения. $A_n$ как commutator $[S_n,S_n]$ при $n\ge 2$ (см. E11).

Используется в. Доказательство простоты $A_n$ (E9); IMC-1998-2 (Klein 4-group obstruction для $n=4$).

### E7. Любая перестановка — произведение двух involutions

Формулировка. Каждая $\sigma \in S_n$ (или биекция бесконечного множества) представима как произведение $\sigma = \alpha\beta$ двух involutions (т.е. $\alpha^2=\beta^2=\mathrm{id}$). Более конкретно: цикл $(a_1\,a_2\,\dots\,a_k)$ — произведение двух «зеркальных» involutions $\alpha=(a_1\,a_k)(a_2\,a_{k-1})\cdots$ и $\beta=(a_1\,a_{k-1})(a_2\,a_{k-2})\cdots$.

Условия применимости. Любая перестановка любого множества; для бесконечной орбиты $\cong \mathbb Z$ — две зеркальные симметрии относительно «полуцелой» и «целой» точек.

Идея доказательства. Достаточно доказать для одного цикла. $k$-цикл — поворот регулярного $k$-угольника на угол $2\pi/k$, представим как композиция двух осевых симметрий. Бесконечный сдвиг $\mathbb Z\to\mathbb Z$, $n\mapsto n+1$, $= (n\mapsto 1-n)\circ(n\mapsto -n)$.

Варианты и обобщения. В диэдральной группе $D_k$ любой элемент — произведение $\le 2$ отражений. Орбиты под действием группы $\langle\alpha,\beta\rangle$ из двух involutions — цепочки между fixed points или чётные циклы (Zagier).

Используется в. IMC-1997-DAY2-5 (буквально это утверждение); комбинаторные интерпретации classical bijections (Pak); счёт involutions (sequence A000085 OEIS).

## Записи - Standard

### E8. Splitting of conjugacy classes $S_n \to A_n$: критерий

Формулировка. Класс сопряжённости элемента $\sigma\in A_n$ в $S_n$ либо целиком сопряжён в $A_n$ (один класс), либо распадается на два класса равной мощности. Распадается $\iff$ cycle type $\sigma$ состоит из *различных нечётных длин* (включая 1-циклы). Эквивалентно: централизатор $C_{S_n}(\sigma)\subseteq A_n$.

Условия применимости. $n\ge 2$, $\sigma\in A_n$.

Идея доказательства. Класс распадается $\iff$ нет нечётной перестановки, коммутирующей с $\sigma$. Если в cycle type есть чётный цикл $(a_1\,\dots\,a_{2k})$ — он сам нечётная коммутирующая перестановка. Если две одинаковые длины $\lambda_i=\lambda_j$ — обмен двух циклов даёт нечётную коммутирующую. Иначе централизатор порождён циклами, все нечётны → лежит в $A_n$.

Варианты и обобщения. В $A_5$: 5-циклы распадаются на два класса по 12 элементов. В $A_6$: 5-циклы распадаются (длины 5,1 — различные нечётные), а 3+3 не распадается (одинаковые длины).

Используется в. Подсчёт characters $A_n$; задачи на conjugacy в $A_5$, $A_6$; в Putnam-стиле — там, где надо понять, почему «два разных типа» $5$-циклов в $A_5$ ведут к разным инвариантам.

### E9. Simplicity of $A_n$ for $n\ge 5$

Формулировка. $A_n$ простая для всех $n\ge 5$, т.е. не имеет нетривиальных нормальных подгрупп. $A_4$ не простая ($V_4 \trianglelefteq A_4$, см. E10), $A_2,A_3$ — циклические простые (порядков $1,3$).

Условия применимости. $n\ge 5$.

Идея доказательства. Стандартная схема Conrad/Lipman: (a) если $H \trianglelefteq A_n$, $H\ne 1$, и $H$ содержит хоть один 3-цикл, то $H \supseteq$ все 3-циклы (через сопряжение в $A_n$ при $n\ge 5$) → $H=A_n$. (b) Если $H$ содержит элемент $\sigma\ne e$, можно построить коммутатор $[\sigma,\rho]$ с тщательно выбранным $\rho$ и получить 3-цикл.

Варианты и обобщения. $A_5$ — наименьшая неабелева простая группа (порядок 60). Изоморфизмы: $A_5\cong PSL_2(\mathbb F_4)\cong PSL_2(\mathbb F_5)\cong I$ (icosahedral). $A_6\cong PSL_2(\mathbb F_9)$, $A_8\cong PSL_4(\mathbb F_2)$.

Используется в. Любые задачи с нормальными подгруппами в $A_n$; галуа-теория ($A_n$ простая ⇒ $S_n$ не разрешима для $n\ge5$ ⇒ нет общей формулы корней многочленов степени $\ge 5$); Schweitzer-стиль контрпримеры.

### E10. Normal subgroups of $S_n$: классификация

Формулировка. Нормальные подгруппы $S_n$:
- $n=1,2$: тривиальная и вся группа.
- $n=3$: $1, A_3, S_3$.
- $n=4$: $1, V_4, A_4, S_4$, где $V_4=\{e,(12)(34),(13)(24),(14)(23)\}$ — Klein 4-group.
- $n\ge 5$: только $1, A_n, S_n$.

Условия применимости. $n\ge 1$.

Идея доказательства. Для $n\ge 5$ — из E9: если $H \trianglelefteq S_n$, то $H\cap A_n \trianglelefteq A_n$, значит $\in\{1,A_n\}$; и плюс возможные нечётные элементы. Для $n=4$ — $V_4$ — единственная пара двойных транспозиций (тип $\lambda=(2,2)$, не расщепляется в $S_4$).

Варианты и обобщения. Нормальные подгруппы $A_n$ при $n\ge 5$: $\{1,A_n\}$ (E9). При $n=4$: $\{1,V_4,A_4\}$.

Используется в. IMC-1998-2 (для $n=4$ Klein 4-group — препятствие к генерации $S_4$ парой $((12)(34),\sigma)$); проверка, что данная подгруппа нормальна.

### E11. Commutator subgroup: $[S_n,S_n]=A_n$, $[A_n,A_n]=A_n$ при $n\ge 5$

Формулировка.
1. $[S_n,S_n]=A_n$ при $n\ge 2$.
2. $[A_n,A_n]=A_n$ при $n\ge 5$ (т.е. $A_n$ совершенна).
3. $[A_4,A_4]=V_4$, $[A_3,A_3]=\{e\}$.

Условия применимости. См. формулировку.

Идея доказательства. (1): $S_n/A_n\cong\mathbb Z/2$ абелева, значит $[S_n,S_n]\subseteq A_n$. С другой стороны, $[(1\,2),(1\,3)]=(1\,2\,3)$ — 3-цикл, и $A_n$ порождается 3-циклами (E6). (2): аналогично, $A_n$ простая некоммутативная, значит совпадает со своим коммутантом.

Варианты и обобщения. Center: $Z(S_n)=\{e\}$ при $n\ge 3$, $Z(A_n)=\{e\}$ при $n\ge 4$. Это видно из E3: единственная перестановка, коммутирующая со всеми 3-циклами — тождество.

Используется в. Доказательство неразрешимости $S_n$ при $n\ge 5$ через derived series; вычисление абелианизации $S_n^{\mathrm{ab}}\cong\mathbb Z/2$.

### E12. Solvability $S_n$, $A_n$ ровно при $n\le 4$

Формулировка. $S_n$ разрешима $\iff$ $A_n$ разрешима $\iff n\le 4$. Derived series: $S_4 \rhd A_4 \rhd V_4 \rhd \{e\}$, факторы $\mathbb Z/2,\mathbb Z/3,V_4$ — все абелевы.

Условия применимости. $n\ge 1$.

Идея доказательства. $\Leftarrow$: явный derived series выше для $n\le 4$. $\Rightarrow$: при $n\ge 5$ derived series стабилизируется на $A_n$ (E11), нетривиальной → не разрешима.

Варианты и обобщения. Galois-теоретическое следствие: общий многочлен степени $\ge 5$ не разрешим в радикалах (Abel–Ruffini). Конкретные многочлены с группой Galois $S_5$, например, $x^5-x-1$.

Используется в. Putnam-1969-B2 (структура «сложных» групп); олимпиадные задачи на иерархию подгрупп.

### E13. Sign — единственный нетривиальный гомоморфизм $S_n \to \mathbb C^\times$ при $n\ge 5$

Формулировка. Пусть $n\ge 5$ и $\varphi\colon S_n \to \mathbb C^\times$ — гомоморфизм. Тогда $\varphi$ либо тривиален ($\varphi\equiv 1$), либо $\varphi=\operatorname{sgn}$.

Условия применимости. $n\ge 5$. При $n=4$ есть ещё гомоморфизмы через фактор $S_4 \twoheadrightarrow S_4/V_4\cong S_3$.

Идея доказательства. $\ker\varphi \trianglelefteq S_n$, по E10 при $n\ge 5$ возможны только $\{e\}, A_n, S_n$. $\ker = S_n$ — тривиальный гомоморфизм, $\ker = A_n$ — $\varphi=\operatorname{sgn}$, $\ker=\{e\}$ — невозможно, ибо $S_n$ некоммутативна, а $\mathbb C^\times$ коммутативна.

Варианты и обобщения. Гомоморфизмы $S_n \to S_m$ при $m<n$ при $n\ge 5$: либо тривиальный, либо через знак (с образом порядка 2). Это даёт жёсткие ограничения на действия $S_n$ на множествах малой мощности.

Используется в. Putnam-style: «есть ли функция $f\colon S_n\to\mathbb C^\times$ с такими свойствами» — обычно сводится к sign.

### E14. Cayley theorem (теорема Кэли) и его уточнения

Формулировка. Любая группа $G$ изоморфна подгруппе $\operatorname{Sym}(G)$. Для $|G|=n$ — подгруппе $S_n$. Уточнение (через действие на $G/H$ для $H\le G$): любая $G$ вкладывается в $S_{[G:H]}$, если $H$ не содержит нетривиальной нормальной подгруппы $G$ (т.е. $H$ — core-free).

Условия применимости. Любая группа.

Идея доказательства. Левое регулярное действие $g\cdot x = gx$ — точное. Уточнение — через действие на смежных классах с ядром $\bigcap_g gHg^{-1}$.

Варианты и обобщения. Минимальная степень вложения $\mu(G)$ — обычно меньше $|G|$ (например, $\mu(\mathbb Z/n)=n$, но $\mu((\mathbb Z/2)^k)=2k$). Для конечной простой $G$ — $\mu(G)$ совпадает с минимальным индексом подгруппы.

Используется в. Стандартный технический инструмент: «если у группы $G$ есть подгруппа индекса $k$, то $|G|$ делит $k!$» — комбо с E10.

## Записи - Exotic

### E15. Outer automorphism of $S_6$ (исключительный)

Формулировка. $\operatorname{Aut}(S_n)=S_n$ (т.е. $\operatorname{Out}(S_n)$ тривиален) для всех $n\ne 2,6$. Для $n=2$ автоморфизмы тривиальны. Для $n=6$: $|\operatorname{Out}(S_6)|=2$ — единственное исключение среди симметрических групп. Аналогично $\operatorname{Out}(A_n)=\mathbb Z/2$ для $n\ne 6$ и $\operatorname{Out}(A_6)=\mathbb Z/2\times\mathbb Z/2$.

Условия применимости. См. формулировку.

Идея доказательства. Outer automorphism $S_6$ переставляет conjugacy classes транспозиций и тройных транспозиций (типа $(12)(34)(56)$) — обе содержат по 15 элементов. Конструкция через 6 «syntheme»-троек или через действие $S_6$ на 6 объектах двумя неэквивалентными способами (Cameron, Hölder 1895).

Варианты и обобщения. Эта аномалия объясняет ряд исключительных явлений: $S_6$ имеет 12 (а не 11) автоморфизмов transitively действующих на $\{1,\dots,6\}$; связь с $\operatorname{PGL}_2(\mathbb F_5)\cong S_5$ и его двумя вложениями в $S_6$.

Используется в. Schweitzer-уровень; обычно эксплуатируется как «парадокс» при подсчёте структур на 6 точках.

### E16. Landau's function $g(n)$: максимальный порядок элемента $S_n$

Формулировка. $g(n) = \max_{\sigma\in S_n}\operatorname{ord}(\sigma) = \max\{\operatorname{lcm}(\lambda_1,\dots,\lambda_k): \lambda_1+\dots+\lambda_k=n\}$. Асимптотика Landau (1903): $\log g(n) \sim \sqrt{n\log n}$, т.е. $g(n)=e^{(1+o(1))\sqrt{n\ln n}}$.

Условия применимости. $n\ge 1$.

Идея доказательства. Порядок $\sigma$ — это $\operatorname{lcm}$ длин циклов (E2 + E8 в Lagrange-заметке). Максимизация $\operatorname{lcm}$ при фиксированной сумме — комбинаторно-теоретико-числовая задача; экстремум достигается на наборе попарно почти-простых $\lambda_i$. Асимптотика — через теорему о простых числах: оптимальные $\lambda_i$ — степени простых $\le \sqrt{n\ln n}$.

Варианты и обобщения. $g(n)$ при малых $n$: $1,2,3,4,6,6,12,15,20,30,30,\dots$ (OEIS A000793). Для $A_n$ — отдельная функция $g'(n)$, отличается множителями 2.

Используется в. Putnam-style оценки «верхняя граница на порядок перестановки $n$ объектов»; задачи о card-shuffling (perfect shuffles).

### E17. $A_5$ как наименьшая неабелева простая группа; характеризация

Формулировка. $A_5$ — единственная (с точностью до изоморфизма) простая неабелева группа порядка $\le 100$. Минимальный порядок простой неабелевой группы — 60 (это $A_5$). Следующая по порядку — $\operatorname{PSL}_2(\mathbb F_7)$ порядка 168.

Условия применимости. Классическая характеризация конечных простых групп.

Идея доказательства. Перебор всех порядков $< 60$ и проверка через Sylow + Cauchy + Lagrange, что простая неабелева группа невозможна. Стандартное упражнение: $|G|=30$ ⇒ есть нормальный 5-Sylow (n_5=1).

Варианты и обобщения. $A_5\cong I$ (rotational icosahedral), $\cong\operatorname{PSL}_2(\mathbb F_4)\cong\operatorname{PSL}_2(\mathbb F_5)$. Изоморфизмы малых рангов — exceptional Lie group correspondences.

Используется в. Любая задача, где нужно «дисквалифицировать» возможность простой группы малого порядка; геометрические задачи на симметрии икосаэдра.

## Self-test

1. Вычисли $\operatorname{sgn}(\sigma)$ для $\sigma=(1\,5\,3)(2\,7)(4\,6\,8\,9) \in S_9$.

2. Пусть $G\le S_n$ транзитивно действует на $\{1,\dots,n\}$ и содержит транспозицию. Должна ли $G$ содержать все транспозиции? (Если нет — минимальный контрпример.)

3. Сколько перестановок в $S_n$ коммутируют с фиксированной перестановкой $\sigma$ cycle type $\lambda=(\lambda_1,\dots,\lambda_k)$, где есть $m_i$ циклов длины $i$?

4. Докажи, что 5-циклы в $A_5$ образуют два класса сопряжённости равной мощности.

5. Пусть $\sigma\in S_n$ имеет порядок $m$. Докажи, что $\sigma$ всегда есть произведение двух involutions, и что одной involution достаточно $\iff m\le 2$.

6. Пусть $G$ — подгруппа $S_7$ с $|G|=21$. Может ли $G$ быть простой?

7. Существует ли гомоморфизм $\varphi\colon S_7 \to S_5$ с нетривиальным образом? Если да — опиши его; если нет — обоснуй.

8. Пусть $n\ge 5$. Докажи, что в $S_n$ нет подгруппы индекса $k$ при $2<k<n$.

9. Найди максимальный возможный порядок элемента в $S_{12}$ и в $A_{12}$.

10. Пусть $G\le S_n$ действует транзитивно. Докажи, что $|G|\ge n$, и что равенство $|G|=n$ означает, что $G$ — регулярное действие (т.е. $G$ изоморфна группе порядка $n$ через Cayley).

## Решения

1. По E4: знак $k$-цикла $=(-1)^{k-1}$. Перемножая по циклам $(1\,5\,3),(2\,7),(4\,6\,8\,9)$ — длины $3,2,4$: $\operatorname{sgn}=(+1)\cdot(-1)\cdot(-1)=+1$. Контроль через формулу $\operatorname{sgn}=(-1)^{n-c(\sigma)}$: $n=9$ (fixed points нет, поскольку $3+2+4=9$), $c(\sigma)=3$, $\operatorname{sgn}=(-1)^{6}=+1$.

2. Не должна. Минимальный контрпример: $G=\langle(1\,2),(3\,4),(1\,3)(2\,4)\rangle\le S_4$ — порядок 8 (диэдральная), транзитивна, содержит транспозиции $(1\,2),(3\,4)$, но не $(1\,3)$ или $(2\,3)$. Однако *примитивное* действие + транспозиция ⇒ $G=S_n$ (классика).

3. $|C_{S_n}(\sigma)|=\prod_i i^{m_i}\cdot m_i!$. Каждый цикл длины $i$ можно сдвигать ($i$ способов), а циклы одной длины переставлять ($m_i!$ способов).

4. Cycle type 5-цикла $=(5)$ — все длины нечётные и различные, по E8 класс расщепляется. Размер каждой части: класс в $S_5$ имеет $5!/5=24$ элемента, оба чётны (5-цикл — чётная перестановка), пополам в $A_5$: $24/2=12$.

5. Существование разложения в две involutions — E7. Одной involution достаточно $\iff$ $\sigma$ сама involution $\iff \sigma^2=e$ $\iff$ все длины циклов $\le 2$ $\iff m\le 2$. При $m\ge 3$ хотя бы один цикл длины $\ge 3$, его квадрат не тождественный.

6. Нет. По теореме Sylow $n_7\equiv 1\pmod 7$, $n_7\mid 3$, значит $n_7=1$ — Sylow-7 нормальна. $G$ не простая (есть $\mathbb Z/7\trianglelefteq G$). Альтернатива: $|G|=21=3\cdot 7$, по E14 в Lagrange-заметке (классификация $pq$) — циклическая или semidirect product, в обоих случаях есть нормальная $\mathbb Z/7$.

7. По E13: при $n=7\ge 5$ единственные гомоморфизмы $S_7\to G$ для абелевых $G$ — тривиальные или sign. Но $S_5$ неабелева, поэтому E13 буквально не работает. Через E10 + Cayley: $\ker\varphi\trianglelefteq S_7$, значит $\in\{1,A_7,S_7\}$. $\ker=\{e\}$ — невозможно, $|S_7|=5040>120=|S_5|$. $\ker=A_7$ — $\varphi$ факторизуется через $\mathbb Z/2$, образ в $S_5$ — порядка $\le 2$. $\ker=S_7$ — тривиальный. Итог: единственный нетривиальный гомоморфизм с образом $\{e,(1\,2)\}$ или подобным, передающий sign $S_7$ как транспозиция в $S_5$.

8. По E14: подгруппа $H$ индекса $k$ даёт гомоморфизм $S_n\to S_k$ через действие на смежных классах, ядро $K\subseteq H$, $K\trianglelefteq S_n$. По E10 при $n\ge 5$: $K\in\{1,A_n,S_n\}$. $K=S_n$ ⇒ $H=S_n$, $k=1$. $K=A_n$ ⇒ $|S_n/K|=2\mid k!$, значит $k\ge 2$, и фактор-гомоморфизм имеет образ $\mathbb Z/2$, значит $H\supseteq A_n$, $k\le 2$. $K=\{e\}$ ⇒ $S_n\hookrightarrow S_k$, $n!\mid k!$, $k\ge n$. Итог: $k\in\{1,2,n,\ldots\}$, нет подгрупп индекса $3,4,\dots,n-1$.

9. $S_{12}$ (E16, Landau): максимизируем $\operatorname{lcm}$ при $\sum\lambda_i=12$. Кандидат $(3,4,5)$: $\operatorname{lcm}=60$, других $\ge 60$ нет (для $\operatorname{lcm}\ge 60$ нужны простые степени $4,3,5$ как делители; единственное разбиение с такими максимальными степенями простых — $(3,4,5)$). Ответ для $S_{12}$: $g(12)=60$.

   Для $A_{12}$: знак $(3,4,5)$ — $(-1)^{(3-1)+(4-1)+(5-1)}=(-1)^9=-1$, не подходит. Перебор больших lcm с положительным знаком: $(2,3,7)$ — $\operatorname{lcm}=42$, $\operatorname{sgn}=(-1)^{1+2+6}=-1$ ✗. $(7,5)$ — $\operatorname{lcm}=35$, $\operatorname{sgn}=(-1)^{6+4}=+1$ ✓. $(2,3,5,1,1)$ — $\operatorname{lcm}=30$. $(8,3,1)$ — $24$. $(6,5,1)$ — $30$, sgn$=-1$. Ответ для $A_{12}$: $g'(12)=35$, реализуется на $(7,5)$.

10. Транзитивность ⇒ stabilizer любой точки $G_x \le G$, $|G|=|G_x|\cdot|G\cdot x|=|G_x|\cdot n$ (orbit-stabilizer), значит $n\mid |G|$. Если $|G|=n$ — то $|G_x|=1$, действие свободное и транзитивное = регулярное; такое действие изоморфно левому регулярному действию $G$ на себе, т.е. это вложение $G\hookrightarrow S_n$ через Cayley.
