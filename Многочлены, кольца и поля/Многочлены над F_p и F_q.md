# Многочлены над $\mathbb{F}_p$ и $\mathbb{F}_q$

## Scope
Техники, специфичные для многочленов над конечными полями. Тождество $x^q - x = \prod_a (x-a)$, эндоморфизм Фробениуса, степенные суммы по $\mathbb{F}_q$, цикличность мультипликативной группы, различие между многочленом и заданной им функцией. Полиномиальный метод: теоремы Шевалле-Варинга, комбинаторный Nullstellensatz, Лукаса, лифтинг Гензеля, многочлены Артина-Шрейера. Все эти факты допустимы на IMC и Putnam без вывода.

## Определения

**Конечное поле.** $\mathbb{F}_q$ — поле из $q$ элементов; $q = p^n$ для простого $p$ и $n \ge 1$. Характеристика $\operatorname{char} \mathbb{F}_q = p$. Подполе $\mathbb{F}_p = \mathbb{Z}/p\mathbb{Z}$. Расширение $\mathbb{F}_{q^m}/\mathbb{F}_q$ всегда циклично, группа Галуа порождена эндоморфизмом Фробениуса.

**Мультипликативная группа.** $\mathbb{F}_q^\times = \mathbb{F}_q \setminus \{0\}$ — группа по умножению порядка $q-1$.

**Порядок элемента.** Наименьшее $k \ge 1$ с $a^k = 1$. Делит $q - 1$.

**Первообразный элемент** (primitive element). Элемент $\mathbb{F}_q^\times$ порядка ровно $q-1$. Существует всегда.

**Кольцо многочленов.** $\mathbb{F}_q[x_1, \dots, x_n]$. Степень одночлена $\prod x_i^{e_i}$ — это $\sum e_i$ (полная степень). Степень многочлена — максимум степеней одночленов.

**Неприводимый многочлен.** $f \in \mathbb{F}_q[x]$ степени $\ge 1$, не раскладывающийся в произведение двух многочленов положительной степени.

**Минимальный многочлен элемента $\alpha \in \overline{\mathbb{F}_q}$.** Нормированный (старший коэффициент $1$) неприводимый многочлен из $\mathbb{F}_q[x]$, для которого $\alpha$ — корень. Единственный.

**Сепарабельный многочлен.** Не имеет кратных корней в алгебраическом замыкании. Эквивалентно: $\gcd(f, f') = 1$.

**Алгебраическое замыкание.** $\overline{\mathbb{F}_p} = \bigcup_{n \ge 1} \mathbb{F}_{p^n}$.

## Записи — Основные

### E1. Эндоморфизм Фробениуса (Frobenius) и «мечта первокурсника» (freshman's dream)

Утверждение. В коммутативном кольце характеристики $p$ отображение $\varphi\colon a \mapsto a^p$ — кольцевой гомоморфизм. В частности:
$$(x+y)^p = x^p + y^p, \qquad (x+y)^{p^k} = x^{p^k} + y^{p^k}.$$

На конечном поле $\mathbb{F}_q$ ($q = p^n$) отображение $\varphi$ — автоморфизм. Группа Галуа $\operatorname{Gal}(\mathbb{F}_{p^n}/\mathbb{F}_p)$ циклическая порядка $n$ и порождена $\varphi$.

Условия применимости. Характеристика поля должна быть простой. В характеристике $0$ утверждение неверно. Пример: над $\mathbb{Q}$ имеем $(x+y)^2 = x^2 + 2xy + y^2$.

Идея доказательства. Биномиальный коэффициент $\binom{p}{k}$ делится на $p$ при $1 \le k \le p-1$ (в числителе $p!$ есть множитель $p$, в знаменателе $k!(p-k)!$ — нет).

Варианты. Относительный Фробениус $a \mapsto a^q$ фиксирует $\mathbb{F}_q$ поэлементно и порождает $\operatorname{Gal}(\mathbb{F}_{q^m}/\mathbb{F}_q)$. Если $\alpha$ — корень $f \in \mathbb{F}_p[x]$ в $\overline{\mathbb{F}_p}$, то $\alpha^p$ — тоже корень; орбита Фробениуса имеет длину, равную степени минимального многочлена $\alpha$.

### E2. Тождество $x^q - x = \prod_{a \in \mathbb{F}_q}(x-a)$ и теорема Вильсона в полиномиальной форме

Утверждение. В $\mathbb{F}_q[x]$ выполнены равенства
$$x^q - x = \prod_{a \in \mathbb{F}_q}(x-a), \qquad x^{q-1} - 1 = \prod_{a \in \mathbb{F}_q^\times}(x-a).$$
Подстановка $x = 0$ во второе равенство даёт теорему Вильсона: $\prod_{a \in \mathbb{F}_q^\times} a = -1$. В случае $q = p$ это $(p-1)! \equiv -1 \pmod p$.

Условия применимости. Поле должно быть конечным. Над $\mathbb{Q}$ или $\mathbb{R}$ аналогичного конечного произведения нет.

Идея доказательства. По малой теореме Ферма $a^q = a$ для всех $a \in \mathbb{F}_q$. Многочлен $x^q - x$ степени $q$ имеет $q$ различных корней — значит, раскладывается на линейные множители целиком.

Варианты. $\gcd(f(x),\, x^q - x)$ выделяет ровно те корни $f$, что лежат в $\mathbb{F}_q$ (см. E12). Если $f(a) = 0$ для всех $a \in \mathbb{F}_q$, то $(x^q - x) \mid f$.

---

### E3. Степенные суммы $\sum_{a \in \mathbb{F}_q} a^k$

Утверждение.
$$\sum_{a \in \mathbb{F}_q} a^k = \begin{cases} -1 & \text{если } k > 0 \text{ и } (q-1) \mid k, \\ 0 & \text{иначе.} \end{cases}$$
По соглашению $0^0 := 1$, поэтому при $k = 0$ сумма равна $q \equiv 0$ в характеристике $p$.

Условия применимости. Поле должно быть конечным. Условие $(q-1) \mid k$ критично.

Пример. При $q = 5$, $k = 1$: $0 + 1 + 2 + 3 + 4 = 10 \equiv 0$, что согласуется с $4 \nmid 1$. При $q = 5$, $k = 4$: $0 + 1 + 16 + 81 + 256 \equiv 0 + 1 + 1 + 1 + 1 = 4 \equiv -1$, что согласуется с $4 \mid 4$.

Идея доказательства. Пусть $g$ — первообразный элемент $\mathbb{F}_q^\times$. Сумма по ненулевым $a$ равна $\sum_{j=0}^{q-2} g^{jk}$. Если $g^k \ne 1$, это геометрическая прогрессия с суммой $0$. Если $g^k = 1$, то $(q-1) \mid k$, и сумма равна $q - 1 = -1$.

Варианты. Для многочлена $f$ степени строго меньше $q-1$ имеем $\sum_{a \in \mathbb{F}_q} f(a) = 0$.

---

### E4. Принцип «слишком много корней»

Утверждение. Многочлен $f \in \mathbb{F}_q[x]$ степени $n$ имеет не более $n$ корней в любом расширении $\mathbb{F}_q$.

Многомерная версия. Если $f \in \mathbb{F}_q[x_1, \dots, x_n]$ имеет степень не выше $d_i$ по переменной $x_i$ и обнуляется на $S_1 \times \dots \times S_n$ с $|S_i| > d_i$, то $f \equiv 0$.

Условия применимости. Кольцо коэффициентов должно быть полем (или хотя бы целостным). Пример без условия: над $\mathbb{Z}/4$ многочлен $2x$ степени $1$ обнуляется в двух точках $0$ и $2$.

Идея доказательства. Одномерный случай — теорема Безу: $f(\alpha) = 0$ означает $(x-\alpha) \mid f$, и линейные множители независимы. Многомерный случай — индукция по числу переменных.

Варианты. Полиномиальная функция степени меньше $q$ из $\mathbb{F}_q$ в $\mathbb{F}_q$ однозначно определяется своими значениями (см. E6).

---

### E5. Цикличность $\mathbb{F}_q^\times$

Утверждение. Группа $\mathbb{F}_q^\times$ циклическая порядка $q-1$. Существует первообразный элемент $g$. Для любого $a \in \mathbb{F}_q^\times$ верно $a^{q-1} = 1$, а порядок $a$ делит $q-1$. Уравнение $x^d = 1$ имеет в $\mathbb{F}_q^\times$ ровно $\gcd(d, q-1)$ решений.

Условия применимости. Поле должно быть конечным. Пример без условия: $(\mathbb{Z}/8)^\times \cong \mathbb{Z}/2 \oplus \mathbb{Z}/2$ — не циклическая. Это потому, что $\mathbb{Z}/8$ — кольцо, а не поле.

Идея доказательства. В конечной абелевой группе экспонента (максимум порядков элементов) достигается на каком-то элементе $g$. Если экспонента $m$ строго меньше $|G|$, все элементы удовлетворяют $x^m = 1$ — многочлен степени $m$ имеет больше $m$ корней, противоречие с E4.

Варианты. То же верно для любой конечной подгруппы мультипликативной группы любого поля.

---

### E6. Многочлен против функции

Утверждение. Отображение «многочлен из $\mathbb{F}_q[x]$ $\to$ функция $\mathbb{F}_q \to \mathbb{F}_q$» сюръективно. Его ядро — главный идеал $(x^q - x)$. Поэтому каждую функцию $\mathbb{F}_q \to \mathbb{F}_q$ задаёт ровно один многочлен степени меньше $q$.

Условия применимости. Только конечные поля. Над $\mathbb{Q}$ разные многочлены задают разные функции — нет такой коллапсации.

Пример различия. Над $\mathbb{F}_2$ многочлены $x$ и $x^2$ формально различны, но задают одну и ту же тождественную функцию.

Идея доказательства. Сюръективность даёт интерполяция Лагранжа. Ядро содержит $(x^q - x)$ по E2; обратное включение — по E4 (многочлен из ядра имеет $q$ корней, делится на $\prod(x-a) = x^q - x$).

Варианты. Индикатор точки $a$ записывается как $1 - (x-a)^{q-1}$. Многочлен, задающий перестановку $\mathbb{F}_q$, — отдельный объект изучения (permutation polynomial); пример: $x^k$ при $\gcd(k, q-1) = 1$, и $x^p$ в характеристике $p$ (это сам Фробениус).

## Записи — Стандартные

### E7. Универсальная факторизация $x^{q^n} - x$ и формула Гаусса для числа неприводимых

Утверждение. Над $\mathbb{F}_q$:
$$x^{q^n} - x \;=\; \prod_{d \mid n}\; \prod_{\substack{f \in \mathbb{F}_q[x] \\ \text{нормированный} \\ \text{неприводимый},\, \deg f = d}} f(x).$$

Сравнение степеней даёт $q^n = \sum_{d \mid n} d \cdot N_q(d)$, где $N_q(d)$ — число нормированных неприводимых многочленов степени $d$ над $\mathbb{F}_q$. Инверсия Мёбиуса:
$$N_q(n) = \frac{1}{n} \sum_{d \mid n} \mu(d)\, q^{n/d}.$$

Следствие. $N_q(n) > 0$ для всех $n \ge 1$: неприводимый многочлен любой степени существует, и значит существует $\mathbb{F}_{q^n}$.

Условия применимости. Поле должно быть конечным.

Идея доказательства. $\alpha \in \mathbb{F}_{q^n}$ тогда и только тогда, когда $[\mathbb{F}_q(\alpha) : \mathbb{F}_q]$ делит $n$. Минимальный многочлен $\alpha$ — единственный неприводимый делитель $x^{q^n} - x$, у которого $\alpha$ — корень. Сепарабельность $x^{q^n} - x$ гарантирует отсутствие кратностей в произведении.

### E8. Теорема Шевалле-Варинга (Chevalley-Warning)

Утверждение. Пусть $f_1, \dots, f_r \in \mathbb{F}_q[x_1, \dots, x_n]$ и $\sum_i \deg f_i < n$. Тогда число общих нулей системы $\{f_i = 0\}$ в $\mathbb{F}_q^n$ делится на $p = \operatorname{char} \mathbb{F}_q$.

Следствие (теорема Шевалле). Если каждый $f_i$ однороден без свободного члена, то $\bar 0$ — решение; вместе с делимостью числа решений на $p$ это даёт существование нетривиального общего решения.

Условия применимости. Степени учитываются полные (по сумме показателей). Строгое неравенство $\sum \deg < n$ существенно. Пример без условия: над $\mathbb{F}_3$ однородная форма $f = x_1^2 + x_2^2$, $n = 2$, $\deg f = 2 = n$. Единственное решение — $\bar 0$ (так как $-1$ — не квадрат в $\mathbb{F}_3$).

Идея доказательства. Положим $g(\bar x) = \prod_i \bigl(1 - f_i(\bar x)^{q-1}\bigr)$. Функция $g$ равна $1$ в общих нулях и $0$ иначе, поэтому $N \equiv \sum_{\bar x \in \mathbb{F}_q^n} g(\bar x) \pmod p$. Степень $g$ меньше $(q-1) n$, значит в каждом одночлене $\prod x_i^{e_i}$ есть переменная с $e_i < q - 1$. По E3 сумма каждого такого одночлена по $\mathbb{F}_q^n$ равна $0$.

Варианты. Уточнение Акса-Каца даёт более сильную делимость. Прямое следствие — теорема Эрдёша-Гинзбурга-Зива (см. E13).

### E9. Комбинаторный Nullstellensatz Алона

Утверждение. Пусть $F$ — поле, $S_1, \dots, S_n \subset F$ конечны, $f \in F[x_1, \dots, x_n]$, $\deg f \le t_1 + \dots + t_n$ при $t_i < |S_i|$. Если коэффициент $f$ при одночлене $\prod x_i^{t_i}$ ненулевой, то $f$ принимает ненулевое значение в какой-то точке $S_1 \times \dots \times S_n$.

Условия применимости. Любое поле. Условие $t_i < |S_i|$ для каждого $i$ существенно.

Идея доказательства. Редукция $f$ по идеалу $(g_1, \dots, g_n)$, где $g_i(x_i) = \prod_{s \in S_i}(x_i - s)$, даёт многочлен $\tilde f$ той же функции на $\prod S_i$ с $\deg_{x_i} \tilde f < |S_i|$. Если $f$ всюду $0$ на $\prod S_i$, то $\tilde f \equiv 0$ по E4. Возврат к коэффициенту при $\prod x_i^{t_i}$ показывает, что он сохраняется при редукции и обязан быть $0$.

Варианты. Теорема Коши-Давенпорта $|A+B| \ge \min(p,\, |A|+|B|-1)$ в $\mathbb{F}_p$ — следствие через подходящий выбор $f$.

### E10. Теорема Лукаса (Lucas)

Утверждение. Пусть $p$ простое, $n = \sum n_i p^i$, $k = \sum k_i p^i$ — записи в системе с основанием $p$. Тогда
$$\binom{n}{k} \equiv \prod_i \binom{n_i}{k_i} \pmod p.$$

Следствие. $p \nmid \binom{n}{k}$ тогда и только тогда, когда $k_i \le n_i$ для всех $i$.

Условия применимости. $p$ должно быть простым. Для составного модуля действовать через простые делители + китайская теорема об остатках.

Идея доказательства. По «мечте первокурсника» (E1) в $\mathbb{F}_p[x]$:
$$(1+x)^n = \prod_i (1+x)^{n_i p^i} = \prod_i \bigl((1+x)^{p^i}\bigr)^{n_i} \equiv \prod_i (1 + x^{p^i})^{n_i}.$$
Сравнение коэффициентов при $x^k$ — единственный способ собрать $k_i$ из $n_i$ в каждой группе разрядов.

Варианты. Теорема Куммера: $v_p\!\binom{n}{k}$ равно числу переносов при сложении $k + (n-k)$ в основании $p$.

### E11. Лифтинг Гензеля (Hensel)

Утверждение. Пусть $f \in \mathbb{Z}[x]$, $a \in \mathbb{Z}$, $f(a) \equiv 0 \pmod p$ и $f'(a) \not\equiv 0 \pmod p$. Тогда для каждого $k \ge 1$ существует ровно один класс $\tilde a \pmod{p^k}$ такой, что $\tilde a \equiv a \pmod p$ и $f(\tilde a) \equiv 0 \pmod{p^k}$.

Условия применимости. Условие $f'(a) \not\equiv 0 \pmod p$ (простой корень) критично. Пример без условия: $f(x) = x^2$, $p = 2$, $a = 0$. Тогда $f'(0) = 0$, и при поднятии до модуля $4$ оба класса $0$ и $2$ обнуляют $f$ — единственности нет.

Идея доказательства. Итерация Ньютона $a_{k+1} = a_k - f(a_k)/f'(a_k)$ работает в $\mathbb{Z}_p$: разложение $f(a_k + p^k t) = f(a_k) + p^k t f'(a_k) + O(p^{2k})$ позволяет однозначно подобрать $t \pmod p$ так, чтобы зануляться по модулю $p^{k+1}$.

Варианты. Многомерная версия требует обратимости якобиана по модулю $p$.

---

### E12. Выделение сепарабельной части и разложение по степеням

Утверждение. Для $f \in \mathbb{F}_q[x]$:
- $\gcd(f,\, x^q - x) = \prod_{a \in \mathbb{F}_q,\, f(a) = 0}(x - a)$ — выделяет корни $f$ из $\mathbb{F}_q$.
- $\gcd(f,\, x^{q^d} - x) = \prod$ (неприводимых делителей $f$, степени которых делят $d$).
- $\gcd(f, f')$ собирает кратные множители; $f / \gcd(f, f')$ — сепарабельная часть $f$.

Условия применимости. Формальная производная может тождественно занулиться в характеристике $p$. Пример: $f(x) = x^p$, тогда $f' = p x^{p-1} = 0$, и $\gcd(f, f') = f$ не отражает реальной кратности $p$.

Идея доказательства. Корень $\alpha$ кратный для $f$ тогда и только тогда, когда $f(\alpha) = f'(\alpha) = 0$. Расширение E2 на $\mathbb{F}_{q^d}$ даёт второй пункт.

## Записи — Редкие

### E13. Теорема Эрдёша-Гинзбурга-Зива (Erdős-Ginzburg-Ziv)

Утверждение. Среди любых $2n - 1$ целых найдутся $n$ с суммой, делящейся на $n$.

Идея доказательства (случай $n = p$ простое). Для $a_1, \dots, a_{2p-1} \in \mathbb{Z}$ рассмотрим систему в $\mathbb{F}_p[x_1, \dots, x_{2p-1}]$:
$$f_1 = \sum_i a_i x_i^{p-1}, \qquad f_2 = \sum_i x_i^{p-1}.$$
Сумма степеней $2(p-1) < 2p - 1$. По Шевалле-Варингу (E8) есть нетривиальное общее решение $\bar x$. На носителе $I = \{i : x_i \ne 0\}$ по малой Ферма $x_i^{p-1} = 1$, поэтому $f_2(\bar x) = |I|$ и $|I| \equiv 0 \pmod p$. Так как $1 \le |I| \le 2p - 1$, остаётся $|I| = p$. Аналогично $f_1(\bar x) = \sum_{i \in I} a_i \equiv 0$.

Общий $n$ — индукция по простым делителям $n$.

### E14. Многочлен Артина-Шрейера (Artin-Schreier) $x^p - x - a$

Утверждение. Над $\mathbb{F}_p$ многочлен $x^p - x - a$ при $a \ne 0$ неприводим. Над $\mathbb{F}_q$ ($q = p^n$) — неприводим тогда и только тогда, когда $a$ не лежит в образе оператора $\wp(t) = t^p - t$. Образ $\wp$ — гиперплоскость коразмерности $1$ в $\mathbb{F}_q$ как $\mathbb{F}_p$-векторном пространстве.

Идея доказательства. Если $\alpha$ — корень, то $(\alpha + c)^p - (\alpha + c) = \alpha^p - \alpha = a$ для каждого $c \in \mathbb{F}_p$ (использовали $c^p = c$ и E1). Значит $\alpha, \alpha+1, \dots, \alpha + (p-1)$ — все $p$ корней. Если $\alpha \notin \mathbb{F}_q$, то расширение $\mathbb{F}_q(\alpha)/\mathbb{F}_q$ степени $p$, циклическое, с группой Галуа, порождённой $\alpha \mapsto \alpha + 1$.

Над $\mathbb{F}_p$ оператор $\wp$ обнуляется тождественно по Ферма, поэтому образ $\wp$ — нуль, и любое $a \ne 0$ даёт неприводимый многочлен.

## Self-test

1. Доказать, что для любого $f \in \mathbb{F}_p[x]$, обнуляющегося во всех точках $\mathbb{F}_p$, выполнено $(x^p - x) \mid f(x)$.
2. Найти все $n \ge 1$, для которых $x^p - x + 1$ делит $x^n - 1$ в $\mathbb{F}_p[x]$ (здесь $p \ge 2$ простое).
3. Доказать, что $x^p - x + 1 \in \mathbb{F}_p[x]$ неприводим для каждого простого $p$.
4. Пусть $f \in \mathbb{F}_p[x]$ имеет степень не выше $p - 2$ и удовлетворяет $\sum_{a \in \mathbb{F}_p} a \cdot f(a) = 0$. Доказать, что коэффициент при $x^{p-2}$ равен $0$.
5. Доказать, что среди любых $2p - 1$ целых чисел найдутся $p$ с суммой, делящейся на $p$ (простое $p$).
6. Доказать, что число нормированных неприводимых многочленов степени $n$ над $\mathbb{F}_q$ равно $\frac{1}{n}\sum_{d \mid n} \mu(d) q^{n/d}$, и положительно для всех $n \ge 1$, $q \ge 2$.
7. Доказать, что $\binom{2^k}{j} \equiv 0 \pmod 2$ при $1 \le j \le 2^k - 1$.
8. Пусть $f \in \mathbb{Z}[x]$ степени не выше $4$ удовлетворяет $f(a) \equiv 0 \pmod 5$ для $a = 0, 1, 2, 3, 4$. Доказать, что все коэффициенты $f$ делятся на $5$.
9. Над $\mathbb{F}_p$ ($p$ нечётное простое) вычислить $\prod_{a \in \mathbb{F}_p^\times} a^2$.
10. Пусть $A, B \subset \mathbb{F}_p$ непусты и $|A| + |B| > p$. Доказать, что $A + B = \mathbb{F}_p$.

## Решения

1. По малой теореме Ферма $a^p = a$ для всех $a \in \mathbb{F}_p$, поэтому $x^p - x$ имеет корни во всех точках поля. Каждый из $p$ линейных множителей $(x - a)$ делит $f$ (так как $f(a) = 0$), а они попарно взаимно просты. Значит, $\prod_a (x - a) = x^p - x$ делит $f$. См. E2, E6.

2. По E14 многочлен $x^p - x + 1$ неприводим над $\mathbb{F}_p$, поэтому факторкольцо $R = \mathbb{F}_p[x]/(x^p - x + 1) \cong \mathbb{F}_{p^p}$ — поле. Образ $x$ в $R$ удовлетворяет $x^p = x - 1$. Итерируя Фробениус (E1):
$$x^{p^k} \equiv x - k \pmod{x^p - x + 1, \; p}, \qquad k = 0, 1, \dots, p.$$
В частности, $x^{p^p} = x$, значит порядок $x$ в $R^\times$ делит $p^p - 1$. С другой стороны, $x \notin \mathbb{F}_{p^d}$ при $d < p$ (иначе $x^{p^d} = x$, противоречие с $x^{p^d} \equiv x - d$ и $d \not\equiv 0$). Поэтому порядок $x$ равен ровно $p^p - 1$. Условие $(x^p - x + 1) \mid (x^n - 1)$ означает $x^n = 1$ в $R$, что равносильно $(p^p - 1) \mid n$.

3. Оператор $\wp(t) = t^p - t$ на $\mathbb{F}_p$ обнуляется тождественно (Ферма). Значит образ $\wp$ — нуль, и $-1$ не лежит в нём. По E14 многочлен $x^p - x - (-1) = x^p - x + 1$ неприводим над $\mathbb{F}_p$.

4. Пусть $f(x) = \sum_{k=0}^{p-2} c_k x^k$. Тогда $\sum_a a \cdot f(a) = \sum_k c_k \sum_a a^{k+1}$. По E3 сумма $\sum_a a^{k+1}$ ненулевая (и равна $-1$) только при $(p-1) \mid (k+1)$. Так как $0 \le k \le p-2$, единственный подходящий случай — $k = p - 2$. Условие $\sum_a a f(a) = 0$ превращается в $-c_{p-2} = 0$, то есть $c_{p-2} = 0$.

5. Рассмотрим в $\mathbb{F}_p^{2p-1}$ систему $f_1 = \sum_i a_i x_i^{p-1}$, $f_2 = \sum_i x_i^{p-1}$. Сумма степеней $2(p-1) < 2p - 1$, поэтому по Шевалле-Варингу (E8) число общих нулей делится на $p$. Решение $\bar 0$ есть, значит существует нетривиальное $\bar x$. На носителе $I = \{i: x_i \ne 0\}$ имеем $x_i^{p-1} = 1$ (Ферма), значит $|I| \equiv 0 \pmod p$. Из $1 \le |I| \le 2p - 1$ получаем $|I| = p$. Аналогично $\sum_{i \in I} a_i \equiv 0 \pmod p$. См. E13.

6. По E7 $x^{q^n} - x = \prod_{d \mid n} \prod_{f \text{ нормир. неприв., } \deg = d} f$. Сравнивая степени, получаем $q^n = \sum_{d \mid n} d \cdot N_q(d)$. Инверсия Мёбиуса даёт $n N_q(n) = \sum_{d \mid n} \mu(n/d) q^d$, то есть нужную формулу. Положительность: $n N_q(n) \ge q^n - \sum_{d \mid n,\, d < n} q^d \ge q^n - n q^{n/2} > 0$ при $q \ge 2$ и $n \ge 1$.

7. По Лукасу (E10) разложим $2^k$ в основании $2$ как $1\underbrace{00\dots0}_{k}$. Для $1 \le j \le 2^k - 1$ в разложении $j$ есть хотя бы один разряд $j_i = 1$ при $i < k$, тогда как соответствующий разряд $(2^k)_i = 0$. Значит $\binom{(2^k)_i}{j_i} = \binom{0}{1} = 0$, и весь продукт $\equiv 0 \pmod 2$.

8. Рассмотрим редукцию $\bar f \in \mathbb{F}_5[x]$. По условию $\bar f$ имеет степень не выше $4$ и обнуляется в пяти различных точках $0, 1, 2, 3, 4$. По E4 многочлен степени не выше $4$ с пятью корнями тождественно равен нулю, значит все коэффициенты $f$ делятся на $5$.

9. По E2 имеем $\prod_{a \in \mathbb{F}_p^\times} a = -1$. Тогда $\prod_a a^2 = (-1)^2 = 1$.

10. Зафиксируем $c \in \mathbb{F}_p$. Множества $A$ и $c - B = \{c - b : b \in B\}$ имеют суммарный размер $|A| + |B| > p = |\mathbb{F}_p|$, поэтому они пересекаются. Любой элемент пересечения даёт $a = c - b$, то есть $a + b = c$. Значит $c \in A + B$ для каждого $c$.
