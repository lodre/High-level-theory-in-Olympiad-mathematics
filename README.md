# High-level-theory-in-Olympiad-mathematics
Сборник теории, который можно использовать для подготовки к соревнования студентов по математике, включая IMC

# Каталог тем

## Вещественный анализ (одномерный)
- Последовательности и ряды: тесты сходимости, Stolz–Cesàro, Abel summation, телескопирование, перестановки, асимптотики (Stirling, Wallis)
- Семейство теорем о среднем (Rolle / Lagrange / Cauchy / IVT) + Taylor, дифференцирование по параметру
- Интегралы: спецзамены, специальные интегралы, gamma и beta функции, тождество $\Gamma(x)\Gamma(1-x)=\pi/\sin\pi x$
- Степенные ряды и равномерная сходимость: перестановка $\lim$, $\int$, $\sum$; аналитические функции
- Функциональные уравнения на $\mathbb{R}$
- Неравенства и выпуклость: Cauchy–Schwarz, Jensen, AM–GM, rearrangement, Chebyshev + интегральные аналоги; convex / log-convex функции, supporting hyperplane
- Lipschitz, Banach fixed point (принцип сжимающих отображений)

## Многомерный анализ и топология
- Частные производные, дифференцируемость, chain rule
- Lagrange multipliers, условные экстремумы
- Якобиан, замена переменных в кратных интегралах, Fubini
- Компактность в $\mathbb{R}^n$ и в общем случае, связность, образ непрерывной функции
- Equicontinuity / Arzelà–Ascoli (опционально)

## Комплексный анализ
- Тождества для голоморфных функций: Liouville, maximum modulus, Schwarz lemma
- Контурное интегрирование и вычеты для вещественных интегралов
- Argument principle, Rouché (счёт корней)
- Корни из единицы, циклотомические тождества

## Линейная алгебра
- Определители: тождества, Vandermonde, блочные, Cauchy–Binet
- Характеристический и минимальный многочлен, Cayley–Hamilton
- Eigenvalues, тождества со следом (trace), spectral radius, ранговые аргументы
- Специальные классы матриц: nilpotent / idempotent / involutive / коммутирующие
- Жорданова форма + спектральная теорема (symmetric / Hermitian / unitary / orthogonal / normal)
- Евклидовы пространства: ортогональная проекция, Gram matrix, неравенство Hadamard
- Positive (semi)definite матрицы: $A=B^TB$, корни, критерии; singular values (базово)
- Матричная экспонента $e^{tA}$ и связь со спектром
- Билинейные и квадратичные формы, signature, закон инерции Sylvester
- Целочисленные матрицы: Smith normal form (light), делимость инвариантных факторов
- Матрицы над $\mathbb{F}_p$
- Tensor / Kronecker product (базово)

## Многочлены, кольца и поля
- Факторизация и неприводимость (Eisenstein, reduction mod $p$)
- Vieta, симметрические многочлены, тождества Newton
- Resultant, discriminant
- Многочлены над $\mathbb{F}_p$ и $\mathbb{F}_q$
- Расширения полей: minimal polynomial, $\mathbb{Q}(\alpha)$, конечные поля

## Теория групп
- Порядки, теорема Lagrange, циклические группы, структура подгрупп
- Group actions, orbit–stabilizer, Burnside
- Теоремы Sylow, $p$-группы (центр нетривиален)
- Классы сопряжённости, нормализатор и централизатор
- Структура конечных абелевых групп
- $S_n$, $A_n$, sign / parity

## Комбинаторика
- Производящие функции (OGF и EGF)
- Формула включений-исключений
- Биективные аргументы и double counting
- Pigeonhole + экстремальный принцип
- Линейно-алгебраическая комбинаторика (dimension arguments, инцидентные матрицы)
- Рекуррентные соотношения и закрытые формы
- Теория графов: деревья, формула Cayley, matrix-tree, Эйлеровы / Гамильтоновы циклы, спектр графа
- Probabilistic method (light)

## Теория чисел (light)
- Lifting the exponent (LTE)
- Квадратичные вычеты, символы Legendre / Jacobi
- Порядок элемента mod $n$, первообразные корни
- Диофантовы трюки: бесконечный спуск, чётность, mod-арифметика
